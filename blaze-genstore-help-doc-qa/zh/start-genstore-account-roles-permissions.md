# 账号权限

您可以通过分配角色，将员工添加到您的店铺中来共同运营，确保每位员工获得适当的权限。

通过精细化的角色和权限管理，您可以根据不同员工的职责和工作需求，控制他们在店铺中能够执行的操作。例如，您可以将员工分为**管理角色**和**普通员工角色**，并为每个角色分配不同的权限（如添加产品、管理订单、查看财务报表等），从而更高效地管理店铺运营。

## 添加员工

### 步骤

1. 登录 Genstore 商户后台，导航至 **设置** -> **账号权限**。
2. 点击 **添加员工**。
3. 输入员工的邮箱，员工将通过该邮箱收到邀请及店铺相关通知。
4. 在 **角色** 部分，至少为其分配一个角色。
5. 【可选】您可点击角色后的 **查看权限**，获取当前角色的所有权限。如果需要，可以编辑现有角色或创建新角色。
6. 点击 **发送邀请**。邀请有效期为 3 天。若未接受，需重新发送邀请。

发送邀请后，员工将自动出现在员工列表中，根据激活状态，员工状态可能为：

- **已加入**：员工已接受邀请。
- **待处理**：邀请已发送，尚未接受。
- **已过期**：邀请已过期，未被接受。
- **已拒绝**：员工拒绝了邀请。

### 接受邀请

被邀请人收到邀请邮件后，应在 3 天内回复此邀请。

1. 在邀请电子邮件中，点击 **接受邀请**。
2. 登录 Genstore 账户。如果您尚未创建 Genstore 账户，请先完成 Genstore 账户创建。
3. 点击 **接受邀请**。

## 员工账号的停用与激活

通过员工账号状态管理功能，管理员可灵活控制后台访问权限：
- **停用账号**：立即终止员工后台登录权限，停用账号不计入员工数量。
- **重新激活**：快速恢复账号权限，支持保留或重置原有配置。

::: tip

仅拥有「账号权限 - 查看&操作」权限的账号可执行此操作。

:::

### 停用员工账号

1. 登录 Genstore 商户后台，导航至 **设置** -> **账号权限**。
2. 在员工列表中找到要停用的员工账号，点击 **停用**。
3. 在弹框中再次点击 **停用** 以确认操作。

### 重新激活员工账号

1. 登录 Genstore 商户后台，导航至 **设置** -> **账号权限**。
2. 在员工列表中找到要停用的员工账号，点击 **重新激活**。
3. 在弹框中再次点击 **重新激活** 以确认操作。

## 管理角色

角色决定了员工的工作职责及所需权限。您可以分配以下两类角色：<!--后续会补充角色清单-->

- 系统角色：由 Genstore 提供的预定义角色，无法编辑。
- 自定义角色：店铺自定义的角色，可对其进行编辑以满足业务需求。

### 自定义角色

1. 登录 Genstore 商户后台，导航至 **设置** -> **账号权限**。
2. 点击 **管理角色**。
3. 在新窗口中，点击 **新建角色**。
4. 填写角色 **标题** 和 **描述**。
5. 在 **权限** 部分，选择 **店铺权限** 和 **应用权限**。
6. 点击 **保存**。

### 删除角色

删除角色仅针对自定义角色，系统角色不可删除。

1. 登录 Genstore 商户后台，导航至 **设置** -> **账号权限**。
2. 点击 **管理角色**。
3. 在要删除的角色后，点击 **...** -> **删除**。
4. 点击 **删除** 进行确认。此操作无法撤销。**注意**：所有分配此角色的用户将失去相关店铺权限。
5. 点击 **保存**。

### 向用户分配角色

您可以为员工分配一个或多个角色，如分配多个角色，多个角色的权限会叠加。

1. 登录 Genstore 商户后台，导航至 **设置** -> **账号权限**。
2. 在 **员工** 列表中选择要分配角色的用户，点击 **...** -> **编辑**。
3. 在员工详情中，**选中** 要分配的角色。
4. 点击 **保存**。

### 删除用户角色

您可以删除用户的角色。删除角色会删除该角色的所有相关权限。

1. 登录 Genstore 商户后台，导航至 **设置** -> **账号权限**。
2. 在 **员工** 列表中选择要分配角色的用户，点击 **...** -> **编辑**。
3. 在员工详情中，**取消选中** 要删除的角色。
4. 点击 **保存**。

## 转移店铺所有权

注意：当前操作仅支持店主操作。

1. 登录 Genstore 商户后台，导航至 **设置** -> **账号权限**。
2. 在 **店铺所有者** 部分中，点击 **转移店铺所有权**。
3. 输入新所有者的邮箱，点击 **转移所有权**。
