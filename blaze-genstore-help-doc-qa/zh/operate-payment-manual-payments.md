# 手动支付

部分顾客更倾向于使用非在线方式完成付款。如果您的商店启用了手动支付，顾客可以在下单后选择线下支付，为不使用信用卡的顾客提供更多选择。您可根据商店需求设置不同的手动支付方式。本章将介绍几种常见的手动支付方式。

:::tip

- 使用手动支付方式时，您将不需要为交易支付第三方交易费用。
- 使用手动支付选项时，建议及时与顾客确认款项已到位，以确保订单处理的顺畅。

:::

## 货到付款 (Cash on Delivery, COD)

货到付款允许买家在收到产品时进行付款。此选项适合那些对在线支付不太信任的顾客。

- **适用场景**：主要用于电子商务平台或本地物流配送，尤其是顾客没有信用卡或不愿使用信用卡支付时。
- **优点**：顾客可以在确认收到产品后再付款，降低了交易风险。
- **缺点**：商家面临较高的拒收风险和支付失败的可能性，特别是在较大的交易中。

## 汇票 (Money Order)

汇票是由银行或其他金融机构签发的支付凭证，持票人可以凭此向指定的付款人索取款项，通常用于大额支付。

- **适用场景**：跨城市或跨国交易，尤其适合电子支付不便的地区或客户。
- **优点**：较高的安全性，支付凭证清晰，适合大额交易。
- **缺点**：处理时间较长，且可能涉及手续费，支付的灵活性差。

## 银行转账 (Bank Transfer)

银行转账是通过银行账户进行的资金划转，包括线上和线下操作。它适用于金额较大的交易。

- **适用场景**：适用于企业与企业或个人间的资金转移，特别是在支付账单或大额款项时。
- **优点**：安全、可靠，尤其适合大额支付。
- **缺点**：需要准确的银行信息，否则可能导致支付延误或失败，且处理时间较长。

:::tip

- 当您的客户使用手动付款向您提交订单时，不会向您收取第三方交易费用
- 当您向您的客户提供手动付款方式时，请务必及时与您的客户联系并确保资金款项已到位

:::

## 激活手动付款

您可以在商店后台轻松创建手动支付方式。具体操作步骤如下：

1. 登录 Genstore 商家后台，进入 **设置 -> 支付**。
2. 在 **更多支付方式** 下，点击 **添加支付方式**。
3. 在 **手动支付方式** 标签页中，点击 **添加手动支付方式**。在弹窗中进行如下设置：
   - 选择 **手动支付类型**：**货到付款** 或 **手动支付**。
   - 填写支付方式名称
   - 设置结账页展示文案，比如为顾客提供关于支付方式的详细信息，例如：
	 - **汇票支付**：提供汇款要求与详细银行信息。
	 - **货到付款**：提供您的联系电话与交货细节。
	 - **银行转账**：提供转账所需的银行账户信息。
   - 设置感谢页展示文案。
4. 点击 **保存** 完成设置。

## 消费者体验

当您成功激活手动支付选项后，消费者在结账时可以选择这种支付方式：

1. 在结账页面，消费者选择手动支付选项后，会看到您设置的相关支付文案。
2. 选择支付方式后，消费者点击 **提交订单** 即可完成下单。

## 停用手动付款

若您不再希望为顾客提供某种手动支付方式，可以在后台轻松停用：

1. 登录您的 Genstore 管理员账号，进入 **设置 -> 支付**。
2. 在 **更多支付方式** 模块下，找到您希望停用的支付方式。
3. 点击 **管理** 按钮，然后在弹窗的左下角点击 **删除**。
