# 变更新的 genmystore.com 域名

当您注册 Genstore 店铺时，系统会为您生成默认的 `.genmystore.com` 域名，作为您在线商店的默认网址。如果您希望展示您的品牌，Genstore 允许您免费修改此 `.genmystore.com` 域名，但每个商店仅有一次修改机会。

## 变更步骤

1. 在 Genstore 后台中，导航至 **设置** -> **域名**。
2. 在 **域名** 页面上，为相关域名点击 **更改为新的 genmystore.com 域名**。
3. 输入新名称，然后点击 **添加域名**。

新的 `.genmystore.com` 域名自动成为您的主域名，之前的 `.genmystore.com` 域名会被自动重定向到新域名。无需创建任何其他 URL 重定向。
