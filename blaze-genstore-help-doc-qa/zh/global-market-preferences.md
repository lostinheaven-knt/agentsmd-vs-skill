# 偏好设置

您可以在 Genstore 的**市场偏好设置**中，配置自动重定向、价格显示、价格舍入、币种控制等功能，以提升多市场客户的本地化购物体验。

这些设置适用于所有 **非主市场**，并可对每个市场独立配置。系统会根据您的设置，在客户访问时自动切换语言、地区和币种，并确保展示统一、合理的价格结构。

## 可配置选项总览

|设置项|说明|
|---|---|
|国家/地区重定向|根据客户 IP 自动跳转至对应市场，提升入口体验|
|语言重定向|根据客户浏览器语言偏好，自动切换为已发布的语言版本|
|当地货币|启用后，客户可使用其国家/地区的本地货币查看和结算订单价格（基于系统汇率转换）|
|价格舍入|对经过汇率换算的价格进行舍入处理，避免小数位不统一，提升价格展示一致性|
|隐藏原价|向欧洲经济区（EEA）客户隐藏与其他国家市场之间的价格差异（满足特定地区法规要求）|

## 操作入口

1. 登录 Genstore 后台，前往 **设置** -> **市场**。
2. 点击页面右上角的 **偏好设置** 进入页面。
3. 根据以下说明，逐项完成设置：
	- [设置国家/地区重定向](./global-market-preferences-auto-redirect#国家-地区重定向)
	- [设置语言重定向](./global-market-preferences-auto-redirect#语言重定向)
	- [启用当地货币显示](./global-market-preferences-pricing#设置当地货币)
	- [开启价格舍入规则](./global-market-preferences-pricing#设置舍入价格)
