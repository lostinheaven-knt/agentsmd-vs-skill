# **设置**

本章介绍 **Genstore** 的通知设置、政策管理、客户隐私管理和内容管理。

## 本节主题

- [通知管理](./settings-notifications.md)：在 **设置 -> 通知** 中配置通知，验证发送邮箱，并开启或关闭邮件、短信和后台通知。
- [素材管理](./settings-media.md)：在 **设置 -> 素材** 中上传和管理图片、视频及文件，支持多种格式，存储空间根据套餐不同（**100GB 至 300GB**）。
- [税务管理](./taxes-overview.md)：在 **设置 -> 税务** 中管理您销售区域的税务设置，包括销售产品时的税费和相关交易费用的征收等问题。
- [市场](./global-market-overview.md)：
  - [按国家/地区设置产品价格](./global-set-product-prices-by-country.md)：您可以根据不同国家/地区设置产品价格，更好地管理面向国际或多地区市场的电商店铺。
  - [按国家/地区查看产品价格](./global-set-product-prices-by-country.md)：查看不同市场的产品价格明细，确保定价的准确性和一致性。
- [域名](./global-domains.md)：在  **设置 -> 域名** 中管理客户访问您商店的域名设置。
- [政策管理](./settings-policy.md)：在 **设置 -> 政策** 中创建和管理退货、隐私、服务条款等政策，使用模板或自定义内容，政策链接将自动显示在结账页底部。
- [客户隐私管理](./settings-policy.md)：通过 **Genstore** 后台的 **客户隐私** 设置，保护客户隐私并确保符合相关隐私法规。
