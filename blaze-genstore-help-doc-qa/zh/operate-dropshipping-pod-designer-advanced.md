# 高级设计器

高级设计器是专为 **品牌化定制** 打造的专业级创意工具。无论您是上传自有作品，还是从零开始创作，它都能帮您生成符合工业印刷标准的生产文件和高品质样机图。

::: tip
对于支持多种工艺（如 DTG、刺绣、满版印刷等）的产品，请务必先确认“打印技术”，以确保设计画布与功能精准匹配。
:::

## 设计制作流程 

1. 登录 Genstore 商户后台，选中具体产品后，点击产品卡的 **开始设计**。
2. 点击页面左上角的 **高级设计** 按钮。
3. 进入设计工具后，选择您需要的 **打印技术**（如 DTG、刺绣、满版印刷等）。
4. 选定打印技术后，您可根据业务需求进行设计，如添加文字、图片、图案等元素。

### 素材与创作工具
* **自定义上传**：支持上传本地设计文件。所有已上传素材都会保存在您的 **文件素材库** 中，方便后续重复使用。
* **文字工具**：支持自定义字体与颜色，并支持为文字 **填充图案效果**。
* **丰富素材库**：提供多主题剪贴画，并支持使用 **付费精选素材**。所有已上传或选定的素材都会保存在你的文件素材库中，方便后续重复使用。
* **快捷模板**：提供一组可自由编辑的预制版式，支持按需快速定制。

### 排版与图层管理
* **多区域印刷**：支持将设计图案放置在正面、背面、衣袖、标签等不同印刷区域。
* **精准编辑**：自由调整元素的尺寸、位置、旋转角度，并设置 **图层叠加（透明度）** 效果。
* **质量保障**：参考 **印刷质量警示**，确保图像分辨率符合生产标准，避免打印模糊。
* **背景填充**：支持为特定印刷区域填充背景色或特定图案。

### 满版印刷 (AOP) 专属工具
- **一键生成平铺图案**：在 **Fill**（填充）标签页中，您可以针对剪贴画、自定义上传或示例文件使用 **Apply background to all placements**（全幅应用背景）功能，将其一键转化为无缝平铺的重复图案。
- **精准布局控制**：通过界面滑块，可自由调整图案的间距、旋转角度及对齐方式，实现完美的视觉排布。
- **跨位置快速同步**：在 **Layer**（图层）面板中，选中设计元素后点击 **⋮** 图标并选择 **Duplicate to**，即可将其一键复制到其他印刷区域，确保各位置设计的高度统一。

## 产品效果图 (Mockup) 

设计完成后，产品效果品（Mockup）将决定产品在店铺中的展示效果：

* **多角度展示**：选择不同角度的样机图，全面呈现产品细节。
* **主图自定义**：在选定的多张样机中，自由指定其中一张作为商品在店铺中的 **主图**。
* **一键推送**：确认后的效果图将随产品信息直接推送至您的 [产品列表](./operate-product-manage.md)。
