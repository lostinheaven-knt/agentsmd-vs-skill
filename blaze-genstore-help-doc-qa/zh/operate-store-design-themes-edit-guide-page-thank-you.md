# 感谢页

感谢页是用户完成支付后看到的确认页面，既承担订单确认功能，也是提升复购与互动的重要场景。本页将帮助您理解感谢页的结构、编辑方式及自定义能力。

## 选择页面

在编辑器顶部导航栏，点击 **主页** 右侧的下拉箭头，可展开当前市场下支持的页面类型列表。

- 选择 **感谢页**，进入默认的感谢页模版。

您也可在页面管理页面，直接进入感谢页。

## 页面结构

感谢页主要由以下区域组成：

- **订单确认信息**：显示订单号、订单状态等基础信息。
- **收货与支付摘要**：展示收货地址、支付方式及相关费用明细。
- **产品清单**：列出用户购买的产品及数量。
- **后续操作入口**：引导用户进行账号注册、订单跟踪或联系客服。

## 全局样式

感谢页的整体样式继承自店铺主题，包括字体与主色调。您也可以对局部元素进行覆盖，在保持品牌一致性的同时突出关键信息。

## 侧边栏功能

- **自定义代码（Custom Liquid）**  
    可在侧边栏添加 Liquid 代码，实现个性化逻辑或动态展示。例如：为不同客户群体展示专属感谢文案或定制推荐。
- **应用嵌入**：在感谢页顶部预留了展示区域，可嵌入第三方应用或营销工具。 例如：添加推荐产品模块或社交分享按钮，鼓励复购和用户传播。
## 实时预览

所有修改都会在编辑器中即时呈现，无需来回切换前台确认。
