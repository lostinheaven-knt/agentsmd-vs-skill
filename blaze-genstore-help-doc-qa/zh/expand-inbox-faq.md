# 常见问题

::: faq

### Q1. Genstore AI Agent 支持哪些语言？

> **A:** AI Agent 可以使用我们的大型语言模型（LLM）支持的多种语言进行消息编写与回复。
>
> 为确保 AI Agent 表现最佳，我们建议您注意以下几点：
>
> - 推荐使用**英语（美式或英式）**撰写知识库与政策类内容，AI 对英文文本的理解更准确、响应更稳定；
> - 若您希望 AI Agent 使用其他语言服务客户，也可以根据业务需求配置多语言内容；
> - 若需上传知识库文档，建议使用英文版本以获得最优效果。

### Q2. 我可以在多家商店中使用 Genstore AI Agent 吗？

> **A:** 可以。每个 Genstore 商店都支持独立启用 AI Agent，且 AI 会基于当前店铺的产品与政策等数据独立训练，不会影响其他商店的数据或表现。

### Q3. AI Agent 可以识别图像、视频或嵌入式媒体内容吗？

> **A:** 暂不支持。AI Agent 无法读取图片、视频或嵌入式媒体内容，只能处理文本信息。
>
> 因此我们建议：
>
> - 在上传 PDF 或撰写帮助中心文章时，确保**关键信息以文本形式呈现**；
> - 请勿依赖图像传达重要政策或操作说明，以免影响 AI 回答的完整性；
> - AI Agent 同样无法读取客户通过邮件或对话中发送的图片、附件等非文本内容。

:::
