# 管理您的 Genstore 套餐

在商店运营过程中，您可能会根据业务需求调整套餐等级，例如：

- 商店即将上线，需解锁更多销售渠道或支付方式
- 活动期间访客流量增长，需要更高并发与营销容量
- 降级为基础套餐以节省成本

您可以在 Genstore 商家后台的 **设置 -> 套餐和积分** 页面管理店铺订阅套餐，包括 **升级/降级套餐** 或 **停用当前套餐（关闭店铺）**。

## 本节主题

- [变更套餐](./start-genstore-account-plan-upgrade.md)：了解如何升级或降级套餐，并查看按比例计费规则。
- [关闭店铺](./start-genstore-account-plan-unsubscribe.md)：了解如何取消订阅并关闭您的 Genstore 店铺。
