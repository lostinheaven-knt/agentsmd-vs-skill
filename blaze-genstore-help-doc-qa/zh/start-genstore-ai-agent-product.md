# 产品 Agent

产品 Agent 是您的产品管理专家。无论您是经营自有品牌的实物或数字产品，还是从事无货源代发（Dropshipping）业务，AI 都能为您提供全自动化的产品创建与上架体验。

## 场景一：无货源代发 (Dropshipping) 

*适用于：希望链接全球供应链，快速寻找并销售高利润爆款产品的商户。*

在 Genstore，选品不再需要繁琐的平台跳转。您可以调动 产品 Agent 作为您的 "金牌选品专家"，直接从百万产品库中精准获取货源。

### 智能选品协作流
* **指令输入**：直接在对话框中输入您的选品需求，例如“帮我推荐 5 款高质量的瑜伽垫”。
* **需求分析 (Process)**：AI 会深度解析您的意图，锁定产品品类、成本区间及节假日趋势，并生成关键词。
* **智能推荐 (Proposal)**：AI 实时检索百万产品库，为您提供一份包含高质量、高性价比产品的**提案**供您决策。
* **一键同步**：在您勾选心仪产品后，AI 会自动将产品数据同步至您的后台列表，并完成上架。

## 场景二：自有实物产品

*适用于：拥有自有库存、手工艺人、工厂卖家或传统零售商。*

您无需手动填写繁琐的表单，只需通过一张图片或一段文字描述，产品 Agent 即可像“产品上架专员”一样为您完成新品创建。

### 1. 图生产品
* **操作方式**：在对话框输入指令（如：生成一个产品）并上传产品实拍图。
* **视觉识别**：AI 视觉引擎自动扫描图片，识别颜色、款式、材质及品牌等关键属性。
* **提案 (Proposal)**：基于识别结果，AI 自动生成符合 SEO 规范的标题、极具吸引力的营销文案及吸睛的主图。
* **确认与执行**：AI 将生成的**提案 (Proposal)** 供您确认。确认后，产品将以“草稿”状态保存到后台待上架。

### 2. 文生产品
* **操作方式**：直接描述产品特征，例如“帮我创建一个真皮男士钱包”。
* **语义分析**：AI 引擎精准提取品类、卖点及定价，自动扩写为专业的营销详情。
* **提案 (Proposal)**：基于碎片化的关键词，AI 自动扩写并生成完成的产品详情文案。
* **确认与执行**：AI 将生成的**提案 (Proposal)** 供您确认。确认后，产品将以“草稿”状态保存到后台待上架。

## 场景三：自有品牌数字产品

*适用于：电子书、在线课程、摄影素材及音乐文件等数字内容创作者。*

产品 Agent 就像您的“内容出版经纪人”，协助您将文件快速转化为高吸引力的商业产品。

* **协作逻辑**：在对话框上传文件（PDF/视频等）并描述资源信息。
* **提案 (Proposal)**：AI 深度解析内容，提炼核心价值点，撰写产品描述并生成配套的质感产品图。
*  **确认与执行**：AI 将生成的**提案 (Proposal)** 供您确认。确认后，产品将以“草稿”状态保存到后台待上架。
