# 冻结/解冻客户

在管理客户时，如发现存在欺诈行为或虚假信息的账户，您可以通过 Genstore 的客户模块对其进行冻结处理。

## 客户冻结

:::tip

- 仅已激活的客户账户支持冻结。未激活客户不会显示冻结按钮。
- 冻结后，客户将无法使用该账户登录您的商店，请谨慎操作。

:::

1. 登录 **Genstore** 后台，导航至 **商店** -> **客户** 菜单。
2. 点击目标客户进入**客户详情页**。
3. 选择 **更多操作**，点击 **禁用账户**。
4. 在弹窗中确认操作，点击 **禁用账户** 以完成冻结。

冻结成功后，客户状态将更新为 **客户已冻结**。

:::tip

冻结仅影响当前账户。客户仍可使用其他邮箱注册新账户。

:::

## 客户解冻

如需解冻已被冻结的客户账户，您可以重新发送账户激活邀请：

1. 登录 **Genstore** 后台，导航至 **商店** -> **客户** 菜单。
2. 点击目标客户进入 **客户详情页**。    
3. 点击 **发送客户邀请**。

客户收到激活邮件后，点击邮件中的激活链接并设置新密码，即可恢复账户登录权限。
