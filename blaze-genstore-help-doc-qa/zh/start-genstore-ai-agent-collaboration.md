# 与 AI Agent 协作

Genstore 以 **AI-native** 的设计理念，彻底重构了商家与后台的交互关系。通过自然语言交互，您可以与 AI 团队高效协作，完成从建站到运营的全链路任务。

## 功能概述
Genstore 将商家在创建和经营店铺过程中高频使用的开店、产品、装修、营销、客服和数据分析等场景全面 **AI 化与自动化**。通过与 AI 协作，商家可以从繁琐的“操作工”升级为店铺的“决策者”，专注于业务增长。

## 关键术语解释
为了让您与 AI 团队的协作更加高效，请熟悉以下定义了 Genstore 核心工作流的基础术语：

| 术语（英文） | 术语（中文） | 解释 |
| :--- | :--- | :--- |
| **LUI** | **自然语言交互界面** | 您可以在首页通过自然语言对话来操作系统，实现 "所说即所得"。 |
| **Chat** | **对话** | 您与 AI 协作的场所。所有的指令下达、方案确认及生成结果均在此呈现。系统保留 Chat 历史作为上下文记忆，确保业务连续性。 |
| **Prompt** | **指令/提示词** | 您在 Chat 中输入的具体文字。Prompt 的准确度决定执行效果；系统会提供 "推荐 Prompt" 辅助您快速启动任务。 |
| **Task** | **任务** | AI 从对话 (Chat) 中识别的具体经营事项，如店铺设置、创建产品等。Task 具有明确的状态流转（进行中、已完成、已终止），让您随时掌握进度。 |
| **Process** | **执行过程** | 可视化的进度反馈。系统会实时展示 AI 当前正在处理的具体步骤（如“正在提取产品信息”），让等待过程透明化。 |
| **Proposal** | **提案** | AI 生成的决策建议草案。基于您的诉求，AI 会先输出方案，待您“确认”或“修改”后，AI 才会继续执行落地操作。 |
| **Preview** | **预览** | 视觉任务专用窗口。在正式发布前，您可以实时查看装修设计效果与布局调整，确保视觉品牌形象一致。 |

## 核心场景与 Agent 能力
通过自然语言指令，您可以直接调用以下核心 AI 能力（建议按照以下顺序开启您的经营之旅）：

### [极速开店引导](./start-genstore-ai-agent-store-setup.md)
**确认预设信息，分钟级开启全球业务。**
* **指令示例**：3分钟完成店铺设置；
* **AI 执行**：启动引导流程，带领您快速确认系统预生成的物流、店名、域名等关键设置，确保无误后完成开业。

### [智能产品创建](./start-genstore-ai-agent-product.md)
无论是管理自有产品还是开展 Dropshipping 业务，AI 都能助您实现选品与上架的自动化。
- **自有产品创建（实物产品/数字产品）**：只需上传一张产品图或输入一段描述，AI 即可自动提取信息并生成专业的产品详情页。
  - **指令示例**：帮我根据这张图创建一个新款连衣裙产品。
  - **AI 执行**：AI 会根据您的指令生成 Proposal，待您确认后，产品将自动同步至您的店铺后台并完成上架。
- **智能选品**：针对 Dropshipping 商家，AI 可从百万级产品库中精准发掘潜力爆品，实现从挑选到上架的全流程自动化。
  - **指令示例**：帮我推荐圣诞装饰类的畅销 Dropshipping 产品。
  - **AI 执行**：从百万产品库中精选爆品进行推荐，选中后，一键上架至店铺销售。

### [AI 店铺装修](./operate-store-design-themes-ai-decorate.md)
**无需设计基础，通过对话即时改变店铺视觉风格。**
* **指令示例**：下周是万圣节，把店铺首页换成暗黑风格，并生成配套的活动 Banner；
* **AI 执行**：自动调整主题配色并设计节日海报，实现一键换肤。

### [视觉 Agent](./start-genstore-ai-agent-visual.md)
**将静态图片转换为动态视频，实现虚拟试穿或全局适配。**
* **指令示例**：帮我生成一个展示这把新衣服的模特视频；
* **AI 执行**：AI 会根据您的指令，自动识别图片中的人物模型，并生成符合要求的视频。

### [自动化营销活动](./start-genstore-ai-agent-discount.md)
**分析竞品并推荐活动方案，自动关联店铺首页公告栏进行快速推广。**
* **指令示例**：帮我设计一个满 $500 打 8 折的清仓活动；
* **AI 执行**：生成包含折扣逻辑与公告栏文案的。


### 智能商家助手
**7×24 小时在线的专属商户服务专家。**
* **指令示例**：如何创建一个草稿订单；
* **AI 执行**：Ella 将直接提供分步操作指引及操作菜单链接，实时解答平台使用疑问。

### 数据洞察
**基于数据表现提供深度分析与经营建议。**
* **指令示例**：查看最近 7 天的销售数据；
* **AI 执行**：生成可视化报表，并根据数据趋势提供具体的经营改进建议。
