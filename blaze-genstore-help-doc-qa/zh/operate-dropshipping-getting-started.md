
# 启用 Genstore Dropshipping

代发货（**Dropshipping**）是一种轻资产的电商经营模式，适合希望快速上架商品、减少库存压力、提升履约效率的商户。
在正式启用 Dropshipping 功能前，请先完成以下前置准备工作，确保后续流程顺畅。

## 开通 Genstore Payments

Genstore Dropshipping 依赖 **Genstore Payments** 完成订单收款与资金结算。
若您尚未开通 Genstore Payments，将无法启用 Dropshipping 功能。

### 支持的主体类型

| 注册地区   | 主体类型                  |
| ------ | --------------------- |
| **美国** | 个人 / 独资经营 / 公司 / 合伙企业 |
| **香港** | 独资经营 / 公司 / 合伙企业      |

> 若您的店铺注册主体不在上述地区，暂无法开通 Dropshipping 功能。

### 开通流程参考

请根据您的经营主体所在地，参考以下指引完成申请：

* [美国商户申请指南](./operate-genstore-payments-usa.md)
* [香港商户申请指南](./operate-genstore-payments-hk.md)

开通过程中，您需提交相关营业资料、身份验证信息及结算账户信息。
Genstore 审核通过后，您即可在后台启用 **Genstore Payments** 并开始使用 Dropshipping 服务。


## 绑定收款账户

Genstore Payments 开通成功后，需绑定您的银行账户以便后续结算资金转入。
您可以在商户后台 **设置 -> 支付 -> Genstore Payments** 中查看并管理结算账户信息。

> 账户信息需与主体注册名称一致。若信息不符，系统将提示验证失败。

## 了解 Dropshipping 工作原理

在启用代发货前，建议您了解 Dropshipping 的基本流程：

1. 在 Genstore Dropshipping 平台选择产品并推送至您的店铺。
2. 客户在前台下单并付款。
3. 系统根据履约设置（自动或手动）将订单同步给供应商。
4. 供应商发货并上传物流追踪信息。
5. 系统同步物流状态至店铺订单。
6. 完成发货后，平台进入清算与结算流程。

## 常见问题

### Q1. 未开通 Genstore Payments 是否可以使用代发货功能？
> A: 不可以。所有 Dropshipping 订单均需通过 Genstore Payments 处理支付与结算。

### Q2. 申请 Genstore Payments 是否需要企业营业执照？
> A: 
> 若以公司或合伙企业名义开通，需要提供相关注册及税务文件；
> 个人主体可使用有效身份证明与银行账户。
