# 撤销库存转移

撤销库存当发生错误记录、过期产品、损坏库存或多计库存时，商家可以通过撤销库存功能调整库存数量，确保系统中的库存水平与仓库的实际库存一致。

## 撤销未接收的库存转移

**操作步骤**

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **转移**，找到 **待收货** 状态下的库存转移单据号。
2. 点击单据号进入详情页。
3. 在 **添加产品** 部分的 **数量** 中手动输入正确的数量。
4. 如果要删除转移中的产品，点击要删除的产品旁边的删除按钮。

## 撤销已接收的库存转移

### 操作步骤

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **转移**，找到 **部分收货** 或 **已收货** 状态下的库存转移单据号。
2. 如果该转移已部分接收，则无法调整已接收的库存数量。
3. 若不再期望收到该转移的剩余库存，打开相应的转移单据，点击 **接受库存**。
4. 在收货页面点击 **拒绝全部转移**，拒绝所有尚未收到的库存。
5. 更新后，该转移单据将被标记为 **已收货**，并同步调整相应产品的传入库存数量。
