# 关联模版与内容

在每套主题模版中，系统会为不同的页面类型（如**产品、产品系列、页面、博客集合和博客文章**）内置一个默认模版，确保店铺能够快速启用并保持一致的展示效果。

如果您希望在默认样式的基础上区分不同场景或风格，可再根据业务需求将其分配到对应的内容对象。未分配时系统会自动使用该类型的默认模版，而一旦指定了自定义模版，展示效果将始终以所选模版为准。

## 适用范围

- 页面
- 产品
- 产品系列
- 博客文章
- 博客集合

## 为产品分配模版

1. 登录 Genstore 管理后台。
2. 导航至 **商店** -> **产品**，点击目标产品进入产品详情页。
3. 在右侧的 **展示模版** 下拉框中，选择要应用的模版。
4. 点击 **保存**。

## 为产品系列分配模版

1. 登录 Genstore 管理后台。
2. 导航至 **商店** -> **产品** -> **系列**，点击目标产品系列 **操作** 列的 **铅笔** 图标。
3. 在右侧的 **展示模版** 下拉框中，选择要应用的模版。
4. 点击 **保存**。

## 为博客文章分配模版

1. 登录 Genstore 管理后台。
2. 导航至 **商店** -> **在线商店** -> **博客文章**，点击目标博客文章 **操作** 列的 **铅笔** 图标。
3. 在右侧的 **展示模版** 下拉框中，选择要应用的模版。
4. 点击 **保存**。


## 为博客集合分配模版

1. 登录 Genstore 管理后台。
2. 导航至 **商店** -> **在线商店** -> **博客文章** -> **管理博客集合**。
3. 点击目标博客集合 **操作** 列的 **编辑** 按钮。
4. 在右侧的 **展示模版** 下拉框中，选择要应用的模版。
5. 点击 **保存**。

## 为页面分配模版

1. 登录 Genstore 管理后台。
2. 导航至 **商店** -> **在线商店** -> **页面**。
3. 点击目标页面 **操作** 列的 **铅笔** 图标。
4. 在右侧的 **展示模版** 下拉框中，选择要应用的模版。
5. 点击 **保存**。

## 为产品和产品系列批量分配模版

对于产品和产品系列，您还可通过批量编辑操作实现模版的批量应用。

### 为产品批量分配模版

1. 登录 Genstore 管理后台。
2. 导航至 **商店** -> **产品**，勾选希望批量应用某个模版的产品。
3. 点击 **批量编辑**。
4. 在批量编辑器中，找到 **展示模版** 列。
5. 在下拉列表中，选择要应用的模版。
6. 点击 **保存**。

### 为产品系列批量分配模版

1. 登录 Genstore 管理后台。
2. 导航至 **商店** -> **产品** -> **系列**，勾选希望批量应用某个模版的产品系列。
3. 点击 **批量编辑**。
4. 在批量编辑器中，找到 **展示模版** 列。
5. 在下拉列表中，选择要应用的模版。
6. 点击 **保存**。
