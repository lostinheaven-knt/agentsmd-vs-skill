# 自动重定向设置

Genstore 支持通过自动重定向功能，为客户自动切换到其所在国家/地区的市场，并匹配其浏览器语言，从而提供更加本地化的购物体验。

例如，若客户的 IP 属于加拿大，浏览器语言为法语，且加拿大市场已发布法语语言版本，则系统会自动将客户重定向至加拿大市场，并以法语显示店面内容。

自动重定向分为两种类型：**国家/地区重定向** 和 **语言重定向**，您可以根据需要分别启用。

## 国家/地区重定向

启用后，系统将根据客户的 IP 地址识别其所在国家/地区，并自动跳转至匹配的市场页面。
- 若客户的 IP 地址不在任何已配置市场的国家/地区范围内，系统将默认跳转至主市场。

### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **市场**。
2. 点击页面右上角的 **偏好设置**。
3. 在 **自动重定向** 部分中，开启 **国家/地区重定向**。

## 语言重定向

启用后，系统会检测客户浏览器的语言首选项，并自动切换为匹配语言版本（前提是该语言已在市场中启用并发布，具体可阅读 [针对市场配置语言](./global-market-settings-lang-domain#配置语言)）。
- 若浏览器语言未被启用或发布，则将显示市场的默认语言版本。

### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **市场**。
2. 点击点击页面右上角的 **偏好设置**。
3. 在 **自动重定向** 部分中，开启 **语言重定向**。
