# 美国商户申请指南

在美国使用 **Genstore Payments** 前，您需要提交有关个人及企业的信息，以满足监管要求并防范金融犯罪。本文将帮助您了解：

- 您应选择哪种主体类型
- 各主体的差异
- 不同主体的文件要求

## 主体类别差异

根据不同国家和地区的法律法规，您需要提供有关您和您企业的信息。这些信息将帮助 **Genstore（后称“我们”）** 确认商店的身份，包括公司实体的所有受益人，并确保其遵守反洗钱、反恐怖主义融资和其他金融犯罪防范的规定。随着法律法规的变化，**Genstore** 和合作伙伴可能会要求您提供额外的信息。

在美国，商户可以通过以下四类主体注册 **Genstore Payments**：

| 类别            |  Sole Proprietorship（独资经营）                           | Company（公司：LLC / Corporation）                                                                                                                                                                                                                           | Partnership（合伙企业）                                      |
| ------------- |  --------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------ |
| **法律地位**      | 不是独立法人，业主和企业为同一主体                                   | 独立法人，公司与股东/成员分离                                                                                                                                                                                                                                         | 不是独立法人，合伙人共同承担责任                                       |
| **责任承担**      |业主无限责任                                              | 股东/成员有限责任，仅以出资额为限                                                                                                                                                                                                                                       | 普通合伙人无限责任，有限合伙人有限责任                                    |
| **IRS 税务处理**  | - 使用 **Form 1040 + Schedule C** 报告收入<br/>- 不需单独报企业税 | - **LLC (single-member)**：Schedule C- **LLC (multi-member)**：Form 1065 + K-1<br/>- **C-Corp**：Form 1120（双重征税）<br/>- **S-Corp**：Form 1120-S + K-1                                                                                                        | - 提交 **Form 1065**<br/>- 利润/亏损分配至合伙人个人税表（Schedule K-1） |
| **注册名称示例**   | - John Smith（个人名）<br/>- DBA: **John’s Coffee**      | - **Emily Johnson LLC**（single-member LLC 官方注册名）<br/>- **Emily Johnson LLC, Sole Mbr**（IRS/银行文件常见标注）<br/>- **Emily Johnson Sole Mbr**（银行/信用报告可能出现）<br/>- **TechVision LLC**（multi-member LLC）<br/>- **Global Trade Inc.**<br/>- **Bright Future Corp.** | - **Smith & Lee Partnership**                          |
| **IRS 识别关键点** | - 报税使用 **SSN 或 EIN**<br/>- 收入直接进入个人报表               | - 必须申请 **EIN**<br/>- 税务处理取决于 LLC / C-Corp / S-Corp 选择                                                                                                                                                                                                   | - 必须申请 **EIN**<br/>- IRS 要求提交 **Form 1065**            |

## 共通要求

无论选择哪种主体，所有商户均需符合以下条件：

### 最低年龄要求

- 必须年满 **18 岁**，确保具备签署合同和管理商业交易的法律能力。

### 银行账户要求

- 必须是 **美国本地银行** 开设的美元支票账户（Checking Account）。
- 账户名称必须与注册主体完全一致：
  - 独资经营 → 银行账户名 = 个人姓名
  - 公司/合伙企业 → 银行账户名 = 法定注册名称
- 银行账户必须符合 **ACH 自动清算资格**。

### 最终受益所有人（UBO）要求

除独资经营外，在组织审核过程中，**Genstore** 会验证与主体相关的所有自然人，包括所有者、控制人和签字人。

| 对象               | 标准                                                         | 所需人数                                                |
| ------------------ | ------------------------------------------------------------ | ------------------------------------------------------- |
| 通过所有权识别 UBO | 直接或间接拥有公司 25% 以上股份、表决权或其他权益的个人。    | 0–4 人，需提供所有符合条件的个人信息                    |
| 通过控制权识别 UBO | 行使公司最终有效控制权的个人。如果无此人，则需识别高级管理人员。 | 至少 1 人（如果没有符合所有权标准的人，或在美国运营时） |
| 签字人             | 法定代表人或授权代表，可以代表组织与 **Genstore** 签订协议。 | 至少 1 人                                               |

## 主体文件要求

根据主体不同，需提交的文件也不同。以下是四类主体的详细进件要求：

### 独资经营（Sole Proprietorship）

**必需文件**

- 公司/业务文件：
  - Schedule C（附在 Form 1040）
  - SS-4 确认信 / CP575（仅在申请 EIN 时提供）
  - 地址证明 (可选)
- 个人文件：
  - 身份证件（美国国籍人士需 SSN；非美国国籍人士可用护照/驾照/国家 ID）
  - 居住地址证明 (可选)
- 银行文件：
  - 存款单、银行对账单截图、银行官方信函或支票

**补充说明**

- 若 Genstore 无法自动验证，可能会要求提供 **DBA filing（商号注册/营业执照）** 或最近的税务申报单。

### 公司（Company：LLC / Corporation）

**必需文件**

- 公司/业务文件：
  - 公司注册文件（Articles of Organization / Incorporation）
  - SS-4 确认信（EIN）
  - Operating Agreement（LLC） / Bylaws（Corporation）
- 个人文件：
  - 董事/股东的身份证件和居住地址证明
  - 所有权证明（如股东名册）
- 银行文件：
  - 存款单、银行对账单截图、银行官方信函或支票

**补充说明**

- 若无法通过文件确认所有权结构，Genstore 可能要求提供 **所有权和控制结构图**（含签字、职位、日期，且不超过 6 个月）。

### 合伙企业（Partnership）

**必需文件**

- 公司/业务文件：
  - Partnership Agreement（合伙协议）
  - SS-4 确认信（EIN）
- 个人文件：
  - 合伙人身份证件、居住地址证明
  - 合伙人持股/权益证明
- 银行文件：
  - 存款单、银行对账单截图、银行官方信函或支票
