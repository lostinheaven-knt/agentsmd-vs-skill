# 快速上线 Checklist

确保您的在线商店快速上线并运行顺利，请参阅以下快速上线 Checklist。

## AI 问询

初始化建站完成后，AI 将主动启动关于独立站经营的问询，以了解您的商家信息，并推荐更富针对性、高效的建站任务。具体可阅读 [与 AI Agent 协作](./start-genstore-account-homepage.md)。

::: tip 

AI 问询将默认使用您的账户语言进行交流。不过，由于 Genstore 采用了先进的 LLM 技术，当您使用不同语言与 AI 互动时，系统会智能识别并自动切换为相应语言，以确保沟通顺畅。

:::

### 问询内容

- 商家的业务基础及准备阶段
- 商家的流量基础及推广预期
- 商家的业务经验
- 商家选择 Genstore 的目的

## 基础设置

在进行网站配置和营销推广前，请先完成以下基础设置，确保店铺信息完整、后台权限合理分配：

- [设置商店信息](./start-genstore-account-create-manage.md)：完善商店名称、Logo、联系方式、公司地址等基础信息
- 设置 [时区](./start-genstore-account-create-manage#管理店铺时区) 与 [币种](./start-genstore-account-create-manage#选择店铺货币)：确保订单和财务数据准确显示
- [添加员工账号](./start-genstore-account-roles-permissions.md)：为团队成员创建账号并分配权限
- 设置 [店铺语言](./settings-languages.md) 与 [翻译](./global-market-language-translate.md)：配置默认语言，并启用其他语言翻译（如多语言站点）
- [选择付费套餐](./start-understand-pricing-plans.md)：了解并选择适合您业务的套餐，正式启用商店

## 建站任务

问询结束后，AI 将为您定制必要的运营准备任务，主要涵盖：

### 在线售卖

- [实物产品](./operate-product-physical.md)：添加一个产品（添加/导入）
- [数字产品](./operate-product-digital.md)：添加一个产品（添加/导入）
- [店铺迁移](./start-migrate-guide.md)：通过 Genstore 店铺迁移应用，导入产品、订单、客户

### 网站基础配置

- 选择 [装修模版](./operate-store-design-themes.md) 或 [AI 建一个首页](./start-genstore-account-homepage.md)
- [链接到社交媒体](./start-genstore-account-brand-assets#社交媒体链接)（如 Instagram、Facebook、X、TikTok）
- [链接到电商平台](./expand-sales-channels.md)（如 Amazon、Tiktok）
- 设置 [运费模版](./operate-fulfillment-shipping-profile-shipping-rate.md)
- 设置 [支付](./operate-payments.md)


### 正式上线
- [关联域名](./global-domains.md)

### 推广与营销

- 推广 SEO/广告
- [自动化营销](./expand-marketing-automated-email-overview.md)

完成以上步骤后，您的商店将准备就绪，可以开始吸引和转化客户。
