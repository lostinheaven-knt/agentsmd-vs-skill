# 选择模版

要打造一个专业、有品牌感的在线商店，第一步就是选择一款合适的主题模版。Genstore 提供了多种风格和行业类型的 [主题模版](https://themes.genstore.ai)，帮助您在建店初期快速搭建页面结构，呈现出符合品牌调性的视觉效果，并提升用户的浏览和购物体验。

您可以根据自己的行业、产品类型或关键词，筛选出与品牌定位和用户需求高度匹配的模版，快速确定店铺的整体布局与页面风格。

## 按行业筛选主题模版

如果您已经明确了自己的产品类型、所属行业和目标用户，可以通过行业分类快速筛选出与之匹配的主题模版，提升建店效率。

Genstore 针对服饰、美妆、家居、电子产品等热门行业，预设了多种优化主题模版。这些模版在页面布局、视觉风格和常用交互上已做出适配调整，能够满足行业特有的展示需求，帮助您快速搭建出专业、协调且具有品牌感的在线商店。

- 登录 Genstore 后台，点击左侧菜单 **商店** -> **在线商店** -> **模版**。
- 在 **推荐模版** 区域，点击 **访问主题商店**。
- 进入 [Genstore 主题商店](https://themes.genstore.ai/zh-CN)。
- 点击顶部菜单的 **浏览全部**。
- Genstore 主题商店将按照行业返回当前站点的所有模版列表。
- 点击选择您所在的行业分类（例如：服装、美妆、电子产品、食品饮料等）。
- 浏览模板列表，根据预览、评分、价格等信息选择合适主题。
- 点击进入模板详情页，查看演示站点（Desktop 和 Mobile），确认是否符合品牌调性。
- 点击 **安装** 即可将该模版应用到商店，您点击模版后的 **预览** 按钮查看实际展示效果。

## 按关键词筛选模版

如果您对模版所需的页面结构或设计风格有明确预期，也可以通过关键词搜索的方式快速定位目标模版。

- 登录 Genstore 后台，点击左侧菜单 **商店** -> **在线商店** -> **模版**。
- 在 **推荐模版** 部分，点击 **访问主题商店**。
- 进入 [Genstore 主题商店](https://themes.genstore.ai/zh-CN)。
- 在首页上方搜索框中输入关键词进行搜索。
- 点击回车或放大镜图标，查看搜索结果。
- 对比模板的功能说明、演示站点、价格等，选出最符合你需求的一个。
- 点击 **安装** 即可将该模版应用到商店，您点击模版后的 **预览** 按钮查看实际展示效果。
