# 外观设置

本页面用于配置你的网站客服聊天窗口（AI Agent / 人工客服入口）的**外观样式与入口文案**。良好的外观设置能显著提升客户的发起咨询率。

## 设置入口
1. 登录 Genstore 商户后台，点击 **销售渠道** -> **Inbox**。
2. 在客服工作台处，点击 **进入**。
3. 在左侧的菜单栏中，点击 **设置** 下的 **外观**，您可：
  - 修改[欢迎语](#欢迎语)
  - 修改[会话入口文案](#会话入口)
  - 修改 [Inbox 控件的位置与大小](#控件样式)

## 欢迎语
当客户点击打开聊天窗口时，首先看到的引导文字。
* **作用**：打破沉默，引导客户开口提问。
* **设置建议**：
    * 保持简短（1-2行），使用亲切的语气。
    * 建议包含品牌特色，如：“Hi, there👋 How can I help you ?”
    * 支持表情符号（Emoji），增加亲和力。

## 会话入口
配置在聊天气泡未展开时，卡片上显示的文案。

* **标题**：建议使用变量 <span v-pre> `{{ AI Agent name }}` </span>。系统会自动调用您在 AI 设置中定义的名称，增强身份感，例如：“Ask `Olivia` for help”。
* **副标题（Subtitle）**：一句话说明服务能力，例如：“AI agent and team can help”。

## 控件样式
您可以根据网站装修风格，调整 Inbox 按钮的物理属性，避免遮挡页面核心内容（如结账按钮）。
| 配置项 | 可选项 | 建议场景 |
| :--- | :--- | :--- |
| **浮球规格** | 小 / 中 / 大 | 默认值为 **大**， 建议根据页面布局调整。|
| **浮球水平位置** | 左 / 右 | 多数站点推荐 **右**，符合右手操作习惯。 |
| **浮球垂直位置** | 低 / 中 / 高 | 若页面底部有 Cookie 弹窗或浮窗，请设为 **高**。 |

## 实时预览
在页面中部的店铺预览区会实时更新效果，您还可点击底部悬浮工作栏中的 **手机** 图标，查看桌面和移动端预览效果，您可根据需要调整规格、位置。确认效果后，点击 **发布** 即可应用。

::: tip
