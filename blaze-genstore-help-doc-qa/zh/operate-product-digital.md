# 数字产品

数字产品指用户可通过电子方式直接获取、下载或在线访问的产品，无需任何实体配送，涵盖所有虚拟产品类型。常见形式包括电子书、软件、音频文件、视频课程、设计资源等，广泛应用于教育、创意设计、软件开发、娱乐音频和商业资源等领域。

通过网络销售数字产品，商家可有效拓展销售渠道，降低物流与库存成本，同时为客户提供灵活、高效的产品交付体验。

## 添加数字产品

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 点击页面右上角的 **添加产品**，选择 **数字产品**。
3. 设置产品标题及产品特色。
4. 从资源库选择或在本地上传介绍图片/视频，支持多张图片上传，并可调整图片顺序，首张图片将作为主图。
5. 使用富文本编辑器详细描述产品的用途、规格等，支持插入图片、链接和表格。
6. 上传与发送产品文件：上传单个文件（最大 5GB，支持图片和视频格式）。可设置文件下载次数限制（默认不限），并选择发送方式：**自动发送文件** 或 **手动发送文件**。
	- 若不添加下载文件，该产品将被视为虚拟产品（如运费补差、服务项目等），不会包含可下载内容。

其他项目的设置可参考 [添加产品](./operate-product-create.md) 页。完成设置后，点击 **保存** 发布产品。

## 筛选数字产品

您可根据产品类型筛选数字产品。

1. 在产品列表中，点击 **筛选**。
2. 选择 **产品类型** 为 **数字产品**，并根据需要在 **数字文件** 中筛选是否含有下载内容。
3. 您可点击产品名称进入数字产品详情页。
4. 在产品列表中，点击 **数字文件** 列下的数字，可查看该产品各规格的已上传文件。

::: tip

如产品表格中未显示“数字文件”列，您可点击右上角的 **自定义显示列** 图标，添加数字文件列。

:::

## 删除数字产品

如需删除一个或多个数字产品，可按以下方式操作：

1. 按上述步骤筛选出数字产品。
2. 勾选目标产品左侧复选框。
3. 点击页面顶部的 **批量删除** 按钮。
4. 系统将提示是否确认删除，操作不可恢复，请谨慎执行。

## 配置数字产品的可选版本

在 Genstore，数字产品同样支持多规格配置。每个规格变体可设置独立价格、库存跟踪，以及上传对应的数字文件（如下载内容或观看资源）。

数字产品规格的设置流程与实物产品基本一致，不同之处在于：

- **规格设置后**，每个规格（SKU）需上传**数字文件**，以供顾客购买后下载或访问。
- 每一个规格后均有 “上传” 入口，支持分别与对应文件绑定。
- 系统不会提供物流信息、重量或原产地等字段。

### 创建步骤

1. 登录 **Genstore 商家后台**，点击 **商店** -> **产品**。
2. 点击右上角 **添加产品**，选择 **数字产品**。
3. 设置标题、产品特色、图片及描述。
4. 在 **规格** 部分，点击 **+ 添加尺寸或颜色等信息**，配置如“格式”、“时长”、“清晰度”等自定义规格项。
5. 系统将自动生成变体（SKU）表格。您可以为每个版本设置：
    - 价格
    - 是否跟踪库存，如跟踪，同时设置库存数量
    - 上传对应的数字文件

### 替换数字文件

如需更新已绑定在数字产品上的文件（如课程视频、电子书、素材包等），您可以通过产品详情页中进行文件替换操作。

1. 在产品列表中，根据产品类型筛选出 **数字产品**。
2. 找到并点击目标数字产品，进入产品详情页。
3. 在规格明细表格中，定位到需要替换文件的产品（SKU）。
4. 在 **数字文件** 列中的文件名称，在弹出的菜单中选择 **替换文件**。
5. 勾选需要替换文件的产品，点击 **确认**。

### **删除数字文件**

#### **操作步骤**

1. 在产品列表中，点击产品类型，选择 **数字产品**。
2. 在目标数字产品后的 **数字文件** 列，点击数字（表示该产品包含的数字文件数量）查看该产品的数字文件。
	> 如数字文件列暂时显示在列表中，您可点击表格右上角的 **自定义显示列** 图标，添加数字文件列。
3. 在弹出的窗口中，点击目标文件后的 **删除** 图标。
