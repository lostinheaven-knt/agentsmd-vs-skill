# 更多支付方式

除 [Genstore Payments](./operate-genstore-payments.md) 服务外，Genstore 还支持多种第三方支付服务提供商，使您能够灵活选择适合您业务的支付解决方案。通过第三方支付服务，您的客户可以直接在您的在线商店中完成支付，无需跳转至第三方页面，提升购物体验。

## 如何设置

1. 登录 Genstore 商家后台，点击 **设置**。
2. 在 **支付** 部分的 **更多支付方式** 部分，点击 **添加支付方式** 。在列表中选择您想使用的支付服务提供商，分成三类
    * **第三方支付提供商**：支付方式将会嵌入到买家的结账页中，买家不需要跳转到外部的页面即可完成付款。
    * **其他支付方式**：支付方式需要买家跳转到外部页面完成付款。
    * **手动支付方式**：您可以添加手动支付或者货到付款的支付方式。

具体设置步骤，请阅读 [配置第三方支付](./operate-payments-third-party-configure.md)。

## 支持的第三方支付服务提供商列表

下表列出了当前 Genstore 支持的各个第三方支付服务提供商，及其支持的支付方式类型：

| 支付提供商名称        | 类型  | 支付方式 |
| :------------- | :-- | :--- |
| PayPal         | 外部  | 电子钱包 |
| Stripe         | 直连  | 卡    |
| Lianlian       | 直连  | 卡    |
| Asiabill       | 直连  | 卡    |
| Pingpong       | 直连  | 卡    |
| Braintree      | 直连  | 卡    |
| Authorize.net  | 直连  | 卡    |
| Oceanpayment   | 直连  | 卡    |
| Airwallex      | 外部  | 卡    |
| UseePay        | 外部  | 卡    |
| AsiaBill Local | 外部  | 本地支付与电子钱包 |
| Payssion       | 外部  | 本地支付与电子钱包 |
| PricPay        | 外部  | 本地支付与电子钱包 |

注意：

- 当您选择使用第三方支付提供商时，请确认已了解各提供商的费率及支付处理时间。
- 如果您删除第三方支付方式的授权关系或者删除账户托管参数，那么历史订单的退款动作很有可能无法执行，请谨慎删除授权或者删除账户托管参数。
