# 将第三方域名连接到 Genstore

如果您已拥有第三方域名提供商（如 GoDaddy、IONOS、Namecheap、Cloudflare 等）提供的域名，您可以将该域名连接到您的在线商店。当第三方域名成功连接到 Genstore 后，系统将免费为您自动创建 TLS 证书。

#### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **域名**。
2. 点击 **连接现有域名**。
3. 在 **域名** 字段中，输入您的域名，例如 `example.com`，然后点击 **下一步**。
4. 点击 **自动连接**，或展开**手动配置**根据指引完成 DNS 记录更新。

域名验证可能最多需要 48 小时才能完成。经过验证后，您的域名将在 **域名** 页面的 **域名** 部分列出，并且状态为**已连接**。

## 自动连接

Genstore 与 Entri Connect 合作，为您提供自动连接域名服务，您无需手动配置，仅需通过登录授权即可将您的域名DNS自动指向Genstore。

使用自动连接功能将消耗 `10` 积分/域名。在自动连接开始前我们将进行积分预扣，并在自动连接成功时正式抵扣。若您在过程中取消或自动连接失败，被预扣的积分会被退回至您的店铺。

## 手动配置

若使用手动配置，您需要登录到域名提供商账户（如 GoDaddy、IONOS、Google Domains等），然后更新 DNS 记录。

更新 DNS 记录后，您需要等待更改生效。这通常在两小时内生效，但可能需要长达两天的时间。

正确的DNS记录值：

| 类型    | 名称  | 正确的记录值                                   |
| ----- | --- | ---------------------------------------- |
| A     | @   | `172.66.1.107` 或 `162.159.141.111`       |
| AAAA  | @   | `2a06:98c1:58::16a` 或 `2606:4700:7::16a` |
| CNAME | www | `shops.genmystore.com`                   |

::: tip

- 请删除域名上的其他任何不正确的 A/AAAA/CNAME 记录值（如果已存在任何此类记录）。
- 若您连接的是子域名，只需要配置相应的 CNAME 记录指向 `shops.genmystore.com`，无需配置 A/AAAA 记录。如：连接 **aaa.example.com**，您只将名称为 **aaa** 的 **CNAME** 记录指向 `shops.genmystore.com`
- 您无需更改 DNS 设置中的 **TTL** 编号。请使用默认值。

:::
