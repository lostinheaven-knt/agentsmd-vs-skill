# 迁移完成后的检查清单

完成迁移后，请使用此清单确认您的商店设置是否齐全，并确保客户可以顺利访问和使用您的在线商店。

::: tip

由于支付功能暂不支持迁移，本检查清单的操作前提是您已在新店铺中完成支付配置。

如尚未配置，请参考 [支付配置](./operate-payments.md) 进行操作。

:::

## 商店展示与内容

[ ] 已选择并应用适合品牌的主题模板
[ ] 已完成模板样式自定义（字体、颜色、按钮等）
[ ] 所有产品均已正确归类至产品系列
[ ] 产品详情页信息准确无误（图片、价格、库存）
[ ] 导航菜单结构合理，页面跳转无误
[ ] 博客内容已迁移且格式正常显示（如适用）

## 商店功能测试

[ ] 已完成至少一笔测试订单（包括成功与失败情形）
[ ] 已测试订单取消、退款与部分发货流程
[ ] 订单可正常归档，状态更新及时
[ ] 支付方式已启用并配置正确（含第三方支付，如 PayPal）

## 客户与账户管理

[ ] 所有客户数据已正确导入，无重复或缺失信息
[ ] 已发送账户激活或重置密码邮件给客户
[ ] 客户可正常登录并下单
[ ] 客户消息可在 Inbox 中接收并及时回复

## 商店发布与访问

[ ] 商店已解除密码保护并对公众开放
[ ] 所有页面在桌面端与移动端均显示正常
[ ] 所有按钮与链接跳转正常，无 404 页面

## 推广与营销设置

[ ] 首页已设置元标题和描述，优化 SEO
[ ] 已启用 Genstore Email 并配置发件信息
[ ] 至少创建一个营销邮件活动
[ ] 已添加折扣码或促销设置
[ ] 已同步销售渠道（如 Facebook、Google 等）

## 持续维护与支持

[ ] 已了解并配置数据备份与导出方式
[ ] 已熟悉帮助中心文档与支持渠道
[ ] 有定期检查订单与客户反馈的流程
[ ] 已设置运营与营销的周期性任务（如每周查看分析数据）

✅ 若全部项已勾选，恭喜您完成迁移后的最后准备工作！建议将此检查清单保存为 PDF 或打印，用于团队内部协作与复查。
