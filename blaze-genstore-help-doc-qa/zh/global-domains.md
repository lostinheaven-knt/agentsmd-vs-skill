# 域名

域名即 URL 或网址，您的客户可通过它找到您的在线商店。使用与品牌相匹配的域名可帮助您在您的在线商店中建立信任感。您可以在**域名**页面上找到您的当前商店 URL。

默认情况下，当您注册 Genstore 店铺时，您的在线商店将获得 `.genmystore.com` URL。若要更改向浏览您在线商店的客户显示的网址，您可以**更改您的 `.genmystore.com` 域名**（注意：只有一次更改机会）；此外，您还可以添加自定义域名（如 `www.example.com`），例如，从任何**第三方域名提供商**处购买的域名。将域名添加到 Genstore 后，您可以在 Genstore 后台的**域名**页面上管理在线商店的域名设置。
