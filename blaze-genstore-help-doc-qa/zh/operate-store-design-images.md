# 图片

Genstore 提供丰富的图片管理功能，帮助您通过上传 **Logo**、**网站图标（Favicon）**、**产品图片**、**幻灯片**、**横幅**、**博客文章图片** 和库存图片，实现店铺的个性化设计。您也可以上传自己的图片，获得更高的自定义空间。

## 上传 Logo 与网站图标（Favicon）

您可以在 **主题设计** 面板中上传店铺的 Logo 和网站图标。上传的 Logo 将作为店铺的主要品牌标识，在所有页面中统一展示。

::: tip

如未上传 Logo，系统将默认显示您的店铺名称。

:::

### Logo 设置

Logo 图片用于代表店铺品牌形象。您可以根据需要调整桌面端展示时的宽度。

### 网站图标（Favicon）

网站图标会显示在浏览器标签页和书签中，建议使用尺寸为 96 × 96 像素的正方形图片。

### 操作步骤

1. 登录 Genstore 商家后台，点击 **商店** -> **在线商店** -> **模板**。
2. 点击所选模板右侧的 **设计** 按钮。
3. 在左侧编辑器面板中，点击第四个图标进入 **主题设计**。
4. 展开 **Logo** 模块，上传 Logo 图片，并根据需要调整宽度滑块。
5. 上传网站图标。
6. 点击右上角 **保存**，保存并应用修改内容。

## 上传图片

您可以上传产品图片、幻灯片、横幅和博客文章图片，并通过裁剪和调整大小来优化图片，灵活自定义媒体文件设置，以更好地展示您的商店。

### 操作步骤

1. 登录 Genstore 商户后台，导航至 **商店** -> **在线商店** -> **模板**。
2. 点击目标模板后的 **设计**。
3. 比如您希望添加或更改主页展示的图片横幅，在模板编辑器的侧边栏中，点击 **模版** -> **图片横幅**。在页面右侧的 **图片横幅** 窗口中完成相关操作。
4. [可选] 如果您希望在商店主页以外的页面添加图片，点击主页下拉菜单，选择要编辑的页面模板，如 **产品**、**产品系列**。
   - 在模板编辑器侧边栏中，点击 **添加分区**，从下拉列表中选择 **图片横幅**，调整图片标题、文本、按钮，或添加更多块。
   - 在页面右侧的 **图片横幅** 窗口中，上传图片，并完成相关设置。
5. 点击 **保存**。

### 图片优化建议
- **图片支持**：WebP、JPG/JPEG、PNG、SVG、AVIF、GIF、HEIC、BMP
- **压缩图片**：使用专业的图片压缩工具（如 [TinyPNG](https://tinypng.com/)、[Squoosh](https://squoosh.app/)），在不损失太多质量的情况下减小文件大小，提高页面加载速度。
- **使用描述性文件名**：例如，将 `IMG_1234.jpg` 改为 `blue-t-shirt-front-view.jpg`，有助于搜索引擎理解图片内容，提升 SEO。
- **优先使用正确格式**：建议产品图使用 JPEG，带透明背景的图标或 Logo 使用 PNG。
- **保持一致比例**：尽量遵守推荐比例，避免图片被裁剪或拉伸影响美观。
