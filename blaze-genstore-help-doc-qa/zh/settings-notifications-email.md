# 管理店铺相关邮箱

电子邮件在接收重要通知和与客户沟通中扮演着关键角色。本页面将指导您如何设置和管理**店铺邮箱**和**发件邮箱**，以便高效地处理业务沟通并保持良好的客户联系。

在 Genstore 中，您将使用三种邮箱：

- **店铺邮箱**：用于 Genstore 与您联系，例如系统通知、账单提醒等。该邮箱默认为您注册 Genstore 时填写的电子邮件地址。
- **发件邮箱**：用于向客户发送订单通知、营销邮件等，客户将通过此邮箱接收来自您商店的所有邮件通知。为提升品牌专业度，建议使用自有域名的品牌邮箱作为发件人邮箱，并完成域名验证。
- **收件邮箱**：用于接收客户对商店邮件的回复。客户对店铺邮件的回复将发送至此邮箱地址。

## 更改店铺邮箱

默认情况下，店铺邮箱是您注册 Genstore 时填写的电子邮件地址。如果您希望 Genstore 通过新的电子邮件地址与您联系，您可以更新该信息。

#### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **通用**。
2. 在 **店铺详情** 部分，点击店铺名称及邮箱后的铅笔图标。
3. 在 **店铺邮箱** 下输入新的邮箱地址。Genstore 将通过此邮箱与您联系。
4. 点击 **保存**。

注：若要更改您用于登录 Genstore 的账号邮箱或密码，可参考 [验证或更换登录邮箱](./start-genstore-account-manage#验证或更换登录邮箱)。

## 更改发件邮箱

发件人邮箱用于向客户发送通知邮件。当客户收到自动通知、订单确认或营销邮件时，此邮箱显示在 **发件人** 字段中。默认情况下，发件人邮箱将显示为类似  `store+123@genstoremail.ai`  的系统邮箱地址，其中 `123` 部分是根据您店铺自动生成的唯一字符串。

如希望使用自有域名的品牌邮箱，可通过如下步骤进行设置：

1. 在 Genstore 后台中，转至 **设置** -> **通知**。
2. 点击 **管理发件人**。
3. 在 **发件邮箱** 部分，输入您的邮箱。
4. 点击 **保存**。

### 验证您的邮箱域名

更改发件邮件为自有邮箱后，需要完成邮箱身份验证，为避免邮件被标记为垃圾邮件，保证邮件顺利送达。

要验证您的域名，并让自定义发件邮箱正常工作，根据引导在域名服务商后台添加新的 DNS记录。

#### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **通知**。
2. 点击 **管理发件人**。
3. 在发件邮箱下方，点击 **验证域名**。
4. 弹窗中将显示详细操作说明，以及需要添加的 DNS 记录。
5. 在新的浏览器标签页中：
	- 登录到您的域名服务商管理后台。 
	- 进入 DNS 管理区域（通常称为“区域编辑器”或“DNS 设置”），添加您从 Genstore 获得的 DNS 记录（保持 TTL 设置不变）。
6. 添加完成后，返回 Genstore 后台标签页，点击 **验证域名** 完成配置。

更改最多可能需要 24 小时才能完成。 如果域名验证失败，请验证您输入到第三方域名提供商中的 DNS 记录是否与 Genstore 后台中提供的记录匹配。

## 更改回复邮箱

回复邮箱用于接收客户对您商店邮件的回复，确保您不会错过客户反馈或问题。您可以在 Genstore 后台更改该邮箱，并通过简单的邮件验证完成配置。

#### 步骤：

1. 在 Genstore 后台中，进入 **设置 -> 通知**。
2. 点击 **管理发件人**。
3. 在 **回复邮箱** 部分，输入新的邮箱地址。
4. 点击 **发送验证邮件**。
5. 系统将向新收件邮箱发送验证邮件，打开邮件并点击其中的**验证邮箱**按钮即可完成验证。
