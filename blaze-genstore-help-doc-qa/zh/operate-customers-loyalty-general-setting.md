# 高级设置

针对订单积分奖励，Genstore 还提供了更多个性化的设置选项，登录商家后台，点击 **商店** -> **客户** -> **忠诚度** -> **设置 -> 通用**。

您可在此定义用于计算积分的订单金额、设定积分发放适用于哪些订单状态，以及在何种情况下应撤销已发放的积分。

## 订单金额

系统默认根据订单的 **小计** 金额计算积分。您可以选择以下费用是否计入积分计算：

- **折扣**：若勾选此项，折扣金额也可用于积分计算。
- **税费**：若勾选此项，税金部分也可用于积分计算。
- **运费**：若勾选此项，运费也可用于积分计算。

## 订单状态

您可以设置在以下订单状态下发放积分：

- **已支付**：订单一经支付完成即自动发放积分，此项默认启用。
- **履约完成**：订单履约完毕后发放积分。

## 积分撤销

您可以设置在以下情况下撤销已发放的积分：

- **退款**：如果订单全额退款，则相关的积分也将全额撤销。
- **部分退款**：如果订单发生部分退款，相应的积分也将按退款金额比例撤销。
- **取消订单**：如果订单取消，相关的积分将全部撤销。

## 邮件提醒

客户忠诚度程序支持您自定义与客户沟通的各种自动化邮件通知。通过合理配置邮件提醒，您可以与客户保持良好的互动关系。

**注意**：当前仅支持对邮件通知场景进行开启或关闭，暂不支持对邮件通知内容进行自定义。

### 系统内置邮件提醒

您可以通过以下路径进行邮件设置：**商店** -> **客户** -> **忠诚度** -> **设置 -> 更多设置 -> 邮箱**。在此页面中，您可以设置发件人名称、邮件接收地址，以及配置各种触发场景的邮件提醒。

#### **基础设置**

- **发件人名称**：建议使用您的品牌名称或商店正式名称，方便客户快速识别邮件来源，名称长度限制为 100 个字符。
- **回件邮箱地址**：用于接收客户邮件的邮件地址，建议使用专门的客服邮箱，以便于集中管理客户反馈。

#### 自动邮件通知场景

- **积分发放**：客户获得积分时自动通知
- **积分过期提醒**：在积分到期前 30 天和 7 天发送提醒
- **生日奖励发放**：客户生日时自动发送，如客户未即使领取，会追加提醒
- **获得奖励**：因积分兑换、VIP 等级达成或推荐成功获得奖励时通知
- **奖励过期提醒**：奖励剩余 30 天以及 7 天时提醒
- **VIP 等级达成**：客户达到新的 VIP 等级时通知
- **完成推荐**：客户成功推荐好友时通知
