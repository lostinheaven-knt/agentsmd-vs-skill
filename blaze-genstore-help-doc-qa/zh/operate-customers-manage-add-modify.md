# 添加/编辑客户

管理客户信息是维持业务效率和确保客户满意度的关键。无论是添加新客户还是更新现有客户数据，准确且及时的信息处理都至关重要。在以下几种情况中，您可能需要添加新客户：

- **新客户注册**：当有新客户通过网站或实体店注册时，需要将其信息录入系统。
- **订单处理**：未注册的游客在下单时提供联系信息，您可以将其作为新客户添加到系统。
- **市场营销活动**：在促销或营销活动中收集到潜在客户信息时，添加这些客户可以后续进行更有效的营销沟通。

本节将指导您如何在 Genstore 系统中添加新客户以及编辑现有客户的信息，并将说明如何批量处理客户数据以提高管理效率。

## 添加客户

**操作路径**：登录 Genstore 商户后台，点击 **商店** -> **客户** -> **添加客户** 。

### 填写客户基础信息

1. **联系信息**：名、姓、邮箱、手机号，至少填写一项。**注意**：邮箱与手机号不得与现有客户重复。
2. **营销订阅**：您可以为客户勾选是否同意接收营销邮件/短信。在勾选前，建议先获取客户的同意。
3. **默认地址**：按实际地址信息填写，可立即填写或稍后补充。地址信息非必填项。
4. **免税设置**：默认为不免税。如需为客户设置免税，请调整免税选项。关于免税更多说明，请参照 [客户免税](./operate-customers-manage-tax-exemption.md)。
5. **客户标签**：在客户标签输入框中直接输入所需标签。关于客户标签更多说明，请参照 [客户标签](./operate-customers-manage-tags.md)。
6. **备注**：添加关于客户的备注信息。

## 编辑客户

**操作路径**：登录 Genstore 商户后台，点击 **商店** -> **客户**，点击目标客户进入客户详情页， 点击 **更多操作** -> **编辑客户信息**。

### 基本信息编辑

- 在客户详情页面点击 **编辑客户信息**，修改客户信息并保存。

### 客户地址管理

- **新增地址**：点击 **添加地址**，为客户添加新的收货地址。
- **编辑地址**：在地址旁点击 **编辑** 按钮，修改信息后点击 **确定** 保存。
- **删除地址**：在地址旁点击 **删除** 按钮进行删除。注意：客户的默认地址暂不支持删除。
- **设为默认**：选择一个地址并点击 **设为默认**，调整默认收货地址。

## 批量编辑客户

除在客户详情页内编辑客户外，您也可以在客户列表页通过勾选多个客户进行批量编辑。

**操作路径**：登录 Genstore 商户后台，点击左侧导航目录中的 **商店 -> 客户**，勾选需要编辑的客户，点击 **批量编辑**。

- 批量编辑支持修改客户的名、姓、邮箱、标签、邮件营销订阅状态、手机号、备注等字段。若您需要编辑的字段不在列表内，您可以点击页面右上角的 **列**，添加其他字段。
- 完成编辑后点击 **保存**。如果出现无法保存的情况，请检查是否有错误提示，并根据提示进行相应修改。
