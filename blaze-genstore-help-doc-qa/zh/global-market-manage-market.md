# 管理市场

Genstore 支持您创建多个市场，以向不同国家和地区分配不同的设置，从而帮助您管理品牌体验。市场中可以包含一个国家/地区，也可以包含一组国家/地区。例如，您可以针对加拿大和美国创建一个名为“北美”的市场，所有国家均采用一组设置。如果您希望仅对日本应用另一组设置，则可以创建另一个仅针对日本这一个国家的市场。

## 添加市场

当您希望为特定地区的客户自定义在线商店体验时，您可以在 **市场** 页面添加新市场。例如，您可能希望将一组国家/地区（例如北美洲）作为一个销售市场，或一个国家/地区（例如加拿大）单独创建一个市场。

:::warning

如果您创建的新市场包含已属于其他市场的国家/地区或区域：

- 该国家/地区会自动从原市场中移除并归入新市场。
- 如移除后，原市场不再包含其他国家/地区，则该市场将被系统自动删除，与其关联的国际价格、域名、子文件夹或自定义项也会一并删除。

:::

### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **市场**。
2. 点击 **添加市场**。
3. 输入 **市场名称**。此名称可帮助您在 Genstore 后台中识别市场，但不会向客户显示。
4. 在 **国家/地区** 部分，搜索要添加的国家和地区，然后选中其名称旁边的复选框。您可以在市场中添加多个国家/地区。
5. 点击 **添加市场**。

## 预览市场

您可以在 Genstore 后台中预览市场的客户体验。您可以使用您添加到该市场的任何语言预览已激活市场和未激活市场。

### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **市场**。
2. 点击进入要预览的市场详情页。
3. 点击 **预览**。

## 更改主市场

在您首次设置店铺时，Genstore 会自动为您创建一个主市场。您可以根据销售策略的调整，将主市场更改为其他国家/地区。

更改主市场后，新的主市场将成为商店的默认市场，并影响默认语言、货币、运费和税率等设置。**请注意**，目标市场必须处于“已激活”状态，且使用的域名为主域名，才可设为主市场。

### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **市场**。
2. 在 **概览** 部分，点击要设为主市场的已激活市场，进入其详情页。
   如果该国家/地区尚未被添加为市场，请先创建并激活相关市场。具体操作，参见 [添加市场](#添加市场)。
3. 点击 **更多操作**。
4. 点击 **设为主市场**。
5. 点击 **确认**。

更改主市场后，原主市场不会被删除，将继续处于已激活状态。

::: tip 限制

- 仅可将“使用主市场域名配置的市场”设为主市场。
  无法将使用子文件夹域名的市场设为主市场，以避免产生 404 错误。
- 仅支持将已激活市场设为主市场。
  无法将未激活市场设置为主市场。

:::

## 停用市场

默认情况下，新添加的市场处于已激活状态（默认启用）。如您暂不计划向某个市场销售产品，可将其设置为未激活。此操作不会删除市场配置（如配送区域等），也不会影响已存在的订单，您仍可随时重新启用该市场。

- 停用后，来自该市场的客户将无法结账
- 停用使用顶级域名或子域名的市场，该域名将重定向至主市场域名
- 停用使用子文件夹域名的市场，其 URL 将无法访问

### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **市场**。
2. 在 **已激活** 部分中，点击您要停用的市场。
3. 点击 **已激活** 下拉菜单，并选择 **未激活**。
   
   

## 删除市场

您可以删除不再需要的市场。删除操作会阻止该市场的客户下单，并使其文件夹 URL 无效，但不会影响已生成的订单。删除市场会将该市场永久性删除。**请注意**，目标市场必须处于“未激活”状态，才可被删除。

### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **市场**。
2. 在 **未激活** 部分中，点击您要删除的市场。
3. 点击 **更多操作**。
4. 点击 **移除市场**。
5. 点击 **确认** 进行确认。

## 编辑现有市场的国家和地区

您可以根据销售策略的变化，在现有市场中添加或移除国家/地区。

### 添加国家或地区

您可以向现有市场中添加国家/地区。例如，原市场包含法国与德国，现在计划拓展至意大利，可将其添加至该市场。

#### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **市场**。
2. 点击您要向其中添加其他国家或地区的市场。
3. 在 **市场详情** 部分中，点击右侧铅笔图案。
4. 在 **编辑市场弹窗** 中，搜索要添加的国家/地区，然后选中其名称旁边的复选框。
5. 点击**保存**。

### 删除国家或地区

您可以从现有市场中删除一个或多个国家/地区，常见的使用场景包括：

1. **手动移除国家或地区**
- 例如，您原有一个“北美市场”，包含美国、加拿大和墨西哥。随着业务策略调整，您决定专注于美国和墨西哥市场，可以将加拿大从“北美市场”中移除。
2. **通过创建新市场自动移除国家或地区**
- 如果您希望将某个国家或地区移出当前市场，并将其纳入新的市场，系统会在您创建新市场时自动完成此操作，无需手动删除。例如：
  - 您原有一个包含多个欧盟国家的市场。随着德国的销售表现持续增长，您希望为德国制定更具针对性的市场策略。此时，您可以为德国创建一个新的市场。系统将自动将德国从原市场中移除并归入新市场，原市场中的其他国家/地区保持不变。

#### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **市场**。
2. 点击要从其中删除国家或地区的市场。
3. 在 **市场详情** 部分中，点击右侧铅笔图案。
4. 在 **编辑市场弹窗** 中，搜索您不再想在此市场中向其销售的国家/地区，然后取消选中其名称旁边的复选框。
5. 点击 **保存**。
