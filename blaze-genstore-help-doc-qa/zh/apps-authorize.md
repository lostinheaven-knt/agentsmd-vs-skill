# 查看应用权限与隐私

权限机制旨在精准控制应用可以访问和修改的商店信息。为了确保商店和客户的隐私安全，Genstore 在处理个人身份信息（PII）时，提供了透明的权限控制。

::: tip

**个人身份信息（PII）** 是指能够唯一识别个人身份的信息，包括但不限于联系人信息、位置数据等。安装自 Genstore 应用商店的应用可能会访问您的商店、客户、订单等数据，并根据应用功能的不同，访问不同类型的 PII。

:::

## 应用可访问的个人信息

每个安装的应用可访问与您的 Genstore 账户关联的个人信息，例如您的联系信息和地址。以下是应用可能会访问的 PII 类型：

* **客户个人信息**：包括姓名、电子邮件、电话、地址等联系信息，及与订单处理相关的地点信息（如 IP 地址和地理位置）和用户代理信息（如浏览器和操作系统）。
* **店主个人信息**：包括店主的所有联系信息，如姓名、电子邮件、电话和地址。
* **员工个人信息**：包含员工的所有联系信息，如姓名、电子邮件和电话。
* **内容提供者个人信息**：涉及博客作者或评论者的信息，如电子邮件、IP 地址、用户代理信息（如浏览器和操作系统）。

例如，管理或发货订单的应用需要访问包含客户 PII 的订单信息，如客户地址和其他联系信息。安装此类应用时，可能会请求访问以下权限：

* **访问最近订单数据**：应用仅限查看过去 60 天内的订单记录。
* **访问所有订单数据**：应用可以查看所有历史和未来的订单记录。

Genstore 会审核所有请求访问完整订单数据的应用，确保它们确实需要这些数据来实现其功能。

如希望了解应用如何访问、使用及删除数据的详细信息，请联系应用开发者或查看应用描述中的隐私政策链接。

## 查看应用隐私和权限

在安装应用时，您可以查看应用所请求的权限以及需要访问的个人身份信息（PII）。安装完成后，您也可以随时在 Genstore 商家后台中查看权限和隐私详情。

* **安装应用时**：在确认安装前，系统会显示应用需要访问的权限和 PII 类型，供您审核是否授予。
* **安装后**：您可以通过以下步骤，查看已安装应用的隐私和权限信息：
  * 登录 Genstore 商家后台中，转到 **应用** 页面 。
  * 在 应用 页面应用列表中，找到要查看权限的应用，点击 ... -> **查看详细信息**。
  * 查看**隐私详细信息**部分，查看应用访问的 PII 类型。
  * 查看**权限详细信息**部分，查看应用有权执行的操作。

## 撤销应用的数据访问权限

您可以通过卸载应用来撤销对数据的访问权限。Genstore 会在 48 小时内向应用开发者发出请求，删除该应用收集的客户个人信息。

若您希望删除单个客户的数据，Genstore 将同时向所有可能涉及该客户信息的应用发送删除请求。要确认数据是否已删除，请直接与应用开发者联系。
