# 安装应用

当您找到需要的应用后，可以通过 Genstore 应用商店将其添加到您的商店。以下是安装应用的详细步骤和后续操作说明。

## 通过 Genstore 应用商店安装应用

### 步骤

1. **登录账户**：使用您的商家账号登录 Genstore 应用商店。
2. **查找应用**：浏览或搜索目标应用，点击进入其描述页面。
3. **阅读详情**：仔细查看应用的功能介绍和用户评价，确认符合您的需求。
4. **点击安装**：在应用描述页面中，点击**安装**按钮。
5. **授权应用**：系统会跳转至您的 Genstore 商家后台，点击**安装**按钮完成授权，允许应用访问您的商店数据。
6. **安装完成**：安装成功后，您可以在 Genstore 商家后台的应用页面查看并使用该应用。

## 后续操作

1. **查看已安装应用**：登录 Genstore 商家后台，转到 **应用** 页面，查看已安装的应用列表。
2. **开始使用应用**：点击应用图标，根据应用指引操作，开始享受其功能和服务，为您的商店带来更多价值。
