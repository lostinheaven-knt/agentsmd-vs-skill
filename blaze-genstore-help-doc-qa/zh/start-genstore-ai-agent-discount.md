# 营促销 Agent

营销 Agent 是您的首席营销策划专家。通过 AI 协作，您可以快速创建高转化的折扣活动，并自动同步到店铺首页公告栏。无论是为了清理库存还是提升客单价，AI 都能为您提供全自动化的营销体验。

借助营销 Agent，您可以跳过繁琐的活动规则设置，直接通过对话让 AI 为您创建高转化的折扣活动。

## 竞品智能创建
- **适用场景**：对标行业竞争对手力度或参考市场热门玩法。
- **指令示例**：“根据市场竞品分析给我生成一个折扣活动。”
- **AI 执行**：AI 会实时分析同品类 TOP 竞品 的玩法，结合您店铺的销售趋势，生成一份更具竞争力的折扣提案。
  - **视觉同步** (Visual sync)：AI 会同步为您设计首页公告栏 (Announcement bar)，营造强烈的促销氛围。
  - **自动排期** (Auto-scheduling)：活动开启时自动生效，结束时即刻下架，确保店铺信息始终准确。

## 直接智能创建
- **适用场景**：已有明确促销目标，如黑五（Black Friday）、新品推广或季末清仓。
- **指令示例**：“帮我做一个黑五促销活动，满 $500 打 9 折。”
- **AI 执行**：AI 会根据您的目标直接构建活动框架，包含门槛设定和折扣力度。 
  - **视觉同步** (Visual sync)：AI 会同步为您设计首页公告栏 (Announcement bar)，营造强烈的促销氛围。
  - **自动排期** (Auto-scheduling)：活动开启时自动生效，结束时即刻下架，确保店铺信息始终准确。
