# 为店铺添加政策页

为了保障店铺的正常运营，尤其是在通过 Facebook Ads、Google Ads 等销售渠道进行推广时，建议您在店铺中完善以下政策信息：

- 退换货政策
- 隐私政策
- 服务条款
- 配送政策
- 联系方式

这些政策信息通常会显示在店铺前台的页面底部，您可以按照以下三个步骤完成设置：先创建政策页面，再将页面加入底部菜单，最后在前台页面中展示底部菜单。

## 创建政策页面（任选以下方式）

### 方式一：使用 Genstore 模板快速生成（适合标准政策内容）

1. 进入后台 **设置** -> **政策**。
2. 按需填写或修改各项政策内容。
3. 完成后点击 **确定** 保存。

### 方式二：自定义创建页面（适合自有政策内容）

1. 登录 Genstore 商家后台，点击 **商店** -> **在线商店** -> **页面** -> **创建页面**。
2. 输入页面标题，并填写政策正文内容。
3. 可根据需要进行格式编辑和排版。
4. 在 **可见性** 中选择 **可见**，设置页面上线状态。
5. 点击 **保存** 完成创建。

### 方式三：通过主题编辑器创建（适合更丰富的视觉样式）

1. 进入 **商店** -> **在线商店** -> **模版**，选择当前主题右侧的 **设计**。
2. 在主题编辑器中，点击最左侧的 **页面管理** 图标（第三个图标），点击 **创建模版**（或选择已有模板）。
3. 如选择新建模版，输入模版名称，选择基础模版，点击 **确定**。
4. 按需添加文字、图片、布局组件等，实时预览编辑效果。
5. 完成后点击右上角 **保存**。

有关页面模板的详细操作说明，可参考 [主题编辑器使用指南 - 页面模板](./operate-store-design-themes-edit-guide#页面模版)。

## 将政策页面添加到底部菜单

1. 进入 **商店** -> **在线商店** -> **菜单**，找到 **页脚菜单**，并进入编辑。
2. 点击 **添加菜单项**，填写菜单名称（即前台显示名称）。
3. 点击 **URL** 输入框，选择对应的政策页面：
	- 使用模板创建的，在 **政策** 中选择；
	- 自定义或主题编辑器创建的，在 **页面** 中选择。
4. 确认后点击 **保存**。

## 在店铺前台显示政策菜单

1. 进入 **商店** -> **在线商店** -> **模版** -> **设计**，进入当前主题。
2. 在 **页脚** 区域添加 **菜单区块**。
3. 选择刚刚配置好的 **页脚菜单**。
4. 点击 **保存**，前台页面即可展示更新后的政策菜单。
