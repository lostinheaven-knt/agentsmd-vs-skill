# Genstore POD

Genstore 提供两种 Dropshipping 模式：
- 如果您希望直接售卖成熟产品，请前往 [选品中心](./operate-dropshipping-product-sourcing.md)；
- 如果您希望打造品牌 IP 或提供定制服务，请使用 Genstore POD (按需印刷) 方案。

Genstore POD 是一个集选品、产品设计、自动/手动订单履约、物流跟踪、统一结算于一体的按需印刷解决方案。

## 业务场景

- **场景一**: 创作者 / KOL / IP / 网红：通过作品、Logo 或梗图制作周边（如 T-shirt、连帽衫、贴纸），无需囤货，也无需提前投入。
- **场景二**: 想做品牌服饰（如潮牌、健身、电商品牌）但没有工厂资源，需要高质量印花（DTG / 刺绣）支持。
- **场景三**: 团体定制：为线下活动或组织提供少量定制服装，要求快速交付且品质稳定。

## 核心功能：选品与设计
在开始售卖前，您可以利用以下工具完成产品的创作与推送：
- 选品池：整合海量可靠供应商，商户可直接挑选适合定制的基础款底衫。
- [快速设计器](./operate-dropshipping-pod-designer.md)：针对无专业设计能力的商户，支持直接上传图片或插入简单文本即可生成产品。
- [高级设计器](./operate-dropshipping-pod-designer-advanced.md)：提供多图层管理、多区域印刷位置设定，通过 Mockup 图满足品牌打造、IP 联名等高阶设计场景需求。

### 设计器对比选择

为了帮助您选择最适合当前业务需求的工具，请参考下表：

| 维度 | [快速设计器](./operate-dropshipping-pod-designer.md) | [高级设计器](./operate-dropshipping-pod-designer-advanced.md) |
| :--- | :--- | :--- |
| **定位** | 极速上手，适合简单图文组合 | 专业定制，适合品牌打造与复杂设计 |
| **核心优势** | **一键渲染 400+ 产品**，批量推送 | **多图层管理**，多位置精准印刷 |
| **设计灵活性** | 基础图片上传、文字编辑 | 支持素材库、付费素材、文字图案填充 |
| **效果预览** | 默认位置实时预览 | **自定义效果图 (Mockup)** 配置 |
| **适用场景** | 快速测试市场热度、KOL 简单周边 | 潮牌主打款、IP 联名、复杂满印产品 |

## 后续操作指引：经营与管理
产品推送至店铺后，您可以通过 Genstore 商户后台完成全生命周期的经营管理：

- [产品管理与定价](./operate-dropshipping-product-management.md)：自定义加价规则，实时监控利润。
- [订单履约](./operate-dropshipping-orders.md)：支持自动发单给供应商，或手动审核后再生产。
- [分账与结算](./operate-dropshipping-fund-splitting.md)：根据您配置的收款方式，自动分账到您的账户，无需手动处理。
