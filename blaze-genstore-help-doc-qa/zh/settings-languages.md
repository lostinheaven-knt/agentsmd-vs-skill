# 管理语言

您可以在 Genstore 后台的 **语言** 部分管理在线商店中的多种语言，还可在 Genstore 后台的 **市场** 部分[为各市场配置语言](./global-market-settings-lang-domain#配置语言)。

::: tip
本文操作适用于在线商店语言设置，即客户在访问您的店铺时所看到的语言，不影响 Genstore 后台管理界面的语言。
:::

## 注意事项

- 发布语言前，请确保已完成对应的翻译。您可使用 [Genstore 翻译](./global-market-language-translate.md) 应用快速完成翻译。
- 添加语言时，系统会自动安装 Genstore 翻译应用。若无需使用，可随时卸载。
- 如已发布语言存在未翻译内容，未翻译部分将在店铺中以默认语言显示。

## 更改在线商店的默认语言

店铺默认语言为您在创建店铺时选择的语言，可根据需要进行更改。更改后，原默认语言将保留在已发布的语言列表中。

1. 在 Genstore 后台，前往 **设置** -> **语言**。
2. 在已发布的语言中，找到默认语言，点击 **...** -> **更改默认值**。
3. 在 **更改默认语言** 对话框中，选择要目标语言。
4. 点击 **保存**。

## 向在线商店添加语言

1. 在 Genstore 后台，前往 **设置** -> **语言**。
2. 点击 **添加语言**。
3. 在 **添加语言** 对话框中，从下拉菜单中选择语言，点击 **下一步**。
4. 将语言分配到市场，在分配弹窗中选择目标市场，点击 **确认**。
5. 添加翻译，可通过以下任一方式：
   - 使用 [Genstore 翻译](./global-market-language-translate.md) 应用添加翻译。
   - 从 Genstore 应用市场安装翻译应用，并按指引完成翻译与发布。
	
## 向在线商店中发布语言

新添加语言默认为未发布状态，客户不可见。您可在完成翻译后发布语言，即可对客户展示。
1. 在 Genstore 后台，前往 **设置** -> **语言**。
2. 在 **未发布的语言** 中，点击目标语言旁的 **...** -> **发布**。
3. 点击 **保存**。

## 删除语言

已发布语言不可直接删除，如需删除，需要先取消发布：

1. 在 **已发布的语言** 中，点击目标语言旁的 **...** -> **取消发布**。
2. 当前语言将出现在 **未发布的语言** 列表中，点击该语言旁的 **...** -> **删除**。

> 删除语言后，相关翻译数据将被永久移除，请谨慎操作。
