# 分享社媒内容

将已创建的内容分享到社交媒体平台，帮助您吸引潜在客户。您可以随时编辑内容草稿并发布。

:::tip

每月可发布 100 条社交帖子。您可在页面右上角的 **每月额度** 部分确定当前余额。

:::

## 发布分享

1. 登录 Genstore 商家后台，导航至 **营销** -> **社媒营销**。
2. 在 **社媒营销** 主页，点击 **创建新帖**按钮。
3. 选择您要分享的 **社媒平台**。
4. 自定义**编辑**您要发布的内容。

## 发布内容类型

- **发布为**：选择内容类型（贴文、短片、限时动态）。
- **发布内容**：编辑要发布的文本内容。您可以：
  - 添加表情符号。
  - 插入站内网页链接。
  - 添加产品链接，推广产品。
- **发布时间**：选择立即发布或设定特定时间。
