# 定价

欢迎了解 Genstore 的定价套餐！我们提供多种套餐选项，满足不同商家需求。每个套餐都包含一系列功能，帮助您快速建立和优化在线商店。

## Genstore 套餐计划

### [Genstore Free 套餐](./start-understand-pricing-plans-free-trial.md)

- 一次性赠送 20 点免费积分
- 每天登陆系统，可自动获得 5 点每日积分
- 每个账号最多可创建 **2 个 Free 套餐 店铺**
- 支持创建店铺、AI 产品创建、AI 店铺装修、AI Droshipping 选品等功能。

### **[Genstore Lite 套餐](./start-understand-pricing-plans-basic.md)**

- 套餐内包含 50 点月度积分。
- 每天登陆系统，可自动获得 5 点每日积分。
- 提供经营在线商店所需的所有基础工具和 AI 能力，您可自由创建和自定义您的在线商店，展示产品，进行 Dropshipping 选品，管理订单与客户信息。可使用 Genstore AI 来轻松应对店铺装修、产品运营、数据分析等日常运营工作，提升运营效率。

### **[Genstore Growth 套餐](./start-understand-pricing-plans-genstore.md)**

- 套餐内包含 200 点月度积分。
- 每天登陆系统，可自动获得 5 点每日积分。
- 支持单独购买积分加油包来获得更多积分。
- 解锁更高的选品推送量、更多的资源配额（如员工账号、库存地点及市场数）以及更优的费率标准，帮助您高效管理和优化网站运营

### **[Genstore Scale 套餐](./start-understand-pricing-plans-advanced.md)**

- 套餐内包含 700 点月度积分。
- 每天登陆系统，可自动获得 5 点每日积分。
- 支持单独购买积分加油包来获得更多积分。
- 解锁更高的选品推送量、最高级别资源配额（如员工账号、库存地点及市场数）以及最优费率方案，旨在满足更大规模业务的运营需求

## 通用功能

所有套餐都包括以下通用功能，助您轻松管理商店：

| 功能             | 描述                                                                           |
| -------------- | ---------------------------------------------------------------------------- |
| Genstore AI    | 内置专业 AI Agent 团队，助力您轻松完成个性化站点装修、产品运营、推广策划及数据分析等工作。                           |
| 开箱即用           | 拥有 10 多款官方及内置应用，开箱即可使用。                                                      |
| 无限量产品          | 无限制的产品数量，可展示您的产品和产品系列。                                                       |
| 产品类型           | 可在在线店铺售卖实物产品、虚拟产品以及提供数字产品下载。                                                 |
| Dropshipping 产品 | 可在 Genstore Dropshipping 选品池内挑选产品并推动至店内售卖。                                    |
| 装修模版           | 提供精美装修主题模版，轻松调整店铺外观。                                                         |
| 折扣             | 提供多样折扣活动选项，包含买 X 得 Y、固定套装、产品折扣、订单折扣、运费优惠等。                                     |
| 弃单恢复           | 自动向未完成支付的客户发送邮件，帮助恢复潜在订单。                                                    |
| 欺诈分析           | 当您使用 Genstore Payments 时，可使用内置的欺诈分析工具，该工具通过识别异常行为来识别并标记潜在的欺诈交易，保护您的业务免受欺诈威胁。 |
| 数据报告           | 提供 40 + 种报告，帮助分析店铺运营情况，优化决策。                                                 |

您可以在各类平台上轻松推广和售卖产品，包括 Facebook、Instagram、Google、亚马逊等。

## 客户体验

通过以下功能，提升您的客户购物体验：

| 特征      | 描述                                           |
| ------- | -------------------------------------------- |
| 产品推荐    | 在不同页面向客户推荐智能、精选、畅销、大家都在看、最近浏览、即将售罄等产品，提升转化率。 |
| 产品与博客评论 | 客户可以对产品和博客文章进行评论，商户可管理这些互动内容。                |
| 折扣推荐    | 在产品详情页、购物车等页面给客户自动推荐折扣活动，增加客户购买动力。           |
| 物流轨迹追踪  | 客户可以轻松查看订单物流，了解配送进度。                         |
