# 应用类型

您可以通过安装应用来强化 Genstore 商家后台，或为商店添加新功能。本节将介绍可以添加到 Genstore 商店的不同应用类型。

## 公共应用

公共应用经过 Genstore 应用审核流程，并可以通过 Genstore 应用商店安装。

- **Genstore 应用**：由 Genstore 开发并提供支持的应用。这些应用通常是免费的，并且与平台高度兼容，确保最佳的使用体验。
- **第三方应用**：由 Genstore 合作伙伴开发的应用，经过审核后可以在应用商店中提供。第三方应用的支持由相关开发者提供。

## 自定义应用

[自定义应用](./apps-type-custom.md) 可为您的 Genstore 后台扩展功能，或通过自定义店面将您的在线商店拓展至其他平台。

- **专属开发**：自定义应用仅为您的 Genstore 商店开发，未在 Genstore 应用商店中列出，适用于满足特定业务需求的个性化功能。
- **外部开发支持**：外部应用开发者可以为您开发自定义应用，且无需访问您的 Genstore 后台或商店的 API 权限。
- **安装方式**：您可以通过应用开发者提供的专属安装链接，轻松将自定义应用添加至您的 Genstore 商店。
