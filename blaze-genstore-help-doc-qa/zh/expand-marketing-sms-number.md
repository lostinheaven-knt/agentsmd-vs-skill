# 申请短信发送号码

为帮助您顺利发送短信，Genstore 提供以下可免费申请的发送号码类型，您可根据需要选择多个国家/地区：

- **美国和加拿大**：本地免费电话号码（TFN）

您只需完成注册流程，即可获得相应的发送号码。

## 开启短信功能

首次使用短信营销功能时，您可以在启动引导中完成注册，也可通过以下路径手动操作：

1. 登录 Genstore 商户后台，进入 **营销** -> **短信营销** 页面。
2. 点击 **免费注册 SMS 号码** 或在页面右上角依次进入 **设置** -> **免费注册 SMS 号码** 开始注册。
3. 选择您计划使用短信的国家/地区（可多选）。
4. 点击 **下一步**。
5. 填写您的品牌发件人 ID（限支持该类型的国家/地区）。
6. 填写公司信息（包括企业信息、联系人信息等）。
7. 点击 **完成** 提交注册。

## 验证与注册

在部分国家/地区，短信发送需经过运营商审批，审核内容包括您提交的企业信息与发件人 ID，以确保真实性和合规性。

- 审核通常需 **15–30 个工作日**
- 无需额外费用，审核通过后即可开始发送短信
