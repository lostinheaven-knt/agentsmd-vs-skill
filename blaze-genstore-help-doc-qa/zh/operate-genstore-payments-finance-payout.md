# 管理付款

付款页面提供了商家将 Genstore 账户中的资金转移到指定银行卡的功能。通过该页面，商家可以查看最新的付款状态、预计到账时间及收款银行账户信息。

本页面内容包括：

- 查看最近付款记录。
- 解读付款详情中的关键字段。
- 追踪预计到账时间及状态。

## 如何访问

登录 Genstore 商家后台，在左侧导航栏中，点击 **财务** -> **付款**。您将在付款页面查看所有与银行账户相关的付款记录及详情。

::: tip

您需要首先 **开通 Genstore Payments** 才能正常使用 **财务** 功能。

:::

## 付款概览

页面顶部展示账户的付款概览，帮助商家快速掌握付款状态和收款账户信息：

- **付款中**：当前正在处理的付款总额。
- **预估到账时间**：下一笔付款预计到达的时间。
- **付款配置（Payout Configuration）**：展示当前设置的收款银行账户信息，包括银行名称和账户信息。
  :::tip

  如果需要更改收款银行账户，请前往 **设置** -> **支付**。点击 [Genstore Payments](./operate-genstore-payments.md) 后的 **银行**，修改账户信息。

  :::

## 最近付款

页面下方展示最近发起的付款记录，包含以下字段：

| 字段名称   | 说明 |
| :---- | :-------------------- |
| 批次 |  付款订单的内部批次号  |
| 开始日期 |  发起付款到卡交易的日期  |
| 预估到账日期 |  预计收到资金的日期。实际的到账日期会根据收款账户的实际情况而有所不同。  |
| 状态 |  付款订单的状态。  |
| 支付参考号 |  发起付款交易的交易单号。  |
| 付款金额 |  预计收款人能够收到的金额。若收款银行有额外手续费，可能造成实际到账金额小于付款金额  |

## 查看付款详情

点击 **最近付款** 下的 **查看** 图标，可进入具体付款详情页面，查看每一笔付款的详细信息。付款详情字段说明如下：

| 字段名称   | 说明 |
| :---- | :-------------------- |
| 付款金额 |  预计收款人能够收到的金额。若收款银行有额外手续费，实际到账金额可能小于付款金额。  |
| 时间 |  发起付款交易的时间。  |
| 收款银行卡 |  与商户设置的收款银行卡信息保持一致。  |
| 状态 |  付款订单的状态。  |
| 付款金额 |  预计收款人能够收到的金额。若收款银行有额外手续费，可能造成实际到账金额小于付款金额。  |
| 转账类型 |  转账到银行卡的交易，默认类型为 “bank”。  |
| 优先级 |  付款处理的优先级程度，默认为 “regular”。  |
| 对账单描述符 |  您在收款银行交易中看到的交易描述。  |
| 预计到账日期 |  预计收到资金的日期。实际的到账日期会根据收款账户的实际情况而有所不同  |
