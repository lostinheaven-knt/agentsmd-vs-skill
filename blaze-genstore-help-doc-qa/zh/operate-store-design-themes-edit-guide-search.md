# 搜索页

搜索结果页用于展示用户输入关键词匹配到的产品，并支持多条件筛选与结果排序，帮助用户快速找到目标产品。通过灵活的布局和交互设置，可以显著提升搜索体验和转化效率。

## 如何访问

1. 登录 Genstore 商家后台。
2. 点击 **商店** -> **在线商店** -> **模版**。
3. 找到目标模版，点击右侧的 **设计** 按钮，进入模版编辑器。
4. 在顶部导航栏点击页面选择器，选择 **搜索**。

**提示**：切换至 **搜索页面模板** 后，可在左侧面板选择 **模板 → 搜索结果**，并在右侧面板调整搜索结果的展示样式、筛选和排序选项。
## 样式与布局配置

搜索结果页提供多项视觉与布局选项，用于优化产品展示效果：

- **展示样式**：可选择“产品小图”或“产品大图”，适应不同的视觉风格和信息密度。
- **产品数量**：调整每页展示产品数量，以及桌面端和移动端的每行列数
- **间距与颜色**：可调整卡片间距（小 / 中 / 大）并应用浅色或深色模式，保持品牌视觉一致性
- **产品卡**：
	- **图片比**：支持适应图、3:4、1:1，满足不同商品类别的展示需求
	- **产品信息**：可设置是否在卡片中展示厂商信息、价格，或启用悬停显示第二张图
	- **快捷操作**：可开启“快速加入购物车”，提升转化效率

## 筛选与排序配置

为帮助用户更高效找到商品，搜索结果页支持以下交互：

- **启用筛选**：开启后，用户可按价格、库存等条件筛选；商家可通过 [Search & Discovery 应用](./operate-store-design-search-discovery.md) 自定义筛选项
- **筛选布局**：桌面端可选择水平、垂直或抽屉式展示
- **启用排序**：允许用户按时间、价格或热度对结果排序
