# 简单设计器

简单设计器是 Genstore POD 的核心创作枢纽。它通过“实时渲染”技术，让商户能够在极短时间内完成跨品类的批量设计与预览。

## 灵活的设计入口

系统提供两种创作路径，满足不同的设计习惯：

* **路径 A：从“创意中心”开始 (推荐)**：在 Print On Demand 页面左侧的 **创意中心** 直接进行创作。当您上传图片或编辑文本时，右侧产品列表中的 **全部产品将同步实时渲染** 您的设计效果。
* **路径 B：从“特定产品”开始**：在右侧商品列表中选中具体产品，点击 **开始设计**。系统将默认打开简单设计器，并以该产品为画布进行创作。

## 核心创作与交互

简单设计器的操作会即时反馈到全线产品上：

| 功能模块 | 功能操作说明 |
| :--- | :--- |
| **上传图片** | 从素材库或本地选择图片。支持在画布中自由 **移动位置、拖动缩放及删除**。 |
| **编辑文本** | 输入自定义文字。支持选择 **字体、颜色、调整大小及旋转角度**。 |
| **模板** | **自定义模板**：将当前设计保存，便于后续同类产品复用；<br>**系统模板**：使用内置设计素材，支持二次调整大小与旋转。 |
| **画布设置** | **画布比例**：提供 3 种比例适配不同产品设计需求；<br>**画布填充**：支持黑色、白色、灰色及透明四种背景切换。 |
| **重置画布** | **一键清空**：仅重置当前编辑区域，不会影响已渲染到选品展示图上的图案。 |
| **预览** | 设计完成后点击 **预览**，可查看图案在选品商品 **默认打印位置** 的真实展示效果。 |

### 二次精修
在通过创意中心完成设计后，若对特定产品的渲染效果不满意，可直接 **点击该产品卡片上的图片或设计图标**，进入针对该单品的微调模式。

## 推送与管理

设计完成后，点击 **推送产品** 即可同步至店铺。

* **默认属性推送**：系统将自动推送该产品的所有颜色与尺寸选项。
* **打印工艺选择**：若产品支持多种工艺（如 DTG 或 Embroidery），系统将 **默认采用第一种打印方式**，并置于固定打印位置。

## 定价逻辑

产品的最终价格将基于您的 **[Dropshipping 定价设置](./operate-dropshipping-product-management.md)** 自动计算，包括：
* 您预设的固定加价金额或加价比例；
* 是否计入预估运费成本。
