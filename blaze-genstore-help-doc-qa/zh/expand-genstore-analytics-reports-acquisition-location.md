# 按地点查看访问量

**按地点查看访问量**报告为您提供基于用户地理位置的详细访问数据，涵盖不同城市、地区和国家/地区的访客数、访问次数和浏览量等关键指标。这份报告能够帮助您了解不同地区的用户活跃度和参与情况，识别出最具潜力和价值的地点。

## 如何使用

- **全球访问概览**：矩形图为您呈现全球各国的访问次数、访客数和浏览量分布情况，一目了然地了解不同国家的用户活跃度。
- **地域深入分析**：点击矩形图中的任一国家，即可**下钻查看该国家下的区域和城市数据**，深入挖掘不同地域层级的访问细节，发现地域差异和特点
- **地点表现洞察**：地点访问量明细列表为您提供每个城市、地区和国家/地区的详细访问数据，包括访客数、访问次数、浏览量等指标，帮助您评估各地的市场潜力和用户参与度。

## 常见术语

**此报告**中使用了一些常见术语，详细信息请参考 [客流量报告 - 常见术语](./expand-genstore-analytics-reports-acquisition.md#常见术语)。

本报告使用了以下附加术语：

| 术语 | 定义 |
| --- | --- |
| **城市（City）** | 指用户访问网站时所在的特定城市，如北京、纽约等。城市数据用于分析城市级别的用户访问情况，帮助商家了解不同城市的用户活跃度。 |
| **地区（Region）** | 指用户访问网站时所在的省/州等地区，如广东省、加利福尼亚州等。地区数据用于分析更广泛地域范围内的用户访问情况，帮助商家识别区域市场潜力。 |
| **国家/地区（Country/Region）** | 指用户访问网站时所在的国家或地区，如中国、美国等。国家/地区数据用于分析全球范围内的用户访问分布，帮助商家制定跨国市场策略。 |

请注意，地理位置数据可能受到用户隐私设置和 IP 地址解析准确性的影响，因此报告中的地理位置数据应作为参考而非绝对精确值。
