# 卸载应用

在 Genstore 中，您可以灵活管理已安装的应用，包括卸载不再需要的应用、重新安装已卸载的应用，以及查看应用的安装和卸载历史记录。

## 卸载应用

您可以通过 Genstore 后台的 **应用** 页面轻松卸载商店中的应用。不过请注意，某些应用可能会对您的商店模板进行修改。在卸载这些应用前，请确认是否需要执行额外的卸载步骤。如有疑问，可通过 **应用** 页面联系应用开发者以获取支持。

### 卸载步骤：

1. 登录 Genstore 商家后台中，转到 **应用** 页面。
2. 找到要卸载的应用，点击应用旁的 **...**（更多选项），然后选择 **卸载**。
3. （可选）在卸载原因下拉菜单中选择卸载原因。
4. 确认无误后，点击 **卸载**，应用将从您的商店中移除。

## 查看已卸载的应用

您可以在 Genstore 中的 **应用** 页面查看商店中已安装和已卸载的应用。

### 步骤：

1. 在 Genstore 商家后台中，进入 **应用** 页面。
2. 点击 **已卸载** 标签以查看已卸载的应用列表。

## 重新安装已卸载的应用

您可以重新安装之前卸载的应用，并查看应用的详细信息，例如安装次数和卸载次数。

**重新安装步骤**

1. 在 Genstore 商家后台中，转到 **应用** 页面。
2. 点击 **已卸载** 标签，找到需要重新安装的应用。
3. 点击 **安装** 按钮以重新安装应用。

## 查看应用的安装或卸载历史记录

在 Genstore 商家后台，您可以查看应用的完整历史记录，包括安装和卸载事件。

**查看步骤**

1. 在 Genstore 后台中，进入 **应用** 页面。
2. 点击 **已安装** 或 **已卸载** 标签以查找应用。
3. 点击目标应用名称，进入应用详情页面。
4. 在应用详情页面的**应用历史记录**部分，查看安装和卸载事件。
   - 若历史记录包含超过 3 个事件，点击 **查看全部** 以访问完整时间线。
   - 点击时间线中的事件以查看更多详细信息，例如卸载原因、卸载描述及操作人员等。
