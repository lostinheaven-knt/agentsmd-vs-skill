# 菜单

导航菜单是商店用户体验的核心，可以链接到特定的页面、产品或产品系列。它帮助客户轻松浏览、找到所需信息、获取相关内容并完成购买。一个结构清晰、高效的导航系统能提升用户满意度、降低跳出率并提高转化率。

## 默认菜单导航

每个在线商店通常设有三个默认菜单：

- **主菜单**：作为商店的核心导航，通常位于网站顶部，帮助用户快速访问产品、分类、品牌信息等关键页面。
- **页脚菜单**：位于网站底部，包含客户服务、商店政策、社交媒体链接等信息，虽不如主菜单显眼，但在增强用户信任、提升购物体验和优化 SEO 方面非常重要。
- **客户账户主菜单**：客户账户主菜单显示在客户登录后的账户页面顶部，方便客户访问其账户相关信息。

| 菜单类型        | 菜单项                 |
| ----------- | ------------------- |
| **主菜单**     | “主页”，“产品目录”，“联系方式”  |
| **页脚菜单**    | “搜索”，“不出售或共享我的个人信息” |
| **客户账户主菜单** | “商店”，“订单“           |

## 自定义菜单导航

除默认菜单导航外，您还可添加自定义菜单导航，链接到搜索页面、主页、产品系列、产品、页面、博客、博客文章、政策等。

**操作步骤**

1. 登录 Genstore 商家后台。
2. 点击 **商店** -> **在线商店** -> **菜单**。
3. 点击 **创建导航菜单**，在新打开的窗口中，设置菜单标题、URL Handle 以及菜单项。注意，**菜单最多支持 3 级**，您可以通过拖动调整菜单项的顺序。
4. 完成设置后，点击 **保存**。

## 管理菜单项

在菜单导航列表中，您可以进行以下操作以管理导航内容：

- **编辑**：点击导航后的 **编辑** 按钮，对导航内容进行修改。
- **删除**：若需要移除某一导航，点击 **删除**。注意：默认导航（主菜单及页脚菜单）不支持删除。
