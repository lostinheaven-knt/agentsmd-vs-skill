# 集成三方 EDM

Genstore 支持与多个主流邮件营销平台的集成，包括 **Klaviyo**、**Mailchimp** 和 **Omnisend**。通过数据同步与自动化流程配置，商家可构建高效、个性化的营销体系，提升用户转化与复购率。

## 平台功能概览

|平台|产品数据同步|订单数据同步|客户数据同步|平台优势|
|---|---|---|---|---|
|**[Klaviyo](./expand-marketing-email-3rd-edm-klaviyo.md)**|✅|✅|✅|支持基于用户行为的精准分群，拥有个性化的邮件模版|
|**[Mailchimp](./expand-marketing-email-3rd-edm-mailchimp.md)**|✅|✅|✅|简洁易用的邮件编辑器，支持受众细分|
|**[Omnisend](./expand-marketing-email-3rd-edm-omnisend.md)**|✅|✅|✅|支持邮件和短信营销，具备行为触发能力|

## 使用流程

所有平台的使用步骤基本一致，包含以下几个阶段：

1. **前置准备**  
    拥有对应平台的账号，并准备好接入所需信息（如 List、Audience、站点绑定等）。
2. **安装应用**  
    在 Genstore 应用市场中搜索并安装所需平台的官方集成应用。
3. **账户授权**  
    登录第三方平台账号，确认数据同步权限，完成账户绑定。
4. **配置客户同步**  
    选择目标 List / Audience，用于后续邮件营销的目标群体管理。
5. **数据同步**  
    同步店铺近三个月的产品、客户和订单数据。后续新增数据将持续自动同步。
6. **配置邮件流程**  
    在 Genstore 后台访问平台的邮件营销模板，跳转至目标平台完成流程配置。

## 注意事项

- 数据同步基于授权后的 API 调用进行，请确保权限设置完整。
- 邮件自动化流程需使用已同步的事件作为触发条件，否则无法正常运行。
- 如看到平台关于“未审核应用”提示，Genstore 官方集成应用均为可信来源，可放心授权。
- 建议定期检查同步状态与流程效果，确保邮件触达的及时性与准确性。
