# 按时间段查看平均订单金额

**按时间段查看平均订单金额报告**展示了在不同时间段内平均订单金额（AOV）的变化情况。

:::tip

- 平均订单金额 (AOV) =（毛销售额 - 折扣）/订单数
- 毛销售额 = 销售产品件数 * 产品价格

\> 其中不考虑下单后的调整，例如编辑或换货。

:::

您可以通过选择分组依据来设置时间单位，包括：小时、天、周、月、季度、年、一天中的某一小时、一周中的某一天或一年中的某个月。时间单位将决定订单金额的分组方式。您还可以比较不同日期范围的数据，深入分析销售趋势。

## 如何使用

- **订单价值趋势**：按日、周、月、季度或年来观察 AOV 的变化，以识别季节性趋势或长期趋势。
- **多维数据透视**：从不同维度（如地区、客户群体、产品类别等）分析平均订单金额，揭示潜在的市场机会和风险。比如可以分析不同产品类别对 AOV 的贡献，了解哪些类别的销售额较高或增长迅速。
- **订单价值明细支持**：基于深入的趋势分析，为定价策略、促销活动和产品线调整提供数据支持，以优化销售绩效。分析不同销售渠道（如在线商店、移动应用、第三方平台）的 AOV 表现。

## 示例说明

假设您的在线商店在某一天内售出了以下产品：

- 3 件 T 恤，每件售价 20 美元
- 2 双运动鞋，每双售价 50 美元

同时，您提供了以下折扣：

- T 恤 10% 折扣
- 运动鞋 20% 折扣

### 计算步骤

1. 计算毛销售额：

	```plaintext
    T 恤的毛销售额 = 3 件 * 20 美元/件 = 60 美元。
    运动鞋的毛销售额 = 2 双 * 50 美元/双 = 100 美元。
    总毛销售额 = 60 美元 + 100 美元 = 160 美元。
    ```

2. 计算折扣：

	```plaintext
    T恤的折扣 = 10% * 60美元 = 6美元。
    运动鞋的折扣 = 20% * 100美元 = 20美元。
    总折扣 = 6美元 + 20美元 = 26美元。
    ```

3. 计算平均订单金额（AOV）：

	```plaintext
    AOV = (总毛销售额 - 总折扣) / 订单数
    AOV = (160美元 - 26美元) / 5 = 134美元 / 5 = 26.8美元。
    ```
