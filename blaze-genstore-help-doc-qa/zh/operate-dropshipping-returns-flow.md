# 售后与退款

当客户对 Dropshipping 产品发起退货申请后，商家审核同意后可接收退回的产品，并在确认收货后，将客户支付的金额退还给客户。

若该订单为 Genstore Dropshipping 产品，商家在完成与客户的退款操作后，还需在订单中针对该产品向 Genstore 提交退货退款申请，将相应货物退还至平台。
平台在确认收货无误后，会将对应的货款退还至商家账户。

:::tip
- 当商户为 Dropshipping 产品发起售后退款时，系统会自动校验商户资金账户余额是否足以支付本次退款金额。若资金账户余额不足，商户需先通过转账或充值补足相应金额，待资金充足后，方可将客户支付的金额退还给客户。
- 同一产品、同一数量仅可发起一次退货申请，无论是部分退款或全部退款，均视为一次有效退货。
- 关于具体的退货政策，请根据您的业务类型参考以下文档:
  - [Genstore Marketplace 退货政策](./operate-dropshipping-after-sales-policy.md)
  - [Genstore POD 退货政策](./operate-dropshipping-after-sales-policy-pod.md)
- 具体退货政策，可参考 [Genstore Dropshipping 退货政策]
:::

## 退货流程

在 Genstore 商家后台创建退货时，系统会根据您的退货规则自动计算退货相关费用。您还可以从退款中扣除特定的退货物流费用和退货手续费。具体操作如下：

1. 登录 Genstore 商家后台，点击 **商店** -> **订单**。
2. 点击目标订单号，进入 **订单详情** 页面。
3. 点击页面右上角的 **退货**。
4. 在新打开的 **退货** 窗口中，输入要退货的产品数量，并选择 **退货原因**。
5. 点击 **退回手续费**  栏的 **编辑** 图标，输入需要扣除的退货手续费，点击 **确定**。
6. 点击 **退回运费** 栏的 **编辑** 图标，输入需要扣除的退货运费，点击 **确定**。
7. 填写退货的物流配送信息和承运商，或选择 **无需物流**。
8. 点击 **创建退货**。
9. 在订单退货详情页中，点击 **退款**。
10. 若要退还运费，请在 **退款金额** 区域调整相应金额。
11. 点击 **退款** 完成退款操作。

## 退款流程

您可以为订单中的 Dropshipping 产品发进行退款。退款时，您可以手动编辑退款金额，包括退货手续费、退货运费以及其他费用（如运费、税费等）。具体步骤如下：

1. 登录 Genstore 商家后台，点击 **商店** -> **订单**。
2. 点击需要退款的订单，进入 **订单详情** 页面。
3. 点击页面右上角的 **退款** 按钮。
4. 输入退款的产品数量。
5. 若要退还运费，请在 **退还金额** 区域调整相应金额。
6. 在 **退款原因** 部分，填写退款原因，该原因将显示给客户。
7. 点击 **退款** 完成退款操作。

在商户完成与顾客之间的退款操作后，针对 Genstore Dropshipping 代发货产品，您还需要向 Genstore 平台提交退货退款申请。

## 向 Genstore 提交退货退款申请

对于已通过代发货供应商履约的订单，商家可向 Genstore 提交退货退款申请，将产品退还至平台。

1. 登录 Genstore 商家后台，点击 **商店** -> **订单**。
2. 点击相关订单，点击进入 **订单详情** 页面。
3. 在订单退货详情页中，点击 **申请 Genstore 退货**，确认需要退货的产品，并填写备注信息。
4. 提交后等待 Genstore 运营的审核。
5. 点击售后卡片上的 **查看审核结果** 来查看申请进度信息。

## 注意事项

- 若退货产品经 Genstore 审核后确认无误，平台将把相应款项退回商家账户。
- 若退款时余额不足，请先通过账户充值补足资金后再重新提交退款操作。
- 退货物流费用与供应商的最终发货成本可能存在差异，建议在产品定价中预留一定运费利润空间。

## 退款处理

当您为订单发起退款时，系统会从账户余额中立即扣除对应金额。请确保余额充足后再操作。

- 手续费不退还：除经 Genstore 审核调整的采购成本外，其余手续费均不退还。
- 余额不足时退款将失败：退款所需资金将从未结算金额与可用余额中同时扣除。若余额不足，请先充值补足。
- 指定订单号充值：您可在充值时填写订单号，以补足该订单退款所需金额。

## 拒付（Chargeback）处理

信用卡拒付指持卡人因交易争议向发卡行申请撤销支付的行为。系统会自动从商户账户余额中扣除拒付金额及相关费用。

- 拒付不退手续费：除采购成本外，手续费不退还，且每笔拒付会额外产生拒付处理费。
- 可能产生负数余额：若余额不足以覆盖扣款，系统会生成负数余额。后续结算资金将优先补足。
- 需及时充值：若出现负数余额，请尽快充值，确保后续订单结算正常进行。
