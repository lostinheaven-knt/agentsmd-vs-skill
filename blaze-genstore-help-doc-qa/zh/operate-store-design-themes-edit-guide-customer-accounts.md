# 客户账户

客户账户页面用于支持客户注册、登录、查看和更新个人信息，是商家构建客户体系并提供个性化服务的核心模块。Genstore 已内置标准客户功能页面，支持基础样式调整，适用于各类 B2C 商家。

## 页面类型

系统预置以下客户功能页面（支持独立模板配置）：

- **客户注册**：用户填写信息或通过社交账号创建账户
- **客户登录**：支持邮箱密码登录或第三方账号登录
- **客户重置密码**：用户通过邮箱验证后重设密码
- **客户地址管理**：查看、添加、编辑、删除收货地址
- **客户账户主页**：查看个人资料、历史订单、管理地址

## 如何访问

1. 登录 **Genstore 商家后台**。
2. 点击 **商店** -> **在线商店** -> **模版**。
3. 找到目标模版，点击右侧的 **设计** 按钮，进入模版编辑器。
4. 在顶部导航栏点击页面选择器，选择 **客户账户**。
5. 展开 **客户账户**，选择目标页面模板。
6. 点击模板后，可在左侧面板编辑分区和块，右侧面板调整样式。

## 样式与布局配置

部分客户功能页面支持基础样式调整，用于优化视觉呈现。常见配置包括：

- **标题样式**：设置标题字号
- **布局样式**：选择水平或垂直布局
- **卡片模式**：开启后增加边框和阴影效果
- **区块填充**：调整上下留白

**提示**：不同页面支持的选项可能有所差异，请在编辑器右侧查看当前页面的可用配置，并通过预览确认效果。
