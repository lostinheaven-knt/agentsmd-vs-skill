# 删除客户

在管理客户资料时，有时候您可能需要删除不再活跃或不必要保留的客户信息。然而，鉴于数据保护法规和历史交易完整性的要求，删除客户资料前应考虑多个因素。

:::warning

客户删除操作是不可逆的，一旦执行，将无法恢复。因此，请在执行前仔细确认。

:::

## 客户删除限制

如果满足以下任一条件，将无法删除相应的客户资料：

- **存在订单历史数据**：如果客户账户下有历史订单数据（包括正式订单和草稿订单），为了保留交易记录的完整性，该客户资料将不能被删除。
- **正在进行合并任务的客户**：正在合并流程中的客户资料无法删除，以免影响合并处理的准确性。
- **存在有效的礼品卡**：如果客户账户下有**已启用且未过期**的礼品卡，为了保护消费者权益，这类客户资料也不能进行删除。

## 删除客户

**删除单个客户**

1. 登录 **Genstore 商户后台**，导航至 **商店** -> **客户** 选项。
2. 点击目标客户进入客户详情页。
3. 选择 **更多操作** -> **删除客户**。
4. 在删除确认弹窗中，点击 **删除** 按钮以完成操作。

**批量删除客户**

1. 登录 **Genstore 商户后台**，导航至 **商店** -> **客户** 页面。
2. 勾选多个需要删除的客户。
3. 点击 **删除客户**。
4. 在删除确认弹窗中，点击 **确定** 按钮。
5. 如果某些客户不符合删除条件，系统将自动跳过这些客户，只删除符合条件的客户资料。
