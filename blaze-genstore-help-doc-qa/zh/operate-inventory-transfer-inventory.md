# 库存转移

库存转移用于管理商店不同地点之间的库存流转，确保库存水平准确，并优化供应链效率。Genstore 提供 **创建库存转移**、**接收库存转移** 和 **撤销库存转移** 三大核心功能，以满足不同的库存管理需求。

| 操作             | 说明                                                         |
| ---------------- | ------------------------------------------------------------ |
| **创建库存转移** | 在 Genstore 商家后台创建库存转移，选择发货地点、目的地点、添加产品和库存数量，并填写配送信息。 |
| **接收库存转移** | 当库存到达时，核实并更新库存数量，选择全部接收或部分接收，以保持库存数据的准确性。 |
| **撤销库存转移** | 发生错误记录、过期产品、损坏库存或多计库存时，可撤销未接收的库存转移，或调整已接收的库存数量。 |
