# 管理支付

支付页面为用户展示了通过 Genstore Payments 交易的全部订单。商家能够了解每一笔订单的交易状态，并对有需要的订单进行处理。本页面主要内容包括：

- 订单状态仪表栏关键指标和说明
- 交易详情页的表格说明

## 如何访问

登录 Genstore 商家后台，在左侧导航栏中，点击 **财务** -> **支付**。进入支付页面后，您可以查看账户中的交易明细和相关资金结算状态。

::: tip

您需要首先 **开通 Genstore Payments**才能正常使用 **财务** 功能。

:::

## 仪表栏

仪表栏部分展示了商家使用 Genstore Payments 进行的交易的概览信息，包括以下关键指标：

- **总收入金额**：展示了用户选定时间范围内的订单金额总量。
- **退款**：展示了用户选定时间范围内的退款订单数量，以及对应的金额。
- **拒付**：展示了用户选定时间范围内的拒付订单数量，以及对应的拒付比率。

:::tip

请注意退款和拒付交易的总量以及比率。过高的退款和拒付比率可能会影响买家的体验，进而引起额外的交易限制、保证金要求等不利影响。

:::

## 支付明细

支付明细部分展示选定时间范围内的交易订单详细信息。每个订单条目包含以下字段：

| 字段名称   | 说明 |
| :---- | :-------------------- |
| 创建时间 |  付款订单下单成功的时间  |
| 支付参考号 |  发起付款交易的支付单号  |
| 状态 |  付款订单的状态  |
| 订单 |  与付款对应的交易订单号  |
| 付款人 |  支付订单的付款人信息  |
| 支付方式 |  付款交易使用的支付工具  |
| 总额 |  产品订单金额经过费项调整后，于付款处理后的总金额  |
| 净额 |  经过支付金额调整后，预计商户能够发起提现的金额  |
| 风险评分 |  Genstore Payments 为每笔支付交易分配的风险评分，分数越高表明潜在风险越高（评分范围：0-100）  |

## 筛选功能

支付明细页面提供了强大的筛选功能，支持按日期和状态对结算记录进行快速过滤，帮助商家高效查找特定记录。点击右上角的筛选按钮，您可以：

1. 按日期范围筛选。
2. 按状态筛选：根据订单状态（如已授权、已捕获）筛选记录。

您也可点击 **清除所有** 清除所有条件并显示完整记录。
