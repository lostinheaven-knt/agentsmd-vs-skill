# Genstore 物流服务

Genstore 提供了的一站式物流履约解决方案，帮助商户在 Genstore 管理后台闭环完成整个发货流程。

## 核心价值

* **一站式履约**：无需跳转第三方平台，直接在 Genstore 后台完成购买面单、打印标签和标记发货的全过程。
* **智能比价**：支持根据发货地和收货地，自动推荐多家承运商（如 UPS, USPS 等），方便商户对比价格与时效。
* **自动化计费**：支持“承运商计算运费”，在客户结账时自动展示最便宜的运输选项。
* **备用费率保障**：内置备用费率（Fallback Rates）机制，确保在物流接口异常时客户依然能顺利下单。

## 物流方案对比

Genstore 支持两种物流服务模式。您可以根据业务需求在 **设置 -> 发货配送 -> 物流商管理** 中进行切换。

| 功能维度 | Genstore Shipping (官方) | Shippo (第三方授权) |
| :--- | :--- | :--- |
| **准入门槛** | 无需额外注册，直接使用 | 需持有 Shippo 账号并完成授权 |
| **账号费用** | 仅限已订阅且非欠费店铺使用 | 需在 Shippo 侧绑定支付信息 |
| **实时报价** | 支持向客户展示承运商实时计费 | 需手动获取费率 |
| **切换逻辑** | 切换后默认激活，且不可关闭 | 切换后运费计算将降级为手动备用费率 |

## 核心操作流程

1. **选择服务商**：在 **物流商管理** 处选择 Genstore Shipping 或授权 Shippo 账号。
2. **配置运费规则**：在[物流模版](./operate-fulfillment-shipping-profile-configuration.md)中设置 **手动费率** 或 **承运商计算费率**，并务必配置**备用费率**。
   - 当您选择承运商计算运费服务时，系统将会识别您的配送区域是否存在可用的物流承运商，若不存在，请注意系统的提示，建议您切换至手动费率，确保您的订单履约正常。
3. **购买面单**：在订单支付后，通过 **商店 -> 订单 -> 发货** 进入待发货列表点击 **购买面单**。
4. **打印与交付**：打印电子面单贴在包裹上，并将包裹交给选定的承运商。
5. **标记发货**：手动标记订单为 **已发货**，系统将自动同步追踪号并通知客户。

## 注意事项
* **必填信息**：为确保获取报价，请务必准确填写发货地电话，并建议将[结账页](./settings-checkout.md)的客户电话设为“必填”。
* **使用限制**：开发者店铺暂不支持购买 Genstore Shipping 的面单。
* **状态修改**：一旦订单标记为“已发货”，后台将无法直接更改发货状态。
