# 产品和定价

当您向多个市场销售产品时，您的客户可以在您的商店中查看价格、在结账时支付订单以及以当地货币接收退款。

## 关键术语

| 概念   | 说明                                                                                              |
| ---- | ----------------------------------------------------------------------------------------------- |
| 店铺货币 | 独立于市场，店铺后台所有价格相关的内容均已店铺货币展示。<br />可在 **设置 -> 通用 -> 展示货币** 进行设置或更改。                              |
| 基础货币 | 指每个市场用于定价和计算的主要货币。所有价格调整、汇率换算均基于基础货币。<br />启用 **以当地货币显示价格** 时，系统会使用动态汇率将基础货币转换为客户所在国家的本地币种进行展示。 |
### 市场价格货币默认规则

- **主市场**：默认使用您在 **设置 -> 通用 -> [展示货币](./start-genstore-account-create-manage#选择店铺货币)** 中设置的币种，作为基础货币。
- **单一国家/地区市场**：默认使用该国家/地区的官方货币作为基础货币。
- **多个国家/地区市场**：默认使用店铺货币作为基础货币，同时启用以当地货币展示价格。例如，若您的市场覆盖美洲和亚洲多个国家，店铺基础货币为美元，并启用了当地货币展示功能，当日本消费者访问店铺时，系统将自动以日元显示产品价格。**注意**：您也可取消使用当地货币展示，统一用 [市场基础货币](#设置市场基础货币) 展示价格。
- **汇率转换**：默认情况下，当市场基础货币和店铺货币不一致时，无论是单一国家/地区市场还是多个国家/地区市场，均使用动态汇率转换规则。您也可以通过 **管理汇率** 手动设置汇率。

## 设置市场基础货币

#### 步骤

1. 在 Genstore 后台中，转至 **设置** -> **市场**。
2. 在 **概览** 部分，点击要设置的市场。
3. 点击  **产品及定价**。
4. 在 **定价** -> **基础货币** 部分，从下拉列表中选择所需货币。
5. 点击 **保存**。

## 设置当地货币

默认情况下，多个国家/地区市场会启用当地货币设置，以当地货币向客户展示价格。必要时，您可以停用当地货币，使市场下的国家/地区以同一种货币（市场基础货币）向客户展示价格。

#### 步骤

1. 在 Genstore 后台中，转至 **设置** -> **市场**。
2. 在 **概览** 部分，点击要调整当地货币设置的市场。
3. 点击 **产品和定价**。
4. 在 **定价** 部分中，完成以下任意一步：
   - 若要启用当地货币，请选中 **以当地货币向客户显示价格** 复选框。
   - 若要停用当地货币，请取消选中 **以当地货币向客户显示价格** 复选框，关闭当地货币展示后，该市场中的所有客户都会看到以该市场的基本货币显示的价格。
5. 点击**保存**。

:::tip

当地货币设置启用时，货币转化将使用动态汇率转换，不可设置手动汇率。

:::

## 设置价格调整

价格调整允许您按百分比增加或减少市场中所有产品的价格，用于应对成本、物流或市场策略差异，价格调整以百分比的形式显示，例如：

- 设置 +100% 将使产品价格翻倍；
- 设置 -50% 将使产品价格减半。

#### 步骤

1. 在 Genstore 后台中，转至 **设置** -> **市场**。
2. 点击要调整价格的市场。
3. 点击 **产品和定价**。
4. 在 **价格调整** 部分，从下拉菜单中选择 **价格上涨** 以设置价格调整增幅，或选择 **价格降低** 以设置价格调整降幅。
5. 输入调整的百分比。
6. 点击 **保存**。

## 设置固定价格（可选）

在某些场景下，您可能希望在特定国家/地区设置固定售价，避免汇率波动或特殊运营策略干扰市场，例如：

- 您计划在法国市场进行长期促销
- 或希望保证高利润产品在特定市场的定价稳定。

如需为特定国家/地区设置某些产品的固定价格，而不使用系统的动态转换与调整逻辑，您可以进入 [按国家/地区设置产品价格](./global-set-product-prices-by-country.md)。
