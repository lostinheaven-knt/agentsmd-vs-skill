# 产品评论

**产品评论**是 Genstore 内置的客户反馈管理工具，**无需安装**，您可直接通过菜单访问并配置评论功能。该功能帮助您收集、展示并管理客户对产品的真实反馈，增强店铺可信度，提升转化率与复购率。

通过 Genstore 评论功能，您可以收集客户反馈，并通过以下三种方式展示产品评论：
- 评论合集：将所有评论展示在首页，增强用户信任感，促进购买决策。
- 产品页评论区：在产品详情页添加评论区，显示星级评分和用户反馈，提升可见度和转化率。
- 星级部件：通过星级评分展示产品总体评价，帮助用户识别优质商品。

此外，除了客户在店铺前台对产品的评论外，您还可以在商户后台针对商品手动添加评论，或导入来自其他平台的历史评论数据，以丰富评论内容，增强产品展示效果。

## 通用设置

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **评论**。
2. 点击 **通用设置**，可配置以下内容：

### 自动发布星级评论
设置是否自动公开评论：
  - 关闭（默认）：评论需商家审核后手动发布。
  - 开启：系统自动发布评论，仅对开启后新评论生效。此外，您可设置星级，例如，自动发布 3 星及以上的评论。
  - 是否展示客户全名。

### 评论及回复权限
  - 无限制：所有访问者均可评论或回复评论
  - 登录后可评论或回复评论
  - 登录并购买过该产品后才能评论或回复评论

### 评论展示设置

针对暂时没有评论的产品，您可以选择以下展示方式：
- 显示：即使没有评论，仍展示评价星级。
- 隐藏：没有评论的产品隐藏评论星级。

## 导入评论

对于其他平台或应用（如来自 Shopify、Loox、Judge.me 等平台的评论），您也可以通过 CSV 文件导入。按照填写指定格式的 CSV 文件填写并上传后，便可轻松将其迁移到 Genstore 中。

您可以通过三种方式导入评论：

1. 登录 Genstore 管理后台，导航至 **商店** -> **产品** -> **评论**。
2. 点击 **导入**，选择任一方式：
	- **从支持的应用导入**（如 Shopify、Loox、Judge.me）
		  - 选择平台后，下载系统提供的 CSV 模板，填写后上传。
	- **从支持的平台导入**（当前支持 Amazon）
		  - 通过插件集成，导入 Amazon 评论内容。
	- **上传本地评论文件（CSV 格式）**
		  - 按照模板准备文件并上传。

上传后进入个人账号，点击左下角 **... -> 任务中心**，可查看导入记录：
   - 成功：评论自动导入至系统。
   - 失败：可下载失败记录并重新上传。

### 导入字段说明

上传本地评论文件时，请先按照以下模板准备文件：
| 字段             | 字段说明                | 是否必填 |
| -------------- | ------------------- | ---- |
| product_handle     | 在产品的搜索引擎下查看 handle 字段 | 必填   |
| rating         | 1-5 星，不支持半星    | 必填   |
| title          | 评论标题，限制 50 个字符         | 必填   |
| body           | 评论内容，限制 1000 个字符       | 必填   |
| reviewer_name  | 限制 100 个字符            | 必填   |  
| reviewer_email | 邮件格式校验              | 必填   |
| review_date    | 格式：YYYY/MM/DD    | 选填   |
| reply          | 最多 1000 个字符           | 选填   |
| reply_date     | 格式：YYYY/MM/DD    | 选填   |
| Media          | 多个图片用 `;` 进行分隔          | 选填   |
| market         | 选择用于展示产品评价的市场，若留空，则默认主市场       | 选填   |
| Status         | Publish （公开）、Hidden （隐藏）    | 必填   |
| like           | 填写整数，最大可填 999999999   | 必填   |

## 商家发布评论

您也可在后台手动为产品发布评论，用于展示用户口碑、丰富产品内容评论展示样式。

操作入口：**商店** -> **产品** -> **评论** -> **发布评论**

| 字段    | 字段说明             | 是否必填 |
| ----- | ---------------- | ---- |
| 选择产品  | 选择发布评论的产品        | 必填   |
| 评论者姓名 | 输入评论者的姓名，限制50个字符          | 必填   |
| 评论时间 | 评论发表时间          | 必填   |
| 评论标题  | 限制 100 个字符         | 必填   |
| 评论内容  | 限制 1000 个字符        | 必填   |
| 评分  | 输入评论者对产品的评分，1-5 星级            | 必填   |
| 媒体  | 限制 8 个图片 1 个视频，可用于展示产品的使用效果或使用场景     | 选填  |
| 点赞数   | 评论的点赞数量，限制 999999999，整数   | 选填  |
| 选择市场  | 选择评论要发布的市场，例如：美国      | 必填   |
| 认证评论  | 选择是否将评论标记为“认证”评论，若未填写，系统将默认处理为‘未认证。’ | 必填   |


## 开启评论展示（装修设置）

在商户后台开启评论功能后，您还需要在装修编辑器中启用评论模块。这样用户才能在前台对产品进行评论、查看其他用户的反馈，并在产品详情页看到评论星级展示。

1. 进入 **商店** -> **在线商店** -> **模版**，点击当前主题后的 **设计** 按钮。
2. 在产品页模板下，点击 **添加分区**，在弹窗中切换到 **应用** 分区，选择 **产品评论**。
3. 配置展示样式：包括评论总数、星级评分、概览样式等。
4. 保存后即可在产品详情页看到评论模块。

## 客户提交评论
开启评论展示后，客户可以直接在前台提交评价，支持以下方式：
1. **星级评分** —— 按 1–5 星进行打分；
2. **文字评论** —— 填写购买体验或使用感受；
3. **多媒体上传** —— 支持上传图片和视频，展示真实使用效果。

## 评论认证
可以将一些来自真实买家，或质量比较高的评论设置为认证评论，您可通过以下步骤进行认证：
1. 评论列表中点击 **编辑** 图标，勾选 **认证评论**。
2. 您也可在评论列表中批量勾选希望认证的评论，点击 **认证** 按钮，列表中会展示已认证标识。
3. 认证完成后，商品详情页也将显示经认证图标，登录客户可见。

## 展示产品评价星级（装修设置）

**星级部件** 用于展示用户对产品总体评分的可视化组件。通常以 1–5 颗星形式呈现，直观体现产品口碑和质量评价。

1. 进入 **商店 -> 在线商店 -> 模版**，点击当前主题后的 **设计** 按钮。
2. 点击左侧功能导航栏中的样式设计图标（竖排第三个），找到 **星级部件**。
3. 配置展示样式：包括图标样式、颜色，以及对没有评论的产品是否隐藏星级部件。
4. 稍后您可在特色产品系列、特色产品以及搜索页控制产品卡是否展示星级评价，显示规则将统一读取全局产品评价设置。

## 展示评论集合（装修设置）
您可在首页添加 **评论集合** 应用分区，用于集中展示全店或指定产品的评论内容。客户可点击具体评论查看详情。登录后，还可对评论进行点赞、回复等交互操作。

操作步骤：

1. 进入 **商店** -> **在线商店** -> **模版**，点击当前主题后的 **设计** 按钮。
2. 在希望展示评论集合的页面，如主页，点击 **添加分区**，在弹窗中切换到 **应用** 分区，选择 **评论集合**。
3. 对该分区应用进行样式设置。
4. 保存后，即可在首页展示评论集合。

## 常见问题

### Q1. 评论功能是否需要安装？
> A：**无需安装**。产品评论是内置功能，默认在后台菜单中显示。

### Q2. 装修中是否需要手动添加评论模块？
> A：是的。需在对应产品模板下添加 **产品评论** 分区，评论模块才会在店铺前台展示。

### Q3. 每个产品都要设置评论模块吗？
> A：不需要。如果所有产品共用同一个模板，仅需设置一次。若需为不同产品模板配置不同展示样式，则需分别设置。

### Q4. 在导入模板时遇到乱码问题，如何解决？
> A：在 Windows 系统下通过模板导入评论时，可能会出现乱码问题。如果历史数据中存在异常评论，您需要删除这些数据，并按照我们最新下载的模板重新导入。请确保在重新导入之前，清空相关历史数据并使用最新版的 CSV 模板文件。
