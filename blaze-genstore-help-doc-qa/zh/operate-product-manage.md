# 管理产品

在 Genstore 平台上，您可以销售实物产品、数字产品和礼品卡。为了更好地展示和销售您的产品，您可以添加详细的产品信息并上传高质量的产品图片。

- **实物产品**：具有物理形态、可触摸、可见并可以通过物流运输的产品（如衣物、电子产品等）。
- **虚拟产品**：无需物流配送的产品，通常是基于软件或在线服务（如电子邮件服务、在线订阅等）。
- **数字产品**：顾客在线购买音乐、电影或其他娱乐内容的数字版本，顾客可以即时下载和使用（如电子书、数字音乐等）。

## 添加产品

### 操作步骤

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 点击页面右上角的 **添加产品**，根据您待发布的产品类型，选择 **实物产品** 或 **数字产品**（含虚拟产品）。
3. 填写产品详细信息。关于各配置内容的详细说明，见 [添加产品](./operate-product-create.md)。
4. 完成产品详情的配置后，您可以设置产品的状态（草稿、上架或未上架）、销售渠道、市场、厂商、标签，以及系列信息。
4. 完成以上设置后，点击 **保存**，完成产品的创建。

创建完成后，产品将自动出现在 **产品** 页的产品列表中，方便后续的管理工作，如编辑、预览、复制以及删除等操作。

## 复制产品

您可以快速创建与现有产品类似的新产品，通过复制现有产品的详细信息并进行修改，节省时间和人力成本。

### 操作步骤：

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 在产品列表中，点击操作列中的 **复制**。
3. 系统将打开一个新的创建产品页，原有标题前会自动添加【复制】。
4. 产品状态默认为 **草稿** 状态。
5. 系统将自动填充其他产品信息。您可根据需要进行调整。
6. 点击 **保存**，完成新产品的创建。

:::tip

**备注**：复制的产品会自动关联市场和相同的销售渠道。如希望调整销售渠道，请在复制后修改并将产品状态设置为已上架。

:::

## 编辑产品

您可以随时修改产品的价格、库存、描述等信息，确保信息准确，修改后会立即生效。

### 操作步骤：

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 在产品列表中，点击操作列中的 **编辑**。
3. 修改需要更新的产品信息，关于各配置内容的详细说明，见 [添加产品](./operate-product-create.md)。
4. 点击 **保存**，完成产品编辑。

Genstore 商家后台同时提供了批量编辑功能，勾选产品前的复选框，可批量编辑产品名称、特色、描述、类别、图片、标签、系列、价格以及原价等信息。

## 预览产品

产品保存后，您可以实时查看其在在线商店中的展示效果。

### 操作步骤：

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 在产品列表中，点击操作列中的 **预览** 图标，即可打开新窗口，查看该产品在 **在线商店** 中的展示效果。
3. 如果该产品分配至多个市场，您可以在预览中切换查看：
    - **已激活市场**：查看真实的前台展示效果。
    - **未激活市场**：提前查看该产品在对应国家/地区的展示样式，便于确认页面表现。
4. 在页面底部提示条，您可以查看产品的状态（如草稿或已发布）、当前主题名称，并选择 **退出预览** 或点击 **设计主题** 进入装修编辑器开始页面定制。
#### 使用限制
- 在预览模式下，您可以查看产品展示效果，但 **不可进行真实下单操作**。
- 若需测试完整购买流程，请退出预览模式，进入店铺前台进行 **模拟下单**。

## 分享产品
Genstore 提供产品分享功能，支持商家将“产品页”一键分享至主流社交媒体渠道，提升产品曝光和转化。

### 操作步骤
1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 在产品列表中，点击进入目标产品的编辑界面。
3. 点击页面顶部的 **分享** 按钮。
4. 选择您要分享的社交媒体平台（目前支持 Facebook、Twitter、Reddit、LinkedIn、Pinterest、WhatsApp）。
5. 您也可以点击 **复制链接**，手动将产品页面链接粘贴到任何社交媒体平台。

### 注意事项
* **移除密码保护**：在分享产品之前，请确保已[移除商店的密码保护](./operate-store-design-preference#店铺密码保护)，否则外部访客将无法查看产品详情。
* **账户登录**：如果您尚未登录对应的社交媒体账户，系统会弹出提示要求您先进行登录。

## 批量上架

对于处理草稿状态的产品，Genstore 商家后台提供了批量上架的功能。

### 操作步骤：

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 勾选产品前的复选框，点击 **批量上架** 产品，完成产品的批量上架。

## 批量编辑产品价格

对于多规格产品，Genstore 支持批量编辑价格。

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 打开目标产品，在规格明细表中勾选需要修改的 SKU，点击 **编辑价格**。
3. 系统提供两种方式：
    - **调整价格**：为所选规格直接设定新价格。
    - **基于原价调整**：在现有价格基础上加减固定金额或百分比。例如：
        - `-5` → 每件减少 $5
        - `+15%` → 每件增加 15%
## 删除产品

您可以删除单个产品或批量删除多个产品。请注意，删除操作不可恢复，请谨慎操作。

### 操作步骤：

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 在产品列表中，点击操作列中的 **删除**。
3. 您可也勾选产品前的复选框，**批量删除** 产品，最多勾选 100 个产品。
