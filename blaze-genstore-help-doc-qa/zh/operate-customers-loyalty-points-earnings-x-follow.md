# 关注 X 积分

启用该积分计划后，当消费者关注商铺的 X 账号时，可获得积分奖励，这有助于提升品牌在社交媒体上的曝光度和互动性。

本文档将说明该功能的配置方法、使用场景及注意事项。

## 配置积分规则

**导航：**

登录 Genstore 商家后台，点击 **商店** -> **客户** -> **忠诚度**。  
在 **积分** 标签下，找到 **获取积分** 部分。

### 设置步骤

点击**新增** -> **关注 X** -> **添加**，进入配置页面：

- **规则名称**：输入积分奖励规则的名称，方便管理。
- **获取积分**：设置客户关注 X 账号时可获得的积分数
- **您的 X 用户名**：输入有效的 X 账户地址，不得为空。

## 预览及发布

在页面的右侧**概要**区域，系统会自动生成并展示当前设置的规则概述，方便实时预览。确认无误后，点击**保存**，发布规则。

规则发布后，您可在积分获取方式列表中查看当前规则、编辑、启用或停用该规则。

:::tip

您也可通过状态开关控制当前规则的启用与关闭。

:::

## 常见问题

### Q1. 如果客户取消关注，是否会扣除积分？
> A：由于技术限制，系统暂时无法跟踪客户是否真实关注或取消关注。因此，关注后获得的积分不会受到取消关注的影响。

### Q2. 我的 X 用户名无法通过验证，该怎么办？
> A：请确保输入了有效的 X 用户名，且格式正确。系统会根据预设的正则表达式校验 URL。

### Q3. 是否可以修改客户获得关注积分的频次？
> A：目前关注积分的频次限制为每位客户仅能获得一次积分，无法更改。
