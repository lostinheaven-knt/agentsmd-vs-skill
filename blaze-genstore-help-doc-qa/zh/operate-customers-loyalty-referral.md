# 好友推荐

在 **Genstore 客户忠诚度计划**中，推荐页面允许商家设置并管理推荐奖励计划。商家可以通过该页面激励现有客户推荐新客户，从而扩大品牌影响力和客户基础。

## 启用推荐计划

1. 进入路径：**商店** -> **客户** -> **忠诚度** -> **推荐**
2. 在**推荐**页签，您可通过顶部状态栏的状态切换按钮，打开或者关闭推荐计划。

在推荐页面，您可设置和查看推荐人和被推荐人的奖励情况，设置推荐链接，具体设定，可查看以下页面。

启用推荐计划后，您可以进一步定制推荐人和被推荐人的奖励，以及社交媒体分享链接。本页面将介绍如何进行相关设定。

## 推荐人与被推荐人奖励

推荐人奖励指的是推荐人在被推荐好友完成首单后可获得的奖励。被推荐人奖励则是指新客户通过推荐链接注册并在首次下单时可使用的奖励。

目前支持的奖励选项包括：

- 折扣金额：设定具体的折扣金额以及使用限制。具体操作请参考 [折扣金额](./operate-customers-loyalty-points-redemption-discount-amount.md)
- 折扣比例：设定折扣金额以及使用限制。具体操作请参考 [折扣比例](./operate-customers-loyalty-points-redemption-discount-percentage.md)
- 免邮：设定可抵扣的最高运费金额以及使用限制。具体操作请参考 [免邮](./operate-customers-loyalty-points-redemption-free-shipping.md)
- 免费礼品：设定免费的礼品奖励。具体操作请参考 [免费礼品](./operate-customers-loyalty-points-redemption-free-gifts.md)
- 积分：设定积分奖励数额，注意：仅推荐人奖励支持配置奖励积分。

奖励设定完成后，会在奖励选项区域显示，您可以点击**编辑**按钮进行修改。

:::tip

- 您只能为推荐人和被推荐人各设置一项奖励。
- 更改推荐奖励将删除当前奖励并添加新的奖励，但不影响已发放的奖励。

:::

## 推荐链接分享

支持客户通过 Facebook 或 X 社交平台分享推荐链接。开启后客户可在会员面板 - 推荐模块对自己的邀请链接进行快速分享至 Facebook 或 X 社媒平台。

- **Facebook**: 因 Facebook 平台限制，用户需要手动添加消息内容。
- **X**：您可自定义分享内容，客户可将其与推荐链接一起发布。

::: faq

## 常见问题

### Q1. 如果我想更改已设置的奖励类型，如何操作？
> A：您可以通过点击奖励选项区域的编辑按钮进行修改。

### Q2. 我可以更换已经设置的奖励类型吗？
> A：可以，点击更换按钮进行操作。**请注意**，修改奖励将删除当前设置并添加新奖励，但不会影响之前已发放的奖励。

### Q3. 推荐奖励何时发放给推荐人？
> A：推荐奖励将在被推荐的好友完成首次下单后发放给推荐人，并通过邮箱进行通知。

### Q4. 如果被推荐人的订单被取消或部分退款，推荐奖励会怎样处理？

> A：针对当前奖励已经使用、未使用，以及被推荐人整单退单或部分退单的情况，可参照下方表格：

| 订单状态               | 推荐奖励处理                                 |
| ---------------------- | -------------------------------------------- |
| 完全退单               | 撤销奖励（如果未使用；如果已使用，则不追回） |
| 部分退款               | 奖励保留                                     |
| 多次部分退款后整单退款 | 撤销奖励（如果未使用；如果已使用，则不追回） |

:::
