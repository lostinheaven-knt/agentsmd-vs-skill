# 物流运费模版

物流运费模版是产品运费计算、物流发货、订单巡仓的基础与前提。了解物流的定义与设置，有助于您更好的为店铺设置符合您预期的发货配置。

## 物流运费模版组成

在物流运费模版中您需要关心以下几个要素：

- **发货点**
	- 发货点即您店铺售卖的产品，需要从哪些仓库或者门店发货。发货点的设置将影响产品的运费价格计算以及产品对应的库存。关于发货点的设置，您可以直接查看 [发货点配置](./operate-fulfillment-shipping-profile-configuration#配置发货点)。
- **适用产品**
	- 物流运费模版中的适用产品范围，将直接影响产品下单后使用哪个物流运费模板来计算运费以及产品巡仓。注意：产品匹配的物流运费模版有且只有一个。更多信息，可参考 [创建自定义运费方案](./operate-fulfillment-shipping-profile-configuration#创建自定义运费)。
- **物流区域**
	- 物流区域即您店铺售卖的产品，支持发货到哪些区域。**客户的收货地址不在物流区域范围内的，将可能导致无法进行正常下单。** 同时考虑物流运输成本，您应该合理规划您的产品可发货的区域范围。更多信息，可参考 [配置物流区域与运费](./operate-fulfillment-shipping-profile-configuration#配置配送区域与运费)。
- **物流运费**
	- 物流运费将直接影响产品配送到该区域范围的运费价格。更多信息，可参考 [配置物流区域与运费](./operate-fulfillment-shipping-profile-configuration#配置物流区域与运费)。

## 物流配送运费模版

在 Genstore 中提供两种类型的物流运费设置：**通用运费** 与 **自定义物运费**。两类运费设置均可为产品提供运费计算，发货，巡仓等依据，具体异同点如下：

| 比较项    | 通用物流运费                      | 自定义物流运费                                                                                                       |
| ------ | --------------------------- | ------------------------------------------------------------------------------------------------------------- |
| 适用场景   | 通用物流发货场景，适配绝大多数产品物流发货配置     | 无法使用通用物流运费设置，需要设置特殊的发货点/物流区域/物流运费的产品。<BR />例如：商店经营家居百货产品，对于玻璃制品或陶瓷制品需要使用特殊的运输发货，并设置区别于其他产品的运费标准，则建议使用自定义运费模版。 |
| 个数限制   | 有且只有一个                      | 可创建多个（目前暂无个数限制）                                                                                               |
| 创建方式   | **系统会自动生成初始化运费模板，需要商户手动修改** | 商户可按需创建                                                                                                       |
| 适用产品范围 | 默认适用店铺所有实物产品以及后续新增实物产品      | 用户自定义产品范围                                                                                                     |
| 模版操作限制 | 限制删除                        | 无限制                                                                                                           |
| 模版优先级  | 自定义运费设置优先级更高                | 自定义运费设置优先级高于通用运费设置。自定义运费因存在适用产品无法重复配置限制，所以在自定义模版之间无优先级问题。                                                           |
