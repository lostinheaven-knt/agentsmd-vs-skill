# 配置第三方支付

如果您选择不使用 [Genstore Payments](./operate-genstore-payments.md)，您仍然可以通过 Genstore 支持的第三方支付服务接收付款。您可以在 Genstore 的支付设置页面激活或更改您的付款提供商。激活过程中，您可能需要向支付提供商提交并验证您的银行账户信息，以确保交易的顺利完成。建议在提供银行信息前，先与支付提供商确认您的商店是否符合接收付款的条件。

## 激活支付服务提供商

:::tip

请注意，以下的支付方式不能被同时激活。
| 支付提供商名称   | 类型  | 支付方式 |
| :-------- | :-- | :--- |
| Genstore Payments | 直连  | 信用卡、Google Pay、Apple Pay    |
| Stripe    | 直连  | 信用卡    |
| Lianlian  | 直连  | 信用卡    |
| Asiabill  | 直连  | 信用卡    |
| Pingpong  | 直连  | 信用卡    |
| Braintree | 直连  | 信用卡    |
| Authorize.net | 直连  | 信用卡    |
| Oceanpayment | 直连  | 信用卡    |


:::

**操作步骤**：

1. 登录您的 Genstore 管理员账户。
2. 导航至 **设置** -> **支付**。在这里您可以直接激活 Genstore Payments、Stripe 以及 PayPal。
3. 如果您希望使用其他的支付方式，请在**更多支付方式** 部分，点击 **添加支付方式** 。在列表中选择您想使用的支付服务提供商
    * **第三方支付提供商**：支付方式将会嵌入到买家的结账页中，买家不需要跳转到外部的页面即可完成付款。
    * **其他支付方式**：支付方式需要买家跳转到外部页面完成付款。
    * **手动支付方式**：您可以添加手动支付或者货到付款的支付方式。
4. 选择支付提供商后，在弹出的窗口中输入账户凭据。
5. 单击**保存**即可。

## 获得支付配置变更通知

支付配置对于商店至关重要，因此我们会在您的支付配置发生变化时，通过邮件或系统通知告知您。如果支付配置的变化并非您所为，建议及时恢复配置并报告。
