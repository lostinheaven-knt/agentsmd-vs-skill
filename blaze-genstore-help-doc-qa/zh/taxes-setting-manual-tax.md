# 手动税务

手动税务一般适用于销售的市场国家税率较为简单且较为固定的场景。您需要为所需要收税的国家或地区配置符合当地税法规定的税率，并在税率有变更时及时手动更新配置的税率。

## 为目标区域配置手动税务

收税区域与您的物流配置深度关联。系统默认从您的 [发货与配送](./operate-fulfillment-shipping-profile-configuration.md) 模版中同步已配置的国家或地区。

1. 登录 **Genstore** 商户后台，进入 **设置** -> **税务**。
2. 找到需要配置税务的国家或地区，点击进入税率配置页面。
3. 点击当前计税方式后的 切换图标，切换至 **手动税务**。

## 设置国家与地区税率

在手动计税模式下，您可以根据目标市场的行政层级灵活配置税率。系统采用“优先级覆盖”原则，确保税费计算的精准性。

### 国家税率
您可以通过设置国家税率，为该国家或地区设定统一的基准产品税率。
- **应用逻辑**：设置的国家税率将自动应用在所有未单独设置税率的二级行政区域（如各省、州）。
- **适用场景**：适用于整个国家/地区实行统一产品税率的情况。

### 地区税率
若国家下属不同行政区（如美国的各州、加拿大的各省）的产品税率存在差异，您可以在对应地区设置特殊税率。
- **优先级原则**：系统在计算时会优先识别并使用 **地区税率**；若该地区未设置税率，则统一使用 **国家税率** 进行计算。

::: tip

- **设置 0 税率**：在已设置国家税率的情况下，若其中某个地区依法免税或不收税，您必须为该地区明确设置 0% 的税率，否则系统将默认应用国家税率。
- **搜索功能**：若二级行政区较多，您可以利用列表上方的搜索框，通过输入地区名称快速定位并编辑税率。 

:::

## 设置自定义税

若您需要为特定产品或运费设置不同于默认税率的税务规则，可使用 **自定义税** 功能进行配置。例如，酒精类产品可能需要单独税率，或运费在特定区域需要征税。

### **自定义产品税**

适用于特定产品需要配置独立税率的情况。

**操作步骤：**

1. 点击 **新增自定义税**。
2. 勾选 **产品**，点击 **选择产品**，在弹出的产品选择框中选择需配置特定税率的产品。
	- **注意：** 已添加过的产品无法重复选择。如需重新添加，请先从已有的自定义税中移除。
3. [可选] 若需要为所有区域配置税率，可勾选 **一键添加所有地区**。
4. 点击 **确定**。
5. 系统将返回当前国家的税务设置页面。对于未勾选 **一键添加所有地区** 的产品，您可继续在每个产品下方添加适用区域。
6. 再次点击 **保存** 完成设置。

### 自定义运费税

适用于希望为运费设置税率的场景。

**操作步骤：**

1. 点击 **新增自定义税**。
2. 勾选 **运费**。
3. [可选] 若希望为所有区域设置运费税率，可勾选 **一键添加所有地区**，系统将自动添加全部可配置区域。
4. 点击 **确定**。
5. 系统将返回当前国家的税务设置页面。如未勾选 **一键添加所有地区**，您可继续手动配置当前运费规则适用的区域。
6. 完成设置后，点击页面右上角的 **保存**。

## 更多设置

**税费展示名称设置**

您可以自定义税费名称，该名称会在用户结算页中显示。默认展示 **Tax**。您也可以输入 **税费说明**，辅助客户下单时对税费进行理解。
