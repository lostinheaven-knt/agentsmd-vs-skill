# 添加自定义域名

除了 `.genmystore.com` URL 外，您可以向您的 Genstore 商店添加最多 20 个域名或子域名。当您通过任何方式将域名添加到 Genstore 时，系统会自动免费创建新的 TLS 证书。

添加自定义域名不仅能帮助树立品牌形象，还能增强客户对您在线商店的信任感。您可以通过以下方式将域名添加到 Genstore：

- **将您的域名连接到 Genstore**：如果您已有第三方提供商提供的域名，您可以将该域名连接到您的在线商店。在这种情况下，您仍将使用第三方域名提供商来管理域名设置、支付域名费用以及续订域名。

  具体设置，可阅读 [将第三方域名连接到 Genstore](./global-domains-custom-domain-third-party.md)。
