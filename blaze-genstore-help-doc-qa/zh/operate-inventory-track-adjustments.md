# 跟踪库存

Genstore 会自动跟踪产品的库存变化，帮助您实时掌握库存状态，减少超卖或库存异常的风险。当产品发生销售、补货、库存转移或手动调整时，系统会同步更新库存数量，并记录对应的库存变动，便于后续核对和管理。

## 工作原理
在库存跟踪生效的情况下：
- 系统会根据订单和库存相关操作自动更新库存数量
- 每一次库存变动都会生成对应的库存记录
- 库存数量可按产品规格和发货地点分别管理

系统会记录每一次库存变动，包括销售、补货、库存转移等操作。具体说明，可参考[查看库存调整历史](./operate-inventory-view-history.md)。

## 设置库存数量

Genstore 会默认启用库存跟踪功能，您只需设置相应库存数量，即可开始跟踪库存变化。

### 为单规格产品设置库存数量
1. 登录到 **Genstore 后台**，点击 **商店** -> **产品**。
2. 在产品列表中，点击目标产品，或点击 **操作** 列的 **编辑** 图标。
3. 在产品详情页中，找到 **库存** 区域，在 **可售库存** 下输入库存数量。
4. 如在多个地点发货，可点击 **管理地点** 添加发货点，并为每个地点分别设置库存数量。
5. 点击 **保存** 保存设置。

### 为多规格产品设置库存数量
1. 登录到 **Genstore 后台**，点击 **商店** -> **产品**。
2. 在产品列表中，点击目标产品，或点击 **操作** 列的 **编辑** 图标。
3. 在产品详情页的 **规格明细** 区域：
    - 找到对应规格，在 **可售库存** 栏输入库存数量。
    - 如需批量操作，可勾选多个规格，点击批量编辑，在 **可售库存** 列填写库存数量。
4. 如在多个地点发货，可点击 **管理地点** 添加发货地点，并通过地点下拉框切换，分别为每个地点分别设置库存。
5. 点击 **保存** 保存设置。

如不需要对该产品进行库存管理，可在产品或规格的库存设置中取消勾选 **跟踪库存**。

## 库存为 0 时的销售设置

当库存数量为 0 时，您可以根据业务需要设置产品的销售行为：
- 允许库存为 0 时继续销售
- 库存为 0 时禁止继续销售

该设置可在产品详情页的中通过勾选 **缺货后继续购买** 进行调整。
对于多规格产品，可在规格明细页面，或批量勾选后点击 **批量编辑** 进行设置。
