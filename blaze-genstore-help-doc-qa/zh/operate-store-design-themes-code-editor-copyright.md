# Genstore 版权说明

在商店主题的页脚（通常位于左下角），系统会默认展示类似以下内容：

```
© 2025, {your-store-name} Made with Genstore AI
```

该版权说明用于标注商店的版权信息及平台技术支持来源。

## 功能价值

- **版权声明**：向顾客展示店铺版权归属。
- **平台署名**：显示由 Genstore 技术支持。
- **自定义空间**：通过编辑主题代码，您可以去除或修改默认文案，让页脚更符合您的品牌定位。

## 如何去除版权说明

::: warning

此操作涉及修改代码，请在操作前做好主题的备份。

:::

1. 登录 Genstore 商家后台，点击 **商店** -> **在线商店** -> **模板**。 
2. 点击目标主题后的 **...** 图标，在下拉列表中，选择 **编辑代码**。
3. 在左侧代码文件列表中，展开 **sections** 文件夹，找到 `footer.liquid` 文件。
4. 使用搜索功能，查找 `footer__copyright` 代码块。
5. 删除或注释该代码块。
    - 删除：直接移除该段代码。
    - 注释：在代码前后添加 `{% comment %}` 与 `{% endcomment %}` 包裹，即可隐藏。
6. 保存文件，刷新商店前台页面，确认版权说明已去除。

::: tip

- 去除版权说明不会影响商店的功能，仅影响前台展示。
- 如果需要替换为自定义文案，可以在删除默认说明后，插入您自己的版权文本。
- 修改代码存在风险，建议在测试主题中先进行操作，再应用到正式主题。
:::
