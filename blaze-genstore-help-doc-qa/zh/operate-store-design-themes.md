# 主题模版

模版是预设计的框架，用于定义在线商店的视觉布局、风格和用户交互。每个模版都设计有特定的布局和风格，旨在满足不同行业和产品需求的展示。

模版通常包括以下几个部分：

- **[标头](./operate-store-design-themes-edit-guide-header.md)**：包含 Logo、导航菜单、搜索栏等。
- **[页脚](./operate-store-design-themes-edit-guide-footer.md)**：包含版权信息、社交媒体链接、辅助导航等。
- **[内容区域](operate-store-design-themes-edit-guide-page.md)**：根据页面类型（如主页、产品页、博客页等）动态展示内容。

通过 Genstore 官方模版或第三方模版，您无需从零开始设计，即可快速搭建出符合品牌风格的电商网站，并可随时自定义以满足不断变化的业务需求。

:::tip

您可在商店中添加多达 20 个模版，但仅有一个模版可被设为活跃状态，即客户在访问您的商店时所看到的界面。

:::

## 通过模版商店选择和安装模版

1. 登录 Genstore 商家后台，点击 **商店** -> **在线商店** -> **模版**。
3. 在 **推荐模版** 部分，点击 **访问主题商店**。
4. 在模版商店中，通过行业标签筛选适合的模版主题。
5. 点击感兴趣的模版，查看详细信息、演示和用户评价。
6. 确定后，点击 **安装** 按钮。
7. 在随即弹出的窗口中，选择您希望安装该模版的店铺，当前模版将被安装到目标店铺中。

更详细的操作步骤，可参考 [选择模版](./operate-store-design-themes-template-select.md)

## 模版预览与发布

安装完成后，您可以在后台的模版库中：

- 点击模版右侧的 **预览**，查看其在店铺中的实际展示效果
- 点击 **发布**，将该模版设为活跃状态，客户访问店铺时将看到该模版界面

::: tip

发布操作无需额外部署，实时生效，但建议在预览确认无误后再进行发布。

:::

## 自定义模版

Genstore 支持通过多种方式对模版进行个性化设置，满足不同用户的定制需求：

### AI 智能装修

Genstore 提供了 AI 智能装修功能，让您无需专业设计知识即可快速创建精美店铺：

- **AI 生成主页**：通过与 AI 对话，描述您的需求，AI 将自动生成符合您品牌风格的完整首页。
- **AI 更换主题**：一键智能部署整套主题模板，AI 会根据您的店铺类目推荐合适的主题。

详细操作说明请参考：[使用 AI 生成店铺首页与切换主题](./operate-store-design-themes-ai-decorate.md)

### 使用装修编辑器定义模版

无需编写代码，您可以通过 Genstore 内置的可视化装修编辑器，对页面结构、样式和内容模块进行调整与配置。

#### 操作路径

1. 登录后台，点击 **商店** -> **在线商店** -> **模版**。
2. 找到目标模版，点击右侧的 **设计**。
3. 系统将打开装修编辑器，可立即进行页面编辑与实时预览。

编辑器功能详见：[装修编辑器使用指南](./operate-store-design-themes-edit-guide.md)

### 编辑主题代码（适合开发人员）

如需进一步定制页面结构或功能（如添加新模板、引入自定义样式、注入 JS 脚本），可通过代码方式进行修改。

#### 操作路径

1. 登录后台，点击 **商店** -> **在线商店** -> **模版**。
2. 找到目标模版，点击右侧的 `...`，选择 **编辑代码**。
3. 在左侧目录中选择 `.liquid` 或 `.json` 文件进行编辑。
4. 修改完成后，点击右上角 **保存**。

代码结构和使用说明详见：[通过代码自定义主题模板](./operate-store-design-themes-code-editor.md)

## 管理模版

在 Genstore，您可以灵活管理在线商店的模版：

- **模版版本**：您可以查看当前模版的版本号。如有新版本可用，系统将在模版名称后显示提示。点击右侧下拉箭头，选择 **立即更新**，即可将当前模版升级为最新版。
	- 升级后，系统会将新模版副本添加至模版库，完整保留您的页面内容、分区结构和主题设置，您可在发布前预览并确认修改。
- **复制模版**：创建模版副本以安全地测试新设计或功能。
- **删除模版**：永久移除不再需要的模版，确保模版库整洁。
- **下载模版**：下载现有模版的完整副本，用于备份或迁移设计。
