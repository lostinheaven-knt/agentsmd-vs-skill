# 礼品卡

**礼品卡**是一种可用于在线商店或线下门店的预付金额消费凭证。商家可在后台生成礼品卡并通过邮箱发送给客户。客户可通过礼品卡页面查看余额、复制兑换码，或使用二维码完成核销。
## 如何访问

1. 登录 Genstore 商家后台。
2. 点击 **商店** -> **在线商店** -> **模版**。
3. 找到目标模版，点击右侧的 **设计** 按钮，进入模版编辑器。
4. 在顶部导航栏点击页面选择器，选择 **礼品卡**。
## 功能说明

- 默认展示品牌名称、礼品卡金额、有效期、兑换码和二维码
- 支持在线使用及门店扫码核销

## 当前限制

该页面暂不支持自定义，包括但不限于样式调整、布局修改和模块增减。未来版本可能开放更多定制功能，敬请期待。
