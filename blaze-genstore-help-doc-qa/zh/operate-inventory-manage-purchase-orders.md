# 采购订单

商家可以直接在后台创建采购订单，记录需要补充的库存及其数量，无需依赖外部工具。清晰的采购记录不仅便于与供应商沟通，还能减少采购过程中的人为错误，提高库存管理效率。

## 创建采购订单

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **采购订单**。
2. 点击 **创建采购订单**。
3. 点击 **选择供应商**，如果没有供应商信息，需先创建供应商，详情查看 [添加供应商](#管理供应商)。
4. 选择 **目的地**，仅可选择活跃地点。
5. 【选填】选择支付条款，如 货到付款、收到发票时付款、提前付款、N 天内全额付款等。
6. 选择供应商货币。
7. 填写配送相关信息，包括 **预计配送日期**、**运单号**（选填）、**运单承运商**（系统可能会根据跟踪编号自动选择，也可手动调整）。
8. 添加产品，最多添加 200 个产品。点击 **选择采购产品** 按钮，在弹窗中勾选待采购的产品，勾选后的产品将自动出现在采购订单页的产品列表中，您可调整 **采购数量**、**供应商 SKU**（选填）、**成本价**（选填）、**税率**（选填）。
9. 您可在备注区域为采购订单增加备注，便于后续管理。
10. 保存为当前草稿单将处于 **草稿** 状态，当您正式向供应商下单后，可在采购单页面右上角点击 **订购**，用于标记采购流程已启动。

### 调整成本

采购订单页的 **成本摘要** 部分，会整体展示当前的采购量、税费情况，您可点击 **添加成本项** 调整管理成本：

- 在 **成本项** 项的下拉列表中选择希望调整的项目，在 **金额** 部分调整当前调整项的具体金额。
- 点击 **添加** 增加更多调整项。
- 如希望移除某个调整项，点击该项后的 **删除**。

## 编辑采购订单

当采购订单处于草稿状态时，如希望调整已创建的采购订单信息，可点击进入订单详情页，在当前页面直接修改。

## 接收库存

当供应商发出的产品到达后，商家可以在后台记录实际接收的库存数量。**接收的库存数量将自动计入可售库存，同时减少传入库存。**

### 库存变化逻辑

- 采购订单 **未接收** 时，传入库存增加，但现货库存和可售库存保持不变。
- 采购订单 **接收库存** 后，现货库存和可售库存增加，传入库存减少。
- 采购订单 **拒绝库存** 时，库存数量不会发生变化。

| 草稿                 | 无变化   | 无变化   | 无变化   |
| -------------------- | -------- | -------- | -------- |
| 已订购（未接收库存） | 无变化   | 无变化   | **增加** |
| 采购订单状态         | 现货库存 | 可售库存 | 传入库存 |
| 已订购（接收库存）   | **增加** | **增加** | **减少** |
| 已订购（拒绝库存）   | 无变化   | 无变化   | 无变化   |

### 相关金额计算说明

采购订单接收产品时，系统会按照**已接收**的产品数量进行计算和跟踪。

- **传入的产品数量** 会根据 **实际接收数量** 变化（无论产品被标记为接收或拒绝）。
- 采购订单的最终金额会基于 **单个产品成本** 和 **税率** 计算。

## 删除采购订单

采购订单仅在**草稿状态**下可删除。如果采购订单已被标记为**已订购**，需先将其**关闭**。

**操作步骤**

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **采购订单**。
2. 选择 **草稿状态** 的采购订单，点击进入采购订单页。
3. 点击删除。

## 关闭采购订单

- **草稿状态的采购订单无法关闭**，只能删除。
- **已订购但未接收库存** 或 **已部分/全部接收库存（包括拒绝库存）** 的采购订单可以关闭。
- **关闭采购订单不会影响库存数量**。

**操作步骤**

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **采购订单**。
2. 选择目标采购订单。
3. 点击 **关闭采购订单**。

## 管理供应商

商家可以在后台维护供应商信息，包括添加、编辑和删除供应商。

### 如何访问

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **采购订单**。
2. 点击 **创建采购订单**。
3. 点击 **选择供应商**。

### 添加供应商

1. 在 **选择供应商** 窗口中，点击 **添加供应商**。
2. 输入供应商名称，其他信息，如国家、地址、城市、详细地址、邮编、联系人姓名、邮箱等为选填项。
3. 点击 **保存** 完成创建。

### 编辑供应商

1. 在 **选择供应商** 窗口中，点击需修改的供应商后的 **编辑**。
2. 您可在供应商详情页更新相关供应商信息。
3. 点击 **保存** 完成更新。

### 删除供应商

1. 在 **选择供应商** 窗口中，点击希望删除的供应商后的 **删除**。
2. 系统将二次弹窗确认，如确认删除，点击 **确定**。
2. 删除供应商后，与该供应商关联的采购订单需重新选择供应商。
