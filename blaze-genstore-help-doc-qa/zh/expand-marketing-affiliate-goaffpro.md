# Genstore GOAFFPRO

GOAFFPRO 是一款专为快速增长设计的全能型联盟营销工具。通过与 **Genstore** 的官方深度集成，您可以一键搭建起专业的联盟分销门户，轻松管理从伙伴招募到订单归因的全流程。

依托 **Genstore** 的高性能系统内核，**GOAFFPRO** 能够精准捕捉每一笔分销订单并实时同步数据。商户无需编写任何复杂代码，即可快速落地自动化、规模化的达人带货体系，将品牌影响力转化为实际销量。

## 前置准备
- **已拥有 GOAFFPRO 账号**：确保您已在 GOAFFPRO 平台注册并登录。如果没有，请先前往 [GOAFFPRO 官网](https://www.goaffpro.com/) 注册账号。

## 安装 GOAFFPRO 应用
您可通过 Genstore 应用商店安装 GOAFFPRO 应用。
1. 打开 [Genstore 应用市场](https://apps.genstore.ai/zh-CN/)，并登录您的 Genstore 账号。
2. 如果您管理多个 Genstore 商店，请先选择目标商店。
3. 搜索 Genstore GOAFFPRO。
4. 在应用页，点击 **安装** 按钮。
5. 您将被导向 Genstore 商户后台，进入应用设置流程。
6. [可选] 点击应用名称旁的图钉图标，可将 GOAFFPRO 固定至后台导航菜单，便于后续访问。


## 获取 Genstore 店铺关键信息
在建立连接前，请记录以下信息（后续绑定需使用）：
- 店铺名称：**设置** -> **通用**，您可在店铺详情部分获取店铺名称。
- 店铺域名：**设置** -> **域名**，复制当前店铺的主域名。

## 在 GOAFFPRO 生成 SDK
1. 登录 GOAFFPRO 后台。
2. 在后台管理面板中，找到并点击「+ 添加店铺」按钮。
3. 链接方式选择自定义 SDK。
4. 输入上述准备好的 Genstore 店铺名称和域名。
5. 系统将自动生成一个唯一的 SDK 链接，点击 **复制**。

## 激活 Genstore 端的追踪配置

1. 返回 Genstore 商户后台，打开已安装的 GOAFFPRO 应用。
2. 在 **集成代码** 配置区，粘贴您刚才复制的 SDK 链接。
3. 点击 **保存**。至此，您的店铺已具备追踪联盟订单的能力。

## 数据同步
成完成后，数据将按照以下路径自动流转：
- **成员加入**：联盟伙伴通过您的专属门户注册并获得推广链接/优惠码。
- **流量转化**：用户点击链接进入 Genstore 店铺完成下单。
- **归因同步**：系统自动识别推广行为，订单数据实时同步至 GOAFFPRO 后台。
- **效果查看**：店铺管理员可实时查看各渠道的订单归因、佣金明细及推广报表。
