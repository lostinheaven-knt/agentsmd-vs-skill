# 按市场配置支付方式

创建市场时，系统会自动加载店铺中已启用的所有支付方式；当新增支付方式时，系统会默认将其应用到所有已创建的市场。

在多个市场销售产品时，您可以为不同国家/地区的客户展示差异化的支付方式，而不是统一展示全部支付方式。

::: tip

- **Genstore Payments**：系统会根据客户所在国家/地区或结算货币自动展示支付方式，无需额外配置。
- **其他第三方支付方式**：您可以为不同市场单独配置展示。
:::

## 配置步骤

1. 在 Genstore 后台中，转至 **设置** -> **市场**。
2. 在 **概览** 部分，选择要配置的市场。
3. 点击进入该市场，选择 **支付**。
4. 在支付方式列表中，勾选或取消勾选您希望该市场展示的支付方式。
5. 点击 **保存**。

## 前端展示示例

假设您在第三方支付网关中同时启用了多个本地支付方式：

- 针对 **日本市场**，仅勾选 **PayPay**；
- 针对 **泰国市场**，仅勾选 **TrueMoney**。

这样，日本客户在结账时只会看到 PayPay，而泰国客户只会看到 TrueMoney，从而确保支付体验符合当地习惯。
