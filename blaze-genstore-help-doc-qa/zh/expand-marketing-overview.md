# 营销

您可以查看营销关键数据来合理规划营销策略，还可以借助 Genstore 丰富的营销工具来完成您的营销计划

应用商店中还有倒计时、徽章、弹窗营销等众多应用，辅助您完成营销与转化。

## 本节主题

- [查看营销活动效果](./expand-marketing-analysis.md)：
	- 您可以在 Genstore 的报告中查看与邮件营销、社媒营销、自动化营销等相关的流量、订单数、销售额和转化率，帮助您分析营销活动的效果。
	- 如已集成 **Facebook 或 Google**，您还可以在对应平台查看广告点击、受众画像、转化路径等更丰富的数据。如需设置 Google 或 Facebook 数据共享，请前往：
		- [设置 Facebook 数据共享](./expand-marketing-meta-data-sharing.md)
		- [设置 Google 数据共享](./expand-marketing-google-data-sharing.md)
- 营销工具
  - [邮件营销](./expand-marketing-email.md)：您可以通过电子邮件向指定的细分客群推广产品、服务或品牌。
  - [短信营销](./expand-marketing-sms.md)：Genstore 短信营销支持免费试用，正式计费按国家和发送条数计费，费用透明、无隐藏成本。
  - [社媒营销](./expand-marketing-social-media.md)：您可以连接到不同的社媒平台，并直接在 Genstore 后台创建营销内容，进行一键发布。
  - [自动化营销](./expand-marketing-automated-email-overview.md)：您可以预设触发规则和流程，系统将自动执行您的营销任务。
  - [集成三方 EDM](./expand-marketing-email-3rd-edm.md)：Genstore 支持与多个主流邮件营销平台的集成，包括 **Klaviyo**、**Mailchimp** 和 **Omnisend**。
  - [联盟营销](./expand-marketing-affiliate-goaffpro.md)：通过 GOAFFPRO 深度集成方案，轻松开启基于绩效的合作伙伴与红人营销体系，实现全球化获客增长。
