# **Genstore Lite 套餐**

Genstore Lite 套餐为您提供经营在线商店所需的所有基础工具。无论您的商店规模如何，都可以通过选择 Lite 套餐，自由创建和自定义您的在线商店，展示产品，管理订单与客户信息。通过 Genstore 后台，您可以高效地运营商店。此外，Genstore AI 将帮助您在商店装修、产品运营、推广方案、数据分析等方面提升效率，轻松应对日常运营挑战。

## 积分

- 套餐内包含 50 点月度积分。
- 每天登陆系统，可自动获得 5 点每日积分。   

## **套餐功能**

除了所有套餐的通用功能外，Genstore Lite 套餐还提供以下功能：

### **Genstore AI**

Genstore AI 是一套智能化工具，旨在帮助商家自动化管理和优化在线商店运营。通过强大的 AI 能力，商家可以轻松进行产品优化、店铺装修、推广方案制定、数据分析等工作，减少手动操作的时间和成本，提升运营效率，确保商店持续增长和竞争力。

| 功能            | 描述                                   |
| ------------- | ------------------------------------ |
| **开店 Agent**  | 帮助您快速完成店铺基础配置，如域名、支付到物流等设置，确保店铺合规上线。 |
| **产品 Agent**  | 通过智能选品与产品创建工具，高效管理您的产品目录。            |
| **装修 Agent**  | 快速执行首页生成与主题切换任务，打造高转化的品牌视觉。          |
| **营促销 Agent** | 精准触发 AI 营销活动与折扣生成逻辑，提升流量转化。          |
| **客服 Agent**  | 提供 7×24 小时全天候技术指导与经营建议的专属商户服务专家。     |
| **数据 Agent**  | 实时分析店铺流量、转化率及销售表现，为您提供直观的数据洞察与经营建议。  |
