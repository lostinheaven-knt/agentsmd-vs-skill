# 语言和域名

在 Genstore 中，您可以为每个市场配置语言和域名，从而为不同国家或地区的客户提供本地化的访问体验。

市场域名配置有以下三种类型：

- **使用主市场配置**
  - 使用主市场配置域名的市场，其语言配置也和主市场保持一致。
  - 默认情况下，新添加市场的语言和域名均使用主市场配置。
- **使用子目录域名**
  - 您可以为市场配置子目录形式的 URL，路径结构为：`[主域名]/[国家/地区代码]/[语言代码]`。
  - 支持为每个市场激活多种语言，每种语言对应不同的子目录路径。
  - 例如：
   	- 如果您的主市场（美国）使用主域名 `example.com`，加拿大市场可使用 `example.com/ca`。
   	- 若为加拿大市场启用法语，则法语页面的路径为 `example.com/ca/fr`。
- **使用域名或子域名**
  - 您可以为每个市场配置独立的域名（如 `example.ca`）或子域名（如 `ca.example.com`）。
  - 配置前，需先在 **设置** -> **域名** 页面中连接目标域名或子域名。
  - 连接后，可将其分配给市场，并为该域名配置所需语言。

> 如果您删除了已分配的域名或子域名，客户访问该市场 URL 时将收到 404 页面。

:::tip

主市场默认使用主域名，且不可变更。若您想要变更主市场相应的域名，请前往 **设置** -> **域名**，将对应的域名设置为主域名。

:::

#### 步骤

1. 在 Genstore 后台中，转至 **设置** -> **市场**。
2. 点击要设置域名和语言的市场。
3. 在 **市场设置** 部分中，点击 **语言和域名**。
4. 点击 **管理域名**，然后选择 **要使用的域名配置**，如子目录、单独的域/子域，或使用主市场配置。

## 配置语言

您可进一步为当前市场配置语言：

步骤：

1. 点击域名后的 **添加语言** -> **添加更多语言**。
2. 在新打开的 **语言** 页面，点击 **添加语言**。
3. 选择您希望添加的语言，点击 **下一步**。
   此时 Genstore 会提示您安装 Translate 应用，帮助完成店铺本土化。
4. 选择将当前语言分配的市场，点击 **完成**。
5. 完成语言的配置后，当前设置仅对您可见，您可在 **语言**  -> **未发布** 的语言下找到该语言。点击 **...** -> **发布**，完成语言的发布。**注意**：您也可在当前页面设置进一步编辑将语言分配至哪些市场。
6. 返回市场，点击进入目标市场店，点击 **语言和域名**，您可看到语言已成功添加。
7. 后续您可点击语言后的  **...**，删除当前语言或将其设为店铺的默认语言。
:::tip

若市场的域名使用了主市场配置，则语言也使用主市场配置，不可单独配置。如需单独管理某个市场的语言，请将该市场的域名设置为使用子目录域名/子域名。

:::
