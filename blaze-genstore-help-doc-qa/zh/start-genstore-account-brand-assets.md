# 设置店铺品牌资产

通过设置店铺的品牌资产，您可以为您的店铺添加 logo、品牌颜色、社交媒体链接等元素，确保店铺的品牌形象在所有销售渠道、主题和应用中保持一致。

## Logo

为您的品牌添加一个独特的 logo。Genstore 提供默认 Logo 和方形 Logo 的上传功能，确保在所有平台和应用中都能完美展示。请注意，不同平台对 Logo 的显示要求可能不同，例如社交媒体平台会将图片裁剪为圆形。

### 上传准备

开始上传前，请确保您上传的 Logo 符合以下要求，以获得最佳展示效果：

| 品牌资产    | 描述                                                                 | 要求                                                     |
| ------- | ------------------------------------------------------------------ | ------------------------------------------------------ |
| 默认 Logo | 适用于大多数常见的 Logo 应用程序，通常显示在店铺首页、购物车等位置。优质的 Logo 与商店名称相得益彰，并采用您的品牌主色。 | - 格式：HEIC, WEBP, SVG, PNG,  JPG<br>- 建议像素：至少 512 x 512 |
| 方形 Logo | 用于社交媒体渠道和资料图片，需上传方形格式的 Logo。如果平台使用圆形图片，Logo 将会被裁剪为圆形。              | - 格式：HEIC, WEBP, SVG, PNG,  JPG<br>- 建议像素：至少 512x512   |

### 上传 logo

1. 登录 Genstore 商户后台，导航至 **设置** -> **通用**。
2. 在 **品牌资产** 部分，点击 **管理**。
3. 在 **Logos** 部分中，上传品牌 Logo 或从素材库中进行选择。您可以使用具体图片名称 **搜索图片**。您还可以从 **本地** 上传图片。
4. 重复步骤 2 以选择或添加方形 Logo。
5. 点击 **保存**。

:::tip

- 如果您上传的 Logo 不符合格式或分辨率要求，系统将提示您进行调整。请确保选择符合要求的文件后再继续操作。
- 每个 Logo 类型仅支持一个文件，您可以随时更新或删除现有的 Logo。
- 首次上传 Logo 后，将自动同步至全站所有展示位置；如之后更新 Logo，需前往[装修编辑器](./operate-store-design-images#上传-logo-与网站图标-favicon)手动修改。

:::

## 颜色

设置品牌色是确保品牌一致性的关键步骤。选择主色、辅助色和对比色，使您的店铺在不同平台上都能维持一致的视觉效果。

### 步骤：

1. 登录 Genstore 商户后台，导航至 **设置** -> **通用**。
2. 在 **品牌资产** 部分，点击 **管理**。
3. 在 **颜色** 区域下，点击添加主色。
4. 【可选】系统会自动为您创建对比色，您可根据品牌风格进一步定制。
5. 重复上述步骤以选择辅助颜色和辅助对比色。
7. 点击 **保存**。

## 社交媒体链接

添加社交媒体链接，方便与客户互动，这些链接通常会显示在店铺页面的页脚或其他位置。

### 步骤：

1. 登录 Genstore 商户后台，导航至 **设置** -> **通用**。
2. 在 **品牌资产** 部分，点击 **管理**。
3. 在 **社交媒体链接** 部分，点击添加社交媒体平台，例如 Instagram、Twitter、Facebook 等。
4. 在提供的字段中输入完整的 URL 链接，例如：`https://instagram.com/genstore` 或 `https://twitter.com/genstore`。
5. 点击 **保存**。
