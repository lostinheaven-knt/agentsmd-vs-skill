# 页面模版

在 Genstore 可视化编辑器中，每种页面类型都有其对应的模版（Template），用于定义该页面的整体结构与视觉样式。商户可针对不同页面类型进行个性化编辑，提升品牌表达力与用户浏览体验。

## 页面模版概览

以下为 Genstore 内置支持的页面模版类型：

| 页面类型                                                                           | 说明                                |
| ------------------------------------------------------------------------------ | --------------------------------- |
| **[主页](./operate-store-design-themes-edit-guide-page-home.md)**                | 店铺首页，可自由组合模块以展示品牌故事、热销产品、活动横幅等    |
| **[产品](./operate-store-design-themes-edit-guide-page-product.md)**             | 每个产品可绑定不同模版，适用于定制化展示（如不同品类需差异化排版） |
| **[产品系列](./operate-store-design-themes-edit-guide-page-collection.md)**        | 可为不同的产品系列指定不同模版，用于强调分类、筛选器、商品布局等  |
| **[产品系列列表](./operate-store-design-themes-edit-guide-page-collection-list.md)** | 展示所有系列分类的汇总页面                     |
| **[礼品卡](./operate-store-design-themes-edit-guide-gift-card.md)**               | 展示店铺发行的礼品卡信息、面额、说明等               |
| **[购物车](./operate-store-design-themes-edit-guide-cart.md)**                    | 展示购物车中商品详情，支持编辑数量、删除等操作           |
| [**结账页**](./operate-store-design-themes-edit-guide-page-checkout.md)           | 用户完成下单前的关键页面，用于填写收货和支付信息，直接影响转化率  |
| [**感谢页**](./operate-store-design-themes-edit-guide-page-thank-you.md)          | 用户完成支付后的确认页面，既提供订单信息，也承载复购与互动机会   |
| **[客户账户](./operate-store-design-themes-edit-guide-customer-accounts.md)**      | 包含注册、登录、找回密码、账户信息、地址管理等子页面        |
| **页面**                                                                         | 用于创建自定义静态内容页面（如品牌介绍、退换货政策、联系我们）   |
| **博客集合**                                                                       | 博客集合列表页，展示所有的博客集合，可分页或瀑布流         |
| **博客文章**                                                                       | 单篇博客内容展示页，支持图文混排、标签、评论功能等         |
| **[搜索页](./operate-store-design-themes-edit-guide-search.md)**                  | 展示搜索关键词匹配到的相关产品或内容                |
| **[404 页面](./operate-store-design-themes-edit-guide-404.md)**                  | 当访问的页面不存在时展示此页面，支持插入导航按钮等内容引导     |

## 页面模版配置说明

- 各页面类型支持在顶部页面选择器中切换，并进入对应模版编辑
- 大多数页面类型（如产品、产品系列）支持创建多个自定义模版，并在产品/系列详情页中进行绑定选择
- 每个页面模版可独立设计布局、模块组合与样式配置，不影响其他页面

## 操作入口

1. 登录 Genstore 商家后台，点击 **商店** -> **在线商店** -> **模版**。
2. 点击目标模版右侧的 **设计** 按钮，进入编辑器。
3. 在顶部工具栏中使用页面选择器，选择您希望编辑的页面类型与模版。
4. 在左侧结构栏中添加、删除、排序模块，在右侧配置栏中调整每个模块的内容与样式。
5. 点击右上角 **保存**，并可使用 **预览** 查看实际效果。

Genstore 主题编辑器还提供了页面管理快捷入口，帮您集中进行模版的创建、绑定与管理操作，具体可参考：[页面管理指南](./operate-store-design-themes-edit-guide-page-manage.md)
