# 接收库存转移

当库存到达后，商家可选择**全部接收**或在产品部分损坏时进行**部分接收**，并通过接收库存功能核实和更新库存数量。这样可以确保系统中每个仓库的库存状态准确，同时减少人为错误，提高库存数据的可靠性。

## 操作步骤

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **库存**，找到库存转移单据号。
2. 进入库存单据，点击 **接受库存**，进入待收货单据界面。
3. 选择合适的操作方式：
   - 点击 **接受全部转移**，全部接收库存。
   - 点击 **拒绝全部转移**，全部拒绝库存。
   - 部分收货：根据实际情况，输入 **已收货** 的数量，并填写因损坏等原因导致的 **已拒绝** 数量。
4. 点击 **保存**，产品的库存数量随您接收的数量进行调整。
5. 确认接收完成：
   - 当所有库存都到达后，转移状态会自动更新为 **已收货**。
   - 系统会自动更新目标仓库的库存数量，同时扣减发件地点的库存。
