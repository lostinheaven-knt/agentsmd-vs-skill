# Genstore 品牌标识

在商店主题页脚（右下角），默认会显示 “Made with Genstore”  品牌标识。该标识旨在帮助建立平台品牌认知，同时向访问者展示商店的搭建平台。如果您已订阅付费计划，则品牌标识会自动隐藏，让商店只展示您的自有品牌。

## 功能价值

- **提升专业感**：去除平台标识后，商店页面只展示您的品牌元素，整体形象更统一。
- **增强独立性**：顾客只会看到您自己的品牌，不会因平台标识产生混淆。
- **专属权益**：此功能属于升级订阅计划的一部分，将自动为您生效，帮助您获得更高级别的定制化体验。

## 品牌标识自动隐藏与恢复显示

::: tip
试用期商户如希望隐藏品牌标识，请先升级为订阅商户，品牌标识将自动隐藏。
:::

- **自动隐藏**：订阅付费计划后，Genstore 品牌标识将自动隐藏，您无需进行任何额外操作。
- **恢复显示品牌标识（如需）**：
  如果您有特定需求需要恢复显示 "Made with Genstore" 标识，请按以下步骤操作：
  1. 登录商户后台，进入 **在线商店** -> **偏好设置**。
  2. 滑动至 **移除品牌标识** 部分，关闭滑钮开关。
  3. 保存设置。

::: tip

- 去除品牌标识仅影响前台展示，对后台配置和搜索引擎优化无影响。
- 修改完成并保存后，设置会在下次访问商店页面时生效。如未看到更新效果，可尝试刷新页面。

:::
