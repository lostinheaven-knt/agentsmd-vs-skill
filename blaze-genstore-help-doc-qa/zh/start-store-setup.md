# 店铺初始设置概览

在 Genstore 上搭建和上线您的商店，主要包括以下两个阶段：

- [搭建店铺基础框架](#搭建店铺基础框架)
- [完成店铺合规与上线准备](#完成店铺合规与上线准备)

## 搭建店铺基础框架

搭建商店基本骨架，完成内容准备与页面布局。

- 注册并创建店铺
- 填写店铺基本信息
- 添加员工账户
- 设置域名
- 添加产品
- 店铺装修与发布

详细指南：[搭建店铺基础框架](./start-store-setup-basic.md)

## 完成店铺合规与上线准备

完善商店政策、物流、支付设置，进行测试并正式上线。

- 设置公共信息页（隐私政策、退换货政策等）
- 配置物流与税费
- 配置支付与结账流程
- 预览与调试店铺
- 创建测试订单
- 正式上线商店

详细指南：[完成店铺合规与上线准备](./start-store-setup-launch-prep.md)

通过完成以上两个阶段的初始设置，您的 Genstore 商店将具备完整的销售、运营与客户服务能力，顺利开启正式运营之路！
