# 库存状态

Genstore 提供了一整套库存状态管理机制，旨在帮助商家和顾客实时掌握产品库存情况，保障购物体验并防止超卖。库存概念同时适用于实物产品和虚拟产品。库存状态根据产品类型和业务场景进行区分，主要包括以下几种：

| 库存类型   | 实物产品 | 虚拟产品 |
| ---------- | -------- | -------- |
| 可售库存   | ✓        | ✓        |
| 现货库存   | ✓        | ✗        |
| 承诺库存   | ✓        | ✓        |
| 不可用库存 | ✓        | ✗        |
| 传入库存   | ✓        | ✗        |

以下为当前库存信息和业务场景的详细说明：

## 可售库存 (Available Inventory)

可售库存为可供客户下单的实时库存数量（仅包含未承诺库存），对实物产品、虚拟产品有效。例如商家有 100 件经典白色 T 恤的库存，其中 10 件已预定，5 件已下单但未发货，那么**可售库存**：

`100 - 10 - 5 = 85 件`

| **项目**     | **数量（件）** | **说明**                     |
| ------------ | -------------- | ---------------------------- |
| 库存总量     | 100            | 产品的总库存数量             |
| 预定库存     | 10             | 已预订但未付款或未发货的库存 |
| 已下单未发货 | 5              | 客户已下单但尚未发货的库存   |
| **可售库存** | **85**         | 供客户下单的实时库存         |

## 现货库存 (On-hand Inventory)

现货库存指仓库中已到货且可随时发货的产品数量，仅适用于**实物产品**，核心价值在于：

- 便于仓库库存管理、补货决策和供应链优化
- 支持预售：当可售库存为 0，但仓库已有入库计划时，可开启预售功能
- 退货管理：退回产品先进入现货库存，经处理后再上架销售

| 产品名称         | **现货库存数量** | **说明**                       |
| ---------------- | ---------------- | ------------------------------ |
| **无线蓝牙耳机** | 100              | 仓库中已到货且可随时发货的库存 |

## 承诺库存 (Committed Inventory)

承诺库存指已被下单但尚未发货的库存，对实物产品和虚拟产品均适用，例如，某商家有 200 件极速遥控车的库存，其中 50 件已被下单但尚未发货，30 件处于预定状态，则**承诺库存**：

`50 + 30 = 80 件`

| **项目**         | **数量（件）** | **说明**                     |
| ---------------- | -------------- | ---------------------------- |
| **库存总量**     | 200            | 产品的总库存数量             |
| **已下单未发货** | 50             | 顾客已下单但尚未发货的库存   |
| **预定库存**     | 30             | 已预订但未付款或未发货的库存 |
| **承诺库存**     | **80**         | 已被锁定，暂不可售的库存     |
| **可售库存**     | **120**        | 供客户下单的实时库存         |

> 发货后，相关库存从承诺库存转移为已发货，系统自动更新状态。

## 不可用库存 (Unavailable Inventory)

不可用库存指无法供客户下单的库存，通常由于特殊情形被预留。该库存仅适用于**实物产品**，并用于确保正常销售库存不受影响。例如，某商家有 500 件产品，其中 20 件因客户预留，10 件因草稿订单，15 件因损坏或过期被标记为不可用，则**不可用库存**计算如下：

`20 + 10 +15 = 45 件`

| **项目**          | **数量（件）** | **说明**                     |
| ----------------- | -------------- | ---------------------------- |
| **库存总量**      | 500            | 产品的总库存数量             |
| **客户预留库存**  | 20             | 预留给特定客户的库存         |
| **草稿订单库存**  | 10             | 草稿订单中暂未正式下单的库存 |
| **损坏/过期库存** | 15             | 无法销售的库存               |
| **不可用库存**    | **45**         | 暂不可售的库存               |
| **可售库存**      | **455**        | 供客户下单的实时库存         |

## 传入库存 (Incoming Inventory)

传入库存指尚未入库、仍在运输或处理中，暂不可售的库存。适用于**实物产品**，用于管理采购订单或库存转移等在途库存。例如，某商家目前有 50 件极速跑鞋的库存，另有 100 件在途，预计几天内到货。

`50 + 100 = 150 件`

| **项目**         | **数量（件）** | **说明**                     |
| ---------------- | -------------- | ---------------------------- |
| **当前库存总量** | 50             | 现有可用库存                 |
| **传入库存**     | 100            | 采购或转移中，尚未入库的库存 |
| **预计总库存**   | **150**        | 当前库存 + 传入库存          |
