# Genstore 翻译应用

**Genstore 翻译应用**帮助您为商店打造专业的多语言内容，提升全球用户的购物体验。借助多语言网站，您不仅可以增加店铺流量、优化 SEO 排名，还能增强国际用户的信任感，助力拓展海外市场。应用支持一键自动翻译，并允许手动调整内容。并排编辑器便于对比原文与译文，高效审校，确保翻译质量。

## 主要特点

- 与 Genstore 系统无缝集成，轻松管理商店内容的多语言版本
- 依托 AI 技术，实现一键自动翻译
- 借助并排编辑器，支持手动翻译、编辑并审阅译文

## 适用场景

Genstore 翻译支持您轻松管理和翻译店铺内容多语言，包括但不限于以下场景：

- **翻译产品描述**：您的店铺需展示多语言产品描述吸引全球买家。多语言描述提升国际市场的销售转化率。
- **网站页面翻译**：网站模板和导航需多语言版本以服务全球用户。多语言页面提升用户体验和访问时长。
- **博客文章翻译**：博客文章需翻译以吸引全球读者扩大品牌影响力。多语言博客增加网站流量和用户互动。
- **邮件通知翻译**：国际客户需接收多语言订单和物流通知。熟悉的语言通知提升客户信任和满意度。
- **政策翻译**：隐私政策和条款需多语言版本以符合全球法规。多语言政策确保用户理解和法律合规。

## 安装应用

当您向商店添加新语言时，系统会自动安装 **Genstore 翻译** 应用。

如果您已向商店中发布语言并且您想安装该应用，请前往 **设置 ->语言**，点击 **安装 Genstore 翻译**。

## 手动翻译内容

您可在 **Genstore 翻译** 应用中，或从具体内容页面直接进入翻译编辑器，手动输入并保存翻译内容。

### 步骤：

1. 在 Genstore 后台中，转到 **应用** 或 **设置 ->语言**。
2. 点击 **Genstore Translate**。
3. 选择要翻译的语言。
4. 选择要翻译的资源。
5. 将已翻译的内容输入相应字段。
6. 点击**保存**。

## 自动翻译内容

您可以使用 **Genstore 翻译** 应用自动翻译内容。发布前建议先审阅翻译内容，确保其准确性。

### 自动翻译某种语言的内容

每种语言需单独开启自动翻译。翻译完成后，内容将在几分钟内显示于店铺前台。

#### 步骤：

1. 在 Genstore 后台中，转到 **应用** 或 **设置 ->语言**。
2. 点击 **Genstore Translate**。
3. 选择要翻译的语言。
4. 点击 **自动翻译**。
	- 如果该语言已有部分翻译，请选择：
		- **仅翻译空字段**：只补充未翻译内容；
		- **覆盖翻译所有字段**：重新生成全部翻译。
	- 点击 **确定**。
5. 翻译完成后，系统会向您的账号邮箱发送通知邮件。

:::tip

- 当新增内容（如新增产品）时，请重新进入应用并点击 **自动翻译**，为该语言生成新内容的翻译。
- 当更新已有内容（如修改产品描述）时：
    - 若之前已有翻译，可选择手动修改；或使用 **覆盖翻译所有字段**；或先清空翻译，再执行 **仅翻译空字段**。
    - 若为新增或之前未翻译的内容，点击 **自动翻译** 并选择 **仅翻译空字段** 即可。

:::

### 自动翻译单项内容

若只需翻译某一项（如单个产品、博客文章），可为该资源单独启用自动翻译。

##### 步骤：

1. 在 Genstore 后台中，转到 **应用** 或 **设置 ->语言**。
2. 点击 **Genstore Translate**。
3. 选择要翻译的内容。
4. 点击 **自动翻译**。
	- 如果该语言已有部分翻译，请选择：
		- **仅翻译空字段**：只补充未翻译内容；
		- **覆盖翻译所有字段**：重新生成全部翻译。
	- 点击 **确定**。
5. 点击 **保存**。

:::tip

- 翻译包含图片或视频的资源时，您可为不同语言选择不同版本的素材。
- URL handle 不支持自动翻译，您需手动为各语言设置。

:::
