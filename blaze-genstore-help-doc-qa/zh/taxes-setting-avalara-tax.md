# Avalara Tax

针对于税务较为复杂的国家，您可以使用 Genstore 提供的基于第三方税务平台 Avalara 的自动化税务解决方案。您只需要连接 Avalara 账号，后续平台所有交易税费计算统一由 Avalara 负责，您无需关心应该在何时何地收取客户多少税费。

::: tip
使用 Avalara Tax 需要您开通并配置 Avalara 账户，建议您使用 Genstore Tax 服务，无需开通和配置账号，更加方便快捷。
:::

## 开通并配置 Avalara 账户

使用自动化税务前，您需要先开通 Avalara 账户。您可以登录 [Avalara 官网](https://www.avalara.com/us/en/index.html) 进行账户注册。

若您已有拥有账号，则可以直接登录 Avalara 平台，并确认以下配置已经完成：

### 在 Avalara 平台设置公司信息及收税地点

1. 登录 Avalara 点击 **Settings** -> **Manage companies**。
2. 添加公司信息。
3. 点击 **Settings** -> **Where you collect tax**。
4. 点击 **Add countries where you report tax** 进行收税国家添加。

### 获取 API 访问凭证

1. 登录 Avalara 点击 **Settings** -> **License and API keys**。
2. 点击 **Generate new key** 生成许可证秘钥。

::: tip

若您有任何关于 Avalara 帐户设置问题，请咨询 Avalara 官方客服寻求帮助。

:::

## 在 Genstore 中连接与配置

获取 Avalara 账号以及生成的许可证秘钥后，请按照以下步骤完成授权：

1. 登录 Genstore 商户后台，进入 **设置** -> **税务**。
2. 在 **税务服务** 模块中找到 **Avalara Tax**。
3. 点击 **管理授权**。
4. 输入 Avalara 账号以及生成的许可证秘钥。
5. 选择账号环境（如不确定账号环境，请咨询 Avalara 客服）：
   - 若账号为正式环境则选择 **正式环境**。
   - 若账号为沙箱环境则选择 **沙箱环境**。
6. 点击 **测试连接**。验证成功后，方可继续操作。
6. 选择在 Avalara 平台已配置的公司主体。
7. 选择计税类型：
   - 仅计税：仅提供税务计算，不会将交易记录同步到税务账户。
   - 计税并提交交易信息：计算税费并将交易信息同步至税务账户，适用于需要跟踪完整交易计税记录用于报税或审计等场景。
8.  设置完成后，点击 **点击授权**。

## 关联税码

自动计税的精准性依赖于产品与 Avalara 税码的正确映射。Genstore 支持按类别进行批量关联：
1. 登录 Genstore 商户后台，进入 **设置** -> **税务**。
2. 在 **税务服务** 模块中找到 **Avalara Tax**。
3. 点击 **管理税码**。
4. 在此页面确认产品的税码关联情况。
5. 对于非标产品，如货运税码，可点击 **非标产品税码** 进行配置。

## 为目标地区启用 Avalara Tax
完成全局配置后，您需要为特定的配送区域开启服务：
1. 登录 Genstore 商户后台，进入 **设置** -> **税务**。
2. 选择目标收税区域，进入税务配置页面。
3. 点击当前计税方式后的切换图标，切换至 Avalara **Tax**。
4. [可选] 点击 **更多设置**，自定义税务展示名称、备注等参数。
5. 点击 **保存**。

## 断开连接

若需断开 Genstore 与 Avalara 的连接：

1. 进入 **设置** -> **税务**，找到 **税务服务**。
2. 点击 **Avalara Tax** 模块后的 **管理授权**。
3. 点击 **去除授权**。

::: warning
删除授权后，您将无法再使用 Avalara 自动计税服务。若您需要重新连接 Avalara 账户，请按照以上步骤重新进行授权。
:::
