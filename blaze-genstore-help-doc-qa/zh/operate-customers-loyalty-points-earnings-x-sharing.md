# 分享 X 积分

分享积分奖励让您的品牌在社交媒体上获得更多曝光。当客户在 X 上分享您指定的店铺链接时，即可获得相应积分。虽然系统暂无法追踪实际分享行为，但这种互动激励机制能有效提升品牌影响力，为店铺带来更多潜在客户。

本文档将说明该功能的配置方法、使用场景及注意事项。

## 配置积分规则

**导航：**

登录 Genstore 商家后台，点击 **商店** -> **客户** -> **忠诚度**。  
在 **积分** 标签下，找到 **获取积分** 部分。

### 设置步骤

点击**新增** -> **分享链接到 X** -> **添加**，进入配置页面：
- **规则名称**：输入积分奖励规则的名称，方便管理。
- **获取积分**：设置客户将链接分享到 X 可获得的积分数。
- **分享链接**：设置分享链接。
- **分享内容**：设置分享内容，选择预设内容或根据店铺风格修改。
- **设置积分获取频次**：设定客户通过链接分享可获得积分的次数，可选择每天、每周、每月、每年可获得 * 次，或仅限首次分享可获得积分。
  :::tip
  系统默认设置为仅首次分享可获得积分，为了防止客户欺诈行为，不建议设置调整为过高频次。
  :::

## 预览及发布

在页面的右侧**概要**区域，系统会自动生成并展示当前设置的规则概述，方便实时预览。确认无误后，点击**保存**，发布规则。

规则发布后，您可在积分获取方式列表中查看当前规则、编辑、启用或停用该规则。

:::tip

您也可通过状态开关控制当前规则的启用与关闭。

:::

## 常见问题

### Q1. 如果客户取消分享，是否会扣除积分？
> A：由于技术限制，系统暂时无法跟踪客户是否真实分享或取消分享。因此，分享后获得的积分不会受到取消分享的影响。

### Q2. 我的分享 URL 无法通过验证，该怎么办？
> A：请确保输入的 URL 是有效的地址，且格式正确。系统会根据预设的正则表达式校验 URL。

### Q3. 是否可以为每个客户设置不同的分享积分频率？
> A：积分获取频率采用统一设置，适用于所有客户，暂不支持针对个别客户自定义调整。同时为防止欺诈，不建议设置过高的频率。
