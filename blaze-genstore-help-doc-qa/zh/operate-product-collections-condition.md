# 自动产品系列筛选条件

Genstore 的 [自动产品系列功能](./operate-product-collections-auto.md) 支持您根据一系列条件自动将符合要求的产品归入相应系列。通过设置条件，您可以根据产品类型、标题、产品特色、描述、价格、库存等多个维度精确筛选产品，并自动将它们添加到指定系列。此功能特别适用于大批量产品管理，能有效提高产品分类效率，节省手动操作时间。

:::tip

- 您可以添加多个条件进行匹配，每个自动产品系列最多可以添加 60 个条件。
- 每个条件都可以设置为 **匹配所有条件** 或 **匹配任意条件**，具体取决于您的需求。
  - 如选择所有条件，产品需要同时满足所有条件才能加入产品系列中。
  - 如选择任意条件，产品只需匹配一个条件即可包含在产品系列中。

:::

## 根据产品类型设置条件

您可以通过筛选产品类型，将符合条件的产品加入系列。可以选择多个条件进行匹配。

| 术语  | 描述                            |
| --- | ----------------------------- |
| 等于  | 下拉框单选设置指定产品类型（实物产品、数字产品、礼品卡等） |

## 根据产品标题、产品特色设置条件

您可以通过筛选产品标题或产品特色，匹配符合条件的产品。可以添加多个条件进行匹配。

| 术语 | 描述|
|-------| -------|
| 等于 | 产品标题或产品特色需要与输入的完整值完全匹配 |
| 不等于 | 产品标题或产品特色与输入的值不一致 |
| 开头为 | 产品标题或产品特色的开头与输入的值匹配 |
| 结尾为 | 产品标题或产品特色的末尾与输入的值匹配 |
| 包含 | 产品标题或产品特色包含输入的值（长度至少为 3 个字符，且该值不能以空格开头或结尾） |
| 不包含 | 产品标题或产品特色不包含输入的值（长度至少为 3 个字符，且该值不能以空格开头或结尾） |

## 根据产品描述设置条件

您可以通过筛选产品描述，匹配符合条件的产品。可以添加多个条件进行匹配。

| 术语 | 描述|
|-------| -------|
| 包含 | 产品描述包含输入的值（长度至少为 3 个字符，且该值不能以空格开头或结尾） |
| 不包含 | 产品描述不包含输入的值（长度至少为 3 个字符，且该值不能以空格开头或结尾） |

## 根据产品类别设置条件

您可以通过筛选产品类别条件，自动将特定类别的产品包含在系列中。选择条件时，请按照以下步骤：

1. 第一个下拉框选择 **等于**。
2. 第二个下拉框选择具体的产品类别。

## 根据产品价格、原价、库存数、重量设置条件

您可以通过筛选产品的价格、原价、库存量或重量，匹配符合条件的产品。条件可为精确匹配、范围匹配或其他操作。

| 术语 | 描述|
|-------| -------|
| 等于 | 与确切值匹配 |
| 不等于 | 与值不匹配 |
| 大于 | 大于此值 |
| 小于 | 小于此值 |
| 大于等于 | 大于等于此值|
| 小于等于| 小于等于此值 |

## 根据产品标签设置条件

您可以通过筛选产品标签，将符合标签的产品包含在系列中。设置步骤如下：

1. 第一个下拉框选择产品标签。
2. 第二个下拉框选择 **等于**。
3. 下拉选择具体产品标签。

## 根据产品厂商设置条件

您可以通过筛选产品厂商设置条件，根据产品的厂商将产品包含在系列中。设置步骤如下：

1. 第一个下拉框选择产品厂商。
2. 第二个下拉框选择 **等于**。
3. 下拉选择具体的产品厂商。
