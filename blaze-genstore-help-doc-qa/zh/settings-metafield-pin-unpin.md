# 管理元字段列表

为了提升后台操作效率，您可以将常用的元字段定义**固定**在模块详情页中，方便直接查看和填写。
## 应用场景

- **自动展示**：固定的元字段将在对应模块（如产品、客户等）的详情页中直接展示；
- **控制顺序**：固定字段的排序即为页面中展示顺序；
- **提升效率**：避免每次都点击“查看全部”查找字段；

默认状态下，仅固定字段会直接显示。未固定的字段需点击 **查看全部** 手动展开。
## 固定规则

- 每个模块最多支持固定 **10 个元字段定义**
- 支持随时固定 / 取消固定
- 支持拖动调整显示顺序

## 固定/取消固定操作

在 **设置 -> 元字段定义** 页面，您可以：

| 操作       | 图标说明                  | 图例                                                  |
| -------- | --------------------- | --------------------------------------------------- |
| **固定字段** | 点击图钉图标将字段加入页面展示区      | ![image-20250723145353592](./assets/icon-pin.png)   |
| **取消固定** | 再次点击图钉图标 取消固定         | ![image-20250723145353592](./assets/icon-unpin.png) |
| **排序调整** | 拖动图标 ☰ 以更改字段在页面中的显示顺序 | ![image-20250723145353592](./assets/icon-sort.png)  |
