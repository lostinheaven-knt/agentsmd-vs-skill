# 店铺通用设置

在开始设置 Genstore 商店之前，您需要创建一个 Genstore 账户并为商店添加必要的业务信息。

## 创建 Genstore 账户

访问 [创建 Genstore 账户页面](https://accounts.genstore.ai/account/signup)，输入有效的邮箱和密码即可开始设置您的店铺。

虽然您不需要立即填写所有详细信息，但请务必添加以下内容：

- 店铺名称和联系方式
- 法定公司名称和公司地址
- 店铺时区
- 默认店铺货币
- 默认重量单位

## 域名

创建店铺时，系统会自动分配一个唯一的 `genmystore.com` 域名，例如 `1a2b3c4-5.genmystore.com`。该域名用于登录后台和确认账户身份。当您希望连接 Genstore 账户与第三方应用或服务时，可能需要提供这个域名作为您的账户 ID。

默认情况下，此域名还会被视为您店铺的主域名，但您可以 [将第三方域名连接到 Genstore](./global-domains-custom-domain-third-party.md) 来替换 `genmystore.com` 域名，作为客户访问您的店铺时所用的网址。

此外，也支持商户自定义域名，具体可参考 [域名](./global-domains.md)。注意：`genmystore.com` 域名仅支持自定义一次。

## 管理店铺名称和联系信息

店铺名称将在您的网站每个页面上显示。默认情况下，商店名称为 **My Store**。

:::tip

更改店铺名称不会更改默认 `genmystore.com` 域名。如希望修改店铺域名，请访问 [域名](./global-domains.md)。

:::

### 步骤：

1. 登录 Genstore 商户后台，导航到 **设置** -> **通用**。
2. 在 **店铺详情** 部分，点击当前店铺名称旁的铅笔图标。
3. 在 **编辑资料** 对话框中，修改店铺名称、联系电话、联系邮箱。
4. 点击 **保存**。

## 管理法定公司名称和营业地址

企业的法定名称和营业地址将显示在 [账单](./start-genstore-billing.md) 中。

### 步骤：

1. 登录 Genstore 商户后台，导航到 **设置** -> **通用**。
2. 在 **店铺详情** 部分，点击 **账单地址** 旁的铅笔图标。
3. 输入您的法定公司名称和营业地址。
4. 点击 **保存**。

## 选择店铺货币

1. 登录 Genstore 商户后台，导航到 **设置** -> **通用**。
2. 在 **默认设置** -> **展示货币** 部分，点击 **...** > **变更店铺货币**。
3. 使用下拉菜单选择新货币单位。
4. 点击 **保存**。

更新商店货币后，请查看并更新以下商店设置：
- **查看并更新产品定价**：由于不同货币的价值通常不等，因此使用新货币可能会导致产品价值发生变化，进而影响您的利润率。
- **查看并调整折扣**：确保与新商店货币保持一致。
- **更新运费设置**：删除现有运费，然后手动重新添加运费以使用新货币显示运费。
- **为余额未用完的客户发放新的礼品卡**：为了确保客户在使用新货币时能够继续购买商品，建议为余额未用完的客户发放新的礼品卡。

## 管理店铺的默认重量单位

Genstore 支持英制（如磅 lb、盎司 oz）和公制（如千克 kg、克 g）作为默认重量单位。注意：后续您可针对具体产品设置不同的重点单位。

### 步骤：

1. 登录 Genstore 商户后台，导航到 **设置** -> **通用**。
2. 在 **默认设置** 部分，选择 **单位制**（英制或公制），然后使用 **默认重量单位** 下拉菜单选择所需单位。

## 管理店铺时区

1. 登录 Genstore 商户后台，导航到 **设置** -> **通用**。
2. 在 **默认设置** 部分，从 **时区** 列表中选择一个新时区。
3. 点击 **保存**。

## 管理店铺品牌资产

您可以为店铺设置 logo、颜色、社交媒体链接等品牌元素，跨销售渠道、主题和应用共享品牌资产。

### 步骤：

1. 登录 Genstore 商户后台，导航到 **设置** -> **通用**。
2. 在 **品牌资产** 部分，点击 **管理** ，转至 **品牌** 页进行设置，各项的设置说明，请阅读 [设置店铺品牌资产](./start-genstore-account-brand-assets.md)。
