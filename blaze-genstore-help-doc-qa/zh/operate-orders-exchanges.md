# 创建换货

换货是指客户在收到产品后申请更换为其他产品，而非退还产品。在 Genstore 商家后台，您可以创建并管理换货流程。创建换货时，您可以为客户提供换货产品，并根据需要为客户提供支付方式。换货服务不仅能提升客户满意度，还能帮助您保留收入，并通过换货增加额外收益。

:::tip

- 不支持自定义产品添加为换货产品。
- 具有关税的订单无法添加换货产品进行换货，但支持创建退货。
- 订单级别折扣不能应用于换货产品。

:::

## 换货流程

您可以在 Genstore 后台中创建换货。换货时可选择需要退回产品和换货产品。具体步骤如下：

1. 登录 Genstore 后台，点击 **商店** -> **订单**。
2. 选择要换货的 **订单号**，进入 **订单详情**。
3. 点击页面右上角的 **退货**。
4. 在新打开的 **退货** 窗口中，输入要退货的产品数量，并选择 **退货原因**。
5. 点击 **退回手续费** 栏的 编辑 图标，输入需要扣除的退货手续费，点击 **确定**。
6. 点击 **退回运费** 栏的 编辑 图标，输入需要扣除的退货运费，点击 **确定**。
7. 在 **换货** 部分，点击 **添加产品**，选择换货产品并点击 **保存**。
8. 填写退货的物流配送信息，或者选择 **无需物流**；
9. 点击 **创建退货**。

## 换货产品发货

在收到退货产品并完成重新入库后，您可以处理并发货换货产品。换货产品发货后，客户会收到电子邮件通知，并且他们可以在客户账户中查看订单详情和订单状态。操作步骤如下：

1. 登录 Genstore 商家后台，点击 **商店** -> **订单**。
2. 在 **订单列表** 中，选择要发货的换货订单，进入 **订单详情**；
3. 在 **未发货** 部分，按常规订单的发货流程发货换货产品。
4. 根据订单调整情况，进行退款或收款：
   - 如果您欠客户款项，发放退款；
   - 如果客户欠您款项，点击 **获取款项**，选择获取款项的方式。


::: faq
## 常见问题
### Q1. 未支付订单是否支持换货？
> A：未支付订单不支持进行换货业务。
:::
