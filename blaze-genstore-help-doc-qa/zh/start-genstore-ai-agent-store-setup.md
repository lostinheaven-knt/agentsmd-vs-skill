# 开店引导 Agent

开店 Agent 是您的开店配置专家。为了让您的在线商店快速投入运营，Genstore 设计了智能化的 **开业引导流程**。通过与 AI 协作，您可以在 **3 分钟内** 完成开业所需的所有基础设置。

## 核心设置模块

在基础设置阶段，您可以通过与开店 Agent 对话，高效完成以下关键配置：

### 店名设置
**AI 智能生成模式**
- 启动方式：在对话区输入"帮我起个店名"
- AI工作流程：
  - AI 会询问您的业务需求
  - 根据您的输入生成 3-5 个推荐店名
  - 提供每个店名的含义解释和品牌联想
  - 支持多次迭代生成直到满意

**自定义输入模式**
- **操作路径**：在对话区输入"帮我更新为XXX店名"
- **功能特点**：
  - 直接输入您已有的品牌名称
  - 实时检查名称可用性
  - 提供品牌命名建议（可选）

### 域名设置
系统提供灵活的配置方案：

**方式一：AI智能生成**
- AI 基于您的店名和行业自动推荐可用域名
- 提供.genmystore固定后缀
- 自动检查域名是否可用

**方式二：自定义输入**
- 输入您已拥有的域名
- 自动检查域名是否可用
- 提供配置指引

**方式三：连接第三方域名**
- AI 询问您的域名
- 提供针对性的 DNS 配置指引
- 生成分步骤的操作指南
- 自动检测连接状态

### 物流、支付与税务设置
在这些专业领域，开店 Agent 可以担任“配置导航助手”角色：
* **物流设置**：AI 自动调出配置页面，根据您的业务所在地推荐默认物流模板。
* **支付设置**：AI 快速定位支付配置界面，引导您完成收款通道绑定。
* **税务设置**：AI 引导至税务配置页面，确保您的店铺符合当地合规要求。

## 智能开业引导流程

这是为您打造的自动化上线路径，确保不遗漏任何关键环节。

### 流程启动方式
* **手动启动**：点击首页底部推荐区域的“开业引导卡片”。
* **对话指令**：直接对 AI 说“开业引导”。

### 协作执行逻辑
进入快速开业视图后，开店 Agent 将为您预执行以下逻辑：
- 自动生成个性化店名。
- 基于店名预选可用域名。
- 匹配并配置适合的默认物流模板。
- 准备好支付设置的引导界面。

###  配置项确认
您可以对 AI 预配置的内容进行快速审查：
- 确认/重设店名与域名：一键应用 AI 生成的结果，或要求重新生成。
- 物流与支付确认：快速确认默认模板或自行编辑细节。

### 开业奖励
* **开业 Credit 奖励**：正式完成开业引导并上线店铺后，您将获得系统发放的 **Credit** 奖励。
