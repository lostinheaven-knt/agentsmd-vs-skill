# 创建博客文章

撰写博客是一种有效的方式，通过与顾客沟通和推广您的产品来提升品牌影响力和网站流量。以下是在 Genstore 平台上撰写并管理博客的步骤：

## 撰写博客

1. 登录 Genstore 商家后台，点击 **商店** -> **在线商店** -> **博客文章**。
2. 点击页面右上角的 **撰写博客文章**。
3. 在新弹出的窗口中：
   - 设置博客的标题和摘要。
   - 撰写博客正文，支付插入图片、超链接、调整字体、字号、颜色等富文本功能。
4. **搜索引擎优化**
   - **页面标题**：确保不超过 70 个字符，避免搜索结果截断。
   - **元描述**：保持在 200 个字符内，以优化搜索展示。
   - **URL handle**：自定义链接，后续如果需要更改，系统提供重定向选项，使旧地址能自动指向新地址。<!--需要确认-->

继续在编辑界面设置：

- **博客状态**：选择博客的可见性，如 **可见**、**隐藏** 或 **指定时间发布**。
- **博客封面**：上传或选择资源库中的图片作为博客封面。
- **作者**：选择或输入文章的作者。
- **博客集合**：选择希望关联的博客集合，或者新建博客集合进行关联。
- **博客文章标签**：为文章添加标签，便于分类和检索，使用逗号分隔标签。每个标签最多 255 个字符，整个博客最多可添加 1,000 个标签。

完成设置后，点击**保存**。新创建的博客将自动显示在**博客文章**列表中。

## 管理博客

在博客列表页面，您可以管理博客内容：

- **编辑**：点击博客条目后的 **编辑** 按钮，对博客内容进行修改。
- **删除**：点击**删除**以移除博客，注意此操作将同时删除任何关联文章，并且无法恢复。
- **预览**：通过点击**预览**，查看博客展示给访问者的样式，确保内容的准确性和吸引力。
