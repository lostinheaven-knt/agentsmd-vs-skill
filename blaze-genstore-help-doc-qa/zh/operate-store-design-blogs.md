# 博客

博客是在线商店中一个关键的内容营销工具。通过定期发布关于产品、行业洞察和实用信息的文章，博客不仅帮助商家建立与顾客之间的信任和沟通桥梁，还能有效提升网站的搜索引擎排名，吸引更多潜在客户。

## 核心优势

- **提高网站 SEO**：通过针对性关键词优化文章，提升搜索引擎排名。
- **增强品牌形象与权威性**：发布高质量内容，展现专业知识，增加品牌信任度。
- **吸引潜在客户**：分享有价值的内容，吸引并留住目标客户群。
- **增强客户关系**：通过评论和互动，加深与客户的联系。
- **支持产品销售**：利用内容推介产品，间接促进销售。

## 撰写博客

1. 登录 Genstore 商家后台，导航至 **商店** -> **在线商店** -> **博客文章**。
2. 点击页面右上角的 **撰写博客文章**。
3. 在新弹出的窗口中，设置博客标题、摘要、正文、SEO、发布时间、封面图片、作者信息等，然后点击 **保存**。新博客将自动出现在**博客文章**列表中。具体操作，请参阅 [创建博客文章](./operate-store-design-blogs-create.md)。

## 预览博客

博客保存后，您可以实时查看其在在线商店中的展示效果。

### 操作步骤

1. 登录 **Genstore 商家后台**，点击 **商店** -> **在线商店** -> **博客文章**。
2. 在博客文章列表中，点击操作列中的 **预览** 图标，即可打开新窗口，查看其在 **在线商店** 中的展示效果。
3. 在页面底部提示条，您可以查看当前博客的状况（是否对客户可见）、当前主题名称，并选择 **退出预览** 或点击 **设计主题** 进入装修编辑器开始定制页面。

## 博客集合

Genstore 允许您将博客组织为集合，使得相关主题的博客可以轻松归类。

1. 登录 Genstore 管理后台，点击 **商店** -> **在线商店** -> **博客文章**。
2. 点击页面右上角的 **管理博客集合**。
3. 在 **博客集合** 部分，您可以查看现有的博客集合，并执行以下操作：
   - **编辑**：点击博客集合旁边的 **编辑**，更新其内容。
   - **删除**：点击 **删除** 来移除博客集合。注意：删除集合将永久移除该博客及其所有关联的文章，此操作无法恢复。
   - **预览**：点击 **预览** 查看博客在访客端的显示效果，确保内容显示正确。
4. 要创建新的博客集合，点击 **添加博客集合**。在创建页面，您可以：
   - 编辑博客集合标题
   - 优化 SEO 设置，提高搜索引擎可见度
   - 管理评论：选择禁用评论、启用评论，或允许评论但需要审批后才能在线显示。

## 预览博客集合

创建博客集合后，您可以实时查看其在在线商店中的展示效果。

### 操作步骤

1. 登录 **Genstore 商家后台**，点击 **商店** -> **在线商店** -> **博客文章**。
2. 点击 **管理博客集合**。
3. 在博客集合列表中，点击操作列中的 **预览**，即可打开新窗口，查看其在 **在线商店** 中的展示效果。
4. 在页面底部提示条，您可以查看当前主题名称，并选择 **退出预览** 或点击 **设计主题** 进入装修编辑器开始定制页面。

## 管理博客评论

1. 登录 Genstore 管理后台，点击 **商店** -> **在线商店**，导航至 **博客文章** -> **管理评论**。
2. 查看博客集合的评论，可以执行以下操作：
   - 标记为 **垃圾评论**：将不适当或无关的评论标记为垃圾评论。
   - **删除**：删除不再需要的评论。

## 添加菜单导航

为便于用户访问博客内容，您可以在商店导航中添加指向博客页面的链接：

1. 登录 Genstore 商家后台。
2. 点击 **商店** -> **在线商店** -> **菜单**。
3. 点击 **创建导航菜单**，在弹出的窗口中填写导航的标题和 URL Handle。
4. 在 **菜单项** 中，点击 **添加菜单项**。添加名称（如“产品博客”），并关联到相应的博客页面，然后点击 **保存**。
5. 完成设置后，点击 **保存** 以创建并保存您的博客菜单导航。
