# 导入/导出库存

Genstore 支持通过 **CSV 文件** 批量管理库存数据，商家可以快速导入或导出库存信息，以便更新库存、进行数据分析或备份。

- **导入库存**：将本地 CSV 文件中的产品信息（包括库存数量、SKU、价格等）导入到系统，以批量更新库存数据。
- **导出库存**：将系统中的库存信息（如库存数量、变体、产品 ID 等）导出为 CSV 文件，便于备份、分析，或修改后重新导入。

## CSV 文件字段

库存 CSV 文件用于唯一标识 **产品** 及其 **多规格**（如颜色、尺寸）在不同地点的库存信息。该格式与其他 CSV 文件类似，但包含不同的字段。

| 字段名称              | 必填  | 说明                                        |
| ----------------- | --- | ----------------------------------------- |
| **Handle**        | ✓   | 产品的唯一标识，**不能包含空格**，仅支持字母、短划线和数字（用于产品 URL） |
| **Title**         | ✗   | 产品标题（选填，可留空）                              |
| **Option1 Name**  | ✓   | 第一规格名称（如 "Color"），如无规格，应填 "Title"         |
| **Option1 Value** | ✓   | 第一规格值（如 "Black"），如无规格，应填 "Default Title"  |
| **Option2 Name**  | ✗   | 第二规格名称（可留空）                               |
| **Option2 Value** | ✗   | 第二规格值（可留空）                                |
| **Option3 Name**  | ✗   | 第三规格名称（可留空）                               |
| **Option3 Value** | ✗   | 第三规格值（可留空）                                |
| **SKU**           | ✗   | 产品的 SKU（选填，可留空，SKU 可重复）                   |
| **Location**      | ✓   | 该产品所属的地点，每个地点的库存需单独列出                     |

> **多规格产品** 需要使用 **Handle、Option1 Value、Option2 Value、Option3 Value** 组合唯一标识。
>
> **CSV 文件格式要求**：必须采用 **.csv** 格式，并符合上述字段要求。

## 导入库存

导入库存功能允许商家通过 CSV 文件批量更新库存信息，提高库存管理效率。

**操作步骤**

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **库存**。
2. 点击 **导入**。
3. 在新打开的窗口中，点击 **上传文件**，上传库存 CSV 文件。
4. 点击 **上传文件**。

## 导出库存

导出库存支持导出当前页面数据、全部数据或筛选后的数据，以便备份、分析或后续导入，例如以下业务场景：

- 生成库存模板，包含当前地点的唯一产品标识和库存信息。
- 获取库存数据，供其他系统或业务流程使用。

**操作步骤**

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **库存**。
2. 点击 **导出**。
3. 在新打开的窗口中，选择导出范围，点击 **导出库存**。
