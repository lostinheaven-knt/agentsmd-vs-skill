# 页面

在 Genstore 平台中，**页面**是商家用于展示和推广其产品和服务的个性化网页。商家可以通过创建各种类型的页面，如产品详情页、品牌介绍页、客户服务页、常见问题解答（FAQ）页及营销活动专页等，以此：

- **提升品牌形象，增强用户信任**
- **优化用户体验，提高转化率**

## 创建新页面 ##

1. 登录 Genstore 商家后台，点击 **商店** -> **在线商店** -> **页面**。
2. 点击 **创建页面**。
3. 填写**页面标题**。
4. 内容部分支持富文本编辑器，您可以插入图片、超链接、调整字体大小和颜色等。
5. 如需编辑页面的 HTML 代码，请点击 `<>` 进入 HTML 视图并进行修改。
6. **搜索引擎优化**（SEO）
   - **页面标题**：确保不超过 70 个字符，避免搜索结果截断。
   - **元描述**：限制在 165 个字符内，以优化搜索展示。
   - **URL handle**：自定义链接，后续如果需要更改，系统提供重定向选项，使旧地址能自动指向新地址。
7. 设置页面可见性：选择使页面公开、隐藏或在指定时间显示。
8. 点击 **保存** 后，新页面将显示在页面列表中。

## 管理页面 ##

在页面列表页，您可以执行以下管理操作：

- **编辑**：点击页面后的 **编辑** 按钮，可以对页面内容进行修改。
- **删除**：若需移除页面，点击对应页面的 **删除** 按钮。

## 预览页面

页面保存后，您可以实时查看其在在线商店中的展示效果。

1. 登录 **Genstore 商家后台**，点击 **商店** -> **在线商店** -> **页面**。
2. 在页面列表中，点击操作列中的 **预览** 图标，即可打开新窗口，查看该页面在 **在线商店** 中的展示效果。
3. 在页面底部提示条，您可以查看当前主题名称，并选择 **退出预览** 或点击 **设计主题** 进入装修编辑器。
## 添加菜单导航

为便于用户访问这些自定义页面，您可以在商店的导航菜单中添加指向这些页面的链接。以下是添加导航的步骤：

1. 登录 Genstore 商家后台。
2. 点击 **商店** -> **在线商店** -> **菜单**。
3. 点击 **创建导航菜单**，在弹出的窗口中填写导航的标题和 URL Handle。
4. 在 **菜单项** 中，点击 **添加菜单项**，输入菜单项名称并在搜索框中选择相应页面，然后点击 **保存**。
5. 完成所有设置后，再次点击 **保存** 以确认并应用您的导航设置。

::: tip

菜单导航最多支持 3 层。

:::
