# 物流模板设置

Genstore Dropshipping 产品（含 **选品中心** 来源产品及 POD 产品）在 **物流配送** 菜单下拥有专属的物流模板，用于与自有产品的运费模板进行区分与独立配置。
通过该模板，系统会根据代发货产品的履约方式和配送范围，自动生成并管理对应的运费规则。

::: tip
除非另有说明，下文所述 **Dropshipping 产品** 均包含选品中心来源产品及 POD 产品。
:::

## 配置前须知

- **模板创建**：Genstore Dropshipping 物流模板无需手动创建。当您刊登第一个 Dropshipping 产品或POD产品至店铺时，系统会自动创建并初始化该模板配置。
- **模板删除**：该模板不可删除。当您删除所有已刊登的 Genstore Dropshipping 产品后，模板会自动隐藏，不再展示于列表中。

## 编辑运费模板

您可以对 Genstore Dropshipping 运费模版的运费金额进行编辑，但其余配置目前暂不支持修改。
1. 登录 **Genstore** 商户后台， 点击 **设置** -> **发货配送**。
2. 定位到 **Genstore Dropshipping 运费模版**模块。
3. 点击对应运费选项，即可对运费进行添加或调整。

### 模版内容说明

* **发货地点**：发货地点默认为 Genstore Dropshipping 服务履约，暂不支持您修改或新增其他发货点。
* **关联产品**：您无需手动关联 Genstore Dropshipping 或POD产品到该模版，系统会自动将您刊登到店铺中的 Genstore Dropshipping 或POD产品与该模版进行映射。
* **配送国家**：目前暂不支持您手动新增或修改配送国家，系统默认读取您 **市场** 中配置的国家范围，并同步至该模版。
* **运费选项**：您可以为 Genstore Dropshipping 或POD产品设置对应的运费，供客户下单时进行运费结算。运费类型您可以参考[配置物流运费方案](./operate-fulfillment-shipping-profile-configuration.md)。

:::tip

1. 店铺前台展示的运费名称与运费说明不可编辑，系统会自动在运费选项中展示相应的物流时效数据作为运费说明。
2. 不同产品及国家对应的时效信息会有所差异，您可在结算页查看最终展示效果。
3. 当前系统仅提供发往 **美国** 的时效数据，其他国家将陆续支持。
4. 当同一订单中包含多个 Genstore Dropshipping 产品时，系统将以 最长期限 的时效作为整体配送参考。
5. Genstore Dropshipping 运费模版目前只支持配置一条运费，且仅支持运费类型为：固定运费或按订单金额设置运费两种。

:::

## 注意事项

1. **设置的运费与供应商实际配送成本不同**
   - 您在模版中设置的运费为客户在下单时支付的金额，与供应商实际发货时产生的运费成本可能存在差异。
   - 为避免运费设置过低导致成本无法覆盖，建议：
     - 在产品定价中预留运费利润空间
     - 根据定价模块中提供的运费参考成本，适当上浮 10%–20% 以确保覆盖实际配送成本。
2. **Genstore Dropshipping 仅支持物流配送**
   - Genstore Dropshipping 产品为一件代发模式，不支持设置自提或与同城配送等方式。
3. **Genstore Dropshipping POD  产品的物流时效**
   - Genstore Dropshipping POD 产品由 Genstore 合作的第三方 POD 供应商负责产品印刷制作和配送，遂物流时效可能较长。您可以参考 POD 产品给出的预计送达时间（预计送达时间 = 预计履约时间 + 预计运输时间）。
