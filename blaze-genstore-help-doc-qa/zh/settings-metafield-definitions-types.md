# 元字段的数据类型

每种元字段类型都支持特定格式的数据值。根据字段的用途和展示方式，您可以从下列类型中选择合适的数据结构：

- 文本
- 数字
- 日期和时间
- 度量
- 引用
- 其他

## 文本类字段

| 数据类型 | 描述                                           | 示例                   |
| -------- | ---------------------------------------------- | ---------------------- |
| 单行文本 | 支持单行纯文本输入，可设置为固定选项列表       | 添加简短的备注和公告   |
| 多行文本 | 支持多行纯文本输入，可设置长度限制和正则表达式 | 添加发货备注或退货政策 |

## 数字类字段

| 数据类型 | 描述              | 示例                 |
| -------- | ----------------- | -------------------- |
| 整数     | 仅支持输入整数    | 书籍页数、库存数量等 |
| 小数     | 支持输入整数+小数 | 珠宝克拉、评级等     |

## 日期和时间字段

| 数据类型   | 描述                                | 示例                                              |
| ---------- | ----------------------------------- | ------------------------------------------------- |
| 日期       | 支持 ISO 8601 格式（yyyy-mm-dd）    | 产品到期日、客户生日等<br />如：2025-05-28        |
| 日期和时间 | 支持 ISO 8601 日期 + UTC 时间戳格式 | 活动或产品发布时间。<br />如：2022-01-01T12:30:00 |

## 度量类字段（支持公制和英制单位）

| 数据类型 | 描述                                         | 示例                               |
| -------- | -------------------------------------------- | ---------------------------------- |
| 重量     | 整数或小数值，如 kg、g、lb、oz               | 原料重量，如：`500g`               |
| 尺寸     | 整数或小数值，如 mm、cm、m、inch             | 包裹高度，如：`80cm`               |
| 体积     | 整数或小数值，如 ml、l、m<sup>3</sup>、us_qt | 水杯容量，如：`350ml` 或 `1 us_qt` |

## 引用类字段（用于关联商店内其他资源）

| 数据类型 | 描述                                                         | 示例                                         |
| -------- | ------------------------------------------------------------ | -------------------------------------------- |
| 客户     | 引用商店中的客户对象                                         | 在 `客户` 模块中添加元字段，关联特定客户对象 |
| 产品     | 引用其他产品（可用于搭配推荐等）                             | 关联 “搭配购买” 产品                         |
| 产品规格 | 引用其他产品的具体规格（如颜色/尺码）                        | 关联相同颜色的服装款式                       |
| 产品系列 | 引用一个或多个产品系列                                       | 创建 “完整搭配” 推荐区域                     |
| 文件     | 引用媒体库中的文件，支持常用格式（详见 [素材要求](./settings-media#素材要求)） | 上传说明书、合规文档或安装文件               |

## 其他类型字段

| 数据类型      | 描述                                         | 示例                        |
| ------------- | -------------------------------------------- | --------------------------- |
| 颜色          | 支持 `#RRGGBB` 格式的 RGB 值                 | 设置背景主色，如`#FF9900`   |
| 金额          | 数值 + 商店币种，支持小数                    | 成本价、批发价，如 `$10.50` |
| URL           | 支持 `HTTP`、`HTTPS`。可限制为限制为特定域名 | 外链视频、品牌官网链接等    |
| true 或 false | 布尔值（是否开启某选项）                     | 是否支持刻字：`true`        |

::: tip 

- 每种字段类型支持的输入结构、交互方式与验证设置均不同。
- 请结合字段用途、前台展示需求以及管理规范合理选择字段类型。

:::
