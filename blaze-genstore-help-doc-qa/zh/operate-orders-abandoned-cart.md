# 弃单

弃单是指客户在结算页填写了邮箱，但未继续操作且在 10 分钟内未完成订单，该行为将被视为弃单。

## **弃单规则**

1. 用户必须于 **结帐页** 填写了邮箱，才会被收集为弃单；
2. 弃单数据保留 3 个月，逾期未召回成功或者未进行召回则系统自动删除；
3. 结帐页金额为 $0.00 的不会被收集为弃单；
4. 使用 Genstore Payments 时
   - 当支付失败原因被判定为 `卡被拒付` 或 `卡余额不足` 时，系统仍会生成订单，但不会计入弃单；若为其他支付失败原因，则仅收集为弃单，不会生成订单，也不会显示在订单列表中。
5. 使用第三方支付网关时
   - 当支付失败原因被判定为 `卡被拒付` 或 `卡余额不足` 时，系统仍会生成订单，但不会计入弃单；若为其他支付失败原因，则仅收集为弃单，不会生成订单，也不会显示在订单列表中。

## 查看弃单

您可以查看弃单记录，了解客户的具体弃单详情。具体操作如下：

1. 登录 Genstore 商家后台，点击 **商店** -> **订单** -> **弃单**。
2. 在弃单页面的 **弃单列表** 区域，选择要查看的弃单，点击 **订单号** 进入到 **弃单详情** 查看弃单信息。

## 自动发送弃单结账链接

您可以在客户放弃结账后自动收集相关交易信息，并通过电子邮件发送恢复结账链接。每封邮件都将包含客户弃单订单的链接，方便客户直接完成结账。具体操作如下：

1. 登录 Genstore 商家后台，点击 **营销** -> **自动化营销**。
1. 点击页面右上角的 **创建自动化营销**。
2. 选择 **弃单** 自动化模板。
3. 点击 **创建任务**。

启动新自动化后，您可以编辑和管理自动化，包括将其关闭和启动。

## 手动分享弃单结账链接

您可以手工复制弃单的结帐链接，并将结帐链接发送给到客户进行支付结帐。具体步骤如下：

1. 登录 Genstore 商家后台，点击 **商店** -> **订单** -> **弃单**。
2. 点击 **订单号** 进入弃单详情页，点击 **复制结帐 URL**。
3. 在 **复制结帐 URL** 弹框中，点击 **复制链接** 按钮。
4. 将复制的链接发送给客户，方便客户完成支付结账。客户可以在结账页链接中编辑结账信息并完成支付。

## **查看自动化流程及召回发送状态**

您可以在 **弃单详情** 中查看与该弃单相关的营销自动化流程及其召回状态。操作步骤如下：

1. 登录 Genstore 商家后台，点击 **商店** -> **订单** -> **弃单**。
2. 进入 **弃单详情**，找到 **自动化营销** 模块，查看已关联的自动化流程。
3. 在 **召回通知发送状态** 中，了解邮件是否已成功发送。
4. 若您尚未开通营销自动化服务，或已开通但尚未创建自动召回流程，系统将提示“自动化流程未启用，请前往开启”。请前往开启服务并配置 [营销自动化的弃单召回流程](./expand-marketing-automated-email-abandoned-checkout.md)。

## 弃单恢复

当客户通过电子邮件中的恢复链接或自行完成结账支付时，弃单将被视为已恢复。
