# 关闭店铺

作为 **店主**，且您的商店符合关闭条件，您可以随时通过 Genstore 商家后台取消订阅并关闭您的店铺。

关闭店铺前，请务必阅读并了解 [服务条款](https://www.genstore.ai/legal/terms)，以确保您清楚相关影响。


## 取消订阅并关闭店铺

::: tip

关闭店铺将立即取消订阅，您将无法继续访问商家后台。若需重新启用店铺，您需要重新订阅套餐进行激活。

:::

### 操作步骤

1. 登录 Genstore 商家后台。
2. 点击 **设置 -> 套餐和积分**。
3. 在 **订阅** 区域点击 **关闭店铺**。
4. 在弹窗中选择关闭原因，并填写补充说明（可选）。
5. 点击 **继续**，确认关闭。

店铺关闭后，您仍可访问并下载历史账单。您的销售、产品和客户数据将在两年内保留，便于您随时重新启用店铺。

## 重新激活店铺

若您决定重新运营您的店铺，可以随时登录账户，选择适合的套餐重新激活。

### 操作步骤

1. 登录 Genstore 商家后台，选择已关闭的店铺。
2. 点击 **继续订阅** 或 **更换套餐**。
3. 完成支付后，店铺将自动恢复为活跃状态。

## 闭店注意事项

### 账户与访问权限

| 项目     | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| 店铺后台 |店铺标记为“停用/Inactive”，无法正常访问店铺后台               |
| 关联账号 | 不影响账号登录，可用于重新激活                               |
| 数据保留 | 所有数据保留 2 年，超期后将自动清除                          |
| 消费者访问体验 | 店铺显示“已关闭”提示，建议引导注册新店或联系客服             |
| 域名处理 | 默认及修改后的 `genmystore.com` 域名不可复用 <br />对于绑定的第三方域名，建议提前移除自定义域名绑定 <br /> 路径：**商户后台 -> 设置 -> 域名** |

::: tip

**唯一域名原则**

创建商店时，系统会为您分配一个默认的 `xxx.genmystore.com` 域名，您也可以在后台对其进行自定义修改一次。但无论是系统分配的默认域名，还是您后续自定义修改的域名，**一旦店铺关闭，该域名都将无法再次使用**。

例如：如果您关闭了名为 **Very Good Things** 的商店，其域名为 `very-good-things.genmystore.com`，即使您重新创建一个同名品牌的新店，**也无法使用该域名**。

:::

### 支付与财务

#### 使用 **Genstore Payments** 的商户

| 项目     | 限制说明                                 |
| -------- | ---------------------------------------- |
| 新支付   | 关闭后买家无法再发起新支付               |
| 资金提现 | 待处理的支付将会正常处理，并根据关店前的商户配置付款到商户银行账户       |
| 退款操作 | 无法在后台发起退款或手动捕获             |
| 拒付争议 | 无法提交抗辩，**所有拒付请求将自动接受** |

#### 使用**第三方支付**的商户

| 项目     | 限制说明                                                     |
| -------- | ------------------------------------------------------------ |
| 新支付   | 关闭后无法再发起新支付                                       |
| 售后操作 | 无法在 Genstore 后台处理退款或拒付，**需前往第三方支付平台自行操作** |

#### 使用**手动付款**的商户

| 项目     | 限制说明                               |
| -------- | -------------------------------------- |
| 新订单   | 关闭后将无法继续使用手动付款方式       |
| 售后操作 | 退款、退货等流程不可执行（同订单限制）|

### 订单与发货

| 状态         | 限制                               |
| ------------ | ---------------------------------- |
| 未完结订单   | 无法继续执行支付、发货、导出等操作 |
| 售后流程     | 无法处理退款、退货、换货等流程     |
| 已购未发面单 | 建议及时发货或取消，避免资源浪费   |
| 已发货订单   | 物流仍更新，但售后不可处理         |

### 营销与销售渠道

| 项目       | 状态                                 |
| ---------- | ------------------------------------ |
| 邮件营销   | 所有尚未进入发送通道的邮件任务将暂停并转为草稿状态。已进入发送流程的邮件将正常发送并计费。           |
| 自动化流程 | 所有流程暂停               |
| 社交帖子   | 计划中的社交发帖取消，转为草稿       |
| 弹窗活动   | 正在运行的活动全部下线               |
| SEO 功能   | 扫描、分析及 robots.txt 规则全部停用 |
| 渠道授权   | 所有销售渠道授权断开，产品停止同步   |
