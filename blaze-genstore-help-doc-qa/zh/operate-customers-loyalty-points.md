# 积分计划

积分计划是 **Genstore 客户忠诚度应用**的核心功能之一，商家可以通过多种积分获取方式（如下单、注册、分享社交媒体等）来激励客户的互动与购买。客户积累的积分可以用于兑换多种奖励，包括折扣、免邮和免费礼品等，增强客户在店铺中的参与感与回头率。

## 启用积分计划

1. 进入路径：**登录 Genstore 后台** -> **商店** -> **客户** -> **忠诚度**。
2. 在 **积分** 页签，您可通过顶部状态栏的状态切换按钮，打开或者关闭积分计划。
3. 此外，您还可以自定义积分展示名称，点击 **设置** 页签 -> **更多设置** ，在 **通用** 部分的**积分标识**，自定义积分符号。

启用积分计划后，您可以设置用户的积分获取方式（线上或社交媒体互动）、积分兑换方式以及积分有效期，您可点击下方页面获取更多信息：

- [获取积分](./operate-customers-loyalty-points-earnings.md)
- [积分兑换权益](./operate-customers-loyalty-points-redemption.md)
- [积分有效期](./operate-customers-loyalty-points-expiration.md)
