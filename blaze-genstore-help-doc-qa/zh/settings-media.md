# 上传和管理素材

在 Genstore 后台的 **素材** 部分，您可以上传、管理和删除店铺素材，包括：

- 模板、博客文章和页面中使用的图片与视频
- 产品和产品系列的图片与视频
- 供客户下载的文件与视频

在编辑模板、页面、博客文章、产品描述或产品系列描述时，您也可以上传素材。您上传的素材将显示在 **素材** 页面中。

若要访问 **素材** 页面，请转到 **设置** -> **素材**。

## 素材要求

上传素材时，确保其符合以下要求：

### 图片要求

| 属性  | 要求                             |
| --- | ------------------------------ |
| 大小  | 最大 20 MB                       |
| 格式  | PNG、JPEG、JPG、GIF、WEBP、BMP、HEIC |

### 视频要求

| 属性  | 要求               |
| --- | ---------------- |
| 大小  | 最大 400 MB        |
| 时长  | 最长 3 分钟          |
| 格式  | MOV、AVI、MP4、WEBM |

:::tip

这些限制不适用于嵌入式 Vimeo 或 YouTube 链接，因此，如希望想分享大于 400 MB 或超过 3 分钟的视频，您可以使用 [YouTube](https://www.youtube.com/) 或 [Vimeo](https://vimeo.com/) 的视频托管服务，然后将视频的分享链接添加到您的内容中。

:::

### 音频要求

| 属性  | 要求          |
| --- | ----------- |
| 大小  | 最大 20 MB    |
| 格式  | MP3、AAC、WMA |

### 其他文件要求

| 属性  | 要求                                                                               |
| --- | -------------------------------------------------------------------------------- |
| 大小  | 最大 20 MB                                                                         |
| 格式  | CSS、RTF、TXT、JSON、CSV、XLS、XLSX、DOC、DOCX、DOTX、POTX、PPSX、PPT、PPSX、PDF、XMIND、ZIP、RAR |

## 在素材页面上传素材

您可以在 **素材** 页面上传素材并将其用于店铺中的各种内容，包括将其用作品牌 logo、产品图片等。

### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **素材**。
2. 点击 **上传**。
3. 最多选择 10 个文件进行上传，然后点击 **打开**。

## 编辑素材

您可以在 **素材** 页面中点击素材缩略图进行预览与编辑。编辑会同步应用到所有使用该素材的地方。

## 删除素材

您可以从 **素材** 页面删除一个或多个素材。

### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **素材**。
2. 选择要删除的素材旁边的复选框。您可以选择多个文件。
3. 点击 **删除**，然后再次点击 **删除** 进行确认。

## 编辑素材但不覆盖现有素材

您可以对图像进行裁剪等编辑操作。如果希望保存更改但不覆盖原始素材，请另存为新素材。

### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **素材**。
2. 选择目标素材，打开素材编辑器。
3. 进行素材编辑。
4. 选择 **另存为**。

另存为将新素材保存到商店的素材库中，同时保留原始图片的原始状态。

## 套餐的素材存储限制

| 套餐       | 每家商店的限制 |
| -------- | ------- |
| Genstore Lite 套餐    | 100 GB  |
| Genstore Growth 套餐 | 300 GB  |
| Genstore Scale 套餐 | 300 GB  |
