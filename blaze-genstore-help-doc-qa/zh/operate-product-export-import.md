# 导入/导出产品

Genstore 平台提供了产品的导入/导出功能，使商家能够高效地管理大量产品数据。通过导出，您可以批量提取产品信息，进行备份、分析或与其他系统集成；而通过导入功能，您可以快速批量上传新产品或更新现有产品信息，极大地简化了操作流程。

## 本地导入产品

### 准备导入文件

下载并准备好您的 CSV 文件，操作步骤如下：

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 点击产品列表上方的 **导入** 按钮。
3. 在 **导入产品** 弹框中，下载导入模板。

根据模板字段设置产品信息，请先确认您的产品属于统一规格还是多规格产品，**产品规格** 是指同一款产品的不同选项或变体，如尺寸、颜色、型号等。例如，一件 T 恤可能有不同的尺寸（S、M、L）和颜色（红色、蓝色、绿色）。每个规格都对应一个独立的 SKU 和库存信息：

- **统一规格产品**：只有一个标准版本，所有产品信息在同一行填写。
- **多规格产品**：具有多个选项（如不同尺寸或颜色），需要在第一行输入产品的基本信息和第一张图片的 URL，后续行填写不同规格的详细信息，可跳过不需要填写的列（如 Title、Body、Vendor 和 Tags）。

**导入模板各列说明**

| 列                                          | 描述                                                         |
|:------------------------------------------ |:----------------------------------------------------------- |
| Handle                                      | 产品的 URL 标识（handle）字段。每个产品的标识（handle）唯一，多规格时标识（handle）相同 |
| Title                                       | 产品标题，不能超过 255 个字符数                              |
| subTitle                                    | 产品特色，不能超过 255 个字符数                            |
| Body (HTML)                                 | 产品描述，不能超过 255 个字符数                              |
| Product Type                                | 产品类型：<BR/>- normal：实物产品<BR/>- virtual：虚拟产品    |
| Vendor                                      | 产品关联厂商                                                 |
| Product Category                            | 产品类别（可以到任何一级），没有填写默认 **未分类**，例如 家居与园艺 -> 床上用品 -> 床品 -> 床单，可使用类别 ID，例如 hg-15-1-2 |
| Tags                                        | 产品标签，多个标签用英文逗号分隔                             |
| collections                                 | 产品系列，多个系列用英文逗号分隔                             |
| Images                                      | 素材中心图片 URL 地址，**注意**：可添加多张图片 URL，每张图片 URL 占一行 |
| Image Position                              | 图片展示顺序，适用于添加多张图片的场景，主图为 1，顺序必须是连续整数，不支持跳跃。没有填写默认为 1 |
| Image Alt Text                              | 图片替换文本，不能超过 300 字符                              |
| Multiple variants                           | 是否是多规格<BR/>- 多规格：True<BR/>- 统一规格：False        |
| Option1 Name                                | 规格项名称 1，例如：颜色、尺寸等。|
| Option1 Value                               | 规格值 1，例如：黑色、L 号等。|
| Option2 Name                                | 规格项名称 2，例如：颜色、尺寸等。|
| Option2 Value                               | 规格值 2，例如：黑色、L 号等。|
| Option3 Name                                | 规格项名称 3，例如：颜色、尺寸等。|
| Option3 Value                               | 规格值 3，例如：黑色、L 号等。|
| Variant SKU                                 | SKU 代码。|
| Variant Barcode                             | SKU 的条形码（如 UPN）。|
| Variant weight                              | SKU 的重量，单位可以是 g、kg、oz 或 lb。|
| Variant Inventory Tracker                   | 是否需要库存跟踪，填写 "TRUE"（跟踪库存）或 "FALSE（不需要跟踪库存）。|

| Variant Price                               | SKU 的价格，不带货币符号，如未设置值，则默认为 0.0，例如 9.99 |
| Variant Compare At Price                    | SKU 的原价。|
| Variant Cost per item                       | SKU 的成本价。|
| Variant Image                               | SKU 规格的图片 URL。|
| Variant Weight Unit                         | SKU 规格的重量单位                                           |
| Status                                      | 产品状态<BR/>draft：草稿<BR/>active：上架<BR/>               |
| SEO Title                                   | 产品的 SEO 标题。|
| SEO Description                             | 产品的 SEO 描述。|
| 产品元字段名称（product.metafields.custom）| 自定义产品元数据字段。<!--这个列名需要确认下-->              |

### 导入产品

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 点击产品列表上方的 **导入** 按钮。
3. 上传准备好的 CSV 文件，点击 **上传文件**。

## 从 Shopify 导入产品

Genstore 支持您将 Shopify 店铺中导出的产品 CSV 文件直接导入产品模块，无需额外修改，轻松实现客户数据的批量迁移与整合。

### 准备导入文件

1. 登录 Shopify 后台，在产品菜单中批量选择需要产品，点击导出，生成 CSV 文件。
2. 如需调整产品信息，可在导出的 CSV 文件中进行编辑并保存。
3. 登录 Genstore 后台，点击 **商店** -> **产品**，点击 **导入**。选择 **从 Shopify 导入**，上传文件开始导入。
4. 导入可能需要一段时间，您可以离开页面，稍后通过页面左下角的个人中心 -> **任务中心** 查看导入进度。

## 导出产品

产品导出功能使商家能够批量管理和更新产品信息，导出的数据可用作备份，也有助于分析销售和库存情况，支持更精确的决策。此外，它还便于跨平台操作，实现与其他系统的数据同步，简化大规模产品管理，提高整体运营效率。

通过产品列表选择要导出的产品数据范围，导出格式为 CSV 文件。支持以下导出选项：

- 当前页面导出
- 全部产品数据导出
- 选择产品导出
- 通过筛选条件导出

**操作步骤**

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 点击产品列表上方的 **导出** 按钮。
3. 在对话框中选择要导出的产品范围。
4. 点击 **导出**。
