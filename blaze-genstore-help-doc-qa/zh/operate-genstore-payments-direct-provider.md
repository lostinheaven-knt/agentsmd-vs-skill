# 配置直连收银台

当您启用直连收银台支付渠道时，消费者在结账页提交支付时，可以直接在您的商店结账页输入信用卡信息，快速完成支付。

## Stripe 支付配置手册

**前提条件**：您必须拥有一个经过验证的 Stripe 账户。

**操作步骤**

1. 登录您的 Genstore 管理员账号，进入 **设置** -> **支付**。
2. 点击 Stripe 后的 **连接** 按钮。
3. 页面将跳转到 Stripe 登录界面。根据 Stripe 提示完成登录并授权。
4. 登录成功后，页面会重定向回 Genstore 商户后台，表示授权已成功。
 
### 绑定常见问题

#### 关于“Stripe 收款功能未开启”的提示

如果您的 Stripe 账户未启用收款能力，在 Genstore 绑定 Stripe 账户时，可能会出现此类错误提示。收款能力是指将资金从其他账户转入您 Stripe 账户的能力。

此类错误需要您联系 Stripe 客户经理，并提供您的 Stripe 账号，以了解具体的限制原因。

#### 关于“Stripe 付款功能未开启”的提示

如果您的 Stripe 账户未启用付款能力，在 Genstore 绑定 Stripe 账户时，可能会出现此类错误提示。付款能力是指将资金从您的 Stripe 账户提现到您的银行账户的能力。

此类错误需要您联系 Stripe 客户经理，并提供您的 Stripe 账号，以了解具体的限制原因。

#### 关于“Stripe 接受卡支付功能未开启”的提示

如果您的 Stripe 账户未启用接受卡支付的能力，在 Genstore 绑定 Stripe 账户时，可能会出现此类错误提示。接受卡支付能力是指让您的商户接受消费者使用信用卡或借记卡进行支付的能力。

建议您登录 Stripe 后台，查看是否已启用相关支付方式。或者，您也可以联系 Stripe 客户经理以获取更多详情。

## PingPong 支付配置手册

**前提条件**：您需要在 PingPong 支付平台注册账户并完成店铺网站审核。

**操作步骤**

1. 登录您的 Genstore 管理员账号，进入 **设置** -> **支付**。
2. 点击 **更多支付方式** 下的 **添加支付方式** 按钮。`
3. 在 **第三方支付提供商** 页签，选择 PingPong Pay 支付。
4. 进入配置页面，填写网站 ID、ClientId、秘钥，选择需要展示的卡组图片。
5. 点击 **保存**。

### 获取 PingPong Pay 账户托管参数步骤

**获取网站 ID 和秘钥信息**

1. 登录您的 PingPong 管理员账号，进入 **系统管理** -> **秘钥管理**。
2. 在网站列表中找到您的店铺域名，获取对应的网站 ID 和秘钥详情。

**获取 Client ID 信息**

1. 登录您的 PingPong 管理员账号，进入 **账户中心**。
2. 在账户详情中，获取对应的 Client ID。

### 消费者体验

配置成功后，您可以进入您的商店，选择产品并体验支付流程。

#### 结账页

在结账页，您可以看到 **信用卡** 支付方式已经激活，选择该方式，输入信用卡信息（三要素），点击 **立即支付** 即可支付成功。
