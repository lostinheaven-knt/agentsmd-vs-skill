# 店铺迁移

Genstore 作为一款强大的电商建站平台，致力于为商家提供一站式的开店解决方案。为了帮助您顺利将现有业务迁移至 Genstore，我们推出了免费的 **Genstore 迁移助手**。

该工具可帮助您 **快速、安全、完整** 地将其他电商平台的店铺数据迁移到 Genstore，或在不同的 Genstore 店铺之间克隆数据，最大程度减少迁移对业务运营和广告投放的影响。

## 店铺迁移流程

1. 绑定待迁移店铺
2. 选择迁移模块
3. 等待数据迁移
4. 检查迁移结果

## 安装 Genstore 迁移助手

您可通过 Genstore 应用商店或 Genstore 商户后台安装 Genstore 迁移助手。

**Genstore 应用市场**

1. 访问 **[Genstore 应用市场](https://apps.genstore.ai/zh-CN/)**。如果您管理多个 Genstore 商店，请点击选择您希望安装迁移助手的商店。
2. 搜索 **迁移助手**（或在商户后台的 **应用** 菜单搜索 **迁移助手**）。
3. 在应用页，点击 **安装** 按钮。
4. 在 Genstore 商户后台确认应用访问权限，点击 **安装** 完成授权。

安装完成后，您可在 **Genstore 商户后台 → 应用 → 已安装** 页面找到 **Genstore 迁移助手（Migration Tool）**。

## 查看迁移记录

完成店铺迁移后，您可在迁移助手应用内查看所有迁移历史记录，包括：

- **迁移时间**：任务发起时间。
- **迁移平台**：迁移平台。
- **店铺域名**：显示您迁移的店铺域名。
- **迁移模块**：列出本次迁移涉及的具体模块，如店铺、客户、产品、订单等。
- **迁移数据结果**：迁移成功和异常的数据量。
- **操作**：可点击 **查看异常详情** 按钮获取具体异常信息，或导出异常数据，便于后续核对与排查。
