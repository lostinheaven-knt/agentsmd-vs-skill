# 手动创建产品系列

手动创建产品系列允许商家将特定的产品手动添加到系列中，适用于需要完全掌控系列内容的情况，例如精选产品或特别策划的产品系列。

- 由商家手动将特定产品添加到系列中。
- 适用于需要完全掌控产品系列的情况，例如某些精选产品或特别策划的产品系列。

## 创建手动产品系列

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **系列**。
2. 点击页面右上角的 **创建产品系列**。
3. 输入系列的 **标题** 和 **描述**。
4. 在 **类型** 部分， 点击 **手动**，进一步设定产品。
5. 您可以通过搜索或筛选选择要加入系列的产品，选择添加支持批量删除。
   - 您可按 **销量**、**字母顺序**、**价格高低**、**创建日期**，或者 **手动拖动排序**。
6. 搜索引擎优化设置：
   - **页面标题**：默认使用系列标题，支持修改，字符长度建议 0-70 个字符。
   - **元描述**：默认使用系列描述，支持修改，字符长度建议 0-200 个字符。
   - **URL handle**：默认生成，最大 255 个字符，支持修改，转换规则如下：
	 - 删除标点符号，特殊字符
	 - 连续空格/下划线转为 `-`
	 - 连续 `-` 替换为单个 `-`
	 - 大小转小写
1. 产品系列会默认在在线商店上线。
2. 在 **产品系列封面图** 部分，您可添加当前系列的封面图，支持从资源库选择、通过链接上传或本地上传。有关可使用的图片尺寸的信息。
3. 点击 **保存** 完成当前产品系列的创建。

## 在产品页面关联产品系列

您也可在 [发布产品](./operate-product-create#产品系列) 或者产品详情页将某个产品添加到产品系列中。

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 在产品列表中，点击希望添加到产品系列中的产品，点击 **操作** 列的 **编辑**。
3. 在产品详情页的右侧 **系列** 部分，选择一个或多个现有产品系列。
4. 点击 **保存**，当前产品将被关联到所选的产品系列中。

## 管理产品系列

所有创建完成的产品系列都将自动出现在 **产品系列** 页的系列列表中，您可在此执行以下操作：

1. **编辑系列**：点击 **操作** 栏的 **编辑**，编辑更新相关信息如：标题、描述、图片、筛选条件、SEO 设置。
2. **预览系列**：点击 **操作** 栏的 **预览**，预览当前产品系列所包含的产品信息。
3. **复制系列**：点击 **操作** 栏的 **复制**，快速基于当前产品系列的设置复制一个新的产品系列，稍后，您可在系列详情页或点击列表中的编辑按钮进一步调整。
4. **删除系列**：点击 **操作** 栏的 **删除**，删除当前产品系列，您可通过勾选产品系列前的复选框，执行批量删除操作。
