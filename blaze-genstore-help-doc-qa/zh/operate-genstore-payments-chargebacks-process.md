# 拒付流程和纠纷处理

拒付（Chargeback）是指买家（持卡人）因对交易存在疑问，向发卡银行申请撤销已完成的付款，导致资金从您的账户中被扣回的过程。拒付通常发生在买家对交易的真实性、产品质量或服务体验存在疑问时。

## 拒付流程 

典型的拒付处理流程如下：

1. **买家提出争议**：持卡人向发卡银行提出信用卡交易争议。
2. **商户接收拒付通知**：发卡银行向信用卡公司发送拒付请求，信用卡公司可能通过Genstore Payment 要求商户补充交易证明材料。您可以在商户后台查看相关通知，并在规定期限内提交材料。
3. **资金暂扣**：Genstore Payment 收到通知后，将从您的账户余额或待结算资金中扣除争议金额以及相关费用。
4. **商户响应**：商户可选择接受拒付结果，或在规定期限内提供证据发起抗辩。
5. **抗辩审核**：信用卡公司审核抗辩材料，通常最长需 75 天。
6. **最终裁决**：
    - 如果商户胜出，争议金额（扣除手续费）返还商户账户。
    - 如果买家胜出，争议金额退还买家。

具体政策将会视乎信用卡公司的差异和交易各方所在的国家和地区的不同而可能有区别。

:::tip
某些情形下，如已退款订单，Genstore Payment 会自动为您发起拒付抗辩，无需您进行额外操作。
:::

## 查看纠纷

您可以通过 Genstore Payment 后台菜单 **财务 -> 支付 -> 全部** 查看交易明细状态，并进入 **纠纷** 页面管理相关纠纷。

纠纷分为三个类别：
1. **NOC（Notice of Chargeback，拒付通知）**：买家对交易真实性、未收到产品或金额有异议，可以向发卡行提出拒付请求。发卡行在审核后会向商户发送拒付通知（NOC），要求商户处理该笔交易。
2. **NOF（Notice of Fraud，欺诈通知）**：当买家声称信用卡账户存在欺诈行为（如盗刷、伪卡交易等），发卡行会向商户发送欺诈通知（NOF），要求商户配合调查并提供相关交易记录。
3. **ROI（Request of Information，信息请求）**：当发卡行对某笔交易存在疑问，或需要进一步了解交易细节时，会向商户发送信息请求（ROI），要求商户提供相关交易记录和证据。
### 纠纷明细表格字段说明

| **字段名**   | **说明**                                               |
| --------- | ---------------------------------------------------- |
| **支付参考号** | 交易的唯一标识符                                             |
| **纠纷参考号** | 该笔纠纷的唯一标识符                                           |
| **原因代码**  | 卡组网络返回的原因代码                                          |
| **支付时间**  | 原始交易完成时间                                             |
| **纠纷时间**  | Genstore Payment 收到纠纷通知的时间                           |
| **拒付金额**  | 被争议的交易金额                                             |
| **支付方式**  | 原始交易使用的支付方式                                          |
| **状态**    | 抗辩状态：<br/>-  `open`（可抗辩）<br/>- `closed`（不可抗辩或已完成）    |
| **抗辩结束于** | 抗辩的截止日期，超过此日期自动判定失败；“-”表示不可抗辩                        |
| **最新事件**  | 拒付的最新状态                                              |
| **操作列**   | - 可抗辩纠纷显示`处理`<br/>- 不可抗辩或已处理的显示`查看`<br/>- 信息请求显示`回应` |

## 接受和抗辩拒付

对于可抗辩的纠纷，商户可选择：

- **接受拒付**：同意拒付结果，放弃申诉，资金返还买家。
- **抗辩拒付**：在指定期限内（通常 14~40 天）提供证明材料进行申诉，流程较长，可能产生额外费用。

:::tip
通常情况下，我们建议您仅针对于有充足证据的、合法的、且金额可观的交易发起抗辩。
:::

如您希望进行抗辩，建议尽可能充分的收集证明材料，包括但不限于：
- 与买家的历史交易记录
- 发货证明（如物流信息）
- 与买家沟通的证据（如邮件或聊天记录）
