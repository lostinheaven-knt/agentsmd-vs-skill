# 导入/导出客户

有效管理客户数据对企业快速响应市场变化至关重要。通过 Genstore 的客户导入功能，您可以批量添加客户信息，快速构建和更新客户数据库；而客户导出功能则支持将客户数据安全备份和用于分析，助力营销策略和业务决策。

## 本地导入客户

本地导入功能使您能够上传本地客户文件批量添加客户到系统，提高数据处理效率并减少错误。为了顺利导入数据，建议您下载并使用 **导入模板** 编辑客户资料。

### 准备导入文件

操作步骤如下：

1. 登录 Genstore 商户后台，导航至 **商店 -> 客户**。
2. 点击 **导入**按钮，在弹出的窗口中，点击 **下载导入模板**。

### 执行导入

1. 在 **客户** 页面点击 **导入**。
2. 点击 **添加文件** 选择您的 CSV 文件。注意：仅支持 CSV 格式，文件大小不超过 50MB。
3. 勾选 **邮箱一致时将覆盖现有客户** 以确认当邮箱相同时是否自动覆盖现有数据。
4. 点击 **导入客户**。注意：若上一个导入任务未完成，则不允许继续导入其他客户资料。

导入可能需要一段时间，您可以离开页面并稍后通过点击页面左下角的个人中心，选择 **任务中心** 查看导入进度。

## 从 Shopify 导入

Genstore 支持您将 Shopify 店铺中导出的客户 CSV 文件直接导入客户管理模块，无需额外修改，轻松实现客户数据的批量迁移与整合。

### 准备导入文件

1. 登录 Shopify 后台，在客户菜单中批量选择需要迁移的客户，点击导出，生成 CSV 文件。
2. 如需调整客户信息，可在导出的 CSV 文件中进行编辑并保存。
3. 登录 Genstore 后台，点击 **商店** -> **客户**，点击 **导入**。选择 **Shopify 导入**，上传文件并点击 **导入客户**。
4. 如需使用模版导入，可在 **Shopify 导入** 页签，点击 **下载 Shopify 客户模版**，填写客户信息后上传并导入。
5. 导入可能需要一段时间，您可以离开页面，稍后通过页面左下角的个人中心进入 **任务中心** 查看导入进度。

## 导出客户

导出功能允许您将客户信息批量保存至外部文件，便于数据分析和备份。

### 操作步骤

1. 登录 Genstore 商户后台，导航至 **商店** -> **客户**。
1. 点击 **导出**。
2. 选择导出范围：
   - **匹配筛选条件导出**：按当前页面输入的筛选条件筛选导出客户，您可根据需要调整需要导出的字段。
   - **导出所有**：导出店铺内的所有客户信息。
3. 点击 **导出客户** 开始导出过程。
4. 完成后，您可以直接下载文件或在 **任务中心** 中查看并下载。
