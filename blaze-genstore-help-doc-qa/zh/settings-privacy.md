# 设置客户隐私

作为 Genstore 商家，确保客户隐私保护并遵守适用的隐私法律至关重要。为帮助您有效管理隐私设置、确保合规性与透明度，您可以在 Genstore 后台自定义 **客户隐私** 设置。注意：由于隐私法和法规因地区而异，因此，您可以通过 **客户隐私** 设针对不同国家和地区自定义隐私设置。

:::warning

此部分内容仅为商家提供有关隐私合规性的一般信息，旨在提供参考，并不构成法律建议。

商家应对自身的合规性负责，并应咨询法律顾问，以确定所在地区及产品销售地区的相关法律对业务的影响。为使用 Genstore 平台进行销售，您必须遵守适用的法律法规、Genstore 服务条款及任何其他相关政策。

:::

## 创建和管理隐私政策

隐私政策是一种法律文件，用于详细说明企业如何收集、使用、存储及保护其客户或网站访客的个人信息。

设置新店铺时，系统会默认启用 Genstore 建议的隐私政策文本。但您可以根据需要修改或手动草拟隐私政策。

### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **客户隐私**。
2. 点击 **隐私政策**。
3. 修改或手动草拟隐私政策。
4. 点击 **保存**。

## 添加 Cookie 横幅

Cookie 横幅是网站上用于告知访客数据收集及追踪活动的通知，并请求其同意。

在设置新店铺时，系统会默认启用 Cookie 横幅，并自动生成 Genstore 建议的 Cookie 横幅内容及适用地区。您可以根据需求修改地区设置或手动草拟 Cookie 横幅内容。

### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **客户隐私**。
2. 点击 **Cookie 横幅**。
3. 在 **区域和内容** 部分，进行设置：
   - 点击 **地区** 部分中的 **编辑**，选择 Cookie 横幅应显示的地区，然后点击 **保存**。
   - 点击 **内容** 部分中的 **编辑**，执行修改，然后点击 **保存**。
4. 调整 Cookie 横幅的外观（包括颜色和偏好设置），使其适应商店的品牌营销内容，然后点击 **保存**。
5. 在 **位置** 部分，选择您的横幅位置，然后点击**保存**。
6. 可选：点击 **预览**，以预览您商店中的 Cookie 横幅。

:::tip

若您需关闭 Cookie 横幅，在 **横幅状态** 栏点击开关即可完成操作。关闭后，您的现有配置会保留，后续可随时重新开启开关，恢复 Cookie 横幅的显示。

需特别注意：一旦关闭，Cookie 横幅将在您店铺的任何场景下均不再出现，请您谨慎操作。

:::

## 添加数据销售选择退出页面

数据销售选择退出页面是一个专门页面，允许访客选择退出向第三方销售或共享其个人数据，帮助客户掌控其信息的使用与分发。当全球隐私控制（GPC）标头存在时，该功能会自动激活。

设置新商店时，系统将默认生成添加数据销售选择退出页面。您可以修改或手动草拟数据销售选择退出页面。

### 步骤：

1. 在 Genstore 后台中，转至 **设置** -> **客户隐私**
2. 点击 **数据销售选择退出页面**。
3. 在 **区域和内容** 部分，进行设置：
   - 点击 **地区** 部分中的 **编辑**，选择 数据销售选择退出页面应显示的地区，然后点击 **保存**。
   - 点击 **内容** 部分中的 **编辑**，执行修改，然后点击 **保存**。
4. 在 **网站导航** 部分，从商店菜单中选择数据销售选择退出页面位置，然后点击 **保存**。

## 了解隐私设置的影响

实现 Cookie 横幅和数据销售选择退出功能可能会影响数据收集：

- **已配置 Cookie 横幅的地区**：只有在获得同意后才能收集非必需数据，这可能会减少可用于分析、营销和个性化设置的数据。例如，这种减少可能体现为访问次数减少和依赖访问数据的其他指标（包括转化率）降低。这适用于需要获得选择加入数据收集同意的地区，例如 EEA 和 UK。
- **未配置 Cookie 横幅的地区**：在无需获得同意的情况下收集数据。这适用于不需要获得选择加入数据收集同意的地区，例如 US。
