# 自定义 Liquid

Genstore 主题编辑器支持在所有页面模板中为分区和块添加 **自定义 Liquid**。该功能专为追求深度个性化设计的商家打造，允许您在无需修改主题底层源代码的前提下，直接在编辑器中插入自定义的 Liquid、HTML、CSS 或脚本代码，实现预设模板之外的专属功能。

## 核心价值
- **灵活扩展专属功能**：突破模板原生组件的限制，轻松嵌入第三方微服务插件、自定义表单或特殊的交互式动态组件。
- **低风险代码实验**：所有的代码编辑仅作用于当前分区，避免了因修改全局主题代码可能导致的网站崩溃风险，让技术尝试更安全、更高效。
- **精准品牌视觉呈现**：商家可根据营销需求编写特定促销逻辑，或通过自定义 CSS 样式精准调优，确保页面每一个细节都符合品牌视觉规范。
- **无缝兼容可视化编辑**：代码渲染后，相关分区仍可被正常定位和选中，相关设置项依然可编辑，不影响主题的整体可视化配置体验。

## 操作步骤

1. 在 Genstore  商户后台，点击 **商店** → **在线商店** → **模板**，点击当前主题后的 **设计** 按钮。
2. 点击顶部页面选择器，切换到您想要编辑的页面（如：主页、产品页）。
3. 在左侧的操作面板中，点击 **添加分区** 或 **添加块**，选择 **自定义 Liquid**。
4. 输入并保存自定义 Liquid 代码，代码校验通过后，编辑器将实时渲染并展示效果。渲染完成后：
   - 分区和块仍可被正常定位、选中
   - 相关设置项依然可编辑，不影响主题可视化配置体验
5. 确认效果后，点击 **保存**。
