# 生日奖励积分

您可以通过设置生日积分奖励，灵活地激励客户在生日时返回店铺互动，并享受专属的生日积分奖励。

本文档将说明该功能的配置方法、使用场景及注意事项。

## 配置积分规则

**导航：**

登录 Genstore 商家后台，点击 **商店** -> **客户** -> **忠诚度**。  
在 **积分** 标签下，找到 **获取积分** 部分。

### 设置步骤

点击**新增** -> **生日有礼** -> **添加**，进入配置页面：

- **规则名称**：输入积分奖励规则的名称，方便管理。
- **奖励规则**：设置客户在生日时可获得的积分数量。**建议根据您的积分体系合理设置奖励数值。**
- 奖励发放方式：
  - **自动**：系统会在客户生日当天，自动将奖励积分发放至客户账户，并通过邮件通知客户。
  - **手动**：系统会发送邮件通知客户登录店铺在会员面板领取生日积分奖励。您可以设置用户可领取生日奖励的时间范围：
	- **仅限生日当天领取**：客户只能在生日当天领取奖励积分。
	- **生日当月任意一天领取**：客户可以在生日月份的任何一天领取奖励积分。
	- **指定时间范围**：您可以设置客户在生日前 X 天至生日后 Y 天的指定日期范围内领取奖励（例如：生日前 7 天至后 7 天）。

## 预览及发布

在页面的右侧**概要**区域，系统会自动生成并展示当前设置的规则概述，方便实时预览。确认无误后，点击**保存**，发布规则。

规则发布后，您可在积分获取方式列表中查看当前规则、编辑、启用或停用该规则。

:::tip

您也可通过状态开关控制当前规则的启用与关闭。

:::

## 常见问题

### Q1. 如何处理未提前登记生日信息的客户？
> A：客户必须在生日前 30 天完成信息登记，否则无法获得奖励。建议在客户注册时就填入生日信息。
> 	::: tip
> 	对于此类用户，建议通过手动的方式发放生日积分。具体设定，可查看 [管理客户](./operate-customers-loyalty-customer-manage.md)。
> 	:::

### Q2. 可以给不同等级的客户设置不同的奖励规则吗？
> A：目前系统仅支持设置统一的奖励规则。每位客户每年只能领取一次生日奖励。


### Q3. 客户错过了领取时间怎么办？
> A：超出设定的领取有效期后，客户将无法领取奖励。建议启用生日奖励邮件提醒功能（默认启用），及时通知客户领取。
> 	::: tip
> 	对于此类用户，建议通过手动的方式发放生日积分。具体设定，可查看 [管理客户](./operate-customers-loyalty-customer-manage.md)。
> 	:::

### Q4. 系统会自动发送生日积分通知邮件吗？
> A：根据**奖励发放方式的设定**，如果为**自动发放**，则将在将在生日**当天**发送生日积分到账邮件。如果设置了领取期限，可将根据设定发送生日积分领取提醒邮件，具体如下表：

| **领取有效期** | **提醒邮件发送时间**  |
| -------------- | --------------------- |
| 生日当天       | 生日当天              |
| 生日月         | 生日当天及月底        |
| 特定时间范围   | 生日当天及生日后 N 天 |
