# 按国家/地区设置产品价格

通过根据不同国家/地区的设定产品价格，商家可以更好地管理面向国际市场或多地区市场的电商店铺。您可以为每个市场配置特定的价格或货币，提升定价灵活性和市场竞争力。

## 关键概念

**转换费**是商家在进行货币转换时，可能会产生的额外费用。在进行定价时，请将该费用考虑在内。如果设置了固定价格，需要在价格中包含货币转换费。

## 设置特定市场货币展示

该功能允许商家为不同市场或国家/地区提供本地化货币支持。启用后，顾客在结账时将看到以本地货币表示的价格，改善用户体验，减少货币转换带来的障碍。

### 操作步骤

1. 登录 Genstore 商户后台，点击 **设置** -> **市场**。
2. 点击目标市场，进入市场设置页。
3. 在新窗口中，找到 **产品及定价** 部分，勾选 **以当地货币向客户显示价格**。
4. 若要停用当前功能，请取消勾选 **以当地货币向客户显示价格**。停用后，该市场的所有客户将看到以该市场的基础货币显示的价格。

## 调整特定市场价格

通过设置价格调整，商家可以根据不同市场的经济状况、税费政策、汇率波动以及运费等因素调整价格。这有助于确保定价符合各地区的消费能力。

### 示例

- 在美国市场，某产品售价为 **$100 USD**，您可以在欧洲市场设置 **10%** 的价格上涨，即 **€110 EUR**（基于汇率和调整后）。
- 在印度市场，您可能选择将价格降低 **5%**，从 **₹750 INR** 调整为 **₹712 INR**。

### 操作步骤

1. 登录 Genstore 商户后台，点击 **设置** -> **市场**。
2. 点击目标市场，进入市场设置页。
3. 在 **价格调整** 部分，选择 **上涨** 或 **降低** 价格。
4. 输入调整的百分比。
5. 点击 **保存**。

## 为产品设置固定价格

商家可以为特定市场或产品设置固定价格，确保在多个国家/地区的价格一致。设置的固定价格将自动覆盖已激活市场的转换后价格。

您可以为部分或所有产品和多多规格下设置固定价格。对于多国家/地区市场，您只能使用

市场的基本货币设置固定价格。您设置的固定价格会自动覆盖您已激活的每个市场的转换后国际价格。

### 示例

假设您销售的 T 恤定价为 **$30 USD**，根据不同市场情况，您可以为其设置如下固定价格：

- 美国市场：**$30 USD**
- 英国市场：**£25 GBP**
- 欧盟市场：**€28 EUR**
- 日本市场：**¥3200 JPY**

这样，客户在不同国家/地区访问商店时，将看到适用于本地市场的价格，您无需担心汇率波动对价格的影响。

### 操作步骤

1. 登录 Genstore 商户后台，点击 **设置** -> **市场**。
2. 点击目标市场，进入市场设置页。
3. 选择待设定固定价格的产品，您可以通过搜索产品名称或根据筛选条件选择需要设置的产品。
4. 在产品旁的价格输入框中，输入新的固定价格，点击 **保存**。注意：您只能使用当前市场的基本货币设置固定价格。
5. 如果您的产品具有多个规格，可以点击 **查看价格**，查看并设置每个规格的价格。
6. 通过切换选择不同的国家/地区，您可查看和设置产品在每个市场的固定价格。
