# 导出客户个人数据

在收到客户申请导出其在店铺的个人数据的需求时，您可在核实客户身份信息后，通过本功能申请导出，并将导出文件发送给客户。

## 操作步骤

1. 登录 Genstore 商户后台，点击 **商店** -> **客户**。
2. 选择需要导出的客户，进入客户详情页。
3. 点击 **更多操作** -> **请求个人数据**，确认后提交申请。
4. 等待导出任务完成。导出完成后，可点击页面左下角头像，进入 **任务中心** 下载文件。
5. 下载文件后，将其手动发送给客户。

## 个人数据范围

以下为可能包含的客户个人数据范围。
请注意：**实际导出内容仅包含客户在店铺内已产生的相关数据**，如部分数据不存在，则对应项不包含在导出文件中。

| **数据内容** | **说明** | 
| -------- | ------------ | 
| 客户数据  | 客户基本信息，例如：姓名、邮箱、手机号、营销订阅、默认地址信息、免税配置、备注等|
| 收货地址     | 客户的所有收货地址信息       | 
| 客户元字段     | 客户元字段信息，例如客户积分、VIP 等       | 
| 客户订单     | 客户在店铺产生的所有订单记录，但不包含已取消的订单      | 
| 客户草稿订单 | 客户在店铺中的所有草稿订单记录       | 
| 客户礼品卡 | 与客户相关的礼品卡内容       | 
|客户折扣|与客户相关的所有折扣|
|客户支付|客户所有订单的支付记录信息|
|客户履约|客户所有订单的履约记录信息|
