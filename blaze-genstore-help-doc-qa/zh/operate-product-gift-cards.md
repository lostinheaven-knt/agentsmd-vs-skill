# 创建礼品卡

Genstore 提供了礼品卡功能，礼品卡是一种预付费卡，客户可以通过它在您的商店购买产品或服务。它通常用于赠送他人或作为奖励，并可在结账时用于抵扣订单金额。

## 手动创建单张礼品卡

该方式适用于由商家主动发放礼品卡的场景，例如作为营销赠品、客户服务补发等。您可为礼品卡指定金额、设置备注，并可选择关联客户，方便发送和后续追踪。

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **礼品卡**。
2. 点击页面右上角的 **创建礼品卡**。
3. 在新打开的窗口中，输入礼品卡的初始金额。
4. 【可选】**过期信息**：选择是否为当前礼品卡设置有效期。**注意**：请确保遵守您所在国家或地区有关礼品卡有效期的相关法律法规。
5. 【可选】**关联客户**：您可当前页面关联客户，在页面右侧的 **客户** 部分，输入客户的姓名、电话号码或邮箱创建或搜索客户。您也可稍后添加客户。
   - 客户必须提供至少一个联系方式（邮箱或电话号码），以便通过电子邮件或短信接收礼品卡代码。
   - 如果客户提供了邮箱和电话号码两项信息，礼品卡将通过电子邮件发送。
6. 您也可在页面右侧的 **备注** 部分，为当前礼品卡设置备注信息。
7. 点击 **保存**，完成礼品卡的创建。

## 创建可购买的礼品卡产品

若您希望顾客可在前台商城中自行购买礼品卡，并由系统自动生成并发送礼品卡代码，可按以下方式创建产品。

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **礼品卡**。
2. 点击页面右上角的 **创建礼品卡产品**。
3. 输入礼品卡产品的标题、产品特色、上传图片，添加文字描述。
4. 设置礼品卡是否有过期日期。
5. **类别** 默认为礼品卡。
6. 在 **规格** 部分，设置礼品卡的面额。
7. 在规格明细表中，针对不同面额的礼品卡完成设置。具体可参考 [产品规格部分](./operate-product-create#产品规格)

其他项目的设置可参考 [添加产品](./operate-product-create.md) 页。

创建完成后，您可在产品列表中筛选查看礼品卡产品，或前往 **礼品卡** 页面点击 **查看礼品卡产品**，系统将自动跳转至对应筛选结果。

## 导出礼品卡

您可将所有礼品卡数据导出为文件，便于做数据分析、备份或线下核对。

### 操作步骤

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **礼品卡**。
2. 点击右上角的 **导出礼品卡** 按钮。
3. 在弹窗中选择导出范围：
    - 导出当前页面所列礼品卡
    - 导出全部礼品卡
    - 导出已勾选的礼品卡
    - 基于当前筛选/搜索结果导出
4. 点击 **导出礼品卡**，系统将开始生成文件。
5. 导出完成后，可在当前页面或 **任务中心** 下载文件。

## 在线兑换礼品卡

顾客在下单时可于结账页面输入礼品卡代码完成抵扣。**注意**：礼品卡代码不区分大小写。

### 兑换规则

- **余额足够支付订单**：如礼品卡余额大于等于订单总额，客户可直接点击 **立即支付** 结账。
- **余额不足**：如礼品卡余额小于订单金额，系统将提示顾客添加其他支付方式（如另一张礼品卡或银行卡）。

### 使用限制

- 礼品卡余额可用于多个订单，直到余额用尽。
- 礼品卡余额适用于订单总额，包括税费和运费。
- 礼品卡可以与折扣码一起使用。
- 一笔订单可使用多张礼品卡。
