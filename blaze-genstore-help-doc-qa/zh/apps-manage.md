# 管理已安装的应用

Genstore 应用提供了丰富的功能，您可以通过后台对应用进行灵活管理，包括打开应用、将应用固定到导航菜单以及查看应用的隐私政策。

## 在 Genstore 中打开应用

您可以通过 Genstore 商家后台的 **应用** 页面查看并打开已安装的应用。

### 步骤

1. 登录 Genstore 后台，进入 **应用** 页面。
2. 在应用列表中，点击您想打开的应用的名称。
3. 在应用页面顶部，点击 **打开应用**。

## 将应用固定到 Genstore 商家后台

将应用固定到后台导航菜单，可以帮助您快速访问常用应用。

### 步骤

1. 登录 Genstore 商家后台，进入要固定的应用详情页面。
2. 点击页面右上角的固定图标，将应用固定到后台导航菜单。
3. 如果需要取消固定，点击导航菜单中的固定图标，即可将应用从菜单中移除。

## 查看应用的隐私政策

您可以从应用的 **关于** 页面查看应用的权限和隐私政策。

1. 登录 Genstore 后台，进入 **应用** 页面。
2. 在应用列表中，找到目标应用，点击 **查看详情**。
3. 在应用详情页面，找到 **数据隐私** 部分。
4. 点击 **查看隐私政策**，阅读该应用的权限和数据隐私信息。
