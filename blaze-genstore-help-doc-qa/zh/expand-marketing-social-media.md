# 社交媒体营销

社交媒体营销功能提供了一站式的内容发布和管理解决方案。通过该平台，你可以轻松创建和发布各种类型的内容，包括图像和视频，直接同步到你的社交媒体账号。这个功能有效整合了社交媒体管理，助力企业实现内容营销的优化，提升整体营销效果。

## 管理社交媒体账号

将社交媒体账号连接至 Genstore 后，您可以直接在平台内发布内容。请按照以下步骤进行设置：

::: tip

重要提示：您可以针对每个社交媒体免费连接一个社交账户。

:::

## 绑定 Facebook

**前置准备**

- 确保您连接的是与 Facebook 业务账户关联的 Facebook 公共主页。点击阅读 [如何创建 Facebook 公共主页](https://www.facebook.com/help/104002523024878)。
- 确保您拥有 Facebook 公共主页的管理员权限。点击阅读 [如何配置管理员权限](https://www.facebook.com/help/1206330326045914)。

**连接步骤**

1. 登录 Genstore 商家后台，导航至 **营销** -> **社媒营销**。
2. 在 **社媒营销** 主页，点击 **创建新帖** 按钮。
3. 点击 **Facebook 渠道** 下的 **绑定账号** 按钮。
   注意：每个渠道只能绑定一个 Facebook 账号。如果已经绑定，需先解绑当前账号，才能绑定新的 Facebook 账号。
4. 确保您满足连接要求，按照说明 **登录** 您的 Facebook 账号。
5. 选择要连接的 **公共主页** 并点击 **继续**。

## 连接 Instagram

**前置准备**

- 您需要 Instagram 业务帐户进行授权。如果您当前登录了个人或创建者帐户，请先切换到企业帐户，然后再继续。点击阅读 [如何创建Instagram业务账户](https://help.instagram.com/502981923235522)。
- 检查您要关联的 Instagram 企业账号是否与您有管理员权限的 Facebook 账户相关联。点击阅读 [如何添加或更改与您的 Instagram 企业账号所关联的 Facebook 页面](https://www.facebook.com/business/help/345675946300334?id=419087378825961)。

**连接步骤**

1. 登录 Genstore 商家后台，导航至 **营销** -> **社媒营销**。
2. 在 **社媒营销** 主页，点击 **创建新帖** 按钮。
3. 点击 **Instagram 渠道** 下的 **绑定账号** 按钮。
   注意：每个渠道只能绑定一个 Instagram 账号。如果已经绑定，需先解绑当前账号，才能绑定新的 Instagram 账号。
4. 确保您满足连接要求，按照说明 **登录** 您的 Facebook 账号。（注意：在连接 Instagram 帐户之前，需要先登录 Facebook 账号，请确保该 Facebook 账号与 Instagram 帐户相关联。）
5. 按照说明选择您要绑定的 Facebook 公共主页和企业账号。（注意：请确保所勾选的 Facebook 公共主页和企业账号与您要连接的 Instagram 相关联。）
6. 选择您要连接的 Instagram 企业账号，然后点击 **继续**。

## 关联 Twitter

1. 登录 Genstore 商家后台，导航至 **营销** -> **社媒营销**。
2. 在 **社媒营销** 主页，点击 **创建新帖** 按钮。
3. 点击 **X （Twitter）渠道** 下的 **绑定账号** 按钮。
   注意：每个渠道只能绑定一个 X 账号。如果已经绑定，需先解绑当前账号，才能绑定新的 X 账号。
4. 按照 Twitter 提示 **登录** 您的 Twitter 帐户。
5. **单击授权**应用以完成绑定。

## 关联 TikTok

1. 登录 Genstore 商家后台，导航至 **营销** -> **社媒营销**。
2. 在 **社媒营销** 主页，点击 **创建新帖** 按钮。
3. 点击 **TikTok** 渠道下的 **绑定账号** 按钮。
4. 系统将跳转至应用授权页：
   - 已有账号：核对登录信息并确认授权。
   - 未有账号：可根据页面提示完成账号注册。
5. 完成授权后，系统将返回 Genstore 商家后台。
   
## 断开社交媒体账户

1. 登录 Genstore 商家后台，导航至 **营销** -> **社媒营销**。
2. 在 **社媒营销** 主页，点击 **创建新帖** 按钮。
3. 点击 **账号解绑**，点击目标账号后的 **取消绑定**。

::: tip

如果您解绑 Facebook 账号，那么该账号下绑定的页面都会被解绑。

:::
