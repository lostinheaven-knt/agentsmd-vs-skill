# 批量更新库存

Genstore 支持批量编辑库存，允许商户一次性调整多个产品、规格和地点的库存数量。批量编辑器类似电子表格，提供直观的数据行与列视图，让库存管理更加高效。

## 通过产品详情页编辑

当需要调整 **某个产品的多个规格** 在不同地点的库存时，可直接进入产品页面进行批量编辑。

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 在产品列表中，点击要跟踪的产品名称进入产品详情页，或 **操作** 列的 **编辑** 按钮。
3. 在 SKU 列表部分，选择要编辑的规格，然后调整可售库存数量。**注意**：更新可售库存时，现货库存也会更新。
4. 如希望希望特定地点的库存，可点击 **管理地点** 按钮，选择库存地点。重复上一步操作，完成相关地点库存的更新。
5. 点击 **保存** 完成库存信息的更新。

## 通过库存列表编辑

如果只需更新少量产品的库存，可在 **库存页面** 通过批量编辑器进行修改。库存列表支持按地点、产品和产品规格进行筛选。

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **库存**。
2. 勾选希望调整的产品规格。
3. 点击 **批量编辑可售库存**。
4. 如希望调整 **指定地点** 的库存，可先选择地点，然后勾选产品规格，并调整可售库存。

## 多规格产品

如需批量调整多规格产品的库存跟踪状态，可在 **产品列表** 进行设置，随后进入产品详情页对各规格的具体库存进行编辑。
1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 点击打开目标产品页面，找到规格明细部分。
3. 勾选希望调整的产品规格，点击 **批量编辑**。
4. 在新打开的窗口中，调整目标产品规格的库存跟踪设置。
5. 点击 **保存** 完成库存更新。
