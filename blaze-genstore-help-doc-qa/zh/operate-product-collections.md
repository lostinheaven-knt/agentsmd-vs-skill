# 产品系列

**Genstore** 提供强大的产品系列功能，帮助商家将产品根据不同条件进行分类与管理。通过灵活的自动化与手动操作方式，商家可以轻松创建和管理产品系列，提升产品管理效率，优化商店展示，增强购物体验。

::: tip
您可以将处于任何状态（草稿、已上架、未上架或归档）的产品加入系列，方便内部管理和提前规划。但在线商店前台仅展示状态为“已上架”的产品，其余状态的产品将自动对客户隐藏。
:::

## **自动创建产品系列**

[自动创建产品系列](./operate-product-collections-auto.md) 功能允许商家通过设置条件和规则，自动将符合条件的产品加入特定系列。通过条件筛选，商家可以根据产品类型、价格、标签、库存等维度进行精准匹配。此功能尤其适用于大批量产品的管理，并能大幅减少人工操作，提高工作效率。

### **自动筛选条件**

在创建自动产品系列时，商家可以根据不同的 [筛选条件](./operate-product-collections-condition.md) 来精确控制哪些产品会被纳入系列。可配置的筛选条件包括：产品类型、价格范围、库存状态、标题关键词等。商家可以选择设置 **所有条件必须满足** 或 **任意条件满足** 来决定系列的内容，最多支持添加 60 个筛选条件。

## **手动创建产品系列**

[手动创建产品系列](./operate-product-collections-manual.md) 则适用于那些商家希望完全控制系列内容的场景。商家可以根据实际需求，自由选择产品并手动添加到系列中。通过这种方式，商家可以针对特定的精选产品、促销活动或特别策划，灵活管理产品的展示。

## 预览产品系列

产品系列保存后，您可以实时查看其在在线商店中的展示效果。 

### 操作步骤
1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **系列**。 
2. 在产品系列列表中，点击操作列中的 **预览** 图标，即可打开新窗口，查看该产品系列在 **在线商店** 中的展示效果。
3. 如果该产品系列分配至多个市场，您可以在预览中切换查看： 
	- **已激活市场**：查看真实的前台展示效果。
	- **未激活市场**：提前查看该产品在对应国家/地区的展示样式，便于确认页面表现。 
4. 在页面底部提示条，您可查看当前主题名称，并选择 **退出预览** 或点击 **设计主题** 进入装修编辑器开始页面定制。

## 应用场景：快速引导顾客浏览新品系列

为了方便顾客快速找到您店铺的新品系列，您可以在在线商店的导航菜单中创建一个下拉菜单，将新品系列链接整合在一起。这样，顾客无需多次点击，即可轻松浏览所有新品。

### 操作步骤

1. 登录 Genstore 商店后台，点击 **商店** -> **产品** -> **系列**。
2. 为店铺新品创建产品系列，例如 “2026 春夏”，具体操作请参考 [手动创建产品系列](./operate-product-collections-manual.md)。
3. 在导航菜单中关联当前产品系列。
   - 点击 **商店** -> **在线商店** -> **菜单**。
   - 点击 **主菜单** 后的 **铅笔** 图标。
   - 输入导航标题（如“2026 春夏”）和 handle（系统会自动生成，也可自定义），然后点击 **添加菜单项** 按钮。
   - 在新弹出的窗口中，输入菜单项名称（如 “2026 春夏”），点击搜索框，在下拉列表中选择 **产品系列**，随即选择目标产品系列。
   - 点击 **保存**。
   - 您可重复上述步骤继续添加其他新品系列。
4. 在导航列表中，您可以通过拖拽来调整菜单项的显示顺序。
