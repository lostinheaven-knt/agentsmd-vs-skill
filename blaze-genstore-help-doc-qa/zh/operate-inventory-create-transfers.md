# 创建库存转移

在现代零售和电商环境中，库存管理至关重要。创建库存转移可帮助企业高效跟踪和接收传入库存，确保供应链顺畅运作。库存转移适用于您的商店地点之间的调拨（如果您有多个地点），以优化库存分配，减少缺货或过量库存的风险。

## 操作步骤

登录 Genstore 商家后台，点击 **商店** -> **产品** -> **转移**，点击页面右上角的 **创建转移**，完成以下内容的配置后，点击 **保存为草稿** 保存。

- **发货地点**：选择活跃发货地点。
- **目的地点**：选择活跃目的地点。
- **添加产品**：输入或选择要添加到转移中的产品规格。
- **库存数量**：选择产品后输入相应规格的库存数量。
- **配送信息**（可选）：填写预计配送日期、运单号、运单承运商。

创建完成后，库存转移单将直接出现在转移页的单据列表中，您可在此查看发货地、目的地、状态、收货数量以及预计配送日期。

在发起库存转移后，可在库存转移页面右上角点击 **标记为待收货**，表示库存转移流程已开始。转移订单列表页面会自动根据转移单的状态进行分类，您可查看某一状态的转移单，或所有转移单，相关状态的解释如下：

| 转移状态 | 描述                           |
| -------- | ------------------------------ |
| 草稿     | 生成草稿库存转移单据           |
| 待收货   | 库存已离开发货地点，正在运输中 |
| 部分收货 | 只有部分库存到达目标地点       |
| 已收货   | 库存已到达目标地点并完成接收   |

## 编辑库存转移

对于 **草稿** 状态的库存转移单，可以进行编辑和更新。

### 操作步骤

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **转移**。
2. 点击希望更新的库存转移单据号，进入详情页，您可对以下内容进行编辑，更新完成后，点击 **保存** 完成更新。
   - **发货地点**：可变更发货地点。
   - **目的地点**：可变更目的地点。
   - **添加产品**：可调整已选产品，删除后重新添加。
   - **库存数量**：可重新调整已选产品的库存数量。
   - **配送信息**（可选）：填写预计配送日期、运单号、运单承运商。
