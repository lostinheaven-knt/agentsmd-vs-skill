# 处理信用卡付款

当客户通过信用卡支付在线订单时，支付处理必须在资金到账之前完成。这个处理过程由您的支付提供商负责，通常分为以下几个阶段：

| 阶段                           | **描述**                                                     | **注意事项**                                                 |
| ------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **授权**<BR/>（Authorization） | 客户使用信用卡支付订单，支付提供商与发卡机构验证信用卡有效性和资金状况。此时支付并未实际发生，只是对交易进行授权。 | 该阶段不涉及资金转移，仅验证信用卡的有效性和账户余额。       |
| **捕获**<BR/>（Capture）       | 在授权后，需要捕获付款，将授权的款项从客户信用卡账户转移到商家账户。支付提供商将付款详情传递至收单机构，开始实际资金转移。 | 捕获必须在授权期限内完成，超时后将无法捕获付款。             |
| **清算**<BR/>（Clearing）      | 收单机构审核付款详情，并向信用卡公司请求资金，信用卡公司再向发卡机构请求资金。经过多个机构的费用扣除后，资金将被传回商家账户。 | 资金在发卡机构、信用卡公司和收单机构之间传递，经过多次扣费后最终到商家账户。 |
| **资金到帐<BR/>**（Funding）   | 收单机构从金额中扣除手续费后，将剩余资金转入商家账户。       | 通常在几个工作日内到账，具体时间取决于收单机构的处理速度。   |

## 自动和手动捕获

Genstore 提供了灵活的付款捕获选项，您可以根据业务需求选择**自动捕获**或**手动捕获**。每种方式都有其特定的优缺点，选择合适的捕获方式可以帮助您更高效地管理订单和资金流动。

### 自动捕获

- **推荐使用场景**：适用于希望简化支付流程的商家，尤其是处理订单较多时，自动捕获能够有效减少操作步骤，提高效率。
- **优势**：自动捕获付款可以确保在订单结算时即时完成资金转移，减少了人为操作失误的风险。
- **适用性**：默认情况下，Genstore 为商家设置了自动捕获，您只需在客户结账时自动完成付款捕获，无需额外操作。

### 手动捕获

- **推荐使用场景**：适用于需要更多控制权的商家，尤其是在处理可能存在欺诈风险的订单或需要手动审查订单时，手动捕获提供了更多灵活性。
- **优势**：允许商家在一定时间内决定是否捕获付款，例如可以在确认订单无误或已完成某些内部审核后进行付款捕获。
- **适用性**：如果您选择手动捕获，则每笔付款都需要在授权期限内手动进行捕获。该模式更适合特殊需求或需要额外审核的订单。

### 捕获付款的时间限制

所有的信用卡支付提供商都有一个授权期（Authorization Period），即从授权开始到捕获必须完成的时间窗口。一般来说，授权期会在几天内结束，具体期限由您的支付提供商规定。

- **Genstore Payments**：提供 7 天的授权期。如果在此期间没有捕获付款，授权将会自动取消，您将无法继续处理此笔交易。
  **汇率注意事项**：当您捕获以外币支付的订单时，金额将按照捕获时的汇率进行转换，而不是授权时的汇率。因此，如果在授权和捕获之间发生汇率波动，您可能会面临资金损失。
- 如使用第三方支付方式，授权期限和第三方交易费用可能会发生变化，请联系支付提供方获取详情。

## 设置自动捕获付款（推荐）

默认设置，如果您曾更改设置并希望恢复自动捕获付款，请按照以下步骤操作：

1. 登录您的 Genstore 管理员账号，进入 **设置 -> 支付**。
2. 在页面底部 **支付捕获方式** 中，点击 **管理** 按钮，在弹窗中选择 **结账时自动捕获**。
3. 单击 **确定** 完成设置。

## 设置手动捕获付款

如果您希望手动捕获付款，可以按照以下步骤更改设置：

1. 登录您的 Genstore 管理员账号，进入 **设置 -> 支付**。
2. 在页面底部 **支付捕获方式** 中，点击 **管理** 按钮，在弹窗中选择 **手动捕获**。
3. 单击 **确定** 完成设置。

:::tip

- 在手动捕获模式下，您将需要在授权期内手动处理每笔付款，避免未及时捕获而导致交易失败。
- 确定切到到手动捕获模式前，请仔细进行业务评估，为减少不必要的风险，推荐使用自动捕获付款。

:::
