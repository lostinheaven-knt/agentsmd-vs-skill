# 订单路由

订单路由规则决定了每个客户订单中产品的发货地点。您可以根据需要添加、编辑、删除以及重新排列这些规则，以确定发货地点的优先级。

此外，您还可以根据常见的配置方式，如始终从距离收货地址最近的地点发货。

::: tip  
仅当 **配置了多个发货点** 时，才需要设置路由规则。如果您的店铺只有一个发货点，则无需配置此功能。  
:::

## 添加订单路由规则

1. 在 **Genstore** 商户后台，转至 **设置** -> **发货配送**。
2. 在 **订单路由** 部分，点击查看订单路由规则。
3. 选择您要添加的规则。
4. 如希望添加多条规则，可重复步骤 2、3。规则将按照优先级，从上到下运行。您可拖动规则旁边的 **拖曳** 图标，调整规则运营优先级。
4. 点击 **保存**。

## 管理订单路由规则

规则添加完成后，您可对其进行编辑和删除操作。

### 编辑发货地址优先级

1. 在 **Genstore** 商户后台，转至 **设置** -> **发货配送**。
2. 在 **订单路由** 区域，可以查看已配置的规则数量。点击进入设置页面。
3. 点击 **配置规则**，添加 **发货点顺序规则**，并选择 **按距离优先** 或 **手动配置顺序**：
	- 若选择 **手动配置顺序**，点击 **编辑** 按钮设置发货点顺序。
	- 如仅有一个发货地址，请先在 **设置 -> 地点** 中添加其他地址后，再进行优先级调整。
4. 点击 **保存**。

### 删除订单路由规则

1. 在 **Genstore** 商户后台，转至 **设置** -> **发货配送**。
2. 在 **订单路由** 部分，点击进入设置页面。
3. 点击您要删除的规则旁边的 **垃圾桶** 图标。
4. 点击 **保存**。

## 常见订单路由配置

以下是一些推荐的订单路由规则配置，您可以根据需求进行设置：

1. **尽量避免拆分发货原则**
   - 同一订单的所有产品应从同一发货点发出，避免订单拆分。只有在无法从单一发货点满足需求时，才考虑拆分发货。
2. **发货点顺序规则**
   - 发货点选择应基于距离或者管理员手动设置的优先顺序。建议合理安排优先顺序，以确保发货顺畅、物流时效。
3. **同一市场原则**
   - 优先选择与收货地址位于同一市场的发货点，以降低跨市场运输成本并提高交付效率。
4. **库存充足原则**
   - 优先选择库存充足的发货点。若某个发货点无法满足订单需求，可选择其他库存充足的发货点，确保订单及时交付。

## 常见问题

### Q1. 草稿订单默认会使用哪个仓库发货？
> A：草稿订单会自动使用您在 [地点](./operate-fulfillment-location.md) 中配置的默认发货仓库，您也可以在发货时修改。如希望修改默认发货地点，可在 Genstore 后台，点击 **设置** -> **地点** 进行修改。
:::
