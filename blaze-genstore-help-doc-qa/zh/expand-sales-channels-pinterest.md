# Pinterest 销售渠道

Genstore 支持通过 Pinterest 销售渠道帮助您推广和销售产品，实现产品信息自动同步至 Pinterest 并发布产品。通过 Pinterest，您可以在 Pinterest 上有效地推广和展示其产品，吸引潜在买家，并增加销售和转化率。此外，您还可以通过 Pinterest 提供的数据分析工具，了解用户对产品的兴趣和行为，优化推广策略，提升市场营销效果。

## 核心优势

- **无缝集成**：商店直接关联 Pinterest 商业账户，确保产品能在 Pinterest 平台展示。
- **提升转化**：将产品快速同步至 Pinterest 目录，通过 Pinterest 推广产品链接更多客户。
- **数据驱动**：借助 Pinterest Tag 跟踪用户转化行为，查看详细的产品表现数据，助力广告优化。

## 前置准备

**Pinterest 账号要求**

使用 Pinterest 销售渠道，您需要具备：

- **Pinterest 商业账户**：用于存储商店及产品数据。
- **Pinterest 广告账户**：用于广告投放和推广服务。

如果您尚未注册账户，请先完成以下步骤：

- [注册 Pinterest 商业账户](https://help.pinterest.com/zh-hans/business/article/get-a-business-account)
- [创建 Pinterest 广告账户](https://help.pinterest.com/zh-hans/business/guide/becoming-an-advertiser-on-pinterest)

**Pinterest 商业账户同步说明**

成功连接 Pinterest 商业账户后，以下信息将从 Genstore 同步至 Pinterest：

- 您的在线商店域名。
- 已配置 Pinterest 销售渠道的店铺产品。有关如何为产品配置销售渠道，请参考 [为产品配置销售渠道](./operate-product-create#销售渠道)。

## 安装与授权

1. 访问 [Genstore 应用市场](https://apps.genstore.ai/zh-CN)。
2. 在搜索框中输入 `Pinterest`，点击卡片进入应用详情页。
3. 点击 ​**安装**​，系统将返回 Genstore 商户后台。
4. 按照提示完成安装并授权，启用 Pinterest 销售渠道。
5. [可选] 点击图钉图标将应用固定至导航栏，便于后续访问。

## 连接账户并同步产品

在使用 Pinterest 销售渠道前，您需要绑定 Pinterest 商业账户与 广告账户，以便您可以与 Pinterest 同步产品。

1. 登录 Genstore 商户后台。
2. 点击 **销售渠道** -> **Pinterest**。
3. 点击 **连接 Pinterest 账户** 并完成授权。
4. 授权完成后，系统会跳转回 Genstore 商家后台。在 **设置** 页面，从 **广告账户** 下拉框中选择您的广告账户。
5. 如未获取到广告账户，可点击提示前往 Pinterest 创建广告账户，创建后返回继续添加。
6. 完成商业账户与广告账户绑定后，系统将自动执行以下操作：
	- 开启网域验证
	- 验证成功后会同步已配置 Pinterest 销售渠道的店铺产品信息到 Pinterest。
		> 有关如何为产品配置销售渠道，请参考 [为产品配置销售渠道](./operate-product-create#销售渠道)。	
	- 自动添加 Pinterest Tag（用于追踪网站访客及广告转化行为）

## 配置 Pinterest 转化 API

Pinterest 转化 API 可帮助您更精确地追踪用户行为。配置步骤如下：

1. 登录 Genstore 商户后台。
2. 在左侧导航栏，点击 **销售渠道** -> **Pinterest**。
3. 点击页面右上角的 **设置**。
4. 在 **转化 API** 模块，点击 **添加**。
5. 连接成功后，数据将实时回传至 Pinterest 广告平台。

## 解绑 Pinterest 账号

如不再需要 Pinterest 销售渠道，可解绑 Pinterest 商业账号：

1. 登录 Genstore 商户后台。
2. 在左侧导航栏，点击 **销售渠道** -> **Pinterest**。	
3. 点击页面右上角的 **设置**。
4. 点击 Pinterest 商业账户后的 **解绑** 按钮。
5. 系统将弹出确认窗口，确认断开操作后，系统将同时移除：
	- Pinterest 广告账户
	- Pinterest Tag
	- 转化 API
