# 获得付款

本文介绍 Dropshipping 订单的付款与支付状态流转流程，包括买家付款方式、支付确认、订单状态变化以及商家收款说明。

## 选择适合您的支付方案

Genstore 为商户提供了两种收款路径。您可以根据业务成熟度、主体资质以及对转化率的要求，灵活选择最适合当前阶段的方案：

| 字段                 | Genstore Payments           | Drop Pay   |
| -------------------- | --------------------------- | ---------- |
| 定位              | 品牌增长/全能方案 | 快速起步/轻量化方案       |
| 进件要求              | 必须，目前仅支持 US 或者 HK | 无需进件       |
| 售卖 Dropshipping 产品 | 支持                        | 支持       |
| 售卖其他产品         | 支持                        | 不支持     |
| 客户结账：快捷结账   | 支持                        | 不支持     |
| 资金分账与结算规则   | 无变化                      | 无变化     |
| 获得付款方式         | 您个人或者公司的银行账户    | PayPal 账户 |

Drop Pay 和 Genstore Payments 都深度兼容了 Genstore Dropshipping 业务，您可根据业务需求来评估使用哪一个支付方式。

## 付款路径概览

当客户在店铺中下单后，系统会根据购物车中的产品类型决定可使用的支付方式：

1. **系统校验产品类型**：若订单中包含代发货产品，系统将仅允许买家使用 **Genstore Payments** 或 **Drop Pay** 支付。
2. **买家完成付款**：系统调用支付网关进行交易授权与扣款。
3. **系统确认支付结果**：支付成功后自动生成订单，并更新订单状态。
4. **同步至商户后台**：订单显示为“已付款”，可进入履约阶段。

## 订单支付状态说明

| 支付状态                  | 说明                                |
| --------------------- | --------------------------------- |
| **待付款（Pending）**      | 订单已创建，买家尚未完成付款。                   |
| **付款处理中（Processing）** | 系统正在等待支付网关返回结果。                   |
| **已付款（Paid）**         | 支付成功，系统已确认付款。                     |
| **支付失败（Failed）**      | 支付未通过或中途失败。                       |
| **已退款（Refunded）**     | 订单支付已退款给买家。                       |
| **拒付（Chargeback）**    | 持卡人向发卡银行申请撤销支付，系统已从商户账户余额中扣除相应金额。 |

## 商户收款说明

* 买家完成付款后，订单款项将由 **Genstore** 暂时托管。
* 系统在清算周期结束后，会自动扣除以下费用并结算至商户账户余额：
  * 支付处理费
  * 采购成本
  * 代发货服务费
* 若订单包含代发货产品：
  - 清算周期为 **45 天**，结算周期为 **5 天**；
- 若仅包含普通产品：
  - 清算周期为 **5 天**，结算周期为 **0 天（每日自动）**。
* 结算完成后，系统会指示银行将资金转入商户绑定的结算账户，银行处理时间可能略有延迟。

> 详情请参考：[代发货服务的分账与结算规则](./operate-dropshipping-fund-splitting.md)。

## 常见问题

### Q1. 买家付款后订单未显示“已付款”，怎么办？
> A: 支付网关可能存在延迟，一般会在几分钟内同步结果。若超过 30 分钟仍未更新，请检查支付状态或联系支持团队。

### Q2. 订单支付失败后是否会自动取消？
> A: 是。若买家未在指定时间内完成付款，系统将自动取消订单并释放库存。

### Q3. 代发货订单与普通订单的付款逻辑是否不同？
> A：不同。代发货订单仅通过 **Genstore Payments** 或 **Drop Pay** 支付，普通订单可使用多种支付渠道。代发货订单的资金清算周期也更长。
