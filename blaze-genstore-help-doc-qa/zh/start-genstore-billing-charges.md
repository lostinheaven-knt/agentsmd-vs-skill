# 账单费用

账单会显示不同类型的费用，具体费用依据您的商店位置和使用的服务而定。您的账单将通过您的 **主要付款方式** 支付。

## 订阅费

订阅费用依您的套餐和订阅周期而定。您可以在 **账单** 页面查看订阅费用。关于 Genstore 各定价套餐及功能说明，参考 [定价套餐](./start-understand-pricing-plans.md)。

## 第三方交易手续费

第三方交易手续费适用于使用 [第三方支付服务](./operate-payments-third-party.md) 的交易。它包括 Genstore 与外部支付服务提供商的集成费用，且费用根据您的套餐而异。注意：交易手续费不包含在第三方支付服务提供商收取的手续费之内。

:::tip

使用 **[Genstore Payments](./operate-genstore-payments.md)** 或 [手动付款方式](./operate-payment-manual-payments.md)（如现金、货到付款等），您无需支付第三方交易手续费或支付手续费。

:::

### 计算方式

Genstore 计算交易手续费的公式为：

```html
 [(产品价格 - 折扣) + 税费 + 运费] x 费率 
```

注意：第三方交易手续费的费率取决于您的定价套餐。

## 抵扣金

抵扣金账户分为以下几类：

- **套餐抵扣金**：用于订阅套餐费用的折扣。
- **交易手续费抵扣金**：用于交易手续费的折扣。

您可以在账单页查看抵扣金的余额。

注意：**抵扣金** 仅适用于未来的账单，无法用于当前的订阅费用或交易费用，抵扣金将应用于账单的相关类别中，一个类别的抵扣金不能应用于其他类别。
