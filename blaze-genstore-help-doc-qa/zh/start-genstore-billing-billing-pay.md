# 支付账单

Genstore 支持使用主要信用卡进行账单支付，包括 Mastercard、Visa、American Express 或 Discover。您可以管理您的付款方式、修改账单地址、调整账单周期，以及添加银行账户信息等。

## 管理付款方式

您可以在 Genstore 后台添加新的付款方式或更新现有付款方式。

**操作步骤**

1. 登录 Genstore 商家后台，点击 **设置** -> **账单**。
2. 点击 **账单资料**。
3. 在 **付款方式** 部分，点击要更改的付款方式的 **修改** 按钮。
4. 在 **添加付款方式** 对话框中，输入所需信息。
5. 点击 **添加银行卡**。

## 管理账单地址

如果信用卡关联的账单地址发生变化，也请在 Genstore 商家后台更新。

### 操作步骤

1. 登录 Genstore 商家后台，点击 **设置** -> **账单**。
2. 点击 **账单资料**。
3. 在 **地址和货币** 部分，点击商店地址的 **管理** 按钮。
4. 请输入新账单地址。
5. 点击 **保存**。

## 更新账单周期

账单期（或账单周期）指的是两笔账单之间的时间间隔，通常默认为每月固定的一天。

要更改账单周期，您需要是店主或拥有**管理套餐**权限的员工。请注意，您只能延长或缩短账单周期，而不能选择在特定日期生成账单。

:::tip

**订阅费用不可退款**：Genstore 订阅套餐的费用不支持退款。具体内容请见 Genstore [服务条款](https://www.genstore.ai/legal/terms) 第 5.9 节。

:::

### 操作步骤

1. 登录 Genstore 商家后台，点击 **设置** -> **套餐和积分**。
2. 在**订阅**中的 **套餐** 部分，点击 **变更套餐**。
3. 在 **选择您的套餐和账单周期** 页面中，选择 **按月付款** 或 **按年付款**。如果未显示这些按钮，您便无法更改当前套餐的账单周期。
4. 查看相关费用说明并点击 **确认更改**。
