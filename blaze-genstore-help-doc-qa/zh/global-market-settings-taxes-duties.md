# 关税和进口税

当您进行跨境运输时，客户在收到货物时可能需要支付额外的关税或进口税。为了提升客户的购物体验并避免交易纠纷，您可以为特定国家或地区提前配置关税和进口税的承担方式。

您可以在每个市场中查看该市场下所有国家/地区的关税和进口税配置状态，并进行相应的添加、编辑或启用/禁用操作。

## 查看关税和进口税

1. 在 Genstore 后台中，转至 **设置** -> **市场**。
2. 在 **概览** 部分，点击打开要设置的市场详情页。
3. 您可在 **关税和进口税** 部分查看当前市场的相关配置。
4. 点击 **关税和进口税**，您可查看税率、计算方式。
5. 您可点击 **编辑** 按钮，点击关税的计算方式、显示名称等。

## 手动设置关税

若您希望在结账时向客户收取关税和进口税，可手动设置关税计算方式。系统支持多种计算方式，适用于不同国家/地区和产品类型。

### 步骤：

1. 在 Genstore 后台，前往 **设置** -> **税务**，启用 **关税设置**。
2. 勾选 **我的承运商支持预付关税**，确保订单可在结账时收取关税。
3. 选择销售国家/地区，进入该地区的关税设置页面。
4. 设置关税计算方式（如 CIF、FOB 或按重量）并输入适用税率。
5. 配置默认税组或为特定产品创建自定义税组，完成后点击 **保存**。

::: tip 提示
启用预付关税前，请确认您的承运商支持 DDP 模式，确保设置合规并与物流流程兼容。
:::

具体操作，可进一步阅读 [手动设置关税](./taxes-setting-manual.md)。
