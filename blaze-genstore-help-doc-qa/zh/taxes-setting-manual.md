# 手动设置关税

您的商店启用关税设置后，您可以手动配置和调整关税计算方式，确保国际客户在结账时预先支付关税和进口税，从而避免客户在收货时遇到意外的额外费用。

## 重要提醒

- **税率设置不等于税务建议**：关税计算方式仅供参考。不同国家/地区的税务规定各异，建议咨询当地税务专家或海关，确保符合目标市场的法律法规。
- **合规性检查**：部分国家/地区有免税政策或最低税收起征点，请务必检查目标市场的关税规定，确保您的设置符合要求。
- **税率与计算方式调整**：税务政策可能会变动，建议定期检查并更新设置，以确保准确性。

## 启用预付关税功能

在手动设置关税时，您需要首先开启预付关税功能，以确保客户在结账时能够支付关税。在该模式下，税和进口税将包含在订单金额中，由承运商负责处理。

:::tip

在调整不同国家/地区或产品的关税前，请确认您的承运商支持 **DDP（Delivered Duty Paid）** 模式，以确保您能收取并支付关税。同时，确保您的税率设置与承运商的流程兼容。

:::

操作步骤：

1. 登录 Genstore 商家后台，点击 **设置** -> **税务**。
2. 切换到 **手动计税**。
3. 滑动按钮开启 **进出口税务** 部分。
4. 勾选 **我的承运商支持预付关税**，以便承运商在结账时计算关税。
5. 点击 **开启关税设置**，在新窗口中，选择您销售的国家或地区。不同国家/地区有不同的关税计算规则，因此，您需要根据具体国家/地区的要求配置税率和计算方式。

## 编辑税务设置

点击目标国家/地区旁的 **编辑** 按钮，您可以调整关税计算方式和税率，以确保合规。

### 选择关税计算方式

关税的计算方式有三种常见选择，您可以根据产品类型和目标市场的要求手动设置：

**基于 CIF（到岸价）计算的从价税**

适用于需要计算 **产品成本、运输费和保险费** 的情况。

计算公式：

```
关税税率 × (产品成本 + 运费)
```

操作步骤：

1. 选择 CIF 作为关税计算方式。
2. 输入目标市场的税率和其他可能的额外费用（如运费、保险等）。
3. 保存后，订单将按 CIF 规则计算关税。

**基于 FOB（离岸价）计算的从价税**

适用于关税仅基于 **产品本身的销售价值**（不含运费和保险）。

计算公式：

```
关税税率 × 产品成本
```

操作步骤：

1. 选择 FOB 作为关税计算方式。
2. 输入产品的销售成本作为计算基础，并为目标市场配置相应的税率。
3. 保存设置后，所有产品将按 FOB 价格来计算关税。

**按重量计算的从量税**

适用于 **按重量或体积** 计算关税的产品。

计算公式：

```
关税税率 × 产品重量
```

操作步骤：

1. 选择按重量作为关税计算方式。
2. 输入每单位重量的税率，并确保正确填写各个产品的重量信息。
3. 完成设置后，所有按重量计算的产品将基于重量来计算关税。

### 配置默认税组与自定义税组

**默认税组**

您可以为所有产品设置一个统一的默认关税税率，适用于所有 **未单独设置** 关税的产品。

操作步骤：

1. 在关税设置页面，选择 **默认税组**。
2. 输入适用的税率和计算方式。
3. 保存后，未指定税率的产品将使用该默认税率。

**自定义税组**

如果某些产品的关税需要特别处理（例如：某些产品免税，或者某些产品需要特殊的税率），可以为这些产品创建自定义税组。

操作方式：

1. 在 **自定义税组** 区域，点击 **添加税组**，进入 **添加自定义税组** 页面。
2. 配置税组名称，方便后续管理。
2. 为特定产品选择关税计算方式（例如，CIF、FOB、按重量等）并设定独立的税率。
3. 保存设置后，特定产品将应用您为其配置的关税税率。

### 自定义税费名称

您可以自定义税费名称，并添加备注以便管理。
