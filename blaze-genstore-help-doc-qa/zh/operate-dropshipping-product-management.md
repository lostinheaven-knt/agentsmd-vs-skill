# 产品管理

当商户完成选品后，可将产品一键推送至已绑定的店铺。
推送时，系统会自动同步产品的主要信息（如标题、描述、图片、价格等），商户可在推送后根据品牌需求进行二次编辑与优化。

## 推送至店铺产品

产品推送至店铺后，系统会自动同步以下信息：

| 字段          | 说明                   | 推送后是否可编辑 |
| ----------- | -------------------- | -------- |
| **产品标题**    | 从选品产品同步，可根据品牌风格调整    | ✅ 可编辑    |
| **产品特色**    | 从选品产品同步，可补充或修改       | ✅ 可编辑    |
| **图片 / 视频** | 从选品产品同步，可替换或新增素材     | ✅ 可编辑    |
| **产品描述**    | 从选品产品同步，可优化文案内容      | ✅ 可编辑    |
| **类别**      | 自动映射至对应的产品类别         | ❌ 不可编辑 |
| **规格信息**    | 仅价格与原价支持编辑           | ⚠️ 部分可编辑 |
| **状态**      | 推送后默认设为“上架”          | ✅ 可编辑   |
| **销售渠道**    | 默认展示渠道为 Online Store | ✅ 可编辑   |
| **厂商**      | 同步供应商信息，可根据实际情况修改    | ✅ 可编辑    |
| **标签**      | 从选品产品同步，可新增或调整       | ✅ 可编辑    |
| **系列**      | 自动或手动关联，可调整展示位置      | ✅ 可编辑    |

## 产品状态与数量限制

推送后的产品默认状态为 **上架**。如您需要调整推送后的产品状态，可在**设置**中进行调整：
1. 导航至 **Dropshipping** -> **设置**。
2. 在**产品状态**下，您可以选择将产品设为**草稿**或**上架**。
   - **草稿**：产品不会在店铺中展示，仅对商户可见，您可在上架前对产品进行编辑。
   - **上架**：产品会在店铺中展示，对客户可见，您稍后可在 **产品** 中对其进行调整。具体操作请参考 [发布产品](./operate-product-create.md)。

每个套餐都有固定的推送数量上限：
  * 超出上限后无法继续推送新产品；
  * 当套餐降级时，系统会提示下架超额的产品。

## 操作建议

* 推送后建议检查标题、描述、图片等内容是否符合品牌风格；
* 可根据市场策略手动调整价格与展示顺序；
* 使用标签、系列等字段提升产品在店铺中的分组与曝光效果。

## 快速识别推送后的 Dropshipping 产品
推送后，Dropshipping 产品将出现在产品列表中，您可通过 **来源** 列快速区分 Dropshipping 产品和自有产品： 
1. 在 Genstore 商户后台，导航至 **商店** -> **产品** 页面。
2. 如 **来源** 列显示为 **Dropshipping**，表示该产品是通过 Dropshipping 推送至店铺的。反之如果为空，则为商户自建产品。
3. 如产品列表页面未显示 **来源** 列，可点击 **列** 图标，添加该列。

## 在店铺前台展示 Dropshipping 产品

通过 Dropshipping 渠道推送的产品将自动绑定为 Dropshipping 产品模板。

- **PC 显示规则**：默认情况下，产品描述区域位于页面左侧，并具备“悬停展开”交互效果。商户还可以自定义 Dropshipping 产品的模板样式。

### 自定义模板

1. 在 Genstore 商户后台，点击 **商店** -> **在线商店** -> **模版**。
2. 点击您当前主题旁的 **设计** 按钮以打开主题编辑器。
3. 切换到 **产品** 模板。
4. 选择 Dropshipping 产品模板，预览并根据需要自定义页面模板。

有关如何自定义模板的更多详情，请阅读 [装修编辑器使用指南](./operate-store-design-themes-edit-guide.md)
。
