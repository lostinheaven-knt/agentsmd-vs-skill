# 设置邮件视觉样式

通过预设邮件的视觉样式，如颜色、字体，您可以为电子邮件营销统一设置视觉风格，包括 logo、字体和品牌配色，确保邮件在各类场景中都保持专业一致的品牌形象。

- **强化品牌形象，统一视觉风格**：通过预设浅色与深色模式，系统将自动优化视觉效果，确保每封邮件都贴合品牌调性、风格统一、专业出彩。
- **提升设计效率，快速定义全局色彩**：只需一次配置，即可一键应用至整个邮件模板，显著提高设计效率。
- **灵活配置，满足多样化需求**：除品牌级统一色彩，您还可为单个邮件活动或组件（如按钮、文本等）设定不同的配色方案，充分释放您的创意空间。

## 功能亮点

| 层级       | 功能说明                                                     |
| ---------- | ------------------------------------------------------------ |
| **品牌级** | 统一设定邮件模板的品牌色彩，支持浅色与深色两种模式，奠定整体视觉风格。 |
| **活动级** | 针对特定邮件活动自定义色彩，使内容更契合主题或促销氛围，强化营销效果。 |
| **组件级** | 精细控制每个组件的颜色配置，增强邮件表现力，适应不同营销创意与场景需求。 |

## 配置指南

### 设置统一品牌模板

1. 在 Genstore 商户后台，点击 **营销** -> **邮件营销**。
2. 点击页面右上角的 **创建邮件活动**。
3. 在 **邮件模板** 页面，点击 **设置**。
4. 点击 **设置品牌模版** 后的 **自定义** 按钮，在此，您可：
	 - 上传或更新品牌 Logo。
	 - 设定模版字体，注意：支持网络安全字体。
	 - 针对**深色模式**和**浅色模式**分别定义您的**主色、辅助色、点缀色和中性色**。
		 - **注意**：为确保您的邮件与品牌形象的一致性，系统会自动引用您在 [品牌资产](./start-genstore-account-brand-assets.md) 中定义的品牌颜色作为基础。
5. 在右侧预览区，您可实时查看样式变更对示例邮件的影响，从而更好把控品牌视觉表达：
	- Logo 展示位置
	- 文案、按钮颜色、字体样式
6. 设置完成后，点击 **保存**。

### 个性化邮件配色（可选）

如需满足特殊活动或节日主题，您可在品牌模板基础上进一步：

- 调整某个 **邮件活动** 的配色
- 为 **特定组件** 设定独立颜色方案（如按钮、标题模块等）

## 温馨提示

- 在设置前，建议先在  [品牌资产](./start-genstore-account-brand-assets.md)  中定义好核心品牌色。
- 调整配色时，请保持良好的对比度与视觉层次，确保阅读体验。
