# **市场设置**

在 Genstore 中，**市场详情页**是您管理各国家或地区销售设置的核心入口，支持查看和配置当前市场的语言、定价、税收、发货、支付等关键信息。无论是创建新市场，还是调整已有市场策略，您都可以通过该页面实现一站式管理与精细化配置。

## 如何访问

1. 登录 Genstore 商户后台。
2. 点击 **设置** -> **市场**。
3. 点击进入目标市场详情页。

## 市场设置

| 项目                                                    | 说明                                 |
| ----------------------------------------------------- | ---------------------------------- |
| [语言和域名](./global-market-settings-lang-domain.md)      | 设置当前市场支持的语言、展示方式和访问路径（主域、子域或子文件夹）。 |
| [产品和定价](./global-market-settings-product-pricing.md)  | 设置该市场的基础货币、是否启用当地货币、价格调整和固定价格策略。   |
| [关税和进口税](./global-market-settings-taxes-duties.md)    | 查看当前市场的关税和进口税设置。                   |
| [发货配送](./global-market-settings-shipping-delivery.md) | 查看当前市场的物流配送方案。                     |


## 店铺设置

|项目|说明|
|---|---|
|[支付](./operate-payments.md) | 设置该市场关联的付款方式。|
|[税务](./taxes-overview.md) | 显示当前市场是否启用税费设置|

## 市场表现

|项目|说明|
|---|---|
|**收入**|展示该市场在过去 30 天内的总收入。<br />计算公式：`收入 = 毛销售额 - 折扣 - 退货金额 + 税费 + 运费 + 其他费用`|
|**销售占比**|表示当前市场对应国家的收入在全部市场中的占比，帮助衡量其整体销售贡献。|
|**转化率**|在市场对应的国家下，成功转化为订单的会话数占总会话数的百分比。|
|**平均订单额**|表示该市场的平均订单金额，便于评估市场价值。<br />计算公式：`平均订单金额 (AOV) =（毛销售额 - 折扣）/订单数`|

> 数据统计时间范围为过去 30 天。
