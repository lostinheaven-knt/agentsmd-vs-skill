# 搭建店铺基础框架

本指南将帮助您完成 Genstore 店铺的基础搭建，包括创建并设置店铺、完成店铺基础信息设置、创建员工账户、添加域名、上传产品和店铺装修，为后续迁移与上线打好基础。

## 注册并创建店铺

在开始配置店铺内容之前，请先完成账户的注册。

- **注册 Genstore 账户**：访问 [Genstore 注册页面](https://accounts.genstore.ai/account/signup)，输入有效的邮箱和密码，完成账户创建。
- 注册完成后，系统将引导您进入 AI 建站流程。您可以跟随 AI 完成快速搭建，也可以点击页面右上角 **跳过** 或 **跳过并建店**，按照传统方式自由配置店铺。
- 在商户后台，点击左下角账户名称，点击 **管理账户**，进行关联第三方登录、切换后台语言等等操作。

## **填写店铺基本信息**

进入商户后台，点击 **设置** -> **通用**，填写并完善店铺基础信息。具体步骤，可参考 [店铺信息设置](./start-genstore-account-create-manage.md)

  - 店铺名称和联系方式（默认名称为 `mystore`）
  - 填写账单地址，包括法定公司名称与营业地址（用于账单和合规）
  - 设置默认货币与重量单位
  - 选择店铺时区（用于订单与报表时间）
  - 上传商店 Logo、设置品牌色彩，并添加社交媒体链接。

> **提示**：完整、准确的基础信息有助于顺利生成账单和使用系统功能。

## 添加员工账户

如果有团队成员协助运营店铺，可以为他们创建独立的后台账户，并分配权限。

**路径**：**设置 -> 账号权限**。具体步骤，可参考 [角色与权限](./start-genstore-account-roles-permissions.md)

- **创建员工账号**：为每位员工分配独立的登录账户，方便分工协作。
- **设置权限控制**：根据角色（如客服、运营、财务）设定访问范围，保护店铺敏感数据。

> **提示**：合理的权限设置不仅提高管理效率，也有助于增强店铺安全性。

## 设置和管理域名

域名是客户访问商店的重要入口。您可以使用 Genstore 提供的默认域名，也可以连接自有域名，打造独立品牌形象。

**路径**：**设置** -> **域名**。具体步骤，可参考 [域名](./global-domains.md)

- **使用默认域名**：注册 Genstore 店铺后，系统会分配一个以 `.genmystore.com` 结尾的默认域名，可 [自定义一次](./global-domains-change-default-domain-name.md)。
- **连接自定义域名**：如果您已有自有域名（如 `yourbrand.com`），可通过 DNS 设置将其 [绑定到您的 Genstore 店铺](./global-domains-custom-domain.md)。绑定前，建议在后台为 [相关页面创建 URL 重定向](./operate-store-design-navigation.md)。

> **提示**：独立域名能有效提升品牌专业度和用户信任度。

## 添加产品

您的店铺需要添加可售卖的产品信息。**路径**：**商店 -> 产品**

- **[单独创建产品](./operate-product-create.md)**：手动输入产品名称、描述、图片、价格、库存信息等。
- **[批量导入产品](./operate-product-export-import.md)**：使用 CSV 文件导入大批量产品，适合库存丰富的商家。
- **[设置产品系列](./operate-product-collections.md)**：将产品归类到不同系列中，优化商店导航与购物体验。

> **提示**：高质量的产品图片和详细的描述可以显著提升转化率。

### 使用产品 Agent 快速生成产品内容

您也可以通过 Genstore 产品 Agent Sara，上传产品图片，由 AI 智能生成产品标题、价格、分类、产品特色与描述。

> **提示**：生成内容可根据需要进一步编辑优化，确保符合品牌风格与市场定位。具体可参考 [与 Agent 的对话](./start-genstore-account-homepage#与-agent-的对话)

## 店铺装修与发布

通过灵活的装修工具，您可以设计独具特色的商店界面。**路径**：**商店 -> 在线商店 -> 模版**

- **[选择模版](./operate-store-design-themes-template-select.md)**：在模板市场选择官方或自定义模板，作为店铺的基础风格。
- **[AI装修赋能](./operate-store-design-themes-ai-decorate.md)**：您也可选择使用 AI智能装修 快速生成完整首页，自动生成包含 Banner 轮播、产品系列展示、热销推荐、品牌故事等关键楼层内容，大幅提升装修效率。
- **编辑页面结构**：使用 [可视化编辑器](./operate-store-design-themes-edit-guide.md) 调整首页、产品页、系列页、博客、自定义页面等布局与内容，或通过 [代码编辑器](./operate-store-design-themes-code-editor.md) 进行深度自定义。
- **预览并发布**：在发布前预览商店效果，确保视觉、功能和导航体验符合预期。

> **提示**：清晰流畅的页面设计能够提升客户体验，促进下单转化。

完成以上步骤后，您的 Genstore 商店将完成基础搭建，具备销售、展示和运营管理的基本能力。

 接下来，您可以继续进行 [店铺公共信息与合规配置](./start-store-setup-launch-prep.md)，为正式上线做好准备！
