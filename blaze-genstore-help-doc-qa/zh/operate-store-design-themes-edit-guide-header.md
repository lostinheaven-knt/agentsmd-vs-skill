# 标头编辑指南

标头（Header）是店铺页面顶部的固定区域，常在所有页面中保持一致显示。它通常包含品牌 Logo、主导航菜单，以及公告栏等辅助信息模块，是用户进入网站后首先接触到的视觉与操作区域。

一个设计良好的标头不仅有助于用户快速识别品牌，还能高效引导用户浏览各类页面，从而提升整体转化效率。

## 如何访问

标头为通用区域，在任一页面中均可进行配置与预览。若要编辑标头区域，请按照以下步骤进入编辑器：

1. 登录 Genstore 商家后台。
2. 点击 **商店** -> **在线商店** -> **模版**。
3. 找到目标模版，点击右侧的 **设计** 按钮，进入模版编辑器。
4. 在顶部导航栏点击页面选择器，选择任一页面类型（如主页、产品页、系列页）。
5. 在左侧操作面板中，您将在 **标头** 分区下找到相关配置区域。

## 标头结构说明

Genstore 中的标头由多个 **分区（Sections）** 与 **区块（Blocks）** 组成，结构清晰且支持灵活扩展。

### 默认结构

| 层级  | 名称        | 说明                                   |
| --- | --------- | ------------------------------------ |
| 分区  | **公告栏分区** | 默认存在的可选区域，用于展示店铺公告内容，支持文本、链接与轮播展示    |
| 分区  | **标头分区**  | 固定存在的核心区域，包含语言选择器、Logo 设置、导航菜单等通用设置项 |

## 公告栏分区

公告栏用于展示全店公告、物流说明、节日促销等内容。

- 可启用/禁用公告轮播
- 支持添加多条公告块
- 每条公告支持文字与跳转链接

## 标头分区

标头分区可进一步配置 **产品**、**产品系列** 和 **超级菜单** 三个区块：

### 产品

产品区块用于在导航中添加特定产品的快捷入口，适合推广新品或热销款。

- 支持添加多个产品入口
- 每项产品可设置跳转链接
- 可启用卡片样式展示：包括产品图、价格、标签等
- 支持快速跳转至产品页

### 产品系列

产品系列区块用于构建分类导航结构（如“女装 / 男装 / 配件”）。

- 添加某个产品系列（如“夏日精选”、“手机配件”）
- 支持封面图、系列名展示样式自定义
- 可配置跳转链接至对应集合页

### 超级菜单

超级菜单用于通过菜单项的层级结构快速组织复杂品类，并在导航中承载营销内容。

- 支持多级结构（如：男装 → 外套 → 羽绒服）
- 可在菜单中插入促销信息、Banner、推荐产品
- 兼具导航与营销作用

关于超级菜单的设置步骤，请阅读 [超级菜单](./operate-store-design-themes-edit-guide-header-mega-menu.md)

::: tip  
建议通过超级菜单在导航中呈现促销活动、精选系列或品牌横幅，在提升导航体验的同时发挥营销价值。

:::

## 默认功能组件

以下组件默认集成于标头，无需手动添加：

- 用户中心图标（登录 / 注册）
- 搜索按钮
- 购物车图标

这些组件将根据不同设备自动适配布局，确保用户操作体验一致性。

## 可扩展分区

点击 **添加分区**，您还可进一步增强标头功能，例如：

- **公告栏**：用于展示店铺重要信息（如促销、物流、服务承诺等）。
- **滚动文本**：以横向滚动方式展示短文本内容，适用于促销口号、品牌标语或用户提示信息
