# 删除元字段

当您不再需要某个元字段时，可以选择将其从 Genstore 后台中删除。删除定义后，该字段将不再出现在任何模块详情页中。

## 删除选项

删除元字段定义时，系统提供两种方式：

- **仅删除字段**：字段定义被移除，但原先填写的字段值仍保留在各模块中（不可见但未清除）；
- **删除字段及其保存的值**：连同所有填写在产品、客户等对象中的值一并删除，彻底清除该字段相关数据。

::: tip 
如果您希望保留历史数据用于备份、审计或未来扩展，建议选择 **仅删除字段**。
:::

## 注意事项

- 删除操作不可撤销；
- 删除后该字段将立即从页面中移除；
- 建议在操作前确认字段是否仍被使用。

## 删除步骤

1. 在 Genstore 后台中，转至 **设置** -> **元字段**。
2. 选择其中包含要删除的元字段定义的模块。
3. 点击要删除的元字段定义。
4. 点击 **删除**，然后选择以下选项之一：
   - 若要在商店的某些部分保留相关值，请点击 **仅删除字段**。
   - 若要完全删除元字段及其所有关联值，请点击 **删除字段及其保存的值**。
