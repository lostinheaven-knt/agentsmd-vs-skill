# 应用

Genstore 提供了一个丰富的应用生态，您可以通过官方应用和第三方开发的应用，轻松构建商店并与外部服务无缝集成。无论是优化店铺管理、提升用户体验，还是整合支付、物流、营销等服务，Genstore 应用都能为您的商店提供全面支持，助力您提升商店的智能化、自动化水平，极大提高运营效率。

- **Genstore 官方应用**：由 Genstore 开发并深度集成，确保应用与平台的兼容性和一致性。这些应用涵盖了商店建设、数据分析、支付处理等多个关键领域。
- **第三方应用**：由外部开发者提供，旨在扩展 Genstore 的功能和服务。无论是个性化定制功能，还是特定行业需求的解决方案，您都能在第三方应用中找到合适的工具。如遇问题，您可以直接联系开发者获取技术支持。
- **自定义应用**：专门针对特定 Genstore 商店开发的应用，不会在 Genstore 应用商店中列出，适用于满足特定业务需求的个性化功能。

此外，您还可以通过自定义应用，进一步拓展商店的功能。借助 [Genstore 的开放 API](https://partners.genstore.ai/zh/docs/api-rest/)，您可以访问商店数据并进行深度定制，甚至实现自动化操作。通过自定义店面，您还可以将商店扩展至其他平台，帮助您触及更广泛的客户群体，推动业务的持续增长。
