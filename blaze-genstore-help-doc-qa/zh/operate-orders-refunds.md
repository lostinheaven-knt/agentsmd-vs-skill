# 订单退款

在 Genstore 后台，您可以为整个订单或部分产品进行退款。退款将会把款项退还给客户，您可以选择不创建退货的情况下进行退款，但一旦退款已发放，您将无法再创建退货。您也可以选择在退款过程中向客户发送通知邮件。

:::tip

退款将仅退回至原付款方式。如果客户不再使用该付款方式，他们需要联系原支付渠道的银行或代理商，领取退款。

:::

## 整单退款

您可以为订单中的所有产品发放全额退款。全额退款时，您可以手动编辑退款金额，包括退货手续费、退货运费以及其他费用（如运费、税费等）。具体步骤如下：

1. 登录 Genstore 商家后台，点击 **商店** -> **订单**。
2. 点击需要退款的订单，进入 **订单详情** 页面。
3. 点击页面右上角的 **退款** 按钮。
4. 输入退款的产品数量。
5. 若要退还运费，请在 **退款金额** 区域调整相应金额。
6. 在 **退款原因** 部分，填写退款原因，该原因将显示给客户。
7. 点击 **退款** 完成退款操作，系统将自动向客户发送通知电子邮件。

## 部分退款

可以针对订单中的部分产品发放退款。您可以分次退款，直到退款总额达到订单的原始金额。发放部分退款后，订单的状态会显示为 **部分退款**。具体步骤如下：

1. 登录 Genstore 商家后台，点击 **商店** -> **订单**。
2. 点击需要退款的订单，进入 **订单详情** 页面。
3. 点击页面右上角的 **退款** 按钮。
4. 输入需要退款的产品数量。如果某些产品不需要退款，请将数量设置为 0。
5. 若要退还运费，请在 **退款金额** 区域调整相应金额。
6. 在 **退款原因** 部分，填写退款原因，该原因将显示给客户。
7. 点击 **退款** 完成部分退款操作，并确保退款金额不超过订单原始金额。
