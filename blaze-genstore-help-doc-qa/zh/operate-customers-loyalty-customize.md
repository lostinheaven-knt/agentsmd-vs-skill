# 品牌装修

品牌装修页面支持您自定义网店的视觉体验，调整色彩、文本和布局以符合您的品牌形象。通过这个页面，您可以定制会员面板、启动器和 VIP 等级卡，为您的客户提供独一无二的购物体验。

## 如何访问

1. 登录 Genstore 店铺管理页面。
2. 进入路径：**商店** -> **客户** -> **忠诚度** -> **设置**。

## 会员面板

在 **会员面板**，您可自定义您的会员面板主题和内容，以更好地符合您的品牌风格。

**操作步骤**：

- 点击进入编辑页面。
- 在 **主题** 部分，您可以选择预设的色彩主题或自定义按钮和背景色。支持纯色或渐变。
- 您可以通过拖放来调整各个模块（如邀请卡、积分、推荐等）的显示顺序和可见性。

| **模块** | **显示时间** | **说明**                                         |
| -------- | ------------ | ------------------------------------------------ |
| 邀请卡   | 登录前       | 向新用户展示加入奖励计划的优势，激励注册和参与。|
| 积分     | 登录后       | 介绍积分获取和兑换方法，促进用户活跃度。|
| 推荐     | 登录后       | 显示推荐奖励，激励用户邀请新客户。|
| VIP      | 登录后       | 展示各 VIP 等级的优势和达成条件。|
| 你的奖励 | 登录后       | 显示用户可用的奖励和详情。|
| 你的活动 | 登录后       | 显示用户的活动历史和积分变动。|

您可通过右侧的预览功能，实时预览各活动控件的展示效果，确保所有设置符合您的品牌需求后，点击**保存**以应用更改。

## 启动器

您可定制启动器的风格外观以及显示位置。

**操作步骤**：

- 点击顶部的开关按钮启用或禁用启动器。
- 自定义启动器的外观，包括类型、文字、图标、尺寸和颜色。
- 设定启动器的显示位置，选择在所有页面还是特定页面显示。

您可通过右侧的预览功能，实时预览各活动控件的展示效果，确保所有设置符合您的品牌需求后，点击 **保存** 以应用更改。

## VIP 等级卡

您可自定义 VIP 等级卡。点击 VIP 登记卡下的 **编辑** 图标进入 **VIP 等级卡** 页面：

**自定义方法**：

- 修改 VIP 等级卡的样式和内容。
- 设置等级卡的名称、背景色、字体样式和进度条字号。

您可通过右侧的预览功能，实时预览各活动控件的展示效果，确保所有设置符合您的品牌需求后，点击 **保存** 以应用更改。

## 多语言

Genstore 客户忠诚度程序支持多语言展示，帮助您更好地服务来自不同语言环境的客户，提升用户体验与参与度。

您可以翻译所有商户自定义内容，包括积分获取方式、奖励名称、VIP 等级名称及 忠诚度程序面板文案。系统将自动翻译并适配其余固定内容。

### 添加语言并自动翻译

**操作步骤**
1. 登录 Genstore 商户后台。
2. 进入路径：**商店** -> **客户** -> **忠诚度** -> **设置**。
3. 在 **设置** 页面的 **品牌样式设置** 区域，点击 **语言** 进入翻译配置页面。
4. 点击 **新增语言**，在弹窗中勾选需添加的一种或多种语言，目前支持：
	- 英语 (en-US)
	- 简体中文 (zh-CN)
	- 繁体中文 (zh-TW)
	- 西班牙语 (es-ES)
	- 马来西亚语 (ms-MY)
5. 添加完成后点击 **保存**，系统将自动初始化翻译内容。

### 编辑翻译内容

添加语言后，点击语言列表中的铅笔图标，进入翻译内容编辑页。您可以对各模块展示内容进行调整和校对，例如：

1. 点击 **欢迎（仅在登录前显示）** 后的下拉箭头。
2. 调整标题、按钮文字、提示语等词条。
3. 支持实时预览登录前/登录后展示效果。
4. 编辑完成后点击右上角 **保存**。

::: tip

某些词条包含动态参数，例如 {Store name}，它们会在页面中自动替换为您的店铺名称。请不要修改或删除参数格式，以确保内容正常显示。

:::

### 更多字段管理入口
 

除在 **设置** -> **语言** 模块外，部分展示字段还可在功能模块中单独维护：

|字段类型|配置入口|
|---|---|
|会员面板文案|品牌装修 -> 会员面板|
|积分标识名称|设置 -> 更多设置 -> 通用 -> 积分标识|
|积分奖励方式命名|积分页签 -> 获取/兑换方式模块|
|VIP 等级名称|VIP 页签 -> VIP 等级管理区域|
|推荐奖励名称|推荐页签 -> 推荐奖励配置区域|

### 设置默认语言

默认语言用于在客户切换的语言未被支持时作为忠诚度应用内容的兜底语言。建议将其设置为与店铺主语言一致的语言。

**操作步骤**

你可以在已添加的语言 **操作** 一栏点击 **设为默认** 按钮，将该语言设置为默认语言。

1. 进入语言管理页面，路径：**商店 -> 客户 -> 忠诚度 -> 设置 -> 品牌样式设置 -> 语言**。
2. 在语言列表中，找到要设置为默认的语言，点击 **默认** 下的开关按钮。 此操作会将原默认语言替换为新设置语言。

::: tip

系统当前默认语言为英文（en-US），不可删除。如需删除，请先将其他语言设为默认语言。

:::

### 删除语言

如需移除某个已添加语言：

- 在语言列表中点击对应语言右侧的删除图标。
- 系统将提示确认操作，确认后即删除该语言的展示配置。

**注意**： 谨慎删除已启用语言，避免在客户切换语言后内容无法正常匹配展示。
