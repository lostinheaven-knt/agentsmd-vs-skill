# 添加元字段定义

要使用元字段，首先需创建 `元字段定义`，即字段的模板。定义用于指定字段的名称、数据类型、验证规则及适用模块（如产品、产品规格、客户）。只有先完成定义，才能为具体对象填写元字段值。

**添加前建议阅读：**

- [元字段定义的组成部分](./settings-metafield-definitions-parts.md)：如命名空间、秘钥、验证规则等
- [元字段的数据类型](./settings-metafield-definitions-types.md)：如文本、日期、文件等

## 添加步骤

1. 在 Genstore 后台中，转至 **设置** -> **元字段**。
2. 选择您要添加元字段的模块，如 **产品**。
3. 点击 **添加定义**。
4. 输入定义 **名称**。
5. 输入 **命名空间和密钥**。
6. 【可选】在 **描述** 字段中，为您的元字段值输入描述。
7. 点击 **类型** 输入框，选择合适的字段类型。
8. 【可选】若该字段需要支持多个值，可点击选中 **值列表**。
9. 【可选】在 **验证** 部分设置规则，用于限制 **元字段填写内容**。支持的限制类型取决于字段的数据类型。
10. 【可选】在 **选项** 部分设置访问权限：
    - **读取**：允许通过 Liquid 模板或 Storefront API 读取该字段。
    - **无访问权限**：字段仅用于后台管理，不可前台访问。
11. 点击 **保存**。
