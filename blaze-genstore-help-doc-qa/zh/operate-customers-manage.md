# 客户管理

本章节旨在全面介绍如何在 Genstore 平台上管理店铺内的客户资产。您将学习到关于客户账户、客户资料管理、客户营销订阅、免税设置、客户标签管理、客户合并、客户删除及客户个人数据的删除等多方面的操作方法。

## 本节主题

- [客户账户](./operate-customers-manage-account.md)：描述客户账户定义，客户账户状态，客户账户邀请，带您了解关于客户账户相关的一系列内容。
- [添加&编辑客户](./operate-customers-manage-add-modify.md)：介绍如何在 Genstore 平台中新增或编辑客户资料，以确保客户信息的准确性和最新性。
- [删除客户](./operate-customers-manage-delete.md)：解释在何种情况下需要删除客户数据，以及如何安全地执行删除操作。
- [导入/导出客户](./operate-customers-manage-import-export.md)：介绍如何对客户数据进行批量导入或导出，这对于数据迁移和备份非常重要。
- [客户合并](./operate-customers-manage-merge.md)：介绍如何合并两个客户账户的资料，以简化客户信息管理并消除重复数据。
- [删除客户个人数据](./operate-customers-manage-erase-personal-data.md)：基于隐私政策要求，提供关于如何删除客户个人数据的详细指导。
- [导出客户个人数据](./operate-customers-manage-request-personal-data.md)：基于隐私政策要求，提供关于如何获取客户个人数据的详细指导。
- [管理客户营销订阅](./operate-customers-manage-marketing-subscription.md)：探讨如何管理客户的邮件和短信营销订阅，包括订阅操作和状态管理。
- [设置客户免税](./operate-customers-manage-tax-exemption.md)：介绍如何为特定客户设置免税状态，包括全域免税和特定地区免税操作。
- [管理客户标签](./operate-customers-manage-tags.md)：阐述如何给客户添加或删除标签，并介绍客户标签的应用场景，帮助您更有效地分组和识别客户群体。
