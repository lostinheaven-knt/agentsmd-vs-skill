# 邮件营销

邮件营销是一种直接营销策略，利用电子邮件作为与目标客户沟通和促进产品或服务的主要手段。它不仅帮助企业以低成本扩展客户基础，增强客户关系管理，还能通过定期发送更新、促销信息和个性化内容，提升客户的忠诚度和购买意愿。

Genstore 为您提供了全面的邮件营销工具，助力您的营销活动达到最佳效果。

## 如何访问

1. 登录 Genstore 商家后台，导航到 **营销**。
2. 点击页面右上角的 **选择营销方式** -> **邮件营销**。

## 营销效果分析

Genstore 通过与商店数据的直接链接，确保您能准确追踪每一封邮件的转化效果。另外，利用 UTM 参数，您还可以在第三方分析工具（例如 Google Analytics）中进一步监控邮件活动，获取更全面的营销洞察。

在 **邮件营销** 首页，您可以快速了解邮件的执行成效与其带来的营销转化，如打开率、点击率和转化率等关键数据。您还可以深入分析每一封邮件的具体效果。

系统将默认为您展示最近 30 天的邮件营销表现，包括以下关键指标：

| 指标  | 描述                                                               |
| :-- | :--------------------------------------------------------------- |
| 打开率 | 指邮件被打开的比例，计算公式为：`打开率 = (累计打开人数/累计成功送达人数) × 100%`                 |
| 点击率 | 指用户点击邮件内链接访问店铺的比例                                                |
| 退订率 | 指用户在收到邮件后选择取消订阅的比例                                               |
| 会话数 | 指用户通过邮件营销活动与您的店铺产生互动的次数，每一次互动都会被记录成一次会话。若用户在 30 分钟内没有活动，则自动开始新会话 |
| 订单数 | 可归因于邮件营销的总订单数                                                    |
| 销售额 | 可归因于邮件营销的总销售额，扣除售后部分                                             |
| 转化率 | 指访问您商店的邮件营销会话中，实际完成购买的比例                                         |

```
- 默认采用 last non-direct 归因模型
```

## UTM 参数配置

您的 Genstore 邮件营销 UTM 链接默认包含以下参数：

|UTM 参数|默认参数值|
|:---|:---|
|utm_source|genstore|
|utm_medium|email|
|utm_campaign|活动 ID|
