# 查找并筛选产品

除了强大的产品管理功能，**Genstore** 还支持智能排序和筛选功能，帮助用户快速定位符合需求的产品、服务或数据，从而提升工作效率和决策精度。

## 搜索产品

您可以通过输入产品名称或关键词，快速搜索并定位到相关产品。

### 操作步骤：

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 在搜索框中输入关键词或产品名称。
3. 按回车键，搜索结果将自动显示。

## 对产品列表进行排序

在 **Genstore** 平台中，产品列表是指所有已创建并保存的产品的集合。每当您完成一个产品的创建，它会自动加入到产品列表中，您可以随时访问和管理这些产品。产品列表展示了每个产品的基本信息，并允许用户对其进行排序、筛选或搜索：

- **创建日期**：按创建时间排序（升序或降序）
- **价格**：按价格排序（升序或降序）

**操作步骤**

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 点击产品列表上方的排序按钮。
3. 选择一个排序条件，产品列表将自动更新。

## 对产品列表进行筛选

通过筛选，您可以轻松找到需要编辑、查看或更新的特定产品。可用的筛选选项包括：

- **标题**：显示标题与输入文本匹配的产品。
- **价格**：根据产品价格区间（设置最高和最低价格）筛选。
- **库存**：筛选有库存、无库存或部分缺货的产品。
- **产品厂商**：筛选特定厂商的产品。
- **标签**：筛选具有指定标签的产品。
- **状态**：筛选处于不同状态（已存档、上架、草稿、未上架）的产品。
- **销售渠道**：筛选包含或排除特定销售渠道的产品。
- **市场**：筛选在特定市场中出售的产品。
- **产品类别**：筛选属于指定类别的产品，支持选择多个类别。
- **产品系列**：筛选属于某一产品系列的产品。
- **礼品卡**：筛选适用于您商店的礼品卡产品。
- **元字段**：筛选具有特定元字段值的产品。

**操作步骤**

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 点击筛选图标。
3. 选择要筛选的条件并设置对应的值，产品列表将自动更新。

:::tip

当设置多个筛选条件时，系统将只显示符合所有条件的产品。

:::
