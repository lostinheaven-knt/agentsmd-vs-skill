# Genstore Growth 套餐

Genstore Growth 套餐为您提供全面的 AI 运营支持，包含 Genstore Lite 套餐的所有功能，并且提供更低的交易手续费，更多的Dropshipping产品推送，帮助您高效管理和优化网站运营。

## 积分

- 套餐内包含 200 点月度积分。
- 每天登陆系统，可自动获得 5 点每日积分。
- 支持单独购买积分加油包来获得更多积分。

## **套餐功能**

除通用功能和 Genstore Lite 套餐所提供的功能外，Genstore Growth 套餐还包括以下附加功能：

### **Genstore AI**

| 功能            | 描述                                   |
| ------------- | ------------------------------------ |
| **开店 Agent**  | 帮助您快速完成店铺基础配置，如域名、支付到物流等设置，确保店铺合规上线。 |
| **产品 Agent**  | 通过智能选品与产品创建工具，高效管理您的产品目录。            |
| **装修 Agent**  | 快速执行首页生成与主题切换任务，打造高转化的品牌视觉。          |
| **营促销 Agent** | 精准触发 AI 营销活动与折扣生成逻辑，提升流量转化。          |
| **客服 Agent**  | 提供 7×24 小时全天候技术指导与经营建议的专属商户服务专家。     |
| **数据 Agent**  | 实时分析店铺流量、转化率及销售表现，为您提供直观的数据洞察与经营建议。  |

### **其他套餐功能**

| 功能              | 描述                                            |
| --------------- | --------------------------------------------- |
| 有竞争力的交易手续费      | 提供较低的交易手续费，具体费用请查看 Genstore 定价页面，了解所在区域的收费标准。 |
| Dropshipping 产品 | 支持最多推送 100 个Dropshipping产品至店铺。                |
| 员工账号            | 支持最多 10 个员工账号，帮助您高效管理团队。                      |
| 库存地点            | 支持最多 15 个用于管理库存的地点。                           |
| 多市场             | 包含 3 个市场，拥有全球化销售能力，从单一店铺管理全球销售。               |
