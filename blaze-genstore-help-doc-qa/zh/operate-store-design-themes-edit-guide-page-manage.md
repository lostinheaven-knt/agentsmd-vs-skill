# 管理页面模版

本页面介绍如何在 Genstore 装修编辑器中使用 **页面管理** 功能，进行页面模版的创建、绑定、切换与维护操作。通过页面管理，商户可为不同类型的页面指定专属模版，实现页面内容与布局的个性化呈现。

## 页面管理入口

1. 登录 Genstore 商家后台，点击 **商店** -> **在线商店** -> **模版**，点击目标模板右侧的 **设计**。
2. 在左侧功能导航栏中，点击第一个图标（页面管理）。
## 页面管理区域说明

页面管理区域主要由以下几部分组成：

- **页面类型列表**：展示当前模板支持自定义模版的所有页面类型（如主页、产品、产品系列、博客等），部分页面类型支持展开查看子模版与已分配页面。
- **模版列表与创建**：每种页面类型下支持创建多个模版。点击 **创建模版** 可新建模版并指定模版名称。
- **模版分配预览**：在每个模版下，可查看当前已绑定的页面数量（如 `已分配 5 件产品`），支持快速跳转。
- **搜索栏**：通过页面顶部的搜索框，可模糊查找模版名称或未分配的页面，提升管理效率。

## 快速创建页面模版

1. 在页面管理中，展开所需页面类型（如产品页）。
2. 点击 **创建模版**，输入模版名称，并选择基于的页面模版，点击 **确定**。
3. 系统将自动跳转至新建的模版页面，您可以在此对模版内容进行编辑。
4. 编辑完成后，点击 **保存**。
## 模糊搜索并绑定具体内容页面

1. 在页面管理顶部搜索框中输入关键词，系统将返回匹配的页面名称或模版名称。
2. 例如针对产品模版，点击 **关联产品**，在弹窗中选择目标产品并完成绑定。

除在当前页面外，您也可以在具体的页面、产品、产品系列、博客文章或博客集合页面为其关联模版，具体可阅读 [关联模版与内容](./operate-store-design-themes-edit-guide-page-link-contents.md)

## 使用建议

- **合理命名模版**：为不同用途的模版使用清晰命名，目前支持支持数字、字母和 "-"。
- **充分利用模版复用能力**：可为多个页面绑定相同模版，实现统一样式的快速复制。
- **预览前后效果**：绑定模版后可随时使用编辑器右上角的“预览”功能查看实际效果。
