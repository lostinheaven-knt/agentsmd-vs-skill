# 产品推荐

产品推荐是提升客户购物体验和增加销售的有效工具。通过智能推荐系统，商家可以向客户展示相关产品或个性化产品，增加产品曝光度和销量，尤其适用于电商平台，能够显著提高转化率和客户满意度。

  - 提升用户购物体验，通过个性化推荐提高转化率与客单价。
  - 基于用户行为和产品数据，帮助商家实现精准营销。
  - 支持多场景推荐（如首页、购物车、产品详情页等）。

## 产品推荐方式

应用提供了 10 种产品推荐方式。每个推荐方式都可以设置设定不同的展示配置页面，相应推荐规则以及排序显示方式，并且可以根据需要进行数据过滤。

| **推荐类型**   | **描述**                                                     | **业务目标**                                                 |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **畅销产品**   | 基于销量数据评估热销产品，展示在多个页面，如首页、产品详情页等。| 提高畅销产品的曝光度，增加购买转化率。|
| **大家都在看** | 展示顾客当前在网站上最受欢迎的产品，基于销量数据、浏览行为和顾客评价推荐。| 吸引顾客关注高人气产品，提高销售机会。|
| **最近浏览**   | 自动记录并展示顾客最近浏览过的产品，基于浏览历史推荐相关产品。| 提升顾客的购物体验，促使他们回到之前感兴趣的产品，增加复购率。|
| **即将售罄**   | 提醒顾客某个产品即将售罄，产生紧迫感，促使顾客更快购买。| 创造购买紧迫感，提高产品的即时销量。|
| **智能推荐**   | 基于顾客的行为、购买历史和浏览记录，自动推荐最相关的产品。| 提供个性化推荐，提升产品的匹配度和购买转化率。|
| **精选产品**   | 商家根据特定策略（如高利润产品）精心挑选的产品进行展示。| 增加高利润产品的曝光度，提升商家利润，提供个性化购物体验。|
| **流行趋势**    |  展示当前店铺中浏览量、点击量、加购量或销量较高的产品，基于近期数据浏览次数增长率、下单增长率。   |  促进转化、提高商品曝光、缩短决策时间  |
| **最受好评**   |  展示评价星级高、评论数多的产品，强调口碑和顾客满意度   |  增强信任感   |
| **新品上市**  |  展示店铺最近新增的产品，基于发布日期（如最近 120 天上架）进行更新。   |  提升新品曝光，鼓励回访顾客用户下单   |
| **关联推荐** |  基于当前浏览或购买的商品，推荐同类、可搭配、组合购买或相似风格的其他商品。可由人工设置或算法推荐。   |   提升客单价，减少跳出率  |

## 安装及卸载应用

1. **安装应用**：在 Genstore 商户后台或 Genstore 应用市场中搜索 **产品推荐**，并安装页面提示完成安装。
2. **固定应用**：在应用列表中找到产品推荐应用，点击 **固定** 按钮，方便快速访问该应用，进行后续的管理和操作。
3. **卸载应用**：若您不再需要产品推荐应用，可以通过在应用列表中点击 **卸载** 按钮将其移除。请注意，卸载后应用的配置和数据将被清除。
4. **重新安装应用**：如果您之后需要重新安装产品推荐应用，您可在 **应用** 页的 **已卸载** 页面找到当前应用并安装。安装后需要进行用户数据授权。产品推荐应用将访问您的店铺数据和个人信息，包括会员、店铺及合作伙伴的信息，确保推荐的准确性和个性化。

## 创建产品推荐

不同推荐策略的设置大致相同，本节将以 **畅销产品** 为例进行说明。

  1. 登录 Genstore 商户后台，导航至 **应用**，在应用列表中找到产品推荐应用（Product Recommendations），点击 **打开应用**。
  2. 点击 **创建推荐**，根据您的需求选择 **产品推荐方式**（以畅销产品为例）。
  3. 启用推荐方式：点击 **状态开关** 启用当前推荐方式。
  4. 在主题装修编辑器中添加展示配置分区，选择相应的展示位置（如产品详情页、购物车页等）。导航至 **商店** -> **在线商店** -> **模版**，点击 **设计** 按钮。在 **产品** -> **模版** 部分，点击 **添加分区**，在 **应用** 页签，选择 **畅销产品**。预览效果并保存。完成配置后，畅销产品推荐将按配置出现在对应的客户端页面。
  5. [选填] 推荐配置，如同时勾选选择指定产品和智能推荐，智能推荐的优先级高于商户指定推荐的产品。
   	- **优先显示指定产品**，将优先推荐指定产品。您可进一步点击 **选择产品**，完成推荐产品配置。
   	- **启用智能推荐**，系统根据智能推荐算法推出的产品将优先展示。
  6. [必填] 展示配置：选择一个展示位置，如产品详情页、产品系列列表页、购物车、购物车抽屉页、404 页面。
  7. [必填] 推荐规则
	   - **设置过去 N 填内最畅销产品**：默认值 7 天，设置范围（1-365 天）
	   - **产品排序方式**：推荐产品按销量、创建日期排序
		:::tip
		 不同推荐方式的推荐规则略有差异，您可根据页面提示进行配置。
		:::
  8. [选填] 过滤设置
	   - **将售罄产品显示在推荐产品**：当勾选推荐产品包含售罄产品。
	   - **价格**：设置推荐产品的价格限制，指定待推荐产品的价格区间，价格取产品销售价格，小数点后 4 位。
	   - **标签**：选择产品标签，希望包含的标签以及希望排除的标签，支持多选。
	   - **库存数量**：设置推荐产品的库存限制，设置最大库存数量和最小库存数量。
