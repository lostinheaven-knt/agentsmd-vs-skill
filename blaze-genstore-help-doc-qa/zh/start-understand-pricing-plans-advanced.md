# **Genstore Scale 套餐**

Genstore Scale 套餐为您的业务增长提供强大支持，包含 Genstore Lite 和 Genstore Growth 套餐的所有功能，助您高效扩展业务。

## 积分

- 套餐内包含 700 点月度积分。
- 每天登陆系统，可自动获得 5 点每日积分。
- 支持单独购买积分加油包来获得更多积分。

## **套餐功能**

除了所有套餐的通用功能外，Genstore Scale 套餐还包括以下功能：

### **Genstore AI**

| 功能            | 描述                                   |
| ------------- | ------------------------------------ |
| **开店 Agent**  | 帮助您快速完成店铺基础配置，如域名、支付到物流等设置，确保店铺合规上线。 |
| **产品 Agent**  | 通过智能选品与产品创建工具，高效管理您的产品目录。            |
| **装修 Agent**  | 快速执行首页生成与主题切换任务，打造高转化的品牌视觉。          |
| **营促销 Agent** | 精准触发 AI 营销活动与折扣生成逻辑，提升流量转化。          |
| **客服 Agent**  | 提供 7×24 小时全天候技术指导与经营建议的专属商户服务专家。     |
| **数据 Agent**  | 实时分析店铺流量、转化率及销售表现，为您提供直观的数据洞察与经营建议。  |

### **其他功能**

| 功能              | 描述                                            |
| --------------- | --------------------------------------------- |
| 有竞争力的交易手续费      | 提供最低的交易手续费，具体费用请查看 Genstore 定价页面，了解所在区域的收费标准。 |
