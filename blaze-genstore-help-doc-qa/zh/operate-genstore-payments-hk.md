# 香港特别行政区商户申请指南

在香港特别行政区使用 **Genstore Payments** 前，您需要了解相关的个人信息和文件要求，以满足法规要求并防范金融犯罪。本文将帮助您：

1. 认清您的主体类别
2. 了解不同主体的文件要求
3. 确保银行账户及最终受益所有人（UBO）信息符合要求

## 主体类别差异

根据不同国家和地区的法律法规，您需要提供有关您和您企业的信息。这些信息将帮助 **Genstore（后称“我们”）** 确认商店的身份，包括公司实体的所有受益人，并确保其遵守反洗钱、反恐怖主义融资和其他金融犯罪防范的规定。随着法律法规的变化，**Genstore** 和合作伙伴可能会要求您提供额外的信息。

在香港特别行政区，商户可选择以下主体注册 **Genstore Payments**：

| 类别            | 独资企业（Sole Proprietorship）                 | 有限公司（Private Limited Company, Ltd） / 无限公司（Unlimited Company）            | 合伙企业（Partnership）                                                  |
| ------------- | ----------------------------------------- | ----------------------------------------------------------------------- | ------------------------------------------------------------------ |
| **法律地位**      | 不是独立法人，业主与企业为同一主体                         | 独立法人，公司与股东/董事分离                                                         | 不是独立法人，合伙人共同承担责任                                                   |
| **责任承担**      | 业主无限责任                                    | 股东有限责任（以出资额为限），董事承担法定责任                                                 | 普通合伙人无限责任，有限合伙人有限责任                                                |
| **税务处理**      | 个人所得税申报：业主直接申报企业收入（IR56B/利得税表格）           | 公司利得税（Profits Tax Return, BIR51/52/54），另需提交周年申报表（NAR1/AR1）              | 需提交合伙税务申报（Profits Tax Return, BIR52），利润/亏损分配到合伙人个人税表               |
| **注册名称示例**    | - Chan Tai Man<br/>- DBA: **Chan’s Café** | - **TechVision (HK) Limited**<br/>- **Bright Future Unlimited Company** | - **Chan & Lee Partnership**                                       |
| **识别关键点（HK）** | - 商业登记证（BR）<br/>- 银行账户名 = 业主姓名            | - 公司注册证明书（CI）、商业登记证（BR）、组织章程（M&A）<br/>- 银行账户名 = 注册法定名称                  | - 商业登记证（BR）<br/>- 合伙协议（Partnership Agreement）<br/>- 银行账户名 = 合伙组织名称 |

> **提示**：认清主体类别后，再根据类别准备相应的文件，有助于顺利完成 Genstore Payments 的进件。

## 共通要求

### 最低年龄要求

- 使用 **Genstore Payments** 的用户必须年满 **18 岁**，确保具备签署合同和管理商业交易的法律能力。
### 银行账户要求

- 银行账户必须在香港开设，并为港币完整支票账户。
- 银行账户名称必须与注册主体一致：
  - 独资企业 → 银行账户名 = 个人姓名
  - 公司/合伙企业 → 银行账户名 = 注册法定名称
- 银行账户必须符合 **本地自动清算资格**。

### 最终受益所有人（UBO）要求

| 对象               | 标准                                                         | 所需人数                                                |
| ------------------ | ------------------------------------------------------------ | ------------------------------------------------------- |
| 通过所有权识别 UBO | 直接或间接拥有公司 25% 以上股份、表决权或其他权益的个人。    | 0 至 4 人。**Genstore** 需要所有符合标准的人的信息。    |
| 通过控制权识别 UBO | 行使公司最终有效控制权的个人。如果找不到此人，则必须识别高级管理人员。 | 至少 1 人（若没有符合所有权标准的人，或在香港运营时）。 |
| 签字人             | 法定代表人或授权代表，可以代表组织与 **Genstore** 签订协议。 | 至少 1 人。                                             |

## 各主体文件要求

**说明**：请先确认您的主体类别，再按对应类别准备文件，以确保顺利完成进件。

### 独资企业（Sole Proprietorship）

- **公司/业务文件**：
  - 商业登记证（BR）
  - 税务文件（利得税表格、BIR52）
  - 如有 DBA，可提供注册证明
- **个人文件**：
  - 香港身份证（HKID）/ 护照
  - 居住地址证明
  - 税号证明
- **银行文件**：
  - 银行对账单
  - 开户证明信（Bank Reference Letter）
  - 支票

> 若 Genstore 无法自动验证，可能会要求提供创立文件或最近税务申报单。

### 公司（有限公司 / 无限公司）

- **公司文件**：
  - 公司注册证明书（CI）
  - 商业登记证（BR）
  - 组织章程（M&A）
  - 周年申报表（NAR1/AR1）
  - 税务文件（BIR51/52/54）
- **个人文件**：
  - 董事及实益拥有人身份证/护照、居住地址证明
  - 所有权证明（如股东名册）
- **银行文件**：
  - 银行对账单
  - 开户董事会决议（Board Resolution）
  - Certificate of Incumbency

> 若无法确认最终所有权，需提供 **所有权和控制结构图**（含签名、职务及日期，6 个月内有效）。

### 合伙企业（Partnership）

- **公司文件**：
  - 商业登记证（BR）
  - 合伙协议（Partnership Agreement）
  - 税务文件（BIR52）
- **个人文件**：
  - 所有合伙人身份证/护照
  - 居住地址证明
  - 合伙人权益证明
- **银行文件**：
  - 银行对账单或开户证明信
  - 部分银行要求合伙协议

## 所有权和控制结构证明

如 Genstore 无法通过上述文件验证最终受益所有人（UBO），需提供 **所有权结构图**，必须包含：

- 所有法律实体及最终受益所有人（UBO）
- 每个实体的官方名称、法律结构、注册国家/地区
- 实体间所有权比例及控制关系
- 如何满足识别 UBO 的标准（如个人持有 25% 股份）
- 签名、职务及签署日期（不超过 6 个月）
