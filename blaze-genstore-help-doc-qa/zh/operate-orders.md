# 管理订单

订单是客户在商家店铺上创建并提交的购买请求，包含客户信息、购买产品、支付状态和配送信息等内容。

在 Genstore 商家后台，您可以通过 **订单页面** 管理所有订单，执行查看、编辑、处理、打印、导出订单等操作，创建草稿订单、获取款项、收集用户弃单等。如果客户需要售后服务，您还可以为用户提供退货、退款和换货服务。

## 搜索和查看订单

登录 Genstore 商家后台，点击 **商店** -> **订单**，进入 **订单列表**。**订单列表** 默认展示全部订单数据，并按照订单的创建时间倒序排列（最新的订单最先显示），在订单列表区域，您可：

- 点击 **订单号** 可进入 **订单详情** 页面，查看订单的详细信息。
- 点击 **创建时间**，可根据订单的下单时间进行正序或倒序排序。
- 点击 **产品**，可弹出产品的详细信息（包括产品图片、名称、单价和购买数量）。

在**订单列表**区域上方的功能区，您可以：

- 点击 **`+`** 来自定义和调整订单列表的视图，以更匹配您的业务需求和场景。
- 点击 **搜索按钮**，根据 订单号、产品名称、客户名称进行订单查询。
- 点击 **自定义显示列** 图标，可根据业务需求调整表头的展示内容和顺序。
- 点击 **筛查**，按日期、渠道、支付状态、配送方式、发货状态、产品类型、标签等条件筛选订单。
- 点击 **排序** 按钮，可以按照 **创建时间（升序）** 或 **创建时间（降序）** 进行排列。

## 编辑订单详情

点击 **订单号** 可进入 **订单详情** 页面，您可以对以下内容进行编辑：

- 备注：点击 **编辑图标** 加备注信息，编辑后点击 **保存** 即可生效。
- 标签：输入关键词搜索标签。入关键词搜索标签，如没有匹配项，可直接在此进行添加操作，保存后当前标签即可生效；如希望删除标签，点击标签右上角的 **x** 即可删除。
- 客户联系信息：点击 **编辑图标** 编辑和更换订单联系人，编辑后点击 **保存**，订单联系人将实时更新。
- 送货地址：点击 **编辑图标** 编辑收货地址，编辑后点击 **保存** 即可生效。

## 管理订单

在 **订单列表** 区域，您可勾选订单前的复选框进行订单管理：

- **未支付状态**：勾选目标订单，点击 **标记已支付**，手动将订单的支付状态更改为 **已支付**，表示订单已完成付款。
- **已授权状态**：勾选目标订单，点击 **捕获支付**，将已授权的支付款项转入您的账户。
- **未发货状态**：勾选目标订单，点击 **标记已发货**，手动将订单的支付状态更改为 **已发货**，表示订单已完成发货。更多信息，可参考 [手动发货](./operate-orders-shipping.md)

### 批量管理订单

您还可以在 **订单列表** 中勾选多个订单，进行以下批量操作：

- **批量标记已支付**：适用于多个 **未支付** 的订单。操作成功后，状态将更新为 **已支付**，失败的订单状态保持不变。
- **批量标记已发货**：适用于多个 **未发货** 或  **部分发货** 的订单。成功后状态更新为 **已发货** 或 **部分发货**，失败则不变。

## 打印订单

您可在 **订单列表** 区域或 **订单详情页** 打印订单：

- **订单列表**：勾选订单前的复选框，选中订单后点击 **打印订单**，弹出打印页面后点击 **打印** 即可打印订单信息。在弹出的订单打印预览页，确认信息无误后，点击 **打印**。
- **订单详情**：点击页面右上角的 **更多** -> **打印订单**，弹出打印页面后点击 **打印** 即可打印订单信息。在弹出的订单打印预览页，确认信息无误后，点击 **打印**。

## 修改订单物流信息

您可在订单中对已发货订单或者部分发货订单中填写的物流信息进行修改。
::: tip
每条物流信息仅支持修改一次，请谨慎操作。
:::

1. 登录 **Genstore 后台**，点击 **商店** -> **订单**。
2. 在 **订单列表** 页面，确定您需要修改物流信息的订单，点击进入**订单详情** 页面。
3. 点击 **修改物流信息** 按钮，进入到 **修改物流信息** 窗口。
4. 确定需要修改的物流信息，输入新的物流单号和物流承运商信息。
5. 点击 **保存** 按钮，完成修改。
