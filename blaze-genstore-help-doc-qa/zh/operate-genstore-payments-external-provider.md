# 配置离站收银台

当您使用离站收银台支付渠道时，消费者在结账页提交支付时，会被引导至对应支付渠道的第三方结账台，在第三方结账台中完成支付后会被再次重定向回您的商店。

## 空中云汇信用卡支付配置手册

**前提条件**：您需要联系空中云汇的客户经理完成支付进件，并开启开发者权限。

:::tip

空中云汇信用卡支付目前仅支持在第三方结账处接受付款。

:::

**操作步骤**

1. 登录您的 Genstore 管理员账号，进入 **设置** -> **支付**。
2. 点击 **更多支付方式** 下的 **添加支付方式** 按钮。
3. 在 **其他支付方式** 页签，选择 Airwallex Credit card，进入配置页面，填写 CLIENT ID、API KEY、Secret key，选择需要展示的卡组图片。
4. 点击 **保存**。

### 获取空中云汇账户托管参数步骤

**获取 ClientID 和 API KEY 信息**

1. 登录您的空中云汇管理员账号，进入 **Account** -> **Developer**。
2. 在 **API Keys** 菜单下找到 **Admin API Key**，获得 **CLIENT ID** 和 **API KEY**。

**获取 SECRET KEY**

1. 登录您的空中云汇管理员账号，进入 **Account** -> **Developer**。
2. 在 **Webhooks** 菜单下点击 **Add Webhook**。
3. 在 **Summary** 表单中填写 **Notification URL**，并勾选所有 **Event**，点击 **Submit** 按钮。
:::tip
	您可联系 Genstore 技术支持获取 Notification URL。
:::
4. 回到 **Webhooks** 页面，获得对应的 SECRET KEY。

### 消费者体验

配置成功后，您可以进入您的商店，选择任意产品进行购买体验支付流程

#### 结账页

在结账页，您可以看到 Airwallex Credit Card 支付方式已经激活，选择该方式，点击 **立即支付** 按钮，会加载当前页面进入空中云汇收银台。

#### 第三方支付页

在空中云汇支付页面，选择指定的支付方式，填充相关信息，点击提交，即可完成支付并重定向回到商店的结账成功感谢页。

## PayPal 支付配置手册

**前提条件**：您必须拥有一个经过验证的 PayPal 企业账户。

**操作步骤**

1. 登录您的 Genstore 管理员账号，进入 **设置** -> **支付**。
2. 点击 PayPal 的 **连接** 按钮。
3. 我们会将页面引导至 PayPal 登录界面，按照 PayPal 的指引登录成功后完成授权。
4. 页面将重定向回到 Genstore 商户后台，表示授权成功。

### 绑定常见问题

#### 关于“PayPal 邮箱未确认”的提示

如果您在 PayPal 注册时使用的邮箱尚未通过官方认证，在 Genstore 绑定 PayPal 账户时，可能会出现此类错误提示。

建议您登录 PayPal 后台，检查该邮箱的验证状态。同时，请查看该邮箱的收件箱，确认是否收到过来自 PayPal 的验证邮件。

#### 关于“PayPal 功能受限”的提示

如果您的 PayPal 账户功能受限，可能会导致无法在 Genstore 成功绑定。此类情况建议您联系 PayPal 客户经理，并提供您的 PayPal 账号，以了解具体的限制原因。

### 消费者体验

配置成功后，您可以进入您的商店，选择任意产品进行购买体验支付流程。

#### 产品详情页

在产品详情页，您可以看到 PayPal Express Checkout 支付按钮，点击 PayPal 按钮，打开 PayPal 支付表单，进行支付。

#### 购物车抽屉和购物车页面

在购车抽屉或者购物车页面，您可以看到 PayPal Express Checkout 支付按钮，点击 PayPal 按钮，打开 PayPal 支付表单，进行支付。

#### 结账页

在结账页，您可以看到在结账页顶部或者结账页支付方式区域看到 Paypal Express Checkout 支付方式已激活。

选择该方式，点击 **立即支付** 按钮，会在当前页面加载 Paypal 收银台页面，支付成功会将您的消费者重定向回到结账成功感谢页。

## PricPay 支付配置手册

Genstore 已支持集成 PricPay 作为外部支付方式，消费者可在结账页选择 PricPay 并完成支付。

:::tip

- **客户在使用 PricPay 结算时需提供手机号**：PricPay 要求所有买家在结账时填写手机号。启用 PricPay 前，请先在店铺后台 [**设置 -> 结账**](./settings-checkout#电话号码) 中开启手机号收集（设为 **展示，必填**），否则可能导致支付失败。
- PricPay 目前仅支持在其商户后台发起退款操作，Genstore 后台暂不支持退款管理。  
- 为了同步退款状态并确保售后流程顺畅，您需要额外设置退款回调地址。
:::

**操作步骤**

1. 登录您的 Genstore 管理员账号，进入 **设置** -> **支付**。
2. 点击 **更多支付方式** 下的 **添加支付方式** 按钮。
3. 在 **其他支付方式** 列表中，选择 PricPay，进入配置页面，填写 Merchant No、APP key，选择需要展示的卡组图片。
4. 点击 **保存** 完成配置。**注意：** 在保存前，请确认已在 **设置 -> [结账页](./settings-checkout.md)** 中开启手机号收集（**展示，必填**），否则 PricPay 支付可能失败。
5. 登录 PricPay 的商户后台，点击 **基本信息**。
6. 在 **回调地址** 栏填入以下链接：  
    `https://pay.genstore.ai/webhook/pricpay/refund`
