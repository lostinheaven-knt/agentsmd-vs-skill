# 管理 Genstore 账号

您可通过登录 [Genstore 商户后台](https://admin.genstore.ai/admin)，访问并管理您的 Genstore 账号。

## 功能概览

Genstore 提供以下账户管理操作：

- 更新或删除您的账户信息，如姓名、邮箱与头像。
- 连接 Apple、Facebook 或 Google 等第三方登录方式。
- 查看与该账号关联的所有商店。
- 选择 Genstore 商户后台的显示语言。

## 更新账号详情

您可以上传或编辑个人资料照片、更新您的姓氏、名字、电子邮件地址，以及重置账号密码。

#### 步骤：

1. 登录 Genstore 商户后台，点击左下角已登录的账号。
2. 点击 **管理账户**。
3. 在 **详细信息** 部分，更新以下任意详情：
   - 如希望添加或更换资料照片，点击 **上传照片**。
   - 如希望更新姓名，请在对应字段中输入新的内容。
4. 点击 **保存**。

## 验证或更换登录邮箱

如邮箱状态显示为“未验证”，请点击页面顶部的 **重新发送验证电子邮件** 按钮，前往邮箱完成验证。

### 修改登录邮箱

若需更换邮箱：

1. 点击邮箱后的铅笔图标。
2. 在弹窗中输入新邮箱与密码。
3. 点击 **发送验证码**，并在邮箱中完成验证。
4. 验证完成后，点击 **保存**。

### 重置密码

1. 点击邮箱旁的 **重置密码** 按钮。
2. 系统将向注册邮箱发送重置密码邮件。
3. 打开邮件并点击 **重置密码**。
4. 设置新密码后即可完成更新，并使用新密码登录 Genstore 后台。

## 连接外部登录服务

为提升账户安全性与登录便捷性，您可以将您的 Genstore 账号连接到外部登录服务，如 Apple、Facebook 或 Google。

::: tip

连接前，请先完成邮箱验证。

:::

#### 步骤：

1. 在 Genstore 商户后台，点击左下角已登录的账号。
2. 点击 **管理账户**。
3. 在 **登录服务** 部分，点击外部登录服务后的 **连接**。目前支持 **Apple**、**Facebook** 和 **Google**。
4. 根据页面提示完成授权。

## 查看关联商店

在 **管理账号** 页面，您可查看和访问和您账号关联的所有商店。

步骤：

1. 在 Genstore 商户后台，点击左下角已登录的账号。
2. 点击 **管理账户**。
3. 在 **店铺** 部分，点击 **查看所有商店**。
4. 在 **商店列表** 页面，您可查看和您账号关联的所有商店，点击即可访问相应的商店后台。

## 更改后台语言

更改账户语言仅会更改 Genstore 商户后台的语言，不会影响您的在线店面的语言。若要更改在线店面的语言，请参考 [语言管理](./settings-languages.md)

#### 步骤：

1. 在 Genstore 后台中，点击左下角已登录的账号。
2. 点击 **管理账户**。
3. 在 **语言** 部分，点击 **语言** 下拉菜单，然后从列表中选择一种语言。
4. 点击 **保存**。

## 删除账号及数据

如您需要删除您在 Genstore 的账号及相关数据，请务必确保您账号关联的所有 Genstore 店铺账单已结清且完全关闭。接收到您的账号及数据删除请求后，Genstore 处理请求可能需要最多 30 天。您可以根据下述步骤，来开始删除流程。

#### 步骤：

1. 在 Genstore 后台中，点击左下角已登录的账号。
2. 点击 **管理账户**。
3. 在 **账号及数据** 部分，点击 **删除账号及数据** 来开始删除流程。

::: warning

删除账户及数据后，数据不可恢复，操作前请务必谨慎！

:::
