# 管理产品 SKU

在 Genstore 中，您可以为产品设置多个规格版本（SKU），以覆盖尺寸、颜色、材质等不同属性组合。通过后台的规格管理功能，您可以灵活地进行 **单独编辑**、**批量调整**、**禁用** 或 **删除** 操作，以应对不同销售阶段的管理需求。

## 如何访问

1. 登录 Genstore 商户后台，点击 **商店** -> **产品**。
2. 找到并点击目标产品，进入产品编辑页。
3. 滚动至 **规格明细** 区域，即可查看和管理产品 SKU。

## 单独编辑 SKU 信息

在产品编辑页面的 **规格明细** 区域，您可以针对任意规格的信息进行逐项编辑，如：

- 图片
- 价格 / 原价 / 成本价
- SKU 编码、条码
- 库存（库存数量、库存跟踪、缺货是否可继续购买）
- HS 编码、原产国/地区
- 重量
- 元字段

## 调整规格显示字段

Genstore 支持对规格字段进行个性化配置：

1. 在 **规格明细** 区域，点击右上角的 **列** 图标。
2. 在下拉菜单中，勾选/取消勾选显示字段。
3. 拖拽字段名称以自定义它们的显示顺序。
4. 系统将按产品保存您的偏好设置。

> **注意**：自定义配置将按产品维度记录的，您可以为不同产品设置不同的显示列。

## 批量编辑 SKU 信息

如需对多个规格同时修改相同字段（如统一定价或库存），可使用批量编辑功能。

### 操作步骤

1. 在 **规格明细** 区域勾选需要批量操作的 SKU（支持多选）。
2. 点击右上角 **批量编辑**。
3. 在弹出的批量编辑窗口中，修改所需字段。
4. 若批量编辑页面中未展示所需字段，可点击右上角 **列** 按钮添加显示项。
5. 修改完毕后，点击 **保存**。

**提示**：除了批量编辑器外，Genstore 还提供了快捷编辑入口。您可对选中的 SKU 快速批量修改价格、库存、图片、编码等核心信息，无需逐一进入规格详情页，管理更高效便捷。


## 批量编辑 SKU 价格

对于多规格产品，Genstore 支持批量编辑价格。

1. 在规格明细表中勾选需要修改的 SKU，点击 **编辑价格**。
2. 系统提供两种方式：
    - **调整价格**：为所选规格直接设定新价格。
    - **基于原价调整**：在现有价格基础上加减固定金额或百分比。例如：
        - `-5` → 每件减少 $5
        - `+15%` → 每件增加 15%

## 删除/禁用 SKU

在产品销售周期中，您可能需要停止某些规格的销售。Genstore 提供两种方式应对：

| 操作     | 适用场景           | 效果                                    |
| -------- | ------------------ | --------------------------------------- |
| 删除 SKU | 永久停售、不再使用 | 永久移除该 SKU，信息不可恢复            |
| 禁用 SKU | 临时停售、断货等   | 隐藏该 SKU 但保留其信息，可随时恢复上架 |

### 删除 SKU

如某个 SKU 不再销售，您可选择将其彻底删除。删除后，该 SKU 将无法在前台或后台访问，且相关数据不可恢复。

#### 操作步骤

1. 登录 Genstore 商户后台，点击 **商店** -> **产品**。
1. 找到并点击目标产品，进入产品编辑页，找到至 **规格明细** 区域。
2. 勾选需要删除的 SKU（支持多选）。
3. 点击右上角 **更多操作（···）** → **删除 SKU。**
4. 在弹窗中确认操作，点击 **确定。**

#### 注意事项

- 删除后，SKU 所有相关信息（库存、定价、条码等）将被永久清除。
- 如果产品仅剩一个 SKU，系统将禁止删除操作。
- 删除的 SKU 不会出现在顾客可见的页面中。

### 禁用 SKU

如果某个 SKU 只是**暂时停止销售**，建议使用「禁用」操作，以便后续随时恢复上架。

#### 功能说明

- 禁用后，SKU 在后台仍可见，前台隐藏，无法被顾客购买。
- SKU 的所有配置信息仍将保留，可通过「启用 SKU」恢复展示与销售。
- 适用于：断货、季节性停售、限量控制等场景。

#### 操作步骤

1. 登录 Genstore 商家后台，点击 **商店** -> **产品**。
2. 找到并点击目标产品，进入产品编辑页，找到至 **规格明细** 区域。
3. 您可通过任一方式禁用 SKU：
    - 勾选单个 SKU 后，点击 **更多操作（···）** → **禁用 SKU。**
    - 点击标题进入规格详情页，在规格详情页，点击页面右上角的 **禁用** 按钮。
4. 被禁用的 SKU 将呈**置灰状态**，并在前台隐藏。
5. 如需重新上架，选中 SKU，点击 **更多操作（···）** → **启用 SKU。**
