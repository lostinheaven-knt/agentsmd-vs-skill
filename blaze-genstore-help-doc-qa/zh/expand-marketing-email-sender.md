# 配置品牌发件邮箱

在邮件营销中，发件人邮箱会直接展示在客户收到的每一封营销邮件中。一个专业、可信的发件人地址，不仅能增强品牌形象，也能显著提高邮件的打开率和送达成功率。

系统默认使用邮箱：

```
store+店铺ID@genstoremail.ai
```
## 为什么建议使用品牌邮箱？

使用自有域名邮箱可带来以下营销优势：

- **提升品牌信任感**：客户更愿意打开熟悉来源的邮件
- **提高送达率**：减少邮件被标记为垃圾邮件的风险
- **统一品牌形象**：发件人与店铺域名一致，更显专业

## 设置步骤

1. 登录您的 Genstore 管理后台。
2. 进入 **设置 -> 通知** 页面。
3. 点击 **管理发件人** 区域，输入您希望使用的邮箱地址。
4. 点击 **对域名进行身份验证**，按引导完成验证。
5. 完成后点击 **保存**。

您也可通过 **邮件营销 -> 创建邮件活动 -> 设置 -> 管理发件人** 进行发件人管理。

::: tip

发件人邮箱设置后，变更最多可能需要 **48 小时** 生效。

:::

## 域名验证说明

- 为确保邮件正常投递，系统会要求您对所使用邮箱的域名进行验证。
- 验证过程需在您的域名 DNS 服务商处添加指定的 DNS 记录。
- 若验证失败，请检查 DNS 设置是否与 Genstore 后台提供的记录完全一致。

 更多说明详见 [管理店铺相关邮箱](./settings-notifications-email.md)
