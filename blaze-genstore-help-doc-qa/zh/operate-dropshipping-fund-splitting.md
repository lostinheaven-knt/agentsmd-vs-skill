# 资金分账与结算规则

当客户购买包含 **Genstore Dropshipping 产品** 的订单并完成支付后，系统会在后台自动执行分账与清算。
客户支付的金额将在扣除各项费用后，进入商家的账户余额。

## 费用结构

每笔 Dropshipping 订单在结算前，会产生以下三类费用：

| 费用类型      | 说明                                     |
| --------- | -------------------------------------- |
| **支付处理费** | 使用 **Genstore Payments** 或 **Drop Pay** 受理支付交易时产生的费用。 |
| **采购成本**  | 商家通过 Genstore 代发货服务向供应商采购产品的成本。        |
| **服务费**   | 使用 Genstore 代发货服务产生的系统服务手续费。           |

商家可通过以下路径查看每笔交易对应的手续费明细：
 - **财务** -> **支付** (如采用 Genstore Payments)
 - **Dropshipping** -> **Balance** (如采用 Drop Pay)

## 清算与结算周期

根据订单中是否包含 **Dropshipping 产品**，系统会采用不同的清算与结算周期：

### 1. 含代发货产品的订单

包括以下两种情况：

* 订单中仅包含 Dropshipping 产品；
* 订单中同时包含 Dropshipping 产品与普通产品。

在此类订单中：

| **阶段**     | **周期** | **说明**                                                                                   |
|--------------|----------|--------------------------------------------------------------------------------------------|
| **清算**     | 45天     | 买家付款后，Genstore 将在 45 天内完成平台与供应商之间的资金清算。                            |
| **结算**     | 5天      | 清算完成后，Genstore 将在 5 天内完成平台与商户之间的资金结算。                               |
| **结算完成** | -        | 结算成功的资金，将会在第二天向商户资金账户转账，转账成功后，系统会根据您的付款计划自动发起提现，并将资金汇入您的指定账户。注意：银行可能需要额外处理的时间。 |


::: tip
资金活动仅在工作日进行。非工作日顺顺延。
:::


### 2. 仅包含普通产品的订单

对于不含代发货产品的普通订单：

| **阶段**     | **周期** | **说明**                                                                                   |
|--------------|----------|--------------------------------------------------------------------------------------------|
| **清算**     | 5天     | 买家付款后，Genstore 将在 5 天内完成平台与供应商之间的资金清算。                            |
| **结算**     | 0天      | 清算完成后，系统会在每日结算周期结束时自动发起银行结算。银行处理可能略有延迟。                     |
| **结算完成** | -        | 结算成功的资金，将会在第二天向商户资金账户转账，转账成功后，系统会根据您的付款计划自动发起提现，并将资金汇入您的指定账户。注意：银行可能需要额外处理的时间。 |

## 注意事项

* 资金在完成结算后，才会进入可用余额。其中可提现部分的金额将按照您的计划配置自动发起提现到您绑定的银行账户。
* 资金结算以清算周期结束为准，未清算的资金不可提现。
* 若订单包含代发货产品与普通产品，将统一按照 **45 天清算周期** 执行。
* 所有交易手续费及扣费记录可通过以下路径查看：
  * **Genstore 商户后台 -> 财务 -> 支付 -> 交易明细** (如采用 Genstore Payments)
  * **Dropshipping** -> **Balance** (如采用 Drop Pay) 
* 银行到账时间可能因地区和账户类型存在差异。
