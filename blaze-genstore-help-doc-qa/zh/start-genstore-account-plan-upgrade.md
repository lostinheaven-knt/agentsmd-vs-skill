# 套餐升级或降级

Genstore 支持随时在线升级或降级套餐。您可以在 Genstore 商家后台的 **设置 -> 套餐和积分** 页面查看和管理您的订阅套餐。

## 查看当前套餐

1. 登录 Genstore 商家后台。
2. 点击 **设置** -> **套餐和积分**。
3. 在 **订阅** 处查看当前使用的套餐。

## 升级套餐

升级套餐操作可随时进行，升级将 **立即生效** 并 **按比例计费**。升级后，您可以立刻使用新套餐提供的功能，并且补偿您两个套餐之间的积分差额。

- 仅 **店主或拥有权限的员工** 可进行套餐变更操作，详见 [角色与权限](./start-genstore-account-roles-permissions.md)。
- 您只能选择 Genstore 当前对外提供的套餐。变更后无法切换回已停用的旧套餐，详见 [Genstore 定价页面](https://www.genstore.ai/pricing)。

### 操作步骤：

1. 登录 Genstore 商家后台。
2. 点击 **设置** -> **套餐和积分**。
3. 在订阅信息处，点击 **变更套餐**。
4. 选择您需要的套餐与账单周期。
5. 点击 **订阅** 完成变更。

## 变更套餐后的计费说明

升级套餐后，系统将立即签发仅包含新套餐费用的账单，并将旧套餐的剩余费用按比例抵扣：

- **立即计费**：系统会立刻生成一张新套餐的账单。
- **即时激活**：新套餐的功能将立即启用。
- **按比例调整**：新套餐费用按账单周期剩余天数比例计算。
- **按比例抵扣**：旧套餐未使用部分将生成等额抵扣金，用于抵消新套餐费用。
- **积分补发**：立即补发两个套餐之间的积分差额

### 计费示例

假设当前套餐为 25/月（每个月发放50积分），账单周期为 7 月 1 日至 7 月 31 日，您在 7 月 11 日升级为 $75/月的套餐（每个月发放200积分），当前账期剩余 20天：

**原套餐抵扣计算**

```
每天费用 = $25 / 31 = $0.81
抵扣金 = $0.81 × 20 = $16.20
```

**新套餐费用计算**

```
每天费用 = $75 / 31 = $2.42
应付金额 = ($2.42 × 20) = $48.40
```

**最终应付金额**

```
$48.40 - $16.20 = $32.20
```

**积分补发**

```
200 - 50 = 150
```

因此，此次套餐变更您需支付 $32.20，升级后立即补发 150 积分。

## 降级套餐

降级套餐不会立即生效，也不会立即扣费，Genstore 会在您下一个续订周期处理订阅降级。


### 计费示例

假设当前套餐为 75/月，账单周期为 7 月 1 日至 7 月 31 日，您在 7 月 11 日降级为 $25/月的套餐，您将保留75/月的套餐订阅权益。直到下一个续订日期（8月1日），您的订阅将续订为25/月的套餐。
