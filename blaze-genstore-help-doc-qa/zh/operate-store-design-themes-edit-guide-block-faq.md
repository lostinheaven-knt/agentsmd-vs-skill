# 为店铺页面添加常见问题分区

FAQ（常见问题）模块用于展示客户最常提出的疑问，例如发货时间、退换货政策、支付方式等。通过在页面中添加 FAQ，不仅能提升用户信任与转化率，也能有效减少客服咨询量。

::: tip

目前 FAQ 仅支持使用 Genstore 官方主题模板。

:::

## 常见展示方式

您可以根据实际需求，在以下位置展示 FAQ：

- **首页**：作为品牌介绍或引导模块的一部分
- **产品详情页**：解答产品相关的常见疑问，提升购买决策效率
- **独立页面**：通过导航菜单跳转，集中展示所有问题和解答内容

## 操作指南

### 在任意页面添加 FAQ 分区
1. 进入 Genstore 商户后台。
2. 点击左侧导航栏 **商店**，导航至 **在线商店** -> **模版**。
3. 在当前主题中点击 **设计** 按钮，进入主题编辑器。
4. 在页面选择器中选择目标页面，如首页、产品页，进入当前页面的编辑工作区。
5. 在左侧操作面板的 **模板** 部分，点击 **添加分区**
6. 搜索或查找 `常见问题` 分区，点击确认添加。
7. 添加成功后，可通过 **添加块** 可进一步增加：
	- **类别**：用于问题分类，如订单类常见问题、支付类常见问题。
	- **问题**：用于增加具体问题。
8. 完成后点击右上角 **保存**。

::: tip

点击分区或块后，您可以在右侧设置面板中进一步调整内容与样式，如展示布局（垂直/水平）、对齐方式、字体大小、颜色风格，以及是否默认展开第一个问题等。

:::

### 创建并设计独立 FAQ 页面

Genstore 会在店铺初始化时自动生成一个常见问题页面（默认隐藏）。您可通过以下步骤进行启用与编辑：

1. 进入 Genstore 商户后台。
2. 点击左侧导航栏 **商店**，导航至 **在线商店** -> **模版**。
3. 在当前主题中点击 **设计** 按钮，进入主题编辑器。
4. 在页面选择中选择 **页面**，并下来选择 **faq** 模版资源。
5. 您可根据需要增加更多 **类别** 与 **问题**。
6. 完成后点击右上角 **保存**。

#### 将常见问题页面添加到导航菜单

1. 返回主题编辑器左上角的 **返回箭头** 返回至商户后台首页。
2. 点击左侧导航栏 **商店**，导航至 **在线商店** -> **菜单**。
3. 点击 **主菜单** 后的 **铅笔图标**，进入编辑页面。 
4. 点击 **菜单项** 后的 **添加菜单项**：
   - 输入菜单项名称。
   - 在搜索页面搜索框中，输入 `faq`，点击选中目标页面。
   - 点击 **保存**。
5. 在 **编辑导航** 页，点击 **保存**。
6. 预览：返回 **模版** 页（**商店** -> **在线商店** -> **模版**），点击当前模版后的 **预览** 查看效果，并根据需要进一步调整页面布局或样式。

## 常见问题

### Q1. 标题、类别、问题等是否支持多语言?

A：支持。 点击 FAQ 区块右上角的 **翻译** 按钮，可跳转至 Genstore 翻译应用，进行多语言内容设置。

### Q2. 是否可以设置 SEO 标题?

A：支持。对于独立的常见问题页面，您可在页面设置中添加页面标题、URL 和元描述内容，提升搜索引擎收录效果。

:::
