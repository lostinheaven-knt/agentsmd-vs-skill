# 店铺通知

您可以通过店铺向客户和员工发送通知。所有店铺通知及其详情可以在 Genstore 后台的 **设置 -> 通知** 中查看和管理。

## 管理发件人

### 步骤

1. 在 Genstore 后台中，转至 **设置** -> **通知**。
2. 点击 **管理发件人**。
3. **输入**发件名称、发件邮箱、回复邮箱，并**验证**邮箱地址。
4. 点击 **保存**。

::: tip
默认情况下，发件名称为您的店铺名称、发件邮箱会显示为`store+123@genstoremail.ai`。建议您的发件邮箱使用品牌专属邮箱，统一专业形象，提升曝光度与用户认知。[了解如何更改发件邮箱](./settings-notifications-email#更改发件邮箱)
:::

## 设置通知通道

- 面向客户的店铺通知，均支持邮件通知，部分支持 SMS。
- 面向员工的店铺通知，均支持邮件通知，部分支持店铺后台通知。

### 步骤

1. 在 Genstore 后台中，转至 **设置** -> **通知**。
2. 找到要设置的通知。
3. 点击 **开关**，开启/关闭相应通道。
