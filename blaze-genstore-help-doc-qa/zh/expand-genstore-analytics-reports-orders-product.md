# 产品订单和退货

**产品订单和退货报告**可帮助您了解哪些产品销售最好，以及哪些产品退货率最高。该报告显示所选时间段内订购的产品总数以及退货数量。

## 如何使用此报告

- **热销产品**：通过查看下单件数最多的产品，您可以快速识别出最受欢迎的产品。
- **高退货量产品**：关注退货数量较多的产品，这可能暗示这些产品存在质量问题、描述不符或其他客户不满意的情况。分析退货原因，及时调整产品策略或改善供应链管理，有助于减少不必要的损失。
- **高退货率产品**：退货率是指特定时间段内退回产品占订购总数的百分比。高退货率的产品可能需要特别关注，因为这可能会影响您的品牌形象和客户满意度。

## 常见术语

**本报告**中使用了一些常见术语，详细信息请参考 [订单报告 - 常见术语](./expand-genstore-analytics-reports-orders.md#常见术语)。

此外，本报告还使用了以下术语：

| 术语名称       | 定义                                                                 |
|----------------|----------------------------------------------------------------------|
| 产品标题       | 已订购产品的名称。                                                   |
| 产品规格SKU    | 产品的唯一标识码，用于区分不同规格、款式或版本的产品。               |
| 产品规格标题   | 对产品规格的简要描述，帮助您快速识别产品的关键特征。                 |
| 下单件数       | 在所选时间段内，订购的产品总数。                                     |
| 退货数量       | 在所选时间段内，退回的产品总数。                                     |
| 净销量         | 下单件数减去退货数量，反映实际销售的最终结果。                       |
| 退货率         | 特定时间段内退回产品占订购总数的百分比，用于衡量产品的退货情况。     |
