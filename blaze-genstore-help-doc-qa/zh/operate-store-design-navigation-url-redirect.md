# URL 重定向

您可以在 Genstore 商家后台配置 URL 重定向，将访问旧地址的流量自动跳转到新地址，有效避免 404 错误、保持搜索引擎权重，并提升用户体验。

系统支持以下两种重定向类型：

| 类型           | 说明                                                         |
| -------------- | ------------------------------------------------------------ |
| **单一重定向** | - 将某一个具体旧地址跳转至一个新地址<BR />- 适用于页面地址更新或合并 |
| **组重定向**   | - 将多个具有相同公共路径前缀的地址，统一跳转至新的路径结构<BR />- 适用于系列改版、分类替换等场景 |

## 配置 URL 重定向

1. 登录 Genstore 商户后台，在左侧菜单中点击 **商店** -> **在线商店** -> **菜单**，点击页面右上角的 **URL 重定向**。
2. 点击页面右上角的 **URL 重定向**，选择重定向类型：
	- **单一重定向**：将一个旧路径定向至一个新路径
	- **组重定向**：将拥有相同路径前缀的多个旧地址统一重定向至新路径结构
3. 填写重定向信息：
	 - 旧网址路径：不包含主域名，如：`/products/old-product-name`
	 - 新网址路径：不包含主域名，如：`/products/new-product-name`
4. 点击 **保存**，系统将自动校验路径有效性。

## 导入 URL 重定向配置

1. 登录 Genstore 商户后台，在左侧菜单中点击 **商店** -> **在线商店** -> **菜单**。
2. 点击页面右上角的 **URL 重定向**。
3. 点击 **导入**，下载导入模版。
4. 按模板格式填写重定向信息（旧链接、新链接），保存为 CSV 文件。
5. 上传 CSV 文件并确认导入。
6. 系统将自动校验每条重定向路径是否符合规范：
	- 必须以 `/` 开头（即相对路径），不能是完整的 `https://` 或 `http:// URL`
	- 不支持包含 `?` 查询参数
	- 不支持包含 `#` 锚点链接
	- 不得包含空格或非法字符（例如 `<>`、`{ }`、`|`、`^`、`[]`）

## 导出 URL 重定向配置

1. 登录 Genstore 商户后台，在左侧菜单中点击 **商店** -> **在线商店** -> **菜单**。
2. 点击页面右上角的 **URL 重定向**。
3. 点击 **导出**，可选择以下导出范围：
    - 导出全部重定向记录
    - 导出选定记录
    - 根据当前筛选或搜索条件导出
4. 确定导出范围后，点击 **导出重定向**，系统将生成对应的 CSV 文件供下载。
