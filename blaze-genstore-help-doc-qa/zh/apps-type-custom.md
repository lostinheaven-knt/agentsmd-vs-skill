# 自定义应用

自定义应用是为您的 Genstore 商店独家开发的应用，区别于服务于多个商店的公共应用。它允许您通过 Genstore API 访问商店数据，扩展平台功能。

## 权限准备

您可邀请协作者或员工创建、编辑或删除自定义应用，并授予开发权限及相关 API 范围设置权限。

### 创建角色权限

1. 登录 Genstore 商户后台。
2. 导航至 **设置** -> **账号权限**。
3. 点击 **管理角色**。
4. 在新窗口中，点击 **新建角色**。
5. 填写角色 **标题** 和 **描述**，如 `自定义应用开发`。
6. 在 **店铺权限** 部分，勾选 **应用开发**
7. 点击 **保存**。

### 向用户分配角色

完全权限的创建后，您可以为协作者或员工分配该角色。

1. 登录 Genstore 商户后台，导航至 **设置** -> **账号权限**。
2. 在 **员工** 列表中选择要分配角色的用户，点击 **...** -> **编辑**。
3. 在员工详情中，**选中** 刚刚创建的 `自定义应用开发` 权限。
4. 点击 **保存**。

## 创建自定义应用

完成角色的创建和分配后，您可开始应用的开发工作，注意：仅店主或拥有 `开发自定义应用` 权限的员工可进行自定义应用开发。

1. 在 Genstore 商户后台，点击 **应用** -> **开发应用**。
2. 在新窗口中，点击 **创建应用**，并填写以下信息：
	- **应用名称**：自定义应用名称
	- **应用开发者**：选择店主/有权限的员工账户
3. 点击 **创建应用**。

## 配置自定义应用

### 配置 API 访问范围

在安装应用之前，您需要设置 API 访问范围。

1. 在 Genstore 商户后台，点击 **应用** -> **开发应用**。
2. 在应用列表中，点击目标应用进入应用详情页。
3. 在 **概览** 页签，点击 **配置 API 访问范围**。在随即打开的 **应用配置** 页面，点击 **编辑**。
4. 勾选所需权限范围（如订单读取、产品管理等）。
5. 点击 **保存**。

## 安装应用

在安装应用之前，您需要先获取 API 凭证。

1. 在 Genstore 商户后台，点击 **应用** -> **开发应用**。
2. 在应用列表中，点击目标应用进入应用详情页。
3. 点击 **安装应用** 按钮，确认安装。
4. 系统将生成 **API Access token**，请立即保存该令牌。**重要**：`Access Token` 仅首次可见，请务必保存。

## 密钥管理

1. 在 Genstore 商户后台，点击 **应用** -> **开发应用**。
2. 在应用列表中，点击目标应用进入应用详情页。
3. 点击 **管理 API 凭证** 按钮或 **API 凭证** 页签，查看：
	- `API Key`
	- `API Secret`
	- `API Access token`（仅首次安装可见，建议查看后保存）

## 应用维护

### 更新 API 范围

1. 在 Genstore 商户后台，点击 **应用** -> **开发应用**。
2. 在应用列表中，点击目标应用进入应用详情页。
3. 您可在 **应用配置** 页签查看当前应用的 API 访问范围。
4. 点击 **编辑** 修改权限范围。**注意：** 修改 API 范围时，已安装应用必须保留至少一个权限。
5. 点击 **保存**。

### 修改基本信息

1. 在 Genstore 商户后台，点击 **应用** -> **开发应用**。
2. 在应用列表中，点击目标应用进入应用详情页。
3. 您可在 **应用设置** 页签修改：
	- 应用名称/描述
	- 主要开发者（员工账户删除后自动转给店主）
	- 开发更新通知邮箱（可选）

修改完成后，点击 **保存**。

## 卸载与删除应用

### 卸载应用

1. 在 Genstore 商户后台，点击 **应用** -> **开发应用**。
2. 在应用列表中，点击目标应用进入应用详情页。
3. 点击 **卸载应用**。
4. 系统将自动撤销 API Token 并删除 Webhooks。

### 永久删除

删除前，请确保应用已卸载。

1. 在 Genstore 商户后台，点击 **应用** -> **开发应用**。
2. 在应用列表中，点击目标应用进入应用详情页。
3. 进入 **应用设置 -> 删除应用**。
4. 确认删除操作，所有相关数据将被同步删除。注意：删除操作不可逆，请谨慎操作。
