# 自定义创建知识库

知识是 AI Agent 回答客户问题的基础。就像培训新员工一样，您可以提供产品、品牌、流程和政策等信息，帮助其更好地为客户提供服务。

您可通过添加片段文本或上传 PDF 文件维护专属的知识库内容，作为 AI 回复的补充信息来源。

## 通过片段文本配置

如果您希望补充运营经验或常见问题说明，可直接在后台新增片段文本，补充 AI Agent 所依赖的知识内容。

1. 在 Genstore 后台，点击 **销售渠道** -> **Inbox**。
2. 点击 **进入** 打开客服工作台，点击 **AI Agent** -> 知识库。
3. 点击右上角的 **自定义创建**，在弹窗中选择 **添加片段**，点击**确定**。
4. 在打开的编辑框中输入您的内容即可。

::: tip

每条片段建议聚焦单一主题，内容越清晰，AI 理解越准确。

:::

### 编写说明

- 支持 Markdown 语法，建议结构清晰（如使用标题、列表、表格等）
- 提供所见即所得的可视化编辑器，即使不了解 Markdown 也能轻松排版
- 请勿插入图片，AI 目前无法识别图像信息

### 编辑或删除：

您可随时编辑片段文本中的内容。

- 在片段列表中，点击名称进入片段详情页。
  - 点击页面右下角的  **编辑**，开始更新内容。
  - 点击 **删除** 删除当前片段。

## 通过文件配置

若您已准备好品牌介绍、操作流程或其他参考材料，可通过上传 PDF 文件的形式添加知识内容。

- **支持格式**：仅支持 PDF（单个文件最大 10MB，每次最多上传 5 个）
- **注意事项**：目前 AI 无法识别 PDF 中的图像信息，建议仅包含文字内容

### 操作步骤
1. 在 Genstore 后台，点击 **销售渠道** -> **Inbox**。
2. 在 **设置 AI Agent** 区域，点击 **知识库**。
3. 点击右上角的 **自定义创建**，在弹窗中选择 **上传文件**，点击 **确定**。
4. 在弹窗中上传 PDF 文件，点击 **确定** 完成添加。

### 管理 PDF 文件

上传后的 PDF 不支持编辑，但可预览、下载或删除。

1. 在 Genstore 后台，点击 **销售渠道** -> **Inbox**。
2. 在 AI Agent 区域，点击 **知识库**。
3. 在知识库列表中点击 PDF 名称。
4. 系统将打开新窗口，供您在线预览或下载该 PDF。
5. 您也可点击页面底部的 **删除** 按钮删除当前文件。
