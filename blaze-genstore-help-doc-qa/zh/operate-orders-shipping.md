# 订单发货

当客户下单后，您需要根据订单内容准备产品，并联系物流商家将产品按客户要求发货至指定收货地址。

Genstore 提供两种便捷的发货方式，以满足不同商家的需求：

- 通过 Genstore 物流平台发货：利用 Genstore 整合的物流资源，快速高效地完成发货。
- 商家自发货：商家自行安排物流，并上传物流信息。

## 通过 Genstore 物流服务发货

您可以通过 Genstore 管理后台直接完成购买面单、标记发货等操作。在开始前，请确保您已完成 [物流商配置](./operate-fulfillment-genstore-shipping-carrier.md)。

### 操作步骤
1. 进入发货管理：登录 Genstore 后台，点击 **商店** -> **订单** -> **发货**。
2. 选择订单：在 **待发货** 列表中，找到目标订单并点击 **购买面单**。
3. 核对发货信息：
   - 地址核验：确认寄件人与收件人地址准确无误。
   - 包裹信息：输入包裹的尺寸与重量（可手动输入或选择预设模板）。
4. 获取运费与线路：
   - Genstore Shipping 用户：系统将自动加载并展示多家承运商（如 UPS, USPS）的实时运费和预计时效。
   - Shippo 用户：选择发货日期后，需点击 **获取运费** 以同步您的 Shippo 账号费率。
5. 购买与打印：确认物流方案后，点击 **购买面单**。购买成功后，您可以直接打印电子面单并贴在包裹上，交给承运商完成发货。

## 商家自发货

如果您选择自行安排物流，您可以按照以下步骤通过订单列表或订单详情页上传物流单号：

1. 登录 Genstore 商家后台，点击 **商店** -> **订单**，进入 **订单列表**。
2. 勾选目标订单，点击 **标记已发货**。您也可点击订单号直接进入订单详情页，点击 **标记为发货**。
3. 在新窗口中，输入物流的**运单号**，单号将展示在商家后台和客户端订单详情中。
4. 填写物流承运商，或通过下拉列表中选择。
5. 确认发货产品和数量。**注意**：当前页面将仅展示未发货的产品和需要发货的数量。点击 **订单发货**。
