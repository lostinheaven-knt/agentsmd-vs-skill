# 获取应用支持

Genstore 仅对由 Genstore 开发的应用提供支持。大多数应用由第三方开发人员构建，如需获取帮助，请直接联系应用开发人员。对于从 Genstore 应用商店安装的应用，您可以通过以下方式获取已安装应用的支持：

## 通过 Genstore 商家后台联系应用开发人员

在 Genstore 商家后台，您可以直接获取已安装应用的支持选项。对于 Genstore 开发的应用，系统会引导您访问 Genstore 帮助中心；对于第三方开发的应用，您可以直接联系应用开发人员。

### 步骤

1. 登录 Genstore 商家后台，进入 **应用** 页面。
2. 在需要获取支持的应用名称旁边，点击 **... -> 获取支持**。
3. 如果应用由第三方开发，您将在页面中看到开发人员的联系方式，直接发送问题详情。

## 通过 Genstore 应用商店联系应用开发人员

在 Genstore 应用商店中，您可以直接访问应用开发人员的支持信息。一些开发人员可能提供支持门户、支持文档或特定的联系方式。

### 步骤

1. 登录 Genstore 应用商店，搜索需要支持的应用。
2. 点击应用名称以打开应用详情页面。
3. 在 **开发者** 部分查看可用支持选项，例如支持门户、电子邮件地址或其他联系方式。
4. 根据开发人员提供的方式，直接发送问题详情。
