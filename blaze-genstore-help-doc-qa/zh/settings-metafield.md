# 元字段

元字段用于存储 Genstore 后台默认不提供的**专门化信息**，帮助您扩展商店功能并自定义页面展示。您可以使用元字段进行内部数据追踪，或在在线商店中以多种形式展示自定义内容，例如：

- 蜡烛商家可在产品页面显示 `燃烧时长`
- 食品商家可添加`产品到期日期`

常见使用场景包括：

- 部件编号
- 颜色样本
- 推出日期
- 相关产品
- 博客文章摘要
- 供下载的文件
- 成分列表

Genstore 支持为以下对象添加元字段：

- 产品（Product）
- 产品规格（Variant）
- 客户（Customer）

## 配置元字段的基本流程

1. **[添加元字段定义](./settings-metafield-definitions.md)**
   - 用于指定字段名称、类型、数据验证规则等内容。
2. **[向元字段添加值](./settings-metafield-adding-values.md)**
   - 为特定产品、规格或客户输入元字段的实际值。
