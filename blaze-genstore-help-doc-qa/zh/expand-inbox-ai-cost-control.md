# AI Agent 成本管控

Inbox 的 AI Agent 回答通常由顾客主动触发。若不加限制，每位顾客都可无限提问，可能导致费用快速上涨。为帮助您有效管理使用成本，Genstore 提供灵活的使用量控制功能。

## 限制 Inbox AI Agent 的使用量

在 Genstore 后台左侧导航栏，进入 **Inbox** 页面，找到 **使用量** 区块，点击 **设置用量限制** 按钮，即可打开使用限制设置面板。

您可以从以下三个维度设置使用上限，按需灵活控制 AI Agent 回答次数：

| 控制维度             | 描述                                                         | 最大可设上限（需订阅 Genstore 版本）                  |
| -------------------- | ------------------------------------------------------------ | ----------------------------- |
| 月度回答上限     | 每个计费周期内，AI Agent 可提供的总回答次数。达到上限后，服务将暂时暂停至下一个计费周期。| 最多 10000 次 / 月 |
| 每日回答上限     | 每日可使用的最大回复次数。该限制有助于均衡分配使用频率，避免短时间内的过度消耗。使用次数将每 24 小时重置一次，重置时间以您的店铺时区为准。| 最多 10000 次 / 日 |
| 每位客户每日回答上限 | AI 每天可为每位客户提供的最多回复次数。该限制有助于保障服务的公平使用，并防止单个客户在 24 小时内过度使用，从而有效管理成本。| 最多 10,000 次 / 顾客 / 日 |

::: tip

**计费说明：** 实际费用由 **月度回答上限** 决定。  
每日上限与每位客户上限仅用于控制使用频率，不影响总费用。  
若您设置了每日/每位客户上限，系统将以 **月度上限** 为基础计算费用，按实际使用量计费，不重复计入。

::: 
## 达到限制后的表现

- **单个顾客限制生效时**：该顾客后续提问将自动转接至人工客服，次日解禁。
- **每日回答上限达成时**：当天所有顾客的 AI 咨询将转接至人工客服，次日自动重置。
- **月度回答上限达成时**：当月所有顾客的 AI 咨询将转接至人工客服，次月恢复。

## 其他可能导致 AI Agent 停用的情况

在以下情况下，即便您的 Inbox AI Agent 尚未达到使用限制，也可能无法响应客户提问：

- **本月费用已达 $1,000 上限**：系统为保障账单安全，将暂停 AI Agent 使用，需等待下个账单周期恢复。
- **店铺存在未支付账单**：请前往账单中心完成支付，以重新启用 AI Agent 功能。
