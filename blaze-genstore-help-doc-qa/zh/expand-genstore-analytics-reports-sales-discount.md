# 按折扣查看销售额

**按折扣查看销售额报告**按折扣名称（如自动折扣名称或折扣类型）分组，展示销售数据。通过分析各类折扣的销售数据，您可以评估促销活动的效果，优化营销策略并提高利润。

:::tip

报告显示的销售额包括整个订单金额，而不仅仅是应用折扣的产品部分。如果您的商店使用组合折扣，订单可能在多个折扣项下重复显示，每个折扣类型都会在报告中单独列出。

:::

## 如何使用

- 查看不同折扣方式或折扣类型的应用频率。
- 了解哪些折扣转化的销售额最高。
- 衡量特定促销活动对收入的影响。

## 常见术语

**本报告**中使用了一些常见术语，详细信息请参考 [销售报告 - 常见术语](./expand-genstore-analytics-reports-sales.md#常见术语)。

此外，本报告还使用了以下术语：

| 术语           | 描述                                                         |
|---------|------------------------|
| 折扣名称 | 在后台创建折扣时输入的名称 |
| 折扣 ID | 折扣的后台 URL 中的 ID。 |
| 折扣等级 | “运费”、“产品”、“订单”或“未知”。 |
| 折扣方式 | 在结账时应用折扣的方式，例如自动应用或通过输入折扣码来应用。 |
| 折扣类型 | 满减满折、满 X 减 Y、产品降价、运费折扣等。 |
| 折扣码 | 用于接收折扣的代码。 |
| 自动折扣标题 | 自动应用的折扣的名称。 |
| 折扣金额 | 所报告的折扣的总价值。 |
| 其他折扣 | 与此折扣应用于同一订单的其他折扣的总价值。 |
| 其他非运费折扣 | 当多项折扣应用于已存在指定折扣的订单时，此字段显示与此折扣应用于同一订单的其他非运费折扣的总价值。 |
| 其他运费折扣 | 当多项折扣应用于已存在指定折扣的订单时，此字段显示与此折扣应用于同一订单的其他运费折扣的总价值。 |
| 非运费折扣 | 使用此折扣的订单中应用的所有非运费折扣的总价值。 |
| 运费折扣 | 使用此折扣的订单中应用的所有运费折扣的总价值。 |
| 非运费脚本折扣 | 由 Script Editor 应用运行的脚本得出的产品折扣的总价值。<br>只有当订单中也使用了折扣码时，系统才会报告此订单的脚本折扣金额。 |
| 运费脚本折扣 | 由 Script Editor 应用运行的脚本得出的运费折扣的总价值。<br>只有当订单中也使用了折扣码时，系统才会报告此订单的运费脚本折扣金额。 |
| 订单 | 应用该折扣的订单数。 |
| 总销售额 | 应用该折扣的所有订单的总销售额。 |
| 发货 | 运费金额，不包括运费折扣。 |
