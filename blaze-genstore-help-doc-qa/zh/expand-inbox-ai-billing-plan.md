# Inbox AI 定价方案

每一个商店都可以免费体验 200 次 AI Agent 的回答，当超出免费额度后，将会按照实际的 AI 回答次数进行收费。

## 什么是一次 “AI Agent 回答”？

每当 AI Agent 对用户消息做出一次完整自动回复，即计为一次“回答”。

- **包含以下情形**：
  - AI 自动生成的文字、链接或建议回复
  - 多轮对话中每一轮 AI 的响应，均单独计为一次
- **不包含以下情形**：
  - 用户提问但 AI 未能做出回复
  - 转接到人工客服的系统提示
  - AI 的系统启动或结束语（如“感谢您的提问”等非个性化内容）

> 示例：用户连续提了三个问题，AI 逐一回答三次，则共计 **3 次回答**

## 免费额度

- **免费额度**：每个商店可终生享受一次 200 次的免费 AI Agent 回答额度
- **适用范围**：所有店铺均可使用，无需订阅任何平台版本
- **有效期**：免费额度永久有效，直到用尽为止

## 收费标准（超出免费额度后）

- **计费方式**：按 AI Agent 实际回答次数计费
- **费用标准**：$0.05 / 次

## 示例说明

若某月内您的店铺产生了 1,528 次 AI Agent 回答，其中包含 200 次免费额度，则需付费的回答次数为 1,328 次。
费用计算方式如下：

```
1,328 × $0.05 = $66.40
```

若免费额度已在此前用尽，则全额计费：

```
1,528 × $0.05 = $76.40
```

## 查看费用

所有 Inbox AI 的使用费用将统一计入 Genstore 账单中。您可在 **设置** -> **账单** 中查看详细账单明细。如需查看详细账单信息，请点击 [了解 Genstore 账单](./start-genstore-billing.md)。
