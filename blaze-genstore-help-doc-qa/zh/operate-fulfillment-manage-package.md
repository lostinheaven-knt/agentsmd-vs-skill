# 包裹管理

在 **Genstore** 后台，您可以设置和管理不同的包装选项，以便在购买运输标签时，能够快速选择最适合的包装。根据您的运费设置（如基于重量或承运商费率），您还可以选择默认包装，确保结账时运费计算准确。

> **注意**：每个 SKU 只能绑定一个包装类型。系统默认提供基础包装，您也可以自定义创建更多包装方案。

## 添加自定义包装

您可以根据实际需要添加自定义尺寸和重量的包装，以确保能够适应不同类型的产品。

### 操作步骤：

1. 在 **Genstore** 后台，前往 **设置** -> **发货配送**。
2. 点击展开 **更多设置**，在 **包装材料** 部分，点击 **添加**。
3. 输入包装名称（例如：“小型盒子”、“标准信封”等）。
4. 选择包装类型（如盒子、袋子等）。
5. 输入包装的 **尺寸** 和 **重量**。
6. 点击 **确认** 保存。自定义包装就会添加至包裹列表中，方便后续使用。

要将此包装设置为默认包装以便结账时自动计算运费，请在列表中开启 **默认** 开关。

## 添加承运商包装

承运商提供的包装（如 USPS、UPS）具有预定尺寸，适用于特定的运输需求。使用这些包装时，您需要先授权 **Shippo** 或其他平台物流账号，以确保系统能够与承运商平台对接并获取最新的包裹列表。

### 操作步骤：

1. 在 **Genstore** 后台，转到 **设置** -> **发货配送**。
2. 点击展开 **更多设置**，在 **包装材料** 部分，点击 **添加**。
3. 选择 **承运商标准包裹**。
4. 选择承运商提供的包装类型。
5. 点击 **确认** 保存包裹。

## 设置默认包装

设置默认包装是确保运费计算准确的关键。客户结账时，默认包装将自动加入产品总重量，影响最终运费计算。您可以在后台设置商店的默认包装，并随时进行调整。

### 操作步骤：

1. 在 **Genstore** 后台，前往 **设置** -> **发货配送**。
2. 点击展开 **更多设置**，找到 **包装材料** 部分。
3. 找到您希望设置为默认的包装，点击 **...**，然后点击选中 **设置为默认**。

**注意**：系统只会选择一个包装进行运费计算，因此确保选择最合适的包装类型。

## 编辑和删除包装

如果您希望更改或删除现有的包装，可随时在 Genstore 商户后台进行调整。请注意，默认包装不支持删除。

### 编辑包装：

1. 在 **Genstore** 后台，转到 **设置** -> **发货配送**。
2. 点击展开 **更多设置**，在 **包装材料** 部分，找到需要编辑的包装，点击 **...**，然后点击 **编辑**。
3. 修改包装的名称、尺寸、重量等信息，点击 **确认** 保存设置。

### 删除包装：

1. 在 **Genstore** 后台，转到 **设置** -> **发货配送**。
2. 点击展开 **更多设置**，在 **包装材料** 部分，找到需要删除的包装，点击 **...**，然后点击 **删除**，确认删除。

## 承运商包装与 Shippo 账号授权

在添加承运商包装时，您需要连接 **Shippo** 账号。Shippo 是 Genstore 支持的第三方承运商，它能够帮助您访问并使用多家承运商的运输标签服务。

### 如何授权 Shippo 账号：

1. 前往 **Genstore 后台** -> **设置** -> **发货配送**。
2. 在 **物流商管理** 部分，点击 **去授权**。
3. 按提示输入您的 **Shippo 账号信息**，进行授权。
4. 完成授权后，您将能够访问承运商包装和运输标签，直接在结账时选择合适的运费选项。

## 为产品和 SKU 分配包装

Genstore 支持您在产品创建或管理过程中为产品及其规格变体（SKU）分配特定包装。这有助于根据每个产品使用的实际包装确保准确计算运费。

### 统一规格产品

1. 登录 Genstore 商家后台，点击 **商店** -> **产品** -> **添加产品**。
2. 在 **物流** 模块中，系统会自动选择 **默认包装**。
3. 如需使用自定义包装，点击下拉菜单并从已保存的包装中选择。
4. 如需管理您的包装，点击 **管理包装** 导航至包装管理页面。
5. 完成产品信息的其余部分并点击 **保存**。

### 多规格产品

对于具有多个规格变体的产品，您可以为每个 SKU 指定不同的包装：

1. 进入产品详情页，滚动至 **规格明细** 部分。
2. 勾选您要分配包装的 SKU。
3. 点击 **批量编辑**。
4. 在批量编辑弹窗中，找到每个 SKU 的 **包装** 字段。
5. 从下拉菜单中为每个 SKU 选择合适的包装。
6. 点击 **保存** 应用更改。

如希望单独为某个产品规格（SKU）指定不同的包装，您可以在规格详情页中进行设置。

1. 进入产品详情页，选择要编辑的规格变体（SKU）。
2. 在规格详情页，滚动至 **物流** 模块。
3. 点击 **包装** 下拉菜单，从已保存的包装中选择。
4. 完成设置后，点击 **保存**。

### 产品导入时的包装分配

通过 CSV 导入产品时，您可以为每个产品指定默认包装。系统将把此包装应用于导入产品的所有 SKU，除非另有说明。
