# 按流量来源查看访问量

**按流量来源查看访问量**报告为您提供了基于用户访问来源的详细数据，包括**直接访问、搜索引擎、电子邮件、社交媒体**等渠道的访问次数和用户活动。这份报告可以帮助您识别哪些渠道为网站带来了最多的流量，以及不同渠道的用户参与度和转化率，从而优化营销策略和资源分配。

## 如何使用

- **主要流量入口识别**：通过流量来源概览，快速识别出为网站带来最多访客、访问次数和浏览量的 Top 20 推荐来源和推荐网站，明确网站的主要流量入口渠道。
- **流量来源表现分析**：借助流量来源明细列表，详细了解每个推荐来源的具体数据表现，包括访客数、访问次数、浏览量等指标，评估不同推荐来源的流量质量和价值。

## 常见术语

**此报告**中使用了一些常见术语，详细信息请参考 [客流量报告 - 常见术语](./expand-genstore-analytics-reports-acquisition.md#常见术语)。

本报告使用了以下附加术语：

| 术语 | 定义 |
| --- | --- |
| **推荐来源（Referrer Source）** | 推荐来源指的是用户访问商店的途径。常见的推荐来源包括：<br/>- **直接访问**：用户直接输入商店 URL 或通过书签访问。<br/>- **搜索引擎**：用户通过搜索引擎结果页面点击链接进入。<br/>- **电子邮件**：用户点击电子邮件中的链接访问商店。<br/>- **社交媒体**：用户通过社交媒体平台的链接进入商店。<br/>这些来源信息通过 Genstore 的 API 捕获，尤其是在用户购买前的最后一次会话中获取。|
| **推荐名称（Referrer Name）** | 推荐网站的具体名称，例如从哪个社交媒体平台、搜索引擎或其他网站链接到您的商店。这个名称可以帮助识别具体的流量来源，以便进行更细致的流量分析和营销优化。|

这两个术语对于理解流量来源和优化营销策略至关重要，它们可以帮助商家识别最有效的用户获取渠道，并据此调整营销活动。

请注意，推荐来源数据可能受到用户隐私设置和跟踪技术限制的影响，因此报告中的推荐来源数据应作为参考而非绝对精确值。
