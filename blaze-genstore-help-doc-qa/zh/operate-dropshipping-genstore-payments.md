# Genstore Payments

如果希望进一步提升您的结账转化率及更全面的品牌背书，您可以选择使用 Genstore Payments 作为 Dropshipping 业务的主要支付方式。

## 核心优势

- **卓越的转化动力**：原生支持快捷结账，为客户提供更丝滑的购物体验。
- **专业品牌背书**：更正式的支付界面与多币种收单支持，增强消费者的付款信任感。
- **全业务覆盖**：不仅支持 Dropshipping 和 POD，还支持您店铺中的所有自营产品。

## 如何开通

1. 登录 Genstore 商户后台。
2. 在左侧导航菜单中，点击 **Dropshipping** -> **Balance**。
3. 在页面右上角找到视图切换按钮，点击 **Genstore Payments** 进行切换。
4. 在 **Genstore Payments** 视图下，点击 **激活 Genstore Payments** 按钮。
5. 系统将引导您至进件页面，请根据您的实际主体（如 US 或 HK）填写表单并等待审查。
6. 核通过后，回到此页面点击 **激活**。此时系统将自动禁用 Drop Pay。

::: tip
激活成功后，后续所有的订单交易款项都将通过 Genstore Payments 进行处理，请确保您的结算银行账户信息准确无误。
:::
