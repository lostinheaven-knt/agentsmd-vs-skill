# 创建退货

退货是指客户已收到产品并申请将产品退还给商家。在 Genstore 商家后台，您可以创建并管理退货流程。创建退货时，您可以向客户发送退货说明或退货标签，并告知预计退款金额。接收和检查退货产品后，您可以发放退款，并将符合条件的产品重新入库。

:::tip

- 仅可为已发货或者部分发货的产品创建退货。
- 同一产品同一数量仅可发起一次退货，不论退款数量和金额是否完全退回或部分退回。
:::

## 退货流程

在 Genstore 商家后台创建退货时，系统会根据您的退货规则自动计算退货运费和退货手续费。您还可以编辑特定退货的退货费用。退货运费只能按退货进行编辑，退货手续费可按产品或退货编辑。具体操作如下：

1. 登录 Genstore 商家后台，点击 **商店** -> **订单**。
2. 点击要退货的 **订单号**，进入到 **订单详情** 页。
3. 点击页面右上角的 **退货**。
4. 在新打开的 **退货** 窗口中，输入要退货的产品数量，并选择 **退货原因**。
5. 点击 **退回手续费**  栏的 **编辑** 图标，输入需要扣除的退货手续费，点击 **确定**。
6. 点击 **退回运费** 栏的 **编辑** 图标，输入需要扣除的退货运费，点击 **确定**。
7. 填写退货的物流配送信息和承运商，或选择 **无需物流**。
8. 点击 **创建退货**。

## 重新入库退货产品

收到退货产品后，您可以检查产品是否符合重新入库的条件。如果符合条件，产品可以重新入库。操作步骤如下：

1. 登录 Genstore 商家后台，点击 **商店** -> **订单**。
2. 选择要重新入库的退货订单的 **订单号**，进入 **订单详情**。
3. 在 **订单详情** 页，在退货卡点击 **重新入库**。
4. 完成产品的重新入库。

## 取消退货

如果您错误地创建了退货，或者客户改变了退货申请，您可以取消退货。取消退货的条件如下：

- 退货尚未退款；
- 退货尚未重新入库。

操作步骤如下：

1. 登录 Genstore 商家后台，点击 **商店** -> **订单**。
2. 点击要取消退货的 **订单号**，进入到 **订单详情** 页。
3. 在退货中卡点击 **更多** ，点击 **取消退货**。
