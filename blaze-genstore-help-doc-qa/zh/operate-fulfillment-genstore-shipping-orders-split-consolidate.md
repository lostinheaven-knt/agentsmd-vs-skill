# 拆单、合单及多仓发货

Genstore 提供灵活的包裹处理功能，帮助您应对复杂的仓库与配送需求。

## 1. 拆单发货
适用于单笔订单含多个产品且需分装的情况。
* **操作**：在 **待发货** 页进入订单详情，选择本次要发货的产品及数量。
* **结果**：拆分后的包裹拥有独立的物流信息和运费，可分别设置不同的发货方式。

## 2. 合单发货
将同一客户、同一地址的多个订单合并为一个包裹，节省运费。
* **操作**：当系统识别到符合条件的订单时，页面顶端会显示 **合并订单** 提示。
* **要求**：收件人、收件地址、下单渠道必须完全一致。
* **撤销**：若误操作，可在“待发货”列表点击 **... -> 取消合并订单**。

## 3. 多仓发货 (Multi-location Fulfillment)
当库存分布在不同仓库时，系统将通过路由逻辑智能处理。
* **自动拆分**：系统根据商品所属仓自动推荐发货点。若产品跨仓，会自动拆分为多个包裹。
* **手动修改**：您可以在发货详情中手动修改系统推荐的发货仓库。

## 常见逻辑问答
* **优先级**：拆单和合单是互斥操作，您只能对一笔订单执行其中一种处理。
* **运费影响**：合单后的包裹将重新计算统一运费，您需要重新确认物流方式。
* **Dropshipping 限制**：Genstore Dropshipping 订单暂不支持拆单或与其他订单合并。
