# 支持的浏览器

访问 **Genstore** 后台时，请确保浏览器为最新版本，以避免功能异常或部分功能不可用。为获得最佳体验，我们推荐使用最新版本的 [Google Chrome](https://www.google.com/intl/en/chrome/browser?utm_source=chatgpt.com)。

## 支持的浏览器列表

Genstore 支持所有主流浏览器的最新版本 (除非另有说明)。通常可以通过以下浏览器访问 Genstore：

### 桌面端浏览器

Genstore 支持以下桌面端浏览器的最新版本：

- [Google Chrome](https://www.google.com/intl/en/chrome/browser)（Windows / macOS）
- [Apple Safari](https://www.apple.com/safari)（macOS）
- [Microsoft Edge](https://www.microsoft.com/edge)（Windows）

### 移动端浏览器

您也可以通过以下移动端浏览器的最新版本访问 Genstore：

- [Google Chrome](https://www.google.com/intl/en/chrome)（Android）
- [Apple Safari](https://www.apple.com/safari)（iOS）
