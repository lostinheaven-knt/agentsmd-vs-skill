# 配置话术库

为帮助客服团队高效应对客户提问、保持回复一致性，您可以在 **Genstore 后台** 配置标准话术。客服可一键调用预设回复，提升响应效率、减少沟通成本。

## 功能价值

- **提升客服效率**：常用回复一键调用，无需重复编写
- **保持服务一致性**：统一话术，避免因人而异的回答差异
- **优化客户体验**：快速、准确的回复让客户感受到专业与重视

## 操作说明

**路径**：Genstore 后台 -> **销售渠道** -> Inbox。点击 **进入** 客服工作台。

点击 **设置** -> **话术库**，您可以进行以下操作：

- **新增话术**：为常见问题配置标准回复内容
- **编辑话术**：随时修改现有话术，确保内容始终准确
- **删除话术**：移除不再适用的回复内容，保持话术库整洁
- **分类管理**：通过创建文件夹对话术进行分类（如物流、售后），便于快速查找和使用

## 编辑方式

- 支持使用 **纯文本** 或 **Markdown 语法** 编写，方便结构化排版
- 不支持直接上传图片或视频
  

## 应用场景

客服人员在处理客户对话时，可从话术库中选择并插入预设回复，常见适用场景包括：

- 解答物流进度、配送周期等问题
- 说明退换货流程、售后政策
- 回复关于支付方式、发票开具等财务问题
- 引导客户了解当前优惠活动或使用规则

## 配置建议

- 建议根据实际业务高频问题，整理并配置好核心话术
- 定期检查话术内容，保持语义准确、信息更新
- 为话术设置清晰的标题和分类，方便客服快速查找
