# 自定义 CSS

Genstore 主题编辑器支持添加自定义 CSS，帮助商户根据品牌需求进一步个性化页面样式。您可选择添加全局 CSS 或为特定分区单独定义样式，实现对页面视觉的精细化控制。

## 添加全局 CSS 样式

全局样式将应用于整个主题模板的所有页面，适用于字体、颜色、组件样式等统一性调整。

### 操作步骤

1. 登录商家后台，前往 **商店 -> 在线商店 -> 模板**，点击目标模板右侧的 **设计**。
2. 在左侧功能导航栏中点击 **主题设计** (第四个图标)。
3. 在侧边操作面板底部，展开 **自定义 CSS** 模块。
4. 在输入框中粘贴或编写您需要的 CSS 代码。
5. 点击右上角 **保存**，更改将实时生效。
## 为特定分区添加 CSS 样式

如需仅对某个页面中的特定分区或模块样式进行定制，可直接为该分区设置局部 CSS。

### 操作步骤

1. 打开主题编辑器，进入需要编辑的页面模版。
2. 在左侧页面结构区域，选择目标分区。
3. 在右侧属性设置面板底部，点击 **自定义 CSS**。
4. 输入分区专属的 CSS 代码。
5. 点击 **保存** 并使用预览功能查看效果。

::: tip

如您不熟悉 CSS 编写，建议由具备前端基础的人员进行操作，或联系 Genstore 服务团队获得支持。

:::
