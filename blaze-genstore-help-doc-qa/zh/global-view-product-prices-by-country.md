# 按国家/地区查看产品价格明细

对于跨国电商商店，能够清晰查看不同市场的产品价格明细是确保定价准确性和一致性的关键工具。Genstore 支持商家查看每个产品在各个市场的价格，从而帮助商家更好地理解在不同市场的定价策略和调整情况。这不仅能确保价格的本地化定制，还能在全球范围内优化定价，提升商家的利润。

通过这一功能，商家可以轻松掌握每个市场的销售价格、汇率波动、价格调整等关键信息，为进一步优化定价和制定更具竞争力的定价策略提供数据支持。

## 操作步骤

1. 登录 Genstore 商户后台，点击 **设置** -> **市场**。
2. 点击目标市场，进入市场设置页。
3. 点击 **产品及定价**。
4. 选择待设定固定价格的产品，您可以通过搜索产品名称或根据筛选条件选择需要设置的产品。
5. 如果您的产品具有多个规格，可以点击 **查看价格**，查看每个产品规格的价格。
6. 在价格明细页，您可点击展开图标查看产品最终价格的计算逻辑。
