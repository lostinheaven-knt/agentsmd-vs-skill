# 404 页面

404 页面用于提示用户访问的页面不存在，是改善错误体验与引导用户回到正确路径的重要页面。Genstore 提供标准 404 模板，支持样式调整和内容替换，帮助商家维持统一的品牌形象并减少跳出率。

## **如何访问**

1. 登录 Genstore 商家后台。
2. 点击 **商店 -> 在线商店 -> 模版**。
3. 找到目标模版，点击右侧的 **设计** 按钮，进入模版编辑器。
4. 在顶部导航栏点击页面选择器，选择 **404 页面**。
## **样式与布局配置**

404 页面支持基础视觉样式调整，常见选项包括：

- **颜色模式**：切换浅色或深色模式，保持视觉一致性
- **配色方案**：选择符合品牌调性的色彩方案
- **区块间距**：调整页面上下填充（无 / 小 / 中 / 大 / 特大）
- **自定义 CSS**：针对高阶用户，支持自定义样式代码

## 扩展分区和应用

在默认分区基础上，您还可以根据业务需求进一步拓展 404 页的内容。通过添加分区，您可以插入图片、视频、富文本等内容模块，打造更具品牌感和信息层次的页面结构。

### 添加分区

在 **模版** 区域点击 **添加分区**，可从以下模块中选择所需内容类型：

| 分区类型   | 说明                                |
| ------ | --------------------------------- |
| 特色产品系列 | 展示精选的产品系列，适合做主题活动或品类推荐            |
| 图片横幅   | 展示品牌大图或促销视觉                       |
| 产品系列列表 | 展示店铺的产品系列课表                       |
| 视频     | 插入品牌故事或产品介绍                       |
| 幻灯片    | 添加轮播图片或广告视觉，用于展示多个品牌亮点或促销信息\|     |
| 联系表    | 引导用户填写咨询/反馈表单                     |
| 富文本    | 添加品牌理念、产品说明等文本内容                  |
| 邮件注册   | 添加邮件订阅入口                          |
| 常见问题   | 展示 FAQ 内容，帮助用户快速了解常见问题和解决方案       |
| 拼贴画    | 多图拼接展示，常用于时尚类或生活方式品牌，增强视觉吸引力      |
| 特色产品   | 推荐其他重点商品                          |
| 分隔线    | 用于模块之间的视觉隔断                       |
| 图片     | 插入单张图片，支持添加跳转链接，适用于宣传图或广告素材       |
| 多列布局   | 自定义图文组合排版                         |
| 博客文章   | 嵌入内容营销文章                          |
| 滚动文本   | 添加横向滚动的短文本内容，适合展示促销标语、品牌口号或用户提醒信息 |

:::tip

为了更好满足商家多样化的页面设计需求，我们会持续根据用户反馈迭代新增分区与区块组件。因此，页面中实际可用的内容组件可能会与本帮助文档略有出入，建议以编辑器中的实际内容为准。此外，不同主题模版可能会展示不同的区块样式和功能，敬请留意。

:::

### 拓展应用

在 **添加分区/添加区块** 弹窗中，点击 **应用** 页签，可添加功能型应用模块，如：

- 关联推荐
- 畅销产品
- 大家都在看
- 流行趋势等
