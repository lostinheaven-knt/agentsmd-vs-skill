# 弃单恢复

Genstore 自动化提供大量的流程模版，帮助您完成特定场景的自动化任务。

本文档将以 **恢复弃单** 为例，介绍如何使用 Genstore 自动化工具实现订单召回，帮助您快速上手。

## 认识弃单

**弃单**指的是顾客进入结账页并提供了联系方式（如邮箱或手机号）后，但未完成支付并最终放弃购买的订单。此类客户通常具有较高的购买意愿，是重点唤回对象。

**恢复弃单的目的**：通过系统的弃单恢复流程，您可以在客户放弃支付后，自动发送提醒邮件或短信，引导其完成订单，从而提升转化率与销售收入。

## 启用弃单恢复模板

1. 登录 Genstore 商户后台。
2. 前往 **营销 -> 自动化营销** 页面。
3. 点击 **创建自动化营销**，选择 **弃单** 模板。
4. 点击 **创建任务**，进入流程画布。
 
您可以查看预设流程并按需调整。

## 默认流程逻辑说明

以下为弃单恢复流程的标准结构：

|步骤|类型|说明|
|---|---|---|
|1|触发场景|客户放弃结账后触发流程|
|2|判断条件|弃单订单金额大于 $0|
|3|延迟|等待 10 小时后执行下一步|
|4|判断条件|客户在弃单后未完成购买|
|5|判断条件|弃单产品仍有库存|
|6|执行动作|向客户发送弃单提醒邮件|

流程中包含三层判断，确保仅在真正的弃单、且有库存时才发送提醒邮件，避免干扰非目标客户。

## 配置流程参数（可选）

您可根据业务需求调整流程中的关键参数，也可直接保留默认值：

- **订单金额判断**：默认仅对金额大于 $0 的订单执行流程。
- **等待时间**：建议设为 6~24 小时，避免立即打扰用户，提升打开率。
- **是否下单判断**：判断弃单后客户是否已重新下单。
- **库存判断**：如弃单产品已售罄，则不会发送提醒。

## 激活邮件（必选）

:::tip
推荐使用 Genstore 提供的弃单邮件营销模板。模板支持灵活调整每个内容分区，帮助您快速搭建个性化邮件样式。
:::  

点击流程画布中的 **发送营销邮件** 节点，进入邮件设置页面：
1. 点击 **选择模板**，系统将在新窗口打开模板详情页，供您选用模板并针对当前流程进行编辑。具体步骤，可参考 [创建邮件营销活动](./expand-marketing-email-create-new.md)。
2. 选定模板后返回自动化流程编辑页，您可在此预览邮件主题及内容；如需修改，请点击 **编辑**。


## 激活流程  

返回流程画布，点击右上角 **激活** 启动该流程。系统将在激活前自动校验流程逻辑的完整性。若存在配置错误，将提示您进行修改。

您也可在 **营销 -> 自动化营销** 页面中找到该流程，点击标题进入流程详情页，点击 **激活** 激活当前流程。

## 客户端用户体验

当流程被激活后，系统将根据客户的弃单行为自动发送邮件，以下是消费者收到邮件后的典型体验：

- 通过自动化流程触发邮件，邮件将自动展示客户放弃结账时的产品信息，包括图片、名称、价格等，帮助客户快速回忆购物意图。
- 邮件内置快速结账按钮，客户点击后可直接跳转至结账页面，便捷完成支付流程。

## 常见不触发情况

系统不触发弃单提醒，主要有以下情况：
- **产品均已售罄**：无可结账的产品。
- **客户已完成购买**：最近一次弃单后客户已重新下单。
- **未订阅营销邮件**：客户未同意接收营销信息。
- **延迟时间未结束**：最近一次弃单的延迟执行时间尚未到达。
- **订单金额为 0**：订单金额为 0 的情况不视为有效弃单。

## 调整流程  

您可以随时查看流程状态、调整流程内容，并追踪弃单恢复的营销效果。

**步骤**：
- 导航至 **营销 -> 自动化营销** 页面。
- 搜索并点击您要查看的自动化流程标题。
- 查看当前流程状态（生效或草稿）。
- 点击 **编辑**，进入流程的草稿版本进行修改。草稿仅在激活后才会生效。
- 确认调整无误后，点击 **激活** 替换正在生效的流程。

## 数据统计和效果反馈  

- 在 **商店** -> **订单** -> **弃单** 页面查看弃单记录，系统会保留数据 3 个月，逾期数据将自动删除。
- 通过 **邮件营销** 的自动化营销页签，查看邮件的打开率、点击率以及召回的销售额和订单数，帮助您评估营销效果并及时调整召回策略。
建议您定期查看弃单恢复流程的数据反馈，并根据用户行为调整判断条件与邮件内容，持续提升订单召回效果。
