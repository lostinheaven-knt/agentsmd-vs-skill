# 税务表格 (Tax Forms)

Drop Pay 是 Genstore 专为代发货业务设计的免审核收单方案。根据[美国税法和国税局 (IRS)](https://www.irs.gov/forms-pubs/about-form-w-9) 的规定，Genstore 必须收集相关税务表格，以确保交易合规及税款缴纳的透明度。

::: tip
重要提示：只有在成功提交并审核通过税务表格后，系统才会按计划向您的 PayPal 账户发放款项。
:::

## 确认您需要提交的文件

| 商户类型        | 税务文件 | 获取地址                                            |
| --------------- | -------- | --------------------------------------------------- |
| 非美国个人      | W-8BEN   | [点击获取 W-8BEN 表格](https://www.irs.gov/forms-pubs/about-form-w-8-ben)   |
| 非美国企业      | W-8BEN-E | [点击获取 W-8BEN-E 表格](https://www.irs.gov/forms-pubs/about-form-w-8-ben-e) |
| 美国个人 / 企业 | W-9      | [点击获取 W-9 表格](https://www.irs.gov/forms-pubs/about-form-w-9)       |

## 操作执行指引

### 步骤 1：获取并签字
- 下载表格：从上表中的 IRS 官方渠道获取对应表格。
- 填写并签字：W-9 等表格需要提供关联公司的有效签名和日期。
  - **工具推荐**：请使用 PDF 阅读器软件（如 Adobe Reader、Foxit Reader）打开文件。
  - **签字方法**：在工具栏中找到 “PDF 签名” 工具完成签字。 从 IRS 官方渠道获取表格

### 步骤 2：在 Genstore 商户后台提交
完成签字后，请按照以下步骤提交至 Genstore：
1. 登录 Genstore 商户后台。
2. 在左侧导航树种，依次点击：**Dropshipping** -> **Balance**。
3. 在 **Drop Pay** 视图下点击 **设置** 按钮。
4. 在弹窗中完成以下配置：
   - **税务居民身份**：选择适用于您的区域（美国/非美国居民）。
   - **法定信息**：填写您的法定全名（如有注册商业名称请可选提供）。
   - **文件上传**：选择对应的税表类型，上传已签字的 PDF 或图片文件（支持 JPEG, PNG, PDF）。
5. 点击 **确定** 完成提交。
  
## 注意事项
- **真实性要求**：您必须提交真实有效的税表以供 IRS 审计，虚假信息将导致账户受限。
- **审核周期**：税表提供后我们将进行合规审查，审查通过后方可激活自动付款计划。
- **文件规格**：建议文件大小在 30MB 以内，确保图片清晰、签字可见。
