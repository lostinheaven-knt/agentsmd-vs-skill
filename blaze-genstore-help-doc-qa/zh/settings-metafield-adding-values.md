# 向元字段添加值

创建元字段定义后，您就可以在 Genstore 后台的相应模块页面中为该字段填写具体的值。例如：为产品添加 “到期日期” 字段后，您可以在产品详情页中录入具体的到期日期。

通过添加元字段值，您可以存储专门化信息，用于内部追踪或展示于在线商店前台。

### 显示规则说明

- **[固定的元字段](./settings-metafield-pin-unpin.md)**：将自动出现在页面的元字段区域，按定义页中的顺序依次展示；
- **未固定的元字段**：不会默认展示，需点击 **查看全部** 后手动展开；
- 每个元字段会显示您在定义中设置的名称、类型与描述信息；
- 元字段值的输入形式（如颜色选择器、日期选择器）取决于其数据类型。参考：[元字段的数据类型](./settings-metafield-definitions-types.md)

## 添加步骤

1. 在 Genstore 后台中，进入目标模块页面（如某个产品或客户详情页）；
2. 滚动至页面底部 **元字段** 区域；
3. 找到需要填写的元字段；
3. 根据字段类型填写值，例如：
   - 文本字段：输入文字内容
   - 日期字段：选择日期
   - 颜色字段：使用颜色选择器
1. 点击 **保存**。

::: tip

如需快速填写常用字段，建议先将其[固定到详情页显示区域](./settings-metafield-pin-unpin.md)。未固定字段仅在点击 **查看全部** 后可见。

:::
