# 结账页

结账页是用户完成下单前的最后一步，对转化率有着关键影响。本页将帮助您理解结账页的结构、编辑方式及自定义能力。

## 选择页面

在编辑器顶部导航栏，点击 **主页** 右侧的下拉箭头，可展开当前市场下支持的页面类型列表。

- 选择 **结账页**，进入默认的结算页模版。

您也可在页面管理页面，直接进入结账页。

## 页面结构

结账页主要由以下区域组成：

- **联系信息**：填写邮箱并选择是否接收优惠信息。
- **配送方式**：输入收货地址并选择配送方案。
- **支付方式**：展示店铺支持的支付方式。
- **账单地址**：填写账单寄送地址。
- **订单概览**：展示产品信息、小计、运费和订单总价。

## 全局样式

结算页整体样式继承自店铺主题，包括字体与主色调。您也可以对局部元素进行覆盖，确保既保持品牌一致性，又能突出关键信息。

## 侧边栏功能

- **Custom Liquid**：可添加自定义 Liquid 代码，实现灵活的逻辑或动态内容展示。例如针对不同客户群体显示 VIP 提示或专属折扣。
- **应用嵌入**：在结算页顶部预留了展示区域，可嵌入第三方应用或营销工具。例如插入优惠券提示或倒计时活动，营造紧迫感，促进转化。

## 实时预览
- 修改内容后可在编辑器内实时查看效果，无需前后台反复切换。
