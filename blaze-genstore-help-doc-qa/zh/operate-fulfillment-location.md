# 管理地点

在 **Genstore** 商户后台，**地点** 指的是用于 **销售、发货或库存存放** 的物理地址。通过添加地点，您可以灵活管理多个仓库、配送中心或门店，精确控制库存和物流流程。

在 **设置** -> **地点** 中，您可以 **添加、停用、启用** 和 **删除** 地点。

## 添加地点

您可以轻松添加新的地点，默认状态为 **启用**，并且可以将其用于发货地点配置 [物流模板](./operate-fulfillment-shipping-profile.md)。

#### 步骤

1. 登录 Genstore 商户后台，点击 **设置** -> **地点**。
2. 点击 **添加地点**，输入地点名称和详细地址。
3. 点击 **保存**。

新增的地点将自动出现在地点列表中，您可以在此管理地点的 **启用状态**、**编辑信息**、**删除地点**。

## 更改默认地点

当无其他指定地点时，默认地点将被作为添加产品时的默认库存地点、默认发货地点。您可以轻松更改默认地点。

### 步骤

1. 登录 Genstore 商户后台，点击 **设置** -> **地点**。
2. 在 **默认地点** 模块，点击 **更改默认地点**。
3. 选择一个可用地点。
4. 点击**保存**。

::: tip

- 创建店铺时系统会自动为您创建一个地点作为默认地点，为避免影响您的业务，建议修改并编辑该地点。
- 仅可用地点可以被设为默认地点，停用地点不可设为默认地点。
:::
## 停用地点

若您不再希望从某个地点进行销售、发货或存放库存，可选择停用该地点。

::: tip

- **至少需保留一个启用的地点**。
- **默认地点** 无法停用。如需停用默认地点，请先修改店铺的默认地点设置。
- 在停用地点前，请确保：
  - **库存已转移**：完成该地点的库存清理或转移。
  - **订单已发货**：确保分配至该地点的订单已完成发货。

:::

#### 步骤

1. 登录 Genstore 商户后台，点击 **设置** -> **地点**。
2. 找到您希望停用的地点，关闭 **状态** 栏下的滑钮开关。

## 重新启用地点

您可以重新启用之前已停用的地点，然后继续从此地点为订单发货。

#### 步骤

1. 登录 Genstore 商户后台，点击 **设置** -> **地点**。
2. 找到您希望停用的地点，打开 **状态** 栏下的滑钮开关。

## 删除地点

如果某个地点不再涉及 **销售、发货或库存存放**，您可以永久删除该地点。

:::tip

删除地点之前，请首先停用当前地址。

:::

#### 步骤

1. 登录 Genstore 商户后台，点击 **设置** -> **地点**。
2. 找到您希望停用的地点，点击 **删除地点** 按钮。
