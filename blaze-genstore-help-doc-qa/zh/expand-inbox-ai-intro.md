# 启用 AI Agent 提升客服效率

Genstore Inbox 的 AI Agent 模块可自动回答客户问题，显著提升客服效率。当 AI 无法回答时，会引导客户补充信息或转接人工客服，确保问题得到妥善解决。

## 核心能力

- **自动应答**：AI Agent 可自动回复客户问题，减少人工客服工作量
- **智能引导**：当无法回答时，会引导客户补充必要信息或转接人工客服
- **持续学习**：AI 可从您的产品信息、店铺政策等数据中学习，不断提升应答质量

## 启用步骤

启用 AI Agent 需使用具备 **Inbox 管理员权限** 的账号。

1. 登录 Genstore 后台。
2. 进入 **销售渠道 -> Inbox**。
3. 在 **设置 AI Agent** 部分，点击 **启用 AI**。
4. 点击启用后，系统将开始初始化 AI Agent，视数据量不同可能需要数分钟。您可关闭当前页面，稍后在 Inbox 首页查看进度。

### 使用准备（建议）

- 所有商家均可免费体验 AI Agent 的基础功能；
- 建议在启用前准备好以下信息：
  - **完善的产品详情内容**，以帮助 AI 理解产品特性
  - **完整的店铺政策**，如配送、退货、隐私政策等，用于提高回答准确性
- 若希望长期使用该模块作为主力客服工具，需具备有效的 Genstore 订阅；
- 建议提前准备丰富的文本素材，如产品详情页、店铺政策、博客文章等，以提升 AI 的理解力与响应质量。

### 启用后配置建议

- 启用后，您可进入 **知识库** 模块进一步配置内容，提升 AI 回答的准确度；
  - 您可通过创建片段文本或上传文件[自定义知识库内容](./expand-inbox-ai-knowledge-base-custom.md)
  - 也可[基于模版创建](./expand-inbox-ai-knowledge-base-template.md)知识库内容。
- 也可进入 **个性化** 设置页面，上传 AI Agent 的头像与客服名称，该信息将展示在客户聊天窗口中。

## 响应数据来源

AI Agent 可综合以下数据源生成准确回复：

- 您的产品描述、详情
- 您的店铺订单数据
- 您的 FAQs
- 您上传的知识库文档
- 您商店的各项政策
- 您的店铺配置，包含支付配置、物流配置、税务配置

::: tip

数据越完整，AI 回答越精准，自动化率越高。

:::

## 持续学习机制

- AI Agent 将自动学习您店铺新增或更新的信息
- 包括新增产品、修改政策文案、上传新知识库等
- 无需额外操作，系统每天同步更新内容

## 品牌一致性

- AI 回答语气可参考您的品牌介绍、FAQ 文案等内容训练
- 回复风格将贴近人工客服，体现专业与信任
