# Transfer inventory

Inventory transfers help manage stock movement between different locations within your store, ensuring that inventory levels are accurate and that your supply chain runs efficiently. Genstore provides three key features for inventory management: **create inventory transfer**, **receive inventory transfer**, and **reverse inventory transfer**, each designed to address different needs in inventory management.

| Action                         | Description                                                  |
| ------------------------------ | ------------------------------------------------------------ |
| **Create inventory transfer**  | In the Genstore admin, create an inventory transfer by selecting the source and destination locations, adding products and quantities, and entering shipping details. |
| **Receive inventory transfer** | When inventory arrives, verify and update the stock quantities, choosing to receive either the full or partial transfer to keep inventory data accurate. |
| **Reverse inventory transfer** | If an error occurs, products are expired, inventory is damaged, or stock is overcounted, you can reverse an unreceived transfer or adjust the quantities of already received stock. |
