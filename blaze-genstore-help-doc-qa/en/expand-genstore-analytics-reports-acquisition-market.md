# Sessions by market

The **Sessions by market** report provides traffic data segmented by geographic markets, including key metrics like visitor count, sessions, and page views. This report helps you understand user engagement across markets and identify high - potential regions.

## Use this report to

- **Identify key markets**: Discover which markets drive the most traffic to your website and uncover growth opportunities.
- **Analyze market performance**: Use the detailed market list to evaluate metrics like visitor count, sessions, and page views per market. Assess traffic quality and user engagement to prioritize resource allocation.

## Report data fields

For general terms, refer to [Acquisition reports - common terms](./expand-genstore-analytics-reports-acquisition.md#common-terms).

This report includes the following additional terms:

| Term | Definition |
| ---- | ---------- |
| **Market name** | Geographic region of users (e.g., North America, Europe, Asia). Used to analyze regional traffic distribution. |
