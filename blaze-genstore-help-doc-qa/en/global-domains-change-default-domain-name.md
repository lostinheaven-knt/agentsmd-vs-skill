# Change your default genmystore.com domain name

When you register a Genstore store, the system automatically assigns a default `.genmystore.com` domain as your online store's URL. If you want to better showcase your brand, Genstore allows you to change this domain for free. However, each store can only make this change once.

## Steps to change your domain name

1. In the Genstore admin, go to **Settings** -> **Domains**.
2. On the **Domains** page, click **Change to a new genmystore.com domain** for the relevant domain.
3. Enter the new domain name and click **Add domain**.

The new `.genmystore.com` domain will automatically become your primary domain, and the previous domain will redirect to the new one. No additional URL redirection is needed.
