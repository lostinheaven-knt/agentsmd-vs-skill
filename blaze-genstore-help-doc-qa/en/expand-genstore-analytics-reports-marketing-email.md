# Email campaign performance

The **EDM campaign performance report** provides insights into data distribution across email marketing campaigns, enabling comparative analysis and strategic decision-making.

## Use this report to

- **Campaign overview**: Evaluate key email marketing metrics including:
  - **Unique opens**
  - **Open rate**
  - **Click rate(CTR)**
  - **Unsubscribes**
  - **Campaign-driven sessions**
  - **Average session duration**
  - **Orders**
- **Performance analysis**: **Filter metrics dynamically** and focus on top 20 campaign titles to identify high-impact initiatives.

## Report data fields

| Term | Definition |
| --- | --- |
| **Sent** | The total number of emails sent in the campaign. |
| **Open rate** | The percentage of recipients who opened the email. Calculated as (Unique opens / Sent) * 100. |
| **Click rate** | The percentage of recipients who clicked on any link within the email. Calculated as (Unique clicks / Sent) * 100. |
| **Unsubscribed** | The number of recipients who opted to unsubscribe from future emails after receiving this one. |
| **Sessions** | The number of times recipients clicked through from the email to visit the website. |
| **Orders** | The number of orders placed by recipients who clicked through from the email. |
| **Total sales** | The total revenue generated from orders placed by recipients who clicked through from the email. |
| **Average order value** | The average amount spent per order placed by recipients who clicked through from the email. |
| **Bounce rate** | The percentage of emails that were not delivered. Calculated as (Bounced / Sent) * 100. |
| **Bounced** | The number of emails that were not delivered due to invalid or non-existent email addresses. |
| **Delivered** | The number of emails that successfully reached the recipients' inboxes. |
| **Delivery rate** | The percentage of emails that were successfully delivered. Calculated as (Delivered / Sent) * 100. |
| **Unique clicks** | The number of unique recipients who clicked on any link within the email. |
| **Unique opens** | The number of unique recipients who opened the email. |
| **Visitors** | The number of unique visitors to the website who arrived via the email campaign. |
