# Metafields

Metafields let you store **specialized information** that isn't captured by default in the Genstore admin. They help you extend store functionality and customize how information appears on your storefront.

You can use metafields for internal tracking or to display custom content on product, variant, or customer pages. For example:
- A candle seller might display a product’s **burn time**
- A grocery store could add a **product expiration date**
### Common use cases include:

- Part numbers
- Color swatches
- Release dates
- Related products
- Blog post summaries
- Downloadable files
- Ingredient lists

Genstore supports adding metafields to the following objects:

- **Products**
- **Variants**
- **Customers**
## Basic setup process

1. **[Add a metafield definition](./settings-metafield-definitions.md)**
