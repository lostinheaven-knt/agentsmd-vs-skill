# Customer referrals

In Genstore Loyalty, the **Referrals** page allows merchants to set up and manage their referral reward programs. This feature helps businesses incentivize existing customers to refer new ones, expanding brand reach and customer base.

## Enable the referral program

1. Log into your Genstore admin.
2. Navigate to: **Store** -> **Customers** -> **Loyalty**.
3. Under the **Referrals** tab, use the toggle switch in the status bar to enable or disable the referral program.

Once on the **Referral** page, you can configure rewards for both referrers and referees, and set up sharing links. Details are covered in the sections below.

## Rewards for referrers and friends

Referrer rewards are given when their referred friend completes their first order. Referee rewards are available to new customers who register through a referral link and can be used on their first purchase.

Available reward options include:

- Fixed discount: Set a specific discount amount and usage conditions. See [Amount discount](./operate-customers-loyalty-points-redemption-discount-amount.md) section for details
- Percentage discount: Configure percentage-based discounts and limitations. See [Percentage discount](./operate-customers-loyalty-points-redemption-discount-percentage.md) section
- Free shipping: Set maximum shipping credit and conditions. See [Free shipping](./operate-customers-loyalty-points-redemption-free-shipping.md) section
- Free gift: Configure free product rewards. See [Free gift](./operate-customers-loyalty-points-redemption-free-gifts.md) section
- Points: Specify the number of reward points. (Note: This option is only available for referral bonuses.)

Once configured, rewards appear in the rewards section and can be modified using the **edit** button.

:::tip

- You can only configure one reward each for the referrer and the friend.
- Changing rewards will replace the current reward with the new one, but won't affect previously issued rewards.

:::

## Referral link sharing

Customers can share referral links through Facebook or X social platforms. Once enabled, customers can quickly share their invitation links to Facebook or social media platform X from the referral module in the member panel:

- Facebook: Due to platform restrictions, users must add message content manually.
- X: You can customize share content that customers can post along with their referral link.

::: faq

## FAQ

### Q1: How do I modify an existing reward type?
> A: Click the edit button in the rewards section to make changes.

### **Q2: Can I switch to a different reward type?**
> A: Yes, use the **Change reward** button to switch types. Note that this will remove the current reward setup and add the new one, but won't affect previously issued rewards.

### Q3: When do referrers receive their rewards?
> A: Referral rewards are issued after the referred friend completes their first order, with an email notification sent to the referrer.

### **Q4: What happens to referral rewards if the referee's order is canceled or partially refunded?**

> A: Reference the following scenarios:

| Order status                                    | Reward handling                                              |
| ----------------------------------------------- | ------------------------------------------------------------ |
| Complete refund                                 | Reward is revoked (if unused; if used, no recovery required) |
| Partial refund                                  | Reward is retained                                           |
| Multiple partial refunds leading to full refund | Reward is revoked (if unused; if used, no recovery required) |

:::
