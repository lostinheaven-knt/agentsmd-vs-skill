# Upload and manage content assets

In the **Contents** section of the Genstore admin, you can upload, manage, and delete assets for your store, including:

- Images and videos used in templates, blog posts, and pages
- Product and product series images and videos
- Files and videos available for customer downloads

You can also upload assets while editing templates, pages, blog posts, product descriptions, or product series descriptions. Once uploaded, your assets will appear on the **Contents** page.

To access the **Contents** page, go to **Settings** -> **Contents**.

## Contents requirements

When uploading content assets, make sure they meet the following requirements:

### Image requirements

| Property | Requirement                          |
| -------- | ------------------------------------ |
| Size     | Max 20 MB                            |
| Format   | PNG, JPEG, JPG, GIF, WEBP, BMP, HEIC |

### Video requirements

| Property | Requirement         |
| -------- | ------------------- |
| Size     | Max 400 MB          |
| Duration | Max 3 minutes       |
| Format   | MOV, AVI, MP4, WEBM |

:::tip

These limits don’t apply to embedded Vimeo or YouTube links. If you want to share videos larger than 400 MB or longer than 3 minutes, use [YouTube](https://www.youtube.com/) or [Vimeo](https://vimeo.com/) for hosting, then add the shareable link to your content.

:::

### Audio requirements

| Property | Requirement   |
| -------- | ------------- |
| Size     | Max 20 MB     |
| Format   | MP3, AAC, WMA |

### Other file requirements

| Property | Requirement                                                  |
| -------- | ------------------------------------------------------------ |
| Size     | Max 20 MB                                                    |
| Format   | CSS, RTF, TXT, JSON, CSV, XLS, XLSX, DOC, DOCX, DOTX, POTX, PPSX, PPT, PPSX, PDF, XMIND, ZIP, RAR |

## Upload content assets

You can upload content assets directly on the **Contents** page and use them for various parts of your store, like logos, product images, and more.

### Steps:

1. In the Genstore admin, go to **Settings** -> **Contents**.
2. Click **Upload**.
3. Select up to 10 files to upload, then click **Open**.

## Edit content assets

Click an asset’s thumbnail on the **Contents** page to preview and edit it. Any changes made will apply to all instances of the content asset.

## Delete content assets

You can delete one or more content assets from the **Contents** page.

### Steps:

1. In the Genstore admin, go to **Settings** -> **Contents**.
2. Select the checkboxes next to the content assets you want to delete (you can select multiple files).
3. Click **Delete**, then click **Delete** again to confirm.

## Edit content assets without overwriting the original

If you want to edit an image, such as cropping it, and save the changes without replacing the original asset, save it as a new file.

### Steps:

1. In the Genstore admin, go to **Settings** -> **Contents**.
2. Select the content asset and open the content editor.
3. Make your edits.
4. Click **Save as**.

**Saving as** will add the edited content asset to your store’s library while keeping the original intact.
