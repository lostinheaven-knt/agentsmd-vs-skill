# Store migration

Genstore, a powerful e-commerce platform, is dedicated to providing merchants with one-stop shop solutions. To help you smoothly migrate your existing business to Genstore, we have launched the free **Genstore Migration Tool**.

This tool can help you migrate store data from other e-commerce platforms to Genstore **quickly, safely, and completely**, or clone data between different Genstore stores, minimizing the impact of migration on business operations and advertising.

## Store migration process

1. Connect the store to be migrated
2. Select migration modules
3. Wait for data migration
4. Check migration results

## Install the Genstore Migration tool

You can install the Genstore Migration Tool through the Genstore App Store or the Genstore admin.

**Genstore App Store**

1. Visit the **[Genstore App Store](https://apps.genstore.ai/en-US)**. If you manage multiple Genstore stores, click to select the store where you want to install the Migration Tool.
2. Search for **Migration Tool** (or search for **Migration Tool** in the **Apps** menu of the Genstore admin).
3. On the app page, click the **Install** button.
4. Confirm the app access permissions in the Genstore admin, and click **Install** to complete the authorization.

After installation, you can find the **Genstore Migration Assistant (Migration Tool)** in the **Genstore admin -> Apps -> Installed** page.

## View migration records

After completing the store migration, you can view all migration history records in the Migration Tool app, including:

- **Migration time**: The time the task was initiated.
- **Migration platform**: The migration platform.
- **Migration store**: Displays the store domain you migrated.
- **Migration modules**: Lists the specific modules involved in this migration, such as stores, customers, products, and orders.
- **Migration data results**: The amount of data that was migrated successfully and the amount of data with errors.
- **Actions**: You can click the **Failure details** button to get specific error information or export error data for easier review and troubleshooting.
