# Shipping discounts

Provide **Free shipping** for eligible orders. Merchants can set multi-dimensional activity thresholds (such as minimum purchase amount or minimum number of items) to determine whether to offer free shipping based on the buyer’s delivery country. This encourages buyers to spend more, increasing the average order value and conversion rate.

## How to access

1. **Log in**: Open your Genstore admin.
2. On the left navigation bar, go to **Marketing** -> **Discounts** -> **Create discount**, and select **Free shipping**.

## Set up your campaign

Configure your campaign on the **Create** page:

### Discount method

Discounts are divided into two types:

- **Discount code**: Require customers to enter a code at checkout.
- **Automatic discounts**: Applied automatically at the checkout page.

### Discount title

- This is the discount title customers will see in their cart when an automatic discount is applied.

### Purchase threshold

- Set whether to set a minimum purchase requirement, either by amount or quantity.

### Country or region

- You can select all countries/regions or specify individual countries/regions.

### Maximum shipping discount
Set whether to apply a shipping discount limit. 
Example: Shipping discount limit = $100
- If the shipping cost in the order ≤ $100 → Free shipping applies
- If the shipping cost in the order > $100 → Free shipping does not apply

### Event time

Set the start and end dates for the promotion. The start date is required, while the end date is optional. If no end date is set, the promotion will remain active indefinitely. **Note:** The time zone defaults to your store’s time zone.
## Advanced settings

In addition, if you want to configure more detailed discount rules—such as purchase limits, combination, or eligible customers—you can do so in **Advanced settings**.
### Usage limits

- **Total usage**: Set the maximum number of times this discount can be used.
- **Limit to one use per customer**: Set the discount to be used only once per customer.

### Combinations

Choose whether to combine this discount with **Product discounts**, **Order discounts**, or **Shipping discount**.
### Target customers

- Select from **All customers**, **Specified customer tag**, or up to 100 **Specified customers**.
## Preview and publish

Review your rule settings in the **Overview** section on the top right of the page. If you want to preview the settings before publish, you can click **Preview** in the top-right corner of the page.

Once finalized, click **Save** to finalize the setting.

Once the rule is saved, it will appear in the **discount list**, where you can edit, activate, deactivate, or delete it.
