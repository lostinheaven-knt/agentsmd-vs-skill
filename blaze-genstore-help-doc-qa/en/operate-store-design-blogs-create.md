# Create blog post

Writing a blog is a great way to connect with customers and promote your products, boosting your brand’s visibility and website traffic. Here’s how to write and manage a blog on the Genstore platform:

## Write a blog

1. Log in to your Genstore admin, then go to **Store** -> **Online Store -> Blog posts**.
2. Click **Create blog post** in the top-right corner.
3. In the new window:
   - Add the blog title and summary.
   - Write the blog content, and use rich text features to insert images, hyperlinks, and adjust font, size, and color.
4. SEO settings
   - **Page title**: Keep it under 70 characters to avoid truncation in search results.
   - **Meta description**: Keep it under 200 characters for better display in search results.
   - **URL handle**: Customize the URL. If you need to change it later, the system will automatically redirect the old URL to the new one.

Continue setting up:

- **Blog status**: Choose the visibility of the blog, such as visible, hide, or set it to display at a specific time.
- **Blog cover**: Upload or choose an image from the media library to use as the cover for your blog.
- **Author**: Select or enter the author’s name.
- **Blog collection**: Choose the blog collection where you want to post the article, or create a new one if necessary.
- **Blog post tags**: Add tags to categorize the article and make it easier to search. Use commas to separate tags. Each tag can be up to 255 characters, and you can add up to 1,000 tags per blog.

Once you’ve finished, click **Save**. The new blog will appear automatically in the blog articles list.

## Manage blogs

On the blog list page, you can manage blog content:

- **Edit**: Click the **Edit** button next to a blog entry to make changes.
- **Delete**: Click **Delete** to remove a blog. This will also delete any associated articles, **and the action cannot be undone**.
- **Preview**: Click **Preview** to see how the blog will appear to visitors and make sure the content is clear and engaging.
