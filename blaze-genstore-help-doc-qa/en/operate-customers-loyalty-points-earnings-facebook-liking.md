# Like on Facebook reward points

Enhance your brand's visibility and engage more effectively with consumers on social media by enabling a points reward system for liking your Facebook page.

This guide outlines the setup process, including operational applications and critical considerations.

## Set point rules

**Navigation**: 

1. Log in to Genstore admin and go to **Store** -> **Customers** -> **Loyalty**. 
2. Under the **Points** tab, find the **Earn points** section.
### Setup steps

Click **Add** -> **Like Facebook page** -> **Add**:

- **Rule name**: Provide a name for this reward rule to facilitate easy management.
- **Earning points**: Specify the number of points awarded for liking your Facebook page.
- **Your Facebook page URL**: Enter a valid URL to ensure points are correctly awarded for actions taken on the correct page. This field must not be empty.

  :::tip

  The system is set by default to award points only for the first share. To prevent fraud, we advise against setting the frequency too high.

  :::

## Preview and publish

Check your settings in the **Summary** section to the right. The system provides a real-time summary here. Once you confirm the details, click **Save** to activate the rule.

You can manage this rule—view, edit, enable, or disable it—via the **Earn points** list.

:::tip

Use the status switch to easily toggle this rule on or off.

:::

## FAQs

### Q1. What happens if a customer unlikes the Facebook page?
> A: Our system doesn't track unliking behavior due to technical limitations, so once points are awarded for liking, they will not be deducted if the like is removed.

### Q2. What if my Facebook page URL fails validation?
> A: Ensure the URL is correct and reflects a valid Facebook page. The system checks URLs against a set pattern. If the URL is correct but validation fails, verify it includes the entire path necessary to navigate directly to your page.

### Q3. Can I adjust how often points are awarded for likes?
> A: Currently, points for liking are awarded only once per customer and cannot be adjusted.
