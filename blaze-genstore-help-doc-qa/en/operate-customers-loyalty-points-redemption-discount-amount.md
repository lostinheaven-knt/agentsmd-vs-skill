# Discount amount reward

The points program of **Genstore Loyalty** allows customers to redeem points for discount amounts, offering two options: fixed value redemption (e.g., 500 points for a $5 discount) and tiered redemption (e.g., 100 points for a $1 discount). Businesses can leverage fixed or tiered redemption models to increase customer spending and loyalty.

## Set discount redemption

**Navigation**: 

1. Log in to Genstore admin and go to **Store** -> **Customers** -> **Loyalty**. 
2. Under the **Points** tab, find the **Redeem points** section.
### Setup steps

Go to **Add** -> **Discount amount** -> **Add**, and you can set it as follows:

- **Rule name**: Create a recognizable name for ease of future management.
- **Redemption rules**
  - **Fixed amount of points**: Redeem a set number of points for a fixed discount, e.g., 500 points for a $5 discount.
  - **Increments of points**: Allocate points to varying discount values, e.g., 100 points for $1. Set minimum and maximum redemption limits to maintain control.
- **limitations on use**
  - **Minimum order amount requirement**: Specify if a discount requires a minimum purchase amount, such as $100, to be valid.
  - **Applies to**: Define if the discount applies universally, to specific products, or to designated collections.
  - **Expiration**: Establish an expiration period for discounts, e.g., 6 months, with an automated reminder email before expiry.
  - **Combines with**: Decide if the discount can be combined with other offers and under what conditions.

	:::tip

	Carefully consider discount combinations to avoid excessive reductions.

	:::

## Preview and publish

Check your settings in the **Summary** section to the right. The system provides a real-time summary here. Once you confirm the details, click **Save** to activate the rule.

You can manage this rule—view, edit, enable, or disable it—via the **Earn points** list.

:::tip

Use the status switch to easily toggle this rule on or off.

:::

## FAQs

### Q1. How can I set point limits for redemption?
> A: Define upper and lower point limits in the **Increments of points** settings to ensure exchanges are within established parameters.

### Q2. Can discount coupons be used if the order doesn't meet the minimum requirement?
> A: Depending on whether you have set **Minimum order amount requirement**, if yes, the order must meet the minimum threshold set for using a discount coupon, if no, there will be no requirement.

### Q3. Can discount coupons be combined with other offers?
> A: Yes, enable this in the **Combines with** settings to allow coupons to be used with other discounts.

### Q4. Do redeemed discount coupons expire?
> A: Depending on your settings, coupons can have an expiration date, with notifications sent before expiry.
