# Market settings

In Genstore, the **market details page** is your central hub for managing the sales settings of each country or region. From this page, you can view and configure key elements such as language, pricing, taxes, shipping, and payments. Whether you’re creating a new market or adjusting an existing strategy, this page enables streamlined, in-depth market management.

## How to access

1. Log in to your Genstore admin.
2. Go to **Settings** -> **Markets**.
3. Click into the market you want to manage.

## Market-level settings

| Setting                                                             | Description                                                                                                                            |
| ------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| [Language and domains](./global-market-settings-lang-domain.md)     | Configure the supported languages, how content is displayed, and how the market is accessed (primary domain, subdomain, or subfolder). |
| [Products and pricing](./global-market-settings-product-pricing.md) | Set the base currency for the market, enable local currencies, adjust prices, and manage fixed pricing strategies.                     |
| [Duties and import taxes](./global-market-settings-taxes-duties.md) | View the current duties and import tax settings for the market.                                                                        |
| [Shipping](./global-market-settings-shipping-delivery.md)           | View and manage shipping options and delivery zones for the market.                                                                    |

## Store-level settings

## Market performance

| Metric                             | Description                                                  |
| -------------------------------- | ------------------------------------------------------------ |
| **Revenue**                    | Total revenue generated in this market over the past 30 days.<BR />Formula: `Revenue = Gross sales - Discounts - Returns + Taxes + Shipping + Other fees` |
| **Sales share**      | The percentage of total store revenue contributed by this market’s country or region. Helps you understand the market’s overall sales impact. |
| **Conversion rate**          | The percentage of sessions in this market that resulted in an order. |
| **Average order value (AOV)** | The average value of orders placed in this market.<BR />Formula: `AOV = (Gross sales - Discounts) / Number of orders` |

> All metrics are based on the last 30 days.
