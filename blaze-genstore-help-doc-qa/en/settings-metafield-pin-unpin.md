# Pin metafields for quick access

To help you work more efficiently in the admin, you can **pin commonly used metafield definitions** so they appear directly in the details pages of products, variants, or customers—no need to click “View all” every time.
## When to pin a metafield

- **Auto display**: Pinned metafields are shown by default in the detail pages (e.g., for products or customers)
- **Custom order**: The pinned order matches the order shown in the admin UI
- **Faster editing**: Avoid digging through the full list by surfacing key fields upfront

By default, only pinned metafields are displayed. To see the rest, click **View all** at the bottom of the section.
## Pinning rules

- You can pin up to **10 metafield definitions** per module
- You can pin or unpin fields at any time
- You can drag to reorder pinned fields
## Operation steps

Go to **Settings -> Metafields** in your Genstore admin. In the **Metafield definitions** section:

| Action                    | What to do                                       | Icon                                 |
| ------------------------- | ------------------------------------------------ | ------------------------------------ |
| **Pin a field**           | Click the pin icon to add it to the visible list | ![Pinned](./assets/icon-pin.png)     |
| **Unpin a field**         | Click the pin icon again to unpin                | ![Unpinned](./assets/icon-unpin.png) |
| **Reorder pinned fields** | Drag the handle ☰ to change the display order    | ![Unpinned](./assets/icon-sort.png)  |
