# Follow X reward points

Once this points program is activated, customers can earn points by following the store's X account, helping boost brand visibility and social media engagement.

This guide outlines the setup process, including operational applications and critical considerations.

## Set point rules

**Navigation**: 

1. Log in to Genstore admin and go to **Store** -> **Customers** -> **Loyalty**. 
2. Under the **Points** tab, find the **Earn points** section.
### Setup steps

Click **Add** -> **Follow X** -> **Add**.

- **Rule name**: Provide a name for this reward rule to facilitate easy management.
- **Earning points**: Specify the number of points awarded for following your X account.
- **Your X username**: Enter a valid URL to ensure points are correctly awarded for actions taken on the correct page. This field must not be empty.

## Preview and publish

Check your settings in the **Summary** section to the right. The system provides a real-time summary here. Once you confirm the details, click **Save** to activate the rule.

You can manage this rule—view, edit, enable, or disable it—via the **Earn points** list.

:::tip

Use the status switch to easily toggle this rule on or off.

:::

## FAQs

### Q1. What happens if a customer unfollows the account, will the points be deducted?
> A: Our system doesn't track whether a customer actually follows the account or not, so once points are awarded for sharing, they will not be deducted if they unfollow the account.

### Q2. What if my X account URL fails validation?
> A: Ensure the URL is correct and reflects a valid Facebook page. The system checks URLs against a set pattern. If the URL is correct but validation fails, verify it includes the entire path necessary to navigate directly to your page.

### Q3. Can I adjust how often points are awarded for likes?
> A: Currently, points for the following behavior are awarded only once per customer and cannot be adjusted.
