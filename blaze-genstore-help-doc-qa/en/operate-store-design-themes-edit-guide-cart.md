# Cart page

The cart page displays the list of products customers have added and the checkout buttons. It connects product browsing with purchase completion and plays a key role in improving the shopping experience and conversion rate. You can customize the layout, style, and buttons on this page to optimize performance.

## How to access

1. Log in to your Genstore admin.
2. Go to **Store** -> **Online Store** -> **Themes**.
3. Find your theme and click **Customize** to open the theme editor.
4. In the top navigation bar, open the page selector and choose **Cart**.

## Features overview

- **Product list**: Shows all items in the cart, including image, name, price, variant details, quantity adjustment, and remove option.
- **Checkout area**: Displays the estimated total and supports tax, discount, and shipping messages. Includes checkout button options.

## Product list settings

| **Category**        | **Description**                                       |
| ------------------- | ----------------------------------------------------- |
| **Title settings**  | Adjust title size (small / medium / large)            |
| **Layout style**    | Choose horizontal or vertical layout                  |
| **Card mode**       | Enable or disable card-style display                  |
| **Color style**     | Switch between light/dark mode and apply theme colors |
| **Section spacing** | Control top and bottom padding                        |

## Checkout section settings

- **Display options**: Show or hide subtotal, checkout button, and dynamic checkout button.  **Note**: When **dynamic checkout** is enabled, the button adapts to payment methods like PayPal.
- **Button alignment**: Align buttons to the left or right.
- **Button style**:
    - Size: Small / Medium / Large / Extra large
    - Type: Solid button or outlined button (Padding / Stroke / Ghost)
    - Color scheme: theme color, custom color, etc.

## Add sections

You can extend the cart page layout by adding sections such as images, videos, or rich text to create a more branded experience.

### Add a section

In the **Sections template** panel, click **Add section** and choose from these content types:

|Section type|Description|
|---|---|
|Featured collection|Showcase selected collections for campaigns|
|Image banner|Add a large promotional or brand image|
|Collection list|Display multiple product collections|
|Video|Embed brand story or product videos|
|Slideshow|Add carousel images for promotions|
|Contact form|Allow customers to submit inquiries or feedback|
|Rich text|Add brand messages or product details|
|Email signup|Insert a newsletter subscription form|
|FAQ|Display frequently asked questions|
|Collage|Combine multiple images for a dynamic visual layout|
|Featured product|Highlight a specific product|
|Divider|Add visual separation between sections|
|Image|Insert a single image with an optional link|
|Multi-column layout|Create custom text-and-image layouts|
|Blog posts|Embed articles for content marketing|
|Image with text|Combine an image with text for CTAs|
|Scrolling text|Add horizontal scrolling text for slogans or promotions|

::: tip  
Available sections may vary depending on your theme. We regularly add new content components based on merchant feedback. Always check the theme editor for the latest options.  
:::

### Add apps

In the **Add section / Add block** pop-up, click the **Apps** tab to add functional components such as:

- Related products
- Bestsellers
- Customers also viewed
- Trending items
