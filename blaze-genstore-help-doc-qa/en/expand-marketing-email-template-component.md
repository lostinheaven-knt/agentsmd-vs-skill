# Template components

Genstore provides a variety of components (e.g., text, images, products, discounts, etc.) to enrich your email content. You can adjust their properties to customize the component styles.

:::tip

Component limit: Each email can use up to 20 components, and each component can be added a maximum of 10 times.

:::

## General components

### Header component

Includes brand display, font style, layout adjustments, and background color settings:

- Header design: Displays product brand marketing information, store name, and logo.
- Font settings: Adjust font size and text color.
- Layout settings: Adjust section and alignment.
- Background settings: Change the component's background color.

### Footer component

Includes store information such as social media links and other relevant details:

- Layout: Offers default and stacked layout options.
- Separator: Set the color of the separator.
- Background color: Change the footer's background color.

### Text component

Customize the text style and layout, including font, text and link colors, alignment, margin adjustments, and background color:

- Font settings: Adjust font size and text color.
- Layout settings: Align text and set top/bottom margins.
- Background settings: Change the component's background color.

### Button component

Customize the appearance and behavior of buttons, including link settings, text style, border and color adjustments, and layout options:

- Button settings: Add a URL link or choose a page. Customize button text.
- Style settings: Set button border color, border radius, and text color.
- Background settings: Change the component's background color.

### Image component

Manage and customize the display of images, including upload, link settings, alt text, layout adjustments, and background color:

- Image upload and selection: Choose an image or drag to upload, and add alt text.
- **[Optional]** Link settings: Add a URL link for the image, with search options for page selection.
- Style and layout: Adjust the image layout and style, such as rounded corners or circular shapes.
- Background settings: Change the component's background color.
- **[Optional]** Alt text: Recommended for better accessibility and SEO.

### Product component

Manage and display your product information, including images, titles, prices, descriptions, etc.:

- Image settings: Replace product images and choose the image style (e.g., rounded corners, oval, or circular).
- Content settings: Show/hide product title, price, and description. Adjust display styles.
- Button settings: Add a button that links to the selected product page. Customize button display style.
- Layout settings: Support column layout, with alignment adjustments.
- Background settings: Change the component's background color.

:::tip
To better showcase and manage product information, it’s recommended to display no more than 9 products per email.
:::

### Discount component

Display and manage your discount activities, including discount titles, activity descriptions, buttons, and layout settings:

- Discount selection: Defaults to manual discount activities.
- Content settings: Set the discount header, title, description, and button. You can also add specific requirements, such as expiration dates.
- Style settings: Adjust top/bottom margins.
- Layout settings: Support section and alignment adjustments.

### Gift card component

Display and manage gift card information, including images, titles, descriptions, and buttons:

- Gift card selection: Defaults to available gift cards.
- Image settings: Replace the gift card image and choose its style (e.g., rounded corners, oval, or circular).
- Content settings: Set gift card title and description.
- Button settings: Add a button linking to the selected gift card page. Customize button display style.

### Text-image component

Display images with text and a button, with customizable layout and style:

- **[Required]** Image settings: Upload or select an image and adjust its style.
- Content settings: Add text and buttons over the image to convey information or guide users. You can hide them if needed. Ensure that all buttons have links if displayed.
- Layout settings: Adjust the image position, such as left, right, or full-width.
- Background settings: Change the component’s background color.

### Multi-column component

Create structured lists with flexible layout and style adjustments:

- List settings: Create content in multiple columns, such as images, text, and buttons.
- Background settings: Change the component’s background color.

### Countdown component

Create and manage countdown functionality, with custom end times, links, style, and background settings:

- Countdown settings: Set the end time for the countdown.
- **[Optional]** Link settings: Add a URL link, with search options for page selection.

## Marketing automation components

When you use marketing automation, the following components will be available in your email templates.

### Viewed but not purchased

Match customers’ browsing history to show them products they viewed but didn’t buy:

- Content settings: Show/hide product name and button.
- Style settings: Choose image display style, such as rounded corners, oval, or circular.
- Layout settings: Adjust layout and alignment.
- Background settings: Customize the component’s background color.

### Added to cart but not purchased

Match customers’ cart additions to show them products they added but didn’t purchase:

- Product matching: Automatically display products that customers added to their cart but haven’t purchased.
- Content settings: Show/hide product name, attributes, cart quantity, and button.
- Style settings: Choose image display style, such as rounded corners, oval, or circular.
- Layout settings: Adjust layout and alignment.
- Background settings: Customize the component’s background color.

### Abandoned checkout

Match customers’ abandoned checkout records to show them products they didn’t complete checkout for:

- Product matching: Automatically display products in abandoned carts.
- Content settings: Show/hide product name, attributes, cart quantity, and button.
- Style settings: Choose image display style, such as rounded corners, oval, or circular.
- Layout settings: Adjust layout and alignment.
- Background settings: Customize the component’s background color.

Make sure to enable the corresponding automation components in emails created through automation workflows to correctly retrieve product data.
