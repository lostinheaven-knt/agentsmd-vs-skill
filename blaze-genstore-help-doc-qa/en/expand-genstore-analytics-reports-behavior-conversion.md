# Online store conversion over time

The **Online store conversion over time** report shows the percentage of online store visitors who made purchases during selected time ranges.

## Data refresh frequency

This report is updated every **4 - 5 seconds.** Refresh the page to view the latest data.

## Use this report to

- **Online store conversion trend analysis**: Use the conversion trend chart to identify fluctuations in conversion rates over time, including overall growth/decline trends and impacts from seasonal factors or promotions. Evaluate the effectiveness of conversion strategies and market dynamics.
- **Conversion funnel insights**: Leverage the funnel model to visualize user journeys from visits to purchases, pinpointing drop-offs at key stages (e.g., browsing, cart additions, checkout) and identifying bottlenecks for optimization.
- **Conversion detail analysis**: Access granular transaction data via the conversion details list, including timestamps, amounts, traffic sources, and customer segments, to uncover patterns in high-value conversions.

## Report data fields

The report includes the following columns to help you understand how your store converts traffic into sales:

| Column | Description |
| ---- | ---------------------------------------------------------------- |
| **Time group** | The selected time unit (e.g., day, week, month). |
| **Sessions** | Total Sessions to your online store. |
| **Added to cart** | Total Sessions where customers added products to their cart. |
| **Reached checkout** | Total Sessions with user interactions during checkout (e.g., clicks, keystrokes). |
| **Converted sessions** | Total Sessions that resulted in a purchase. |
| **Conversion rate** | Percentage of Sessions that led to a purchase. |

:::tip

**Converted sessions** count the total session visits where customers made purchases. A single customer might complete multiple purchases in one visit. Thus, your **orders** and **converted sessions** may differ.

:::
