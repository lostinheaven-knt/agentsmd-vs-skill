# Digital products

A digital product is an item that customers can access, download, or use online without physical delivery. It covers all types of virtual products, such as eBooks, software, audio files, video courses, and design assets. These products are widely used in fields like education, creative design, software development, entertainment, and business resources.

Selling digital products online allows merchants to expand their sales channels, reduce logistics and inventory costs, and provide a more flexible and efficient delivery experience for customers.

## Add digital products

1. Log in to the **Genstore admin** and go to the **Store** -> **Products** page.
2. Click on the **Add product** button at the top-right corner, and select **Digital products**.
3. Set the product title and key features.
4. Choose an introduction image/video from the resource library or upload from your local system. Multiple images can be uploaded and the image order can be adjusted. The first image will be the main image.
5. Use the rich text editor to provide a detailed description of the product's purpose, specifications, etc., with support for inserting images, links, and tables.
6. Upload and deliver product file: Upload a single file (max 5GB; images and videos supported). Set download limits (default is unlimited), and choose how to send the file—**auto-send** or **manually send**.
	 - If no file is uploaded, the digital product will be treated as a virtual item—commonly used for shipping fee adjustments, service charges, or other non-downloadable purposes.

For additional settings, refer to the [Add product](./operate-product-create.md) page. Once completed, click **Save** to publish the product.

## Filter digital products

You can filter digital products by product type.
- On the **Products** page, click the **Filter** icon.
- Under **Product type**, select **Digital products**, and use the **Digital file** filter to choose whether the product includes a downloadable file.
- Click the product name to view its details.
- In the **Digital file** column, click the number to view uploaded files for each variant.

::: tip
If the **Digital file** column is not visible, click the **Manage columns** icon in the top right corner of the table and enable it.
:::

## Delete digital products

To delete one or more digital products:
1. Filter digital products using the steps above.
2. Select the checkbox next to the target products.
3. Click **Bulk delete** at the top of the page.
4. Confirm the deletion in the pop-up. This action is irreversible—please proceed with caution.

## Configure digital product variants

In Genstore, digital products also support variant configurations. Each variant (SKU) can have its own price, inventory tracking, and associated digital file (e.g., downloadable content or videos).

The variant setup process is similar to physical products, with the following differences:
- After adding variants, each SKU requires an uploaded digital file, which customers can download or access after purchase.
- The system provides an **Upload** button for each row in the variant table to bind a file to the corresponding SKU.
- Fields such as shipping, weight, or country of origin are not available for digital products.

### Operation steps

1. Log in to your Genstore admin and go to the **Store** -> **Products** page.
2. Click **Add product** in the top-right corner and select **Digital products**.
3. Set the product title, key features, images, and description.
4. In the **Variants** section, click **Add options like size or color** to add options such as “Format,” “Duration,” or “Resolution.”
5. The system will automatically generate a variant (SKU) table. For each variant, you can set:
    - Price
    - Whether to track inventory (if enabled, set inventory quantity)
    - Upload the corresponding digital file

## Replace a digital file

To update a file already associated with a digital product (e.g., a course video, or eBook):
1. Filter digital products from the **Products** list.
2. Click the desired digital product to open its detail page.
3. In the SKU table, locate the variant with the file you want to replace.
4. In the **Digital file** column, click the file name and select **Replace file** in the popup.
5. Then upload the new version or select from the media library.

## Delete a digital file

### Steps:

1. On the **Products** page, filter by **Digital products** under product type.
2. In the **Digital file** column for the target product, click the number (which indicates the number of digital files associated with the product) to view all files.
   - If the **Digital file** column is not visible, click **Manage columns** icon in the top right of the table and enable it.
3. In the pop-up window, click the **Delete** icon next to the file you want to remove.
