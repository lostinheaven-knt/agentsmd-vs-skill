# Custom Liquid

The Genstore theme editor supports adding **Custom Liquid** to all page templates as both sections and blocks. This feature is designed for merchants seeking deep personalization, allowing you to insert custom Liquid, HTML, CSS, or scripts directly within the editor without modifying the theme's underlying source code.

## Core value

* **Flexible extension of exclusive features**: Break through the limitations of native template components by easily embedding third-party microservice plugins, custom forms, or unique interactive dynamic components.
* **Low-risk code experimentation**: All code edits are contained within the specific section or block, avoiding the risk of site-wide crashes caused by modifying global theme files. This makes technical testing safer and more efficient.
* **Precise brand visual presentation**: Tailor marketing logic or use custom CSS to fine-tune styles, ensuring every detail on the page aligns perfectly with your brand's visual identity.
* **Seamless integration with visual editing**: Once rendered, sections and blocks can still be located and selected normally. Related settings remain editable, ensuring the visual configuration experience of the theme is preserved.

## Operation steps

1. In the Genstore merchant admin, click **Store** → **Online Store** → **Themes**, then click the **Customize** button for your current theme.
2. Use the page selector at the top to switch to the target page you wish to edit (e.g., Homepage).
3. In the left-hand operation panel, click **Add section** or **Add block** and select **Custom Liquid**.
4. Input and save your custom Liquid code. Once the code passes validation, the editor will render and display the effect in real-time. After rendering:
   * The section or block can still be positioned and selected as usual.
   * Related settings remain editable, ensuring the visual configuration experience of the theme is preserved.
5. After confirming the effect, click **Save**.
