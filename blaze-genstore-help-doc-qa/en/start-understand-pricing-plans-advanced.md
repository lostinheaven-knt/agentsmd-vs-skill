# **Genstore Scale plan**

The Genstore Scale plan is designed for business growth. It includes all the features of the Genstore Lite and Genstore Growth plans, helping you scale efficiently.

## Credits

- 700 monthly credits.
- 5 daily credits for daily logins.
- Extra credits available by purchasing a separate Credit Pack.

## **Plan features**

In addition to the features available on all plans, the Genstore Scale plan offers the following features:

### **Genstore AI**

| Feature            | Description                                                                                                |
| ------------------ | ---------------------------------------------------------------------------------------------------------- |
| **Launch Agent**   | Fast-track foundational settings, including domains, payments, and shipping, to ensure a compliant launch. |
| **Product Agent**  | Manage your catalog efficiently with smart product sourcing and creation tools.                            |
| **Campaign Agent** | Trigger AI-driven marketing campaigns and generate discount codes to boost conversions.                    |
| **Design Agent**   | Generate homepages and switch themes instantly to build high-converting brand visuals.                     |
| **Analytics agent**     | Analyze traffic, conversion rates, and sales performance in real-time for actionable business insights.    |
| **Support agent**  | Your 24/7 dedicated merchant expert for technical guidance and operational advice.                         |

### **Additional features**

| Feature                 | Description                                                                                        |
| ----------------------- | -------------------------------------------------------------------------------------------------- |
| Lowest Transaction Fees | Offer the lowest transaction fees. See the Genstore pricing page for details based on your region. |
| Dropshipping Products   | Push up to 1000 dropshipping products to your store.                                               |
| Staff Accounts          | Support up to 20 staff accounts for better team collaboration and management.                      |
| Locations               | Manage your inventory across 20 locations effectively.                                             |
| Markets                 | Include 5 markets and manage them from a single site.                                              |
