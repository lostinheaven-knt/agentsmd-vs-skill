# Sessions by referrer

The **Sessions by referrer** report provides detailed data on how users arrive at your website, including traffic metrics (e.g., sessions, user activity) from **direct visits, search engines, emails, social media**, and other channels. This report helps identify top - performing traffic channels, assess user engagement and conversion rates across sources, and optimize marketing strategies and resource allocation.

## Use this report to

- **Identify primary traffic channels**: Quickly pinpoint the top 20 referral sources and websites driving the most visitors, sessions, and page views through a traffic overview.
- **Analyze referral performance**: Use the detailed referral list to evaluate metrics like unique visitors, sessions, and page views per source, assessing traffic quality and value.

## Report data fields

For general terms, refer to [Acquisition reports - common terms](./expand-genstore-analytics-reports-acquisition.md#common-terms).

This report includes the following additional terms:

| Term | Definition |
| ---- | ---------- |
| **Referrer source** | The pathway through which users access your store. Common sources include:<br/>- **Direct**: Users typing your URL directly or using bookmarks.<br/>- **Search engine**: Users clicking links from search engine results.<br/>- **Email**: Users arriving via email campaign links.<br/>- **Social media**: Users redirected from social platforms.<br/>This data is captured via Genstore’s API, particularly from the last session before a purchase. |
| **Referrer name** | The specific name of the referring website or platform (e.g., "Facebook," "Google," "PartnerSite.com"). Helps identify exact traffic sources for granular analysis and campaign optimization. |

:::tip

Referral data accuracy may be affected by user privacy settings (e.g., ad blockers) and tracking limitations. Treat this data as indicative rather than fully precise.

:::
