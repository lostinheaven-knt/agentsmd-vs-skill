# Themes

Themes are pre-designed frameworks that define your online store's layout, style, and user interactions. Each theme template is tailored to different industries and product needs, providing a solid foundation for your store design.

A typical theme includes:

- **[Header](./operate-store-design-themes-edit-guide-header.md)**: Displays your logo, navigation menu, search bar, etc.
- **[Footer](./operate-store-design-themes-edit-guide-footer.md)**: Contains copyright info, social media links, and additional navigation options.
- **[Main content area](./operate-store-design-themes-edit-guide-page.md)**: Dynamically shows content based on the page type (e.g., homepage, product page, blog).

Using theme templates allows you to quickly build and launch a professional eCommerce website. With extensive customization options, you can adjust elements to align with your brand identity and business goals, creating a unique experience for your customers.

You can install official or third-party themes from the Genstore [Theme Store](https://themes.genstore.ai/), or customize your store’s theme using Genstore’s powerful theme editor.

::: tip

You can add up to 20 themes to your store, but only one can be active at a time—that’s the theme customers see when they visit your store.

:::

## Choose and install a theme from the theme store

1. Log in to your Genstore admin.
2. Click **Store -> Online Store -> Themes**.
3. In the **Recommended themes** section, click **Visit Theme Store**.
4. Browse Genstore Theme Store, using industry tags to find relevant themes.
5. Click on a theme to see more details, demos, and user reviews.
6. Once you’ve made your choice, click **Install**.
7. In the pop-up window, select the store where you want to install the theme. It will then be added to that store.

For detailed instructions, see: [How to choose a theme](./operate-store-design-themes-template-select.md)

## Preview and publish a theme

After installation, go to the theme library to:

- Click **Preview** to see how the theme looks on your store
- Click **Publish** to set the theme as your active storefront design

::: tip

Publishing is instant and does not require redeployment. We recommend previewing carefully before publishing.

:::

## Customize a theme

Genstore supports multiple ways to customize your theme to meet different user needs:

## AI-powered theme agent

Genstore offers AI theme agent that allow you to quickly create beautiful stores without professional design knowledge:

- **AI-generated homepage**: Through conversation with AI, describe your needs, and AI will automatically generate a complete homepage that matches your brand style.
- **AI theme switching**: One-click intelligent deployment of an entire theme template package. AI will recommend suitable themes based on your store category.

For detailed instructions, see: [Generate homepage or switch theme with AI](./operate-store-design-themes-ai-decorate.md)

### Customize a theme with theme editor

No coding is required. Use Genstore’s drag-and-drop visual editor to modify layouts, colors, content modules, and more.

**Steps:**

1. Go to **Store -> Online Store -> Themes**.
2. Find your theme and click **Customize.**
3. The visual editor will open in a new interface.

Learn more in: [Customize with theme editor](./operate-store-design-themes-edit-guide.md)

### Code editor (For developers and advanced customization)

For full control over page structure, styles, or dynamic behavior, use the code editor to modify `.liquid`, `.json`, or asset files.

**Steps:**

1. Go to **Store -> Online Store -> Themes**
2. Click the `...` menu next to the theme and select **Edit code**.
3. Select and edit the relevant `.liquid` or `.json` file.
4. Click **Save** when done.

Learn more in: [Customize with code](./operate-store-design-themes-code-editor.md)

## Manage your themes

Genstore makes it easy to manage your themes:

- **Theme version**: You can view the current theme version and check for the latest version from the dropdown menu. If a new version is available, a notification will appear next to the theme name. To upgrade, click the dropdown arrow on the right and select **Update now**.
  - Safe Update: The system will create a new copy of the theme in your library. This ensures your existing page content, section structures, and theme settings remain fully intact. You can preview and confirm all changes before going live.
- **Duplicate theme**: Create a copy to test new designs or features safely.
- **Delete theme**: Remove themes you no longer need to keep your library tidy.
- **Download theme**: Save a full copy of a theme for backup or migration.
