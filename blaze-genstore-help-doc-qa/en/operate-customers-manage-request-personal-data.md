# Export customer personal data

When a customer requests to export their personal data from your store, you can verify the customer's identity and use this feature to submit a data export request. Once the export is complete, you can send the file to the customer.

## Steps

1. Log in to the Genstore admin and go to **Store** -> **Customers**.
2. Select the customer whose data you need to export and open the customer details page.
3. Click **More actions** -> **Request personal data**, review and confirm the request.
4. Wait for the export task to complete. Once ready, click the avatar in the lower-left corner to access the **Task center** and download the file.
5. Send the downloaded file to the customer manually.

## Data scope

The exported personal data may include the following types of information.

**Note:** Only data that actually exists for the customer in your store will be included. If certain data types are not applicable, they will not appear in the exported file.

| **Data type**       | **Description**                                              |
| ------------------- | ------------------------------------------------------------ |
| Customer info       | Basic customer details, such as name, email, phone number, marketing subscriptions, default address, tax exemption settings, notes, etc. |
| Shipping addresses  | All shipping addresses associated with the customer          |
| Customer metafields | Custom customer data, such as loyalty points, VIP status, etc. |
| Orders              | All completed orders (canceled orders are excluded)          |
| Draft orders        | All draft orders associated with the customer                |
| Gift cards          | Gift cards linked to the customer                            |
| Discounts           | Discount records associated with the customer                |
| Payments            | Payment records for all orders                               |
| Fulfillments        | Fulfillment records for all orders                           |
