# Post-migration checklist

After completing the migration, use this checklist to ensure your store is properly set up and accessible to customers.

::: tip

Since payment settings are not migrated automatically, please make sure you’ve completed the payment setup in your new store.  
If not, refer to [Payment setup](./operate-payments.md) to proceed.

:::

## Store display and content

[ ] A brand-appropriate theme template has been selected and applied
[ ] Template styles have been customized (font, color, buttons, etc.)
[ ] All products have been properly assigned to collections
[ ] Product detail pages are accurate (images, pricing, inventory)
[ ] Navigation menu is structured logically with correct page links
[ ] Blog content has been migrated and displays correctly (if applicable)

## Store functionality testing

[ ] At least one test order has been placed (including success and failure scenarios)
[ ] Order cancellation, refunds, and partial fulfillment flows have been tested
[ ] Orders are archived correctly and statuses update in real time
[ ] Payment methods have been enabled and configured properly (including third-party methods like PayPal)

## Customer and account management

[ ] All customer data has been imported correctly with no duplicates or missing entries
[ ] Account activation or password reset emails have been sent to customers
[ ] Customers can log in and place orders successfully
[ ] Customer messages are received and responded to promptly via Inbox

## Store launch and accessibility

[ ] Password protection has been removed and the store is publicly accessible
[ ] All pages display correctly on both desktop and mobile
[ ] All buttons and links work as expected, with no 404 errors

## Promotion and marketing setup

[ ] Homepage meta title and description have been set for SEO optimization
[ ] Genstore Email has been enabled and sender information configured
[ ] At least one email campaign has been created
[ ] Discount codes or promotional settings have been added
[ ] Sales channels have been connected (e.g., Facebook, Google)

## Ongoing maintenance and support

[ ] Data backup and export methods have been reviewed and configured
[ ] Help center documentation and support channels are familiar to the team
[ ] Regular review processes are in place for orders and customer feedback
[ ] Scheduled operational and marketing tasks are set (e.g., weekly data reviews)

✅ If all items are checked, congratulations! You’ve completed the final steps of your store migration. We recommend saving or printing this checklist for internal use and future reviews.
