# Sessions by landing page

The **Sessions by landing page** report allows you to analyze traffic to specific landing pages, including visit volume and trends over time.

This helps you understand user entry points, optimize page design, enhance user experience, and refine marketing strategies with actionable insights.

## Data refresh frequency

This report is updated every **4 - 5 seconds**. Refresh the page to view the latest data.

## Use this report to

- **Top landing pages overview**: Identify the top 20 most - visited landing pages, including their URLs, types (e.g., homepage, product detail page, category page), and resources (e.g., images, videos), to prioritize optimization efforts.
- **Traffic attribute analysis**: Segment landing page traffic by visitor type (new vs. returning), source (direct, search engines, social media), device (mobile, tablet, desktop), or marketing campaign to uncover preferences and tailor strategies.
- **Granular visit details**: Access detailed metrics like visit counts, session durations, and bounce rates for each landing page to evaluate engagement and performance.
