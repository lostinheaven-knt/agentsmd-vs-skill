# Order reports

**Order reports** provide insights into order volume, fulfillment, shipping, delivery, and returns.

## Data update frequency

The report refreshes automatically every **4-5** seconds. You may manually refresh the page to ensure  data recency.

## How to access

1. Log in to your Genstore Merchant dashboard.
2. Navigate to **Analytics** -> **Reports** in the left menu.
3. Locate financial reports under the **Orders** section on the Reports Summary page.
4. Click any report title to view comprehensive metrics and granular data.

## Report categories

Common report types include:

- **[Orders over time](./expand-genstore-analytics-reports-orders-time.md)**
- **[Product orders and returns](./expand-genstore-analytics-reports-orders-product.md)**

## Common terms

Order reports use the following standardized terminology:

| Term | Definition |
|------|------------|
| Orders  | Total number of orders placed within the specified date range. |
| Returned quantity | Total quantity of products returned by customers within the specified date range. |
| Average units ordered | Average number of products per order. Formula: `Total product quantity / Total orders`. |
| Average order value | Average transaction value per order. Formula: `(Total sales - Total discounts) / Total orders `. Includes post-purchase edits, exchanges, and returns. |
