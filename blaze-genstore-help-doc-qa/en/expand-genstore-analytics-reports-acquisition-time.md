# Sessions over time

The **Sessions over time** report provides a time-based visualization of website traffic patterns, enabling you to analyze visit trends at granular time intervals.

This report allows multi-dimensional analysis across **device types, traffic sources, marketing campaigns, geographic regions, and user behavior patterns**. It also supports exporting detailed visit data for deeper analysis and strategic decision-making.

:::tip

Changes to website content or marketing campaigns during the reporting period may cause fluctuations in visit metrics, reflecting the impact of these updates.

:::

## Use this report to

- **Identify traffic trends**: Detect visit patterns such as peak hours, low-traffic periods, and holiday fluctuations using time-series data.
- **Multi-dimensional analysis**: Evaluate traffic distribution across dimensions like device types, traffic sources, campaigns, and regions.
- **Drill into visit details**: Use customizable columns in the detailed list to analyze user interactions (e.g., page views, session duration) and optimize content.
