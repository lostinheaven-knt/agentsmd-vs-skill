# Sessions by device

The **Sessions by device** report provides traffic data segmented by user device types, including mobile phones, tablets, and desktop computers. This report helps you understand user engagement patterns and browsing behaviors across devices, enabling optimization of cross - device user experiences and website performance.

## Use this report to

- **Device & browser traffic ranking**: Identify the top 20 device types and browsers driving the most traffic to your website.
- **User behavior analysis**: Analyze **behavioral metrics** (e.g., session duration, page views, bounce rate) across devices and browsers to assess engagement levels. Use these insights to refine UX design and tailor marketing strategies for specific device segments.

## Report data fields

For general terms, refer to [Acquisition reports - common terms](./expand-genstore-analytics-reports-acquisition.md#common-terms).

This report includes the following additional terms:

| Term | Definition |
| ---- | ---------- |
| **Device type** | The category of devices used to access the website (e.g., mobile, tablet, desktop). |
| **Device browser** | The browser name and version used during the visit (e.g., Chrome 85, Safari 14). |

:::tip

Device type data accuracy may be affected by user agent string inconsistencies or device fingerprinting limitations. Treat this data as directional rather than fully precise.

:::
