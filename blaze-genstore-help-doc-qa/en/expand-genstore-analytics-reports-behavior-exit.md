# Exit page analysis

**Exit page analysis** helps identify the last pages visited by users before they leave your website. These pages are crucial for understanding the behavior of visitors just before they exit.

## Use this report to

- **Top 20 exit pages**: Ranked by sessions, visitors, and pageviews in descending order, displayed with bar charts for quick identification of the most frequently exited pages.
- **Proportion of exits**: Shown with pie charts to illustrate the distribution of exit sessions across different pages.
- **Exit page details**: A detailed list displaying metrics for each exit page, including all the fields covered in the report.

## Report data fields

The **exit page analysis report** includes the following columns:

| Column                | Description                                                                 |
|-----------------------|-----------------------------------------------------------------------------|
| Exit page        | The name of the last page a visitor views during a session.                  |
| Sessions (exits) | The number of times visitors leave the site from a specific page.            |
| Visitors         | The number of unique users who accessed the page.                           |
| Pageviews            | The total number of times the page was loaded. Increments with each visit.    |
| Average session duration | The average time visitors spent on the exit page before leaving. Measured in seconds. |
| Exit rate             | The percentage of sessions that ended on a particular page. Calculated as: `exit rate = (exits sessions / total sessions) × 100%` |
