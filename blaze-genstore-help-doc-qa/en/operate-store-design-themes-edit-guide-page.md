# Customize your store pages

In the Genstore visual theme editor, each page type is associated with a **template** that defines its overall structure and visual style. You can customize templates for different page types to enhance brand presentation and improve the browsing experience for your customers.

## Overview of page templates

Genstore provides built-in templates for the following page types:

| Page type                                                                               | Description                                                                                                                                               |
| --------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **[Homepage](./operate-store-design-themes-edit-guide-page-home.md)**                   | Your store’s main landing page. Freely combine modules to showcase your brand story, bestsellers, banners, and more.                                      |
| **[Product](./operate-store-design-themes-edit-guide-page-product.md)**                 | Each product can use a different template—ideal for customized layouts across categories.                                                                 |
| **[Collection](./operate-store-design-themes-edit-guide-page-collection.md)**           | Assign unique templates to each collection to emphasize filters, layouts, or featured categories.                                                         |
| **[Collection list](./operate-store-design-themes-edit-guide-page-collection-list.md)** | Displays all your product collections in one summary view.                                                                                                |
| **[Gift card](./operate-store-design-themes-edit-guide-gift-card.md)**                  | Displays available gift cards, denominations, and descriptions.                                                                                           |
| **[Cart](./operate-store-design-themes-edit-guide-cart.md)**                            | Shows the items in the customer’s cart, with options to adjust quantity or remove items.                                                                  |
| [Checkout page](./operate-store-design-themes-edit-guide-page-checkout.md)              | The key page before order completion, where customers enter shipping and payment information, directly impacting conversion rates.                        |
| [Thank you page](./operate-store-design-themes-edit-guide-page-thank-you.md)            | The confirmation page customers see after completing payment, which provides order details and creates opportunities for repeat purchases and engagement. |
| **[Customer accounts](./operate-store-design-themes-edit-guide-customer-accounts.md)**  | Includes pages for sign up, login, password recovery, account info, and address management.                                                               |
| Page                                                                                    | Create static content pages such as About Us, Return Policy, or Contact Us.                                                                               |
| Blogs                                                                                   | Blog index page showing all published posts, with pagination or waterfall layout options.                                                                 |
| Blog post                                                                               | A single blog article page supporting images, text, tags, and comments.                                                                                   |
| **[Search results](./operate-store-design-themes-edit-guide-search.md)**                | Displays matching results for search keywords—products, content, etc.                                                                                     |
| **[404 page](./operate-store-design-themes-edit-guide-404.md)**                         | Shown when a page isn’t found. You can insert buttons or links to guide users back to active content.                                                     |

## Template configuration

- Use the **page selector** in the top toolbar of the editor to switch between page types and access their corresponding templates.
- Most page types (like product or collection) support **multiple custom templates**. You can create and assign different templates to specific products or collections.
- Each template can have its own layout, modules, and styling—independent of other pages.
## How to edit

1. Log in to your Genstore admin and go to **Store** -> **Online Store** -> **Themes**.
2. Click the **Customize** button next to your target theme to open the editor.
3. Use the **page selector** in the top toolbar to choose the page type and template you want to edit.
4. In the **left panel**, you can add, remove, or reorder modules. In the **right panel**, configure the content and style of each module.
5. Click **Save** in the top right, and use **Preview** to see the changes in action.

The Genstore theme editor also provides a shortcut to the page management panel, where you can create, assign, and manage templates in one place. For details, see [Manage page template](./operate-store-design-themes-edit-guide-page-manage.md)
