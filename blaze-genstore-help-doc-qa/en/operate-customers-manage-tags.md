# Manage customer tags

In Genstore, you can manually add tags to customers with similar characteristics. This classification method helps you manage customer data more effectively and target specific groups for marketing campaigns.

## Use cases

Customer tags can be applied in various marketing and management scenarios, such as:

- **Marketing campaigns**: Send targeted [marketing emails or SMS](./operate-customers-manage-marketing-subscription.md) to customer groups with specific tags.
- **Discount campaigns**: In Genstore's [discount](./expand-sales-discounts.md) settings, choose **specific customer tags** to apply discounts directly to customers with those tags.

## Add customer tags

1. **Navigate to customer details**: In the Genstore admin, go to **Store -> Customers**, click on the target customer, and enter their detail page.
2. In the **Customer tags** section, type the tag (case-insensitive) and press **Enter** to confirm the addition. Note: Each tag can be up to 100 characters long, and a maximum of 120 tags can be added per customer.
3. You can also click the edit button next to **Tags** to select multiple tags from the existing list and add them in bulk.

## Bulk add customer tags

Genstore allows you to bulk add tags to multiple customers at once.

1. On the **Store** -> **Customers** page, select multiple customers.
2. Click **Add tag**, choose the desired tags, and save. Note: If the selected customers already have the tag, the system will not duplicate it.

## Delete customer tags

Regularly deleting unnecessary tags helps maintain clean and organized customer information:

1. In the Genstore admin, go to **Store** -> **Customers**, click on the target customer, and enter their detail page.
2. Click the **Delete** button next to a tag to remove it.
3. Alternatively, click the edit button next to **Tags**, uncheck the tags to be removed, and click **Confirm** to save the changes.

## Bulk delete customer tags

Genstore also supports bulk deleting tags for multiple customers.

- On the **Store -> Customers** page, select the customers from which you want to remove tags.
- Click **Delete tag**, check the tags you want to remove, and confirm the action.
