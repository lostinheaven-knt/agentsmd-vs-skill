# Local delivery

To offer better service to local customers, you can enable local delivery for online orders. When a customer's address is within your delivery range, they will see and can select the local delivery option at checkout.

To set up local delivery, you need to enable the relevant settings for each shipping location. You can choose to define your delivery area by distance or specific ZIP codes. After setting up shipping, you can handle deliveries yourself, assign them to staff, or use third-party delivery services to meet customer needs and ensure a smooth shipping process.

## Set up local delivery

You can set up local delivery options for customers within your service area. If a customer's address is eligible and within your delivery range, they will be able to select local delivery during checkout.

Steps:

1. Navigate to **Settings** ->**Shipping and delivery**. Locate the **Local delivery** section and click to enter the settings page.
2. In the new window, click the edit icon besides the location:
   - Use the toggle switch to enable or disable local delivery for the location.
   - Set conditions for when the local delivery option is available, that is, a minimum order amount at checkout.
   - Define the delivery area using ZIP codes or a delivery radius.
	 - **Distance radius**: Set the maximum distance from the shipping location to the customer’s address. The radius can’t exceed 160 km (100 miles). You can also enable the option to **Include neighboring states or regions**.
	 - **ZIP code area**: Enter a list of ZIP codes in your delivery area. You can define ranges using an asterisk or space.

Once you’ve set up the areas, you can adjust delivery prices, set minimum order amounts, and add multiple pricing rules for each region.

## Checkout experience

Once local delivery is enabled, eligible customers can select it as a shipping option at checkout.

For a customer to select local delivery, the following conditions must be met:

- All physical items in the cart must be shipped from the same location.
- The inventory at the shipping location must be enough to fulfill the order.
- The customer’s address must be verified via Google.
- The customer’s address must be within the set delivery radius. Alternatively, the address must fall within the specified ZIP code range and be in the same country.

## Store display

After setup, customers will see and can select the **Local delivery** option in the **Shipping method** section during checkout.

## Local delivery notes

- Customers can’t choose partial item delivery in the same order (e.g., part of the order for local delivery and the rest for standard shipping). They need to place two separate orders.
- Customers can add delivery instructions at checkout and must provide a contact number for communication.

## Order confirmation

You can include local delivery details in the order confirmation email template and display specific delivery instructions based on your settings. Be sure to update the **Genstore** admin [notification settings](./settings-notifications.md) with the relevant information.
