# Edit inventory in bulk

Genstore supports bulk inventory editing, allowing merchants to adjust the stock quantities of multiple products, variants, and locations at once. The bulk editor is similar to a spreadsheet, providing an intuitive view of data in rows and columns to make inventory management more efficient.

## Edit via product detail page

When you need to adjust the inventory of **multiple variants of a single product** across different locations, you can directly edit from the product page.

1. Log in to your **Genstore admin**, then go to **Store** -> **Products**.
2. In the product list, click the name of the product you want to edit or select **Edit** from the **Action** column.
3. In the SKU list section, select the variant you want to edit. **Note**: When you update the available inventory, the on-hand inventory will also update.
4. If you wish to update inventory for a specific location, click the **Manage location** button, select the location, and repeat the previous step to update the stock for that location.
5. Click **Save** to complete the inventory update.

## Edit via inventory list

If you only need to update the inventory for a few products, you can use the bulk editor in the **Inventory page**. The inventory list allows you to filter by location, product, and product variant.

1. Log in to your **Genstore admin**, then go to **Store** -> **Products** -> **Inventory**.
2. Select the product variants you want to adjust.
3. Click **Bulk update available inventory**.
4. If you want to adjust the inventory for a **specific location**, select the location first, then choose the product variants and adjust the available inventory.

## Multi-variant products

To adjust inventory tracking settings for multi-variant products, use **Bulk edit** from the product details page.
1. Log in to your Genstore admin, then go to **Store** -> **Products**.
2. Open the target product to access the product details page.
3. In the Variant details section, select the variants you want to update and click **Bulk edit**.
4. In the bulk edit window, update the **Track inventory** setting for the selected variants.
5. Click **Save** to apply your changes.
