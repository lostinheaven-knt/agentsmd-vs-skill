# Product orders and returns

The **Product orders and returns report** helps you understand which products are your best sellers, and which get returned the most often. The report displays total number of products that were ordered over the selected time period and how many of them were returned.

## Use this report to

- **Best-selling products**: Analyze products with the highest order quantities to quickly identify market favorites.
- **Products with high return quantity**: Focus on products with elevated return counts, which may indicate quality issues, description mismatches, or customer dissatisfaction. Investigate return reasons (e.g., “size discrepancies” or “defective items”) and correlate with supplier/vendor performance data.
- **Products with high return rate**: Prioritize products where Return Rate (returned units ÷ total orders × 100%) exceeds category benchmarks.

## Report data fields

For standardized terminology used in this report, see [Order Reports - Common Terms](./expand-genstore-analytics-reports-orders#common-terms).

This report includes the following additional terms:

| Term                | Definition                                                                                   |
|---------------------|----------------------------------------------------------------------------------------------|
| Product title       | Official name of the ordered product.                                                        |
| Variant SKU    | Unique identifier for specific product variants (size/color/configuration).                   |
| Variant title  | Descriptive label for product specifications (e.g., "iPhone 15 Pro Max 256GB - Space Gray"). |
| Order quantity      | Total items purchased within the selected period.  **Sum of all order line items.** |
| Return quantity     | Total items with completed return process. **Excludes pending/contested returns.** |
| Net quantity        | Final validated sales volume. **Units Ordered - Units Returned.**            |
| Return rate         | Product performance health indicator. **(Return quantity  ÷ Order quantity) × 100%.** |
