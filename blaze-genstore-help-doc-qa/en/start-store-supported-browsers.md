# Supported browsers

To access the **Genstore** admin, make sure you are using the latest version of your browser. Using an outdated version may cause certain features to malfunction or become unavailable. For the best experience, we recommend the latest version of [Google Chrome](https://www.google.com/intl/en/chrome/browser?utm_source=chatgpt.com).

## List of supported browsers

Genstore supports the latest versions of all major browsers (unless otherwise specified). You can typically access Genstore with:

### Desktop browsers

Genstore supports the latest versions of the following browsers on desktop:

- [Google Chrome](https://www.google.com/intl/en/chrome/browser) (Windows / macOS)
- [Apple Safari](https://www.apple.com/safari) (macOS)
- [Microsoft Edge](https://www.microsoft.com/edge) (Windows)

### Mobile browsers

You can also access Genstore from your mobile device using the latest versions of:

- [Google Chrome](https://www.google.com/intl/en/chrome/browser) (Android)
- [Apple Safari](https://www.apple.com/safari) (iOS)
