# Store design

Store design is a key feature of the Genstore platform, defining your online store’s look, structure, and layout. With AI-powered tools, you get smart theme recommendations and one-click design options to quickly create professional pages. Customize your pages, navigation, and templates to strengthen your brand, improve the customer experience, and drive higher conversions.

This section provides an overview of all store design features and the recommended setup order.

## Recommended setup flow

1. **Choose and install a theme** – Define the visual style and layout of your store.
2. **Customize theme layout and page content** – Adjust the homepage, product pages, blog layout, and more.
3. **Upload brand visuals** – Add your logo and other assets.
4. **Create custom pages** – Build static pages like brand introduction, FAQs, and service information.
5. **Set up the blog** – Share updates and insights to improve SEO and engage your audience.
6. **Configure navigation menus** – Help users quickly find key content and products.
7. **Edit store preferences** – Set the store title, SEO description, and access restrictions.

## Topics of this chapter

### [Themes](./operate-store-design-themes.md)

Themes define the overall style, layout, and interaction patterns of your store.

- Install official or third-party themes
- Customize using the built-in theme editor
- Manage multiple themes and publish with one click

### [Pages](./operate-store-design-pages.md)

Create custom static pages to display brand or service-related content.

- Build pages for brand info, customer service, FAQ, promotions, and more
- Configure SEO settings and visibility options
- Add page links to navigation menus

### [Blog](./operate-store-design-blogs.md)

The blog module enables content marketing and improves organic search performance.

- Organize posts into collections
- Add cover images, summaries, and SEO metadata
- Configure comment settings and publish status

### [Menus](./operate-store-design-navigation.md)

Set up navigation menus to define how users browse your store.

- Support up to five menu levels
- Link to pages, products, collections, blogs, and more
- Manage the main menu, footer menu, and custom menus

### [Media library](./operate-store-design-images.md)

Manage all store images and visual assets.

- Upload logo, banners, slideshows, and blog images
- Crop, replace, and configure text and buttons for each image
- Apply visuals across homepage, product pages, and other layouts

### [Store preferences](./operate-store-design-preference.md)

Configure basic site information and access settings.

- Set the homepage title and meta description for SEO
- Enable or disable password protection
- Define the access password for store preview or limited access
