# Add values to a metafield

After you've created a metafield definition, you can begin adding values directly from the relevant module pages in your Genstore admin. For example, if you've created an “Expiration Date” metafield for products, you'll be able to enter specific expiration dates on each product detail page.

Metafields let you store custom information for internal tracking or display it on your storefront.

## How metafields appear

- **[Pinned metafields](./settings-metafield-pin-unpin.md)** appear by default in the metafields section of a page, following the order set in the definitions list
- **Unpinned metafields** are hidden by default and can be viewed by clicking **Show all**
- Each field shows the name, type, and description you defined earlier
- The input method (text box, calendar, color picker, etc.) depends on the field type. See: [Metafield types](./settings-metafield-definitions-types.md)

## Operation steps

1. In your Genstore admin, go to the detail page for the item (e.g., a product or customer).
2. Scroll down to the **Metafields** section.
3. Locate the metafield you want to update.
4. Enter the value based on the field type:
    - Text: Type your content
    - Date: Use the calendar picker
    - Color: Use the color selector
5. Click **Save**

::: tip

To make common fields easier to access, [pin them to the detail page layout](./settings-metafield-pin-unpin.md). Unpinned metafields require clicking **Show all** to view or edit.

:::
