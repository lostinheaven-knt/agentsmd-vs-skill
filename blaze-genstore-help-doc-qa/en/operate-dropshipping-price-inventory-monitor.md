# Monitor price & inventory

In **Genstore Dropshipping**, you can set monitoring thresholds for supplier **price changes** and **inventory levels**.
The system checks data **daily** and sends **in-app** and **email** alerts when thresholds are met, helping you adjust strategy in time and avoid margin loss or stockouts.

## Why it matters

* Automatically track supplier cost and inventory changes
* Detect abnormal fluctuations early to protect profits and supply stability
* Set custom thresholds per your product strategy
* Receive dual alerts via in-app notifications and email

## Price monitoring

Set alert thresholds for supplier **cost increases/decreases**.
Genstore runs a daily check; when the change exceeds your threshold, alerts are sent automatically.

### Steps

1. Go to **Settings → Price monitoring**.
2. Enter percentage thresholds for **Increase** and **Decrease** (e.g., 10%).
3. Click **Save**.
4. When a product’s cost change exceeds the set threshold, the system triggers in-app and email alerts.

### Alert behavior

* Daily scan of supplier cost changes
* If change ≥ threshold, Genstore sends:
  * **In-app notification** (visible in the message center)
  * **Email alert** (to the merchant’s registered email)

> **Recommendation:**
> Use tighter thresholds (e.g., 5%) for high-value or high-volume items to respond faster.

## Inventory monitoring

Set an inventory **warning threshold**.
When available stock drops below the threshold, the system alerts you to restock or unpublish affected items.

### Steps

1. Go to **Settings → Inventory monitoring**.
2. Enter the inventory threshold (e.g., 5).
3. Click **Save**.
4. When a product’s stock falls below the threshold, the system triggers in-app and email alerts.

### Alert behavior

* Daily scan of all Dropshipping product inventory
* If stock < threshold, Genstore sends:
  * **In-app notification** (message center)
  * **Email alert** (registered email)

> **Recommendation:**
> Adjust thresholds by product turnover.
> For best-sellers, set **higher** thresholds so you can replenish earlier.
