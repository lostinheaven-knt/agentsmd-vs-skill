# Manage store languages

You can manage multiple languages for your online store in the **Languages** section of your Genstore admin. You can also [assign languages to each market](./global-market-settings-lang-domain#configure-languages) in the **Markets** section.

::: tip

This article covers language settings for your online store (storefront), which determine the language customers see when visiting your store. It does not affect your Genstore admin interface language.

:::

## Before you start

- Make sure all translations are complete before publishing a language. You can use the [Genstore Translate](./global-market-language-translate.md) app to translate your store content.
- When you add a language, the Genstore Translate app will be installed automatically. You can uninstall it anytime if it's not needed.
- If any published language has untranslated content, those parts will display in your store’s default language.

## Change the default store language

Your store's default language is set when you first create your store. You can update it anytime. The previous default language will remain in your list of published languages.

1. In the Genstore admin, go to **Settings** -> **Languages**.
2. Under published languages, find the current default language and click **...** -> **Change default**.
3. In the dialog, select the new default language.
4. Click **Save**.

## Add a store language

1. In the Genstore admin, go to **Settings** -> **Languages**.
2. Click **Add language**.
3. In the dialog, select a language from the dropdown and click **Next**.
4. Assign the language to one or more markets. Select the target markets and click **Confirm**.
5. Add translations using one of the following methods:
	- Use the [Genstore Translate](./global-market-language-translate.md) app.
	- Install a translation app from the Genstore App Store and follow the in-app instructions.

## Publish a store language

Newly added languages are unpublished by default and not visible to customers. Once translations are complete, you can publish the language to make it available to customers.

1. In the Genstore admin, go to **Settings** -> **Languages**.
2. Under **Unpublished languages**, click **...** next to the language you want to publish and select **Publish**.
3. Click **Save**.

## Delete a language

Published languages cannot be deleted directly. You must unpublish them first:

1. In the **Published languages** list, click **...** next to the language and select **Unpublish**.
2. The language will move to the **Unpublished languages** list. Click **...** next to the language and select **Delete**.

> Once deleted, all translation data for the language will be permanently removed. Please proceed carefully.
